How to: Add a Workplan to a Folder

Purpose
Adding a workplan to a folder allows the practice to standardize processes in the office. When a document is added to a specific folder via scanning or file import, the workplan is kicked-off. Folder workplans can require a review step by the ordering provider, as in the case of a Lab Reports folder, or can automatically close the associated treatment as in the case of radiology reports.

Before You Begin
Folders have a context assigned to them. This represents the piece of information within the chart with which the folder is connected: Patient, Encounter, Assessment, and Treatment. The workplans you attach to a folder need to 'match' the context of the folder.

Details:

Steps
Follow the steps below:

  1. From the Office View, select the Configuration tab. The Configuration list appears.
  2. Select Folders. The Folder Configuration screen appears.
  3. Select the desired folder on the left. The previously defined folder information, such as Description and Workplans, appear on the right.
  4. Select the gray box for Workplans. The Attachment Workplan screen appears.
  5. Select Add Work plan and search for the desired work plan.
  6. Choose the appropriate work plan from the list. If the workplan has a step that requires a specific user or role recieve a Task, the Select User screen appears.
  7. If you know the user or role that should be the recipient of any related Tasks, select it from the list. If you want the user to select the recipient when an item is saved to the selected folder, select Cancel.
  8. To automatically dispatch that work plan when a user posts to a folder, choose Yes for Work plan required. The No option allows the user to choose whether or not to dispatch the work plan.
  9. More than one work plan may be added to one folder. Repeat step 5 through 7 to select additional workplans.
  10. Select Finished. The Folder Configuration screen reappears. The workplan selected appears in the gray box. next to the work plan indicates that the user will pick the owner of the work plan at the time of dispatch.
  11. Select Finished to return to the Configuration list.
