How to: Fix a Bad Immunization Location Domain

Solution:

If an immunization is using a location domain that it isn't visibly configured for in the vaccine procedure configuration:

  • go to o_preferences
  • Filter for either Text filters -> contains ->  location_domain     or     *location_domain*
  • The records that you want to edit are the ones where the preference_ids are (vaccine_id or drug_id)_location_domain
  • Change the preference_value to !IMMUN
