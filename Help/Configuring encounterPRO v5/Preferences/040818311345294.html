How to: Set Window Sizing Preferences

Purpose
There are two preferences that allow users to re-size the encounterPRO window. One allows users to interactively size the window. The other preference maximizes the encounterPRO screen when the user starts the application.

Before You Begin
You can set this Computer Preference globally (if you have sufficient permissions) which will change the setting for all users in the practice. Be sure to consult with the other providers in the practice before making global preference changes.

Alternatively, you can set this preference for the office location or computer.
You can set these two preferences independently. For example, you can set the resize preference globally to allow all users to resize their screens while setting only the computer in the nursing station to start maximized.

Details:

Steps
Follow the steps below:

  1. From the Configuration tab, select the Preferences tab. The Preferences screen appears.
  2. Select Computer Preferences on the left.
  3. You can set the resizing option for the entire practice, the office location, or computer. Select the level you want to change the preference for at the bottom of the screen:
    • Global -- changes the preference for all users
    • Office -- changes the preference for all users who log in to the current office location
    • Computer -- changes the preference for all users who log in to the specific computer
  4. Select Window Resizable (page 3 of the list).
  5. If you want users to be able to interactively re-size the encounterPRO window, select Yes.
  6. Select Finished. The Preferences list reappears.
  7. You can set the start maximized option for the entire practice, the office location, or computer. Select the level you want to change the preference for at the bottom of the screen:
    • Global -- changes the preference for all users
    • Office -- changes the preference for all users who log in to the current office location
    • Computer -- changes the preference for all users who log in to the specific computer
  8. Select Window Start Maximized (approximately page 3 of the list).
  9. If you want the encounterPRO window to display maximized at login, select Yes.
  10. Select Finished. The Preferences list reappears.
  11. Select OK to return to the Configuration list. You will see the change immediately. Other users will need to log out of encounterPRO in order for the change to take effect for them.
