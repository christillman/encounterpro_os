Purpose

This solution gives the instructions for selecting the default list that appears when searching for a user: the short list or the Care Team. The default is the Care Team.

Preference Levels
Each of the preference types can be applied to up to five different levels. The level controls the scope of the configuration change being made.

Details: Steps
Follow these steps:
  1. From the Office View, select the Configuration tab. The Configuration List appears.
  2. Select Preferences. The Preferences screen appears.
  3. Select General Preferences on the left, a list of available preferences is displayed.
  4. Select the preference level at the bottom of the screen.
  5. Select the User Search Default to Care Team preference. A preference parameters screen appears.
  6. Select No to change the default to the user short list. Select Yes to default to the Care Team (when there is one). Select Finished. The list of preferences reappears.
  7. Select Finished. The Preferences screen reappears.
