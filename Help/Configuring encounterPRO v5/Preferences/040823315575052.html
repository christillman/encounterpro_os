How to: Allow Users to Delete Attachment Folders

Purpose
The ability to remove folders from the database is restricted. The Allow Folder Delete preference must be changed in order to allow Super Users to remove attachment folders.

Details:

Steps
Follow these steps:

  1. From the Office View, select the Configuration tab. The Configuration List appears.
  2. Select Preferences. The Preferences screen appears.
  3. Select Attachment Preferences on the left and a list of available preferences is displayed.
  4. Select the Global as the level at which you want to apply the preference at the bottom of the screen.
  5. Select the Allow Folder Delete from the available list. (If the preference name appears in italics, it is not available at the level selected. Change the preference level at the bottom of the screen and re-try.) A preference parameters screen appears.
  6. Select Yes to allow folder deletion. 
  7. Select Finished. The Preferences screen reappears.

NOTE: Some preferences will be apparent to the user making the change immediately, others require you to restart the encounterPRO EHR. Other users in the practice will generally need to restart the encounterPRO EHR.
