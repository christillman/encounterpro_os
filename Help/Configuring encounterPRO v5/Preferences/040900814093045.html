How to: Set the Preferences to Purge Recent Text Entries

Purpose

Many data entry screens have text entry fields that retain a list of previous entries. This is intended to help the users reduce typing as it allows the user to select a recently entered text string instead of re-typing. The more popular entries will appear at the top of the list.

There are two Maintenance Preferences in encounterPRO that can help a practice manage these recent entries. One preference allows the system to purge the recent text entries during the nightly maintenance job. The second preference tells the system how many entries to retain, if any, when the purge preference is activated. This allows a practice to determine the maximum number of text entries that will be saved and available to select.

Details: Steps
Follow these steps:
  1. From the Office View, select the Configuration tab. The Configuration List appears.
  2. Select Preferences. The Preferences screen appears.
  3. Select the Attachment Preferences on the left and a list of available preferences is displayed.
  4. Select Global as the preference level at the bottom of the screen.
  5. Select Purge Recent Picks from the available list. A preference parameters screen appears.
  6. Select 'Yes' to turn on the purge preference. The list of preferences reappears.
  7. Select 'Purge Recent Picks Keep #' from the available list. A preference parameters screen appears.
  8. Enter the maximum number of recent text entries that you want to retain. The default is 30.
  9. Select Finished. The Preferences screen reappears.

NOTE: Turning on the purge recent picks preference will allow the system to purge the text lists the following night and should be apparent to all users the next day.
