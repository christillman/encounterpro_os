How to: Configure an Outstanding Treatment List for a Specific Treatment Type

Purpose
There are already lists for oustanding Labs and Tests, Prescription Medications, and Referrals. Some practices may want to add another list to track outstanding treatments for another treatment type. For example, if the practice sends patients out for x-ray services, an Outstanding X-rays list could be useful in keeping track of those treatments.

There are three main stages to the procedure:

  1. Add the observation type in the tables.
  2. Assign the new observation type to the Treatment Types you want to include in the new Outstanding Treatment list.
  3. Add a button for the new Outstanding Treatment list to the Utilities menu.
Details:

Steps
Follow these steps:
To add the observation type:

  1. Go to the c_observation_type table and add a record for the treatment type(s) for which you want to create the new outstanding treatment list. For example, if there is not an observation type of Xray, you will need to add 'Xray' to c_observation_type. If there are several treatment types that you want to group together on one Outstanding list, make up a name for that group and create an observation type for them.


To assign the new observation type to the treatment types: 

  1. In encounterPRO select Configuration from the Office View. The Configuration List appears.
  2. Select Treatment Types. The Select Treatment Type screen appears.
  3. Select the first treatment type from the list that you want to include in the new outstanding treatment list. The Treatment Type Definition screen appears.
  4. On the Properties tab, select the Observation Type. A list of available observation types (from c_observation_type) appears.
  5. Select the observation type for this outstanding treatment list.
  6. Select OK
  7. Repeat steps 2 - 6 for any additional treatment types you want to include in this outstanding treatment type list.

To add the button to the Utilities list:

  1. In encounterPRO turn on Configuration Mode.
  2. Select Utilities from the Office View. The Utilities List appears.
  3. Select the small text near the bottom of the screen that is the menu's system name. The Menu Properties screen appears.
  4. If there is not already a Local Version select Create Local Copy. Otherwise, select (highlight) the Local Version button on the right side of the screen.
  5. Select Edit Menu.
  6. Select New. The Menu Item Properties screen appears. A list of Menu Item Types appears.
  7. Select Service from the Menu Item Type list.
  8. Select General from the list of available service types. A list of general services appears.
  9. Select the Outstanding Labs/Tests bar. A parameters screen appears.
  10. Select the gray bar for Observation Type. A list of observation types appears. Select the new observation type you created in the first procedure in c_observation_type.
  11. Select Next.
  12. If you want to select the service that encounterPRO will perform when the user selects one of the items on the new outstanding treatment list, select the Treatment Service bar. A list of available services appears. The default service is to display the treatment dashboard for the item selected. Leave blank to keep the default.
  13. Select Finish. The Menu Item Properties reappears.
  14. Change the Button Title as needed. This is the text that will appear on the Utilities List.
  15. Select Finished. The Menu Properties screen reappears. The new button will appear in the list of Menu Items, often at the bottom of the list.
  16. If you want to re-order the buttons on the Utilities List, select a button you want to move and select the Move button on the right side of the screen. Use the arrow keys to position the button where you want it to appear on the Utilities List.
  17. When Finished adding and re-ordering buttons, select OK. The first Menu Properties screen reappears.
  18. Select Finished. The Utilities List reappears.
