How to: Turn on Configuration Mode

Purpose
In order to customize observations, add buttons to menus or reports, and other configuration functions you must have Standard Configuration Mode turned On (Yes).

You can turn on/off Configuratoin Mode two ways:

  • Through General Preferences
  • Using the F3 hotkey

Since the encounterPRO EHR will run slower while in (Config)  mode, be sure to go back to the Configuration Mode preference and set it back to (No) when you are done configuring on the fly.

Before You Begin
When Configuration Mode is set to Yes, you can see small black text at the bottom of most EncounterPRO menu screens. Clicking on that text enables you to configure menus and reports on the fly. When Configuration is set to No, the black text will no longer appear at the bottom of the encounterPRO EHR's screens.

Details:

Steps
Follow the steps below:
To turn on Configuration Mode from an encounter screen:

  1. From most encounter screens, press F3. A debugging screen appears.
  2. Select the bar 'Config Mode Off' to change it to 'Config Mode On.'
  3. Select OK. The original screen reappears.


To turn on Configuration Mode from the Configuration list:

  1. From the Office View, select the Configuration tab. The Configuration list appears.
  2. Select Preferences. The Preferences screen appears.
  3. Select General Preferences as the Preference Type on the left.
  4. At the bottom of the screen, select Computer. You want to only set the Configuration Mode for the workstation you are using.
  5. Select Configuration Mode.
  6. Select Yes, then Finished. The Preferences screen reappears.
  7. Select OK to return to the Configuration list. When you are done configuring, you should repeat this process and turn off Configuration Mode. Working in Configuration Mode can slow your computer when documenting encounters.
