How to: Add a New Consultant.

Purpose
Consultants are used in encounterPRO when ordering a referral, sending a patient to an outside laboratory, and when documenting the provider who referred the patient to your practice.

Before You Begin
You will need to know the name and mailing address and telephone number for the consultant. Other information such as the fax number and insurance carriers that the consultant accepts are optional.

Details:

Steps
Follow the steps below:

  1. From the Configuration tab, select Consultants. The Select User to Edit screen appears. The display defaults to Active users.
  2. Select the New Consultant button. A window appears to allow you to select whether this is an organization or an individual person. Select Organization or Person.
  3. Enter the consultant name and select OK. The New Consultant Definition screen appears.
  4. Enter all other appropriate information for this consultant.
  5. Select the Specialty for this user. This is very important, since it will aid in searching for the consultant.
  6. Select the Contact Info tab. Enter the address and telephone information for this consultant. If a Fax or Email address is known, enter that information as well. Fax and email address can be useful when sending documents using the Document Manager function.
  7. Select the Payers tab. This tab allows you to select which insurance carriers are accepted by this consultant. Select the three-dot button (...) to display a list of carriers that you can select from, then select OK.
  8. Select the Routes tab. This information is for use with Document Manager.
  9. Select the ID Numbers tab. If there are any identification numbers known for this consultant, enter those here.
  10. Select the Person/Organization tab.
  11. Review the information for the new consultant. When you are satisfied that the information is complete and accurate, select OK to save.
