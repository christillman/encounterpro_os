How to: Manually Refresh the Reports Library.

Purpose

EncounterPRO version 5 offers a new method of adding reports to your system. The Reports Library is an ever-expanding repository of reports developed by encounterPRO users as well as EHR, Inc. This list of reports will be refreshed every night during the nightly sync process. If a new report has been added to the list since the nightly sync, or the nightly sync failed you can manually refresh the Reports Library list.

Shareware Reports

EncounterPRO users who have developed their own reports can choose to share those reports with the encounterPRO community by uploading the reports to the Reports Library. Those reports will be labeled 'Shareware' in the Reports Library. See How to: Export a Report to the Reports Library.

CAUTION: Shareware reports are not warranted or maintained by EHR Product Support. If there is a defect in a shareware report, there will be a fee charged if you wish EHR Product Support to troubleshoot or repair the report.

Details:

Steps
Follow these steps:

  1. From the Office View, select the Configuration tab. The Configuration list appears.
  2. Select Reports. The Select Report screen appears.
  3. Select Library under the Search Options. A keyboard screen appears.
  4. Leave the field blank and select OK. A list of matching reports appears.
    NOTE: If the report has been created by an encounterPRO user rather than EHR, Inc. the report name bar will display 'Shareware.' These reports are not warranted or maintained by EHR Product Support.
  5. Select the Refresh Library button  above the Library Search Option. A prompt appears alerting the user that the refresh can take several minutes. Select Yes to proceed.
  6. When the refresh function is complete, the list of matching reports is updated.
  7. At this point, you can search for a specific report in the Library.
  8. When finished, select Cancel to return to the Configuration list.
