How to: Add a New encounterPRO User

Purpose
The User Configuration screen allows Super Users to enter and edit the identity, credentials, roles, and blanket privileges of all encounterPRO users. The providers' personal identification # (PIN), access code, color, prescription certification, and supervisor assignment for mid-level providers are assigned from the User Configuration screen as well.

Before You Begin
If this user needs to be able to write prescriptions, you will need all the identification numbers (license, certification, DEA,UPIN, NPI, and e-Prescribing ID,  etc.). You may also need to set up the signature stamp for each provider. See Screen: How to: Add a Signature Stamp for a User for detailed instructions.

Details:

Steps
Follow the steps below:
To enter the basic information:

  1. From the Configuration tab, select Users. The Select User to Edit screen appears. The display defaults to Active users.
  2. Select the New User button. A window appears to allow you to enter the user's name.
  3. Enter the user's name and all other appropriate information and select Finished. The User Definition screen appears.
  4. Select the Specialty for this user. This is very important, since it controls which short lists the user will be able to view. (Short lists are tied to specialties.)
  5. Select the user assigned to supervise the new user, if any.
  6. Select the gray bar for the Username. Select Create New Username.
  7. Enter the Username. This is the name that appears on the User Selection screen and other areas in encounterPRO.
  8. Enter the email address.
  9. Select the Reset Password button. A number pad appears. Enter the passcode for this user. You will be prompted to enter it twice to confirm the number. The system will acknowledge if the passcode was accepted or notify you that the passcode was rejected.
  10. Review the Display Format information:
    • Full name
    • Short name
    • Initials
  11. Select the Color bar and assign the color for this user. Generally providers will have their own unique color. Other office staff will have the color assigned to their main role.

To Enter the Contact Information:

  1. From the User Definition screen, select the Contact Info tab.
  2. Enter the user's address, phone, and email address.

To Set up the User's Roles:

  1. From the User Definition screen, select the Roles tab. The User Roles list appears.
    NOTE: All users start with the User role. Do not remove the User role, it is very important to many workflows.
  2. Select the Add button then select each role applicable to this user. Repeat until all applicable roles have been added, then select OK. Each role that you add will add a tab to the Task list for that user.
    NOTE: All providers and nurses should be assigned the role of Clinical Staff in addition to their other roles.

To Set the User's Permissions:

  1. From the User Definition screen, select the Permissions tab. All permissions are initially set to No.
  2. You can set different permissions for a user by office location. Select the office location for which you would like to set permissions (if your practice has multiple locations).
  3. Under Special Privleges, select the No button to change to Yes:
    • If the user is allowed to access clinical (patient) data
    • If the user is allowed to prescribe medications (be sure to enter the user's ID numbers)

  4. Under EncounterPRO Privileges, select the No button to change to Yes for the functions you want this user to be able to perform:
    • User may read audit records
    • User may edit common standard exams
    • User may edit common treatment lists
    • User may set or modify computer preferences
    • User may use encounterPRO in Configuration Mode
    • User may edit procedure charges
    • User may edit common short lists
    • User may set or modify global preferences
    • User may scribe for other users
    • User has all privileges and can access all services (the Super User setting)
    • User may set or modify preferences for other users

If the User is Prescription Certified, enter the ID Numbers:

  1. From the User Definition screen, select the ID Numbers tab.
  2. Enter the following identification numbers for this user:
    • DEA #
    • Certification #
    • License #
    • UPIN (Unique Provider Identification Number)
    • NPI (National Provider Identifier)


    NOTE: This information can be configured to appear on the prescriptions written by this provider.
  3. If your practice is using e-Prescribing, select the Register for E-Prescribing button. Follow the prompts to register this user. The user's e-Prescribing ID appears when the process is complete. See E-Prescribing for details.

To Save the New User:

  1. From the User Definition screen, select the User tab.
  2. Review the information for the new user. When you are satisfied that the information is complete and accurate, select OK to save.
