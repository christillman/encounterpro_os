How to: Change Refill Buttons and Links on RX Treatment Dashboard to Not Use Doc Mgr

Solution:

In PedsStandard 1.1 the refill buttons and links are set up to use E-RX and Document Manager as follows:

Links at the top of the dashboard:

The items in blue above will order a NewRX document and send it to Document Manager. The Print Now link has the flag to Print Now to the default printer.

Buttons included at the bottom (Services on the Treatment type):

These are all set up to use Document Manager and to add a progress note that a refill was ordered and either Printed or sent to Doc Mgr. the Print Now buttons use Doc Mgr, but are set to Print Now and should print ot the default printer.

This is assuming that the default treatment mode for Prescriptions is the E-RX treatment mode.

Details:

What If the Practice Doesn't User E-RX or Document Manager?

If the practice does not want to use e-RX and Doc Mgr here's what you need to do:

Go to the Prescription Treatment Dashboard and edit the Display Script.

Enable the line for  Disable the lines for and

Then for the buttons (services) at the bottom, go to Configuration>Treatment type and select Prescription Drugs -- Go to the Services tab. Delete the buttons for and  and and turn to Yes the settings for the  button that is currently hidden.
