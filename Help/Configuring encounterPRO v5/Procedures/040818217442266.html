How to: Modify an Existing Procedure

Purpose
You may need to update your procedures to reflect new CPT codes or change billing or E & M coding information.

Before You Begin
When encounterPRO is installed, there is no E & M coding information entered for procedures. If you want the system to consider Risk Level in calculating E & M coding levels you will need to enter a Risk Level for each procedure.

Details:

Steps
Follow the steps below:

  1. From the Configuration tab, select Procedures. The Procedure Configuration screen appears.
  2. Search for the procedure you want to edit. You can search based on the procedure type to be sure that the system searches the entire procedures database. The matching procedures are displayed.
  3. Select the procedure you want to modify. A menu of options appears.
  4. Select the Edit Procedure button. The procedure definition screen for the type of procedure you are updating appears. For example Collect Procedure or Vaccine Admin Procedures.
  5. Make the changes needed, then select OK. The Procedure Configuration screen reappears.
  6. Repeat steps 2 through 5 to modify additional procedures. When finished, select OK to exit.
