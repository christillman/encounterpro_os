How to: Set a Duplicate Treatment Warning

Purpose

Setting the number of days for the Duplicate Warning to a value greater than zero causes encounterPRO to display a warning when same treatment of that type is ordered for the same patient within that number of days. For example, if your practice configures the Lab treatment type to have a Duplicate Warning and sets the number of days to 180, if a provider orders the same lab treatment within 180 days for the same patient a warning will appear reminding the provider when the lab was most recently ordered. The lab can be canceled or ordered if needed.

Details:

Steps

Follow these steps:

  1. From the Office View, select the Configuration tab.
  2. Select Treatment Types. The Select Treatment Type screen appears.
  3. Select the desired Treatment Type from the list.
  4. Go to the Properties tab.
  5. Enter  the number of days in the Duplicate Warning field.
  6. Select OK.
