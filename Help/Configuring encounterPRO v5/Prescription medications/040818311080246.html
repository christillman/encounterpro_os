Purpose
Dispense amounts on your prescriptions can be displayed as either numeric only or as numeric and textual amounts. This is determined by a Prescription Preference set globally for the entire practice.

Before You Begin
You must set this preference globally (if you have sufficient permissions) which will change the setting for all users in the practice. Be sure to consult with the other providers in the practice before making global preference changes.

Details:

Steps
Follow the steps below:

  1. From the Configuration tab, select the Preferences tab. The Preferences screen appears.
  2. Select Prescription Preferences on the left.
  3. You can only set this default for the entire office. Make sure that Global is at the bottom of the screen.
  4. Select Display RX Dispense Amount Textually.
  5. If you want to display the dispense amounts on your prescriptions numerically only, select No. If you want to display the dispense amounts both numerically and textually, select Yes.
  6. Select Finished. The Preferences list appears.
  7. Select OK to return to the Configuration list.
