Description
This screen is used to configure simple or single prescription medications. There are several tabs that allow the drug configurer to set the basic information, drug packages, administration, categories and specialties for the new medication.

Details:

How to Access This Screen
Access this screen by selecting Configuration from the Office View, then selecting Drugs.

Screen Example

 To view the other tabs for this screen, select one of the links below:

  Single Drug Definition - Packages tab
  Single Drug Definition - Administration tab
  Single Drug Definition - Categories tab
  Single Drug Definition - Specialties tab

The tabs for HCPCS, Makers and Drug Codes are used to configure in-office medications and vaccines and are not used to configure single drugs.
