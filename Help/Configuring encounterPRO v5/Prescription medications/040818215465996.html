Purpose
You may need to make changes to existing prescription drugs in the database to better meet the needs of your practice. For example, you may need to add another package option or administration rule to a drug in the database. 

Details:

Steps
Follow the steps below:

  1. From the Office View, select the Configuration tab. The encounterPRO Configuration list appears.
  2. Select Drugs. The Drug Configuration screen appears.
  3. Select one of the search option buttons.
    • If you know the Trade Name of the drug, select that button, enter the trade name or a partial name and select OK. Note the option to search by Begins With or Contains.
    • If you know the Generic Name, select that button, enter the generic name or a partial name and select OK. Note the option to search by Begins With or Contains.
    • If you want to search by type of drug, select the Category button then select a category from the list.
  4. The matching drugs are shown. If no matches were found under your specialty, select the Specialty/All button above the Category button to search the entire database. Select the drug you want to update. A menu of options appears.
  5. Select the Edit Config button. The Drug Definition screen appears.
  6. Make changes as needed. You can add or change package options, administration rules, duration, dosage, etc.
    • How to: Add a New Prescription Medication Package
    • How to: Document Administration of In-office Prescription Medications.

    NOTE:  It is recommended that you do not change the Trade name of the drug.
  7. Select OK to save your changes.
