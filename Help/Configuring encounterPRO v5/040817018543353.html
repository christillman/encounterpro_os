Description
Use this list to select the area you want to configure. The items appearing on the list will depend on privileges assigned to you.

NOTE: Configuring certain areas of encounterPRO requires advanced customization training. A few areas are reserved exclusively for EHR personnel.

How to Access This Screen
Access this screen by selecting the Configuration tab on the Office View.
