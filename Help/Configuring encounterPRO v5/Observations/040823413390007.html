How to: Configure Result Ranges

Purpose
Super Users are able to configure lab and test result observations to display pre-determined normal result ranges in the result tables that display on reports and the treatment dashboards. This allows the provider to have the normal result range available when reviewing the patient's results. Additionally, the system can evaluate the numeric results entered and flag the result as Normal, Abnormal, Low, Very Low, High or Very High. These result ranges can be based on criteria such as gender and age range.


Example: Result display grid

There are two options when setting a result range:

  1. Display the normal range as text only: Enter a normal result range that is for display purposes only and does not evaluate the individual results entered. This text appears in the result display grid in the Normal Range column.
  2. Advanced result range options: Set up normal and/or abnormal result range rules that can be age and/or gender-specific and will flag the patient's result as abnormal if it falls outside the entered range. The system will dynamically display text in the Abn and Normal Range columns of the result display grid based on the rule used to evaluate the data entered.

When to Use the Advanced Result Options
You will want to consider using the advanced result options when:

  • There is no single reference range that applies to everyone because the tests performed may be affected by the age and sex of the patient.
  • You want encounterPRO to evaluate the results entered and set the Abnormal/Normal flag based on the individual result entered.

Working with Multiple Rules

When you use advanced result options you will be setting up rules that the system uses to evaluate the result data entered. It is important to note:

  • The system will examine the rules in the order they are entered (sequence #). When the system finds a rule for which the patient in question meets all criteria, the system uses that rule to evaluate the data and will not continue to look at any subsequent rules. So if rule #1 has no sex or age criteria, then the system always use that rule (unless it find a patient missing an age or sex designation).
  • When setting up multiple rules, it is best to create a blank/blank (no sex, no age range) rule as the LAST rule. This way, if the rules do not happen to be 100% mutually exclusive, there is still a rule that the system can use to evaluate the data.

Steps

NOTE: It will be easiest if you perform this configuration in the context of a patient encounter using a 'test' patient. This will allow you to enter results and immediately see if the configuration was successful.

Follow these steps:
To enter the normal or reference range:

  1. From the Patient Chart, order the lab or test for which you want to enter a normal result range.
  2. Select the blue description of the treatment to access the treatment dashboard.
  3. Select the blue description of the treatment to access the data entry screen.
  4. Select the Configure button at the bottom of the results entry screen (you may need to select Other Options first). The Observation Definition screen appears.
  5. Select the gray box for Results. The Result Set Definition screen appears.
  6. If the result for which you want to enter a normal result range has already been created, select the bar for that individual result, then select Edit. The Result Definition screen appears.

    If not, select New Result. The Result Definition screen appears.
  7. The Result Type can be Normal or Abnormal.
    CAUTION: Remember, you are setting only a text result to display the normal range on a result grid, the system will not use this to evaluate whether the data entered are normal or abnormal. If you need the system to evaluate the data entered, you will need to set the Advanced Result Options for this result.
  8. If this is a numeric result, set Result has Value to Yes. Select the button for the type of value represented by this result.

    If this result is textual, then set Result has Value to No. An example of a textual value would be a color or other qualifier, like clear or cloudy.
  9. Select the gray bar next to Normal Range. A text entry box appears.
  10. Enter the text that you want to appear in the Normal Range column for this specific result. Maximum length is 40 characters.
  11. Select OK. The text appears in the Normal Range field.
  12. Select OK to return to the Result Set Definition screen.
  13. Repeat steps 6 through 12 to enter the normal result range for any other results for this observation.
  14. When finished, select OK. The Observation Definition screen reappears.
  15. Select OK. The result data entry screen for the lab or test appears.
  16. Enter sample data in the result fields for which you entered the normal range text, then select Finished. The treatment dashboard screen reappears. The result grid should display your normal range text in the Normal Range column.

Follow these steps:
To enter the Advanced Result Options:

  1. From the Patient Chart, order the lab or test for which you want to enter advanced result options.
  2. Select the blue description of the treatment to access the treatment dashboard.
  3. Select the blue description of the treatment to access the data entry screen.
  4. Select the Configure button a the bottom of the results entry screen (you may need to select Other Options first). The Observation Definition screen appears.
  5. Select the gray box for Results. The Result Set Definition screen appears.
  6. If the result for which you want to enter the advanced result options has already been created, select the bar for that individual result, then select Edit. The Result Definition screen appears.

    If not, select New Result. The Result Definition screen appears.
  7. The Result Type should be Normal.  If the data entered for the result is outside the result ranges entered for the advanced result options, the system will automatically change the Normal to Abnormal on the result display grid.
  8. Select Yes for Result has Value. Select Number as the type of value represented by this result. (Only numeric results can be evaluated using advanced result options.)

    TIP: You should enter a text result in the Normal Range field on this screen in the event that a patient does not meet any of the result range criteria entered on the Result Advanced Options screen. This is a precaution so that the provider will be alerted if certain information about the patient's age or gender is not available.

    1. Select the gray bar next to Normal Range. A text entry box appears.
    2. Enter the text that you want to appear in the Normal Range column for this specific result. Maximum length is 40 characters. Suggestion: Patient data incomplete - no range available
    3. Select OK. The text appears in the Normal Range field.
  9. Select the Advanced Options button at the bottom of the screen. The Result Advanced Options screen appears.
  10. Select New Rule. A set of fields for you to enter the rule criteria appears.
  11. Select the sex that this rule will apply to or leave blank if sex is not part of the result range criteria.
  12. Select an Age Range from the drop-down list or leave blank if age is not part of the result range criteria.

    If the age range you need does not appear on the drop-down list, select the New Age Range button. Select the Age From and Age to and select OK. The new age range should appear on the drop-down list for selection (you may have to select OK and then Advanced Options again to refresh the list).
  13. Enter the Low and High values for the Normal Range.
  14. Enter the text you want to appear in the Normal Range column of the result display grid for patients who meet the gender and age criteria entered.
  15. If you want to set additional ranges for Low/Very Low and High/Very High enter those values.
  16. Enter the text you want to appear in the Abn column of the result display grid for patient's whose results fall in the Low/Very Low range in the Low Nature and Very Low Nature fields respectively (8 characters maximum).
  17. Enter the text you want to appear in the Abn column of the result display grid for patient's whose results fall in the High/Very High range in the High Nature and Very High Nature fields respectively (8 characters maximum).
  18. Repeat steps 10 through 18 to add another rule.
    TIP: You should always add a last rule that has no sex or age criteria as a 'catch-all' rule for any patient who is missing the required age or sex criteria. The Normal Range can be 0 to 0 and the suggested text is 'Patient data incomplete - no range available.'
  19. When you have entered all the rules for this result, select OK. The Result Definition screen reappears.
  20. Select OK to return to the Result Set Definition screen.
  21. Repeat steps 6 through 14 to enter the normal result range or any other results for this observation.
  22. When finished, select OK. The Observation Definition screen reappears.
  23. Select OK. The result data entry screen for the lab or test appears.
  24. Enter sample data in the result fields for which you entered the normal range text, then select Finished. The treatment dashboard screen reappears. The result grid should display your normal range text in the Normal Range column.
