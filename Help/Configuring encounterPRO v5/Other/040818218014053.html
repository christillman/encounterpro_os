How to: Require Supervisory Review

Purpose
In certain cases you may have a user who's encounters must be reviewed by another provider. Residents and new providers who do not have their own billing number in the billing system will generally require you to set up the supervisory review process.

This is a two-part process:
  1. Select the user's supervisor on the User definition screen
  2. Set the Workflow Preferences to require supervisory review

Before You Begin
Your workplan configuration will control what happens during the Supervisory Review. There are several options, the most common being:
  • Send patient to Checkout, send a ToDo to the supervisory provider to review the chart
  • Leave patient in exam room, create a service bar for the supervisory provider to review the chart before it can go to Checkout
  • Move the patient to a Pre-checkout room with a service bar for the supervisory provider to review the chart before it can go to Checkout
Details:

Steps
Follow the steps below:
To Set the Supervisor for a User:

  1. From the Configuration tab, select Users. The Select a User to Edit screen appears.
  2. Select the user who needs to have a supervisor assigned. You may need to use the Search options to locate the desired user. The User Definition screen appears.
  3. Select the gray bar for Supervisor. The Select User screen appears.
  4. Select the user who will be supervising the current user. The User Definition screen reappears.
  5. Select OK to return to the Select User to Edit screen.
  6. Select OK to return to the Configuration list.

To Set the Supervisory Preference:

  1. From the Configuration tab, select Preferences. The Preferences screen appears.
  2. Select Workflow Preferences from the selections on the left.
  3. Select User as the preference level (along the bottom of the screen). The user id of the current user appears.
  4. If you want to set this preference for a different user, select the name for the Preference Key. The Select User screen appears.
  5. Select the user who needs to be supervised. The Preferences screen reappears with the selected user as the Preference Key.
  6. Select Supervisor Review. Type 'True' in the Supervisor Review field then select Finished.
  7. Select OK to save your changes and return to the Configuration list.
  8. Repeat steps 1 through 6 for each user for which you want to activate supervisory review.
