How to: Add a Move Assessment Button

Purpose
The Assessment Move button can be added to the Billing Review screen or the Patient Chart menu, if desired. This button will allow the provider to rearrange the order of assessment on the Patient Chart and Billing. See How to: Rearrange Assessments on the Patient Chart for details on reordering assessments.

Details:

Steps
Follow these steps:

  1. Turn on Config Mode. See How to: Turn on Configuration Mode.
  2. From the Patient Chart select the Exit tab to display the Billing tab.
  3. Select the gray text box with the billing summary. The Billing Review screen appears.
  4. Select the small text near the bottom of the screen that is the menu's system name. The Menu Properties screen appears.
  5. If there is not already a Local Version select Create Local Copy. Otherwise, select (highlight) the Local Version button on the right of the screen.
  6. Select Edit Menu.
  7. Select New. The Menu Item Properties screen appears. A list of Menu Item Types appears.
  8. Select Service from the Menu Item Type list.
  9. Select Patient from the list of available service types. A list of patient services appears.
  10. Select Assessment Move. There are no parameters to select.
  11. Select Finished.
  12. Select Finished again to return to the Menu Properties screen.
  13. Change the Button Title as needed. This is the text that will appear on the Utilities List.
  14. Select Finished. The Menu Properties screen reappears. The new button will appear in the list of Menu Items, often at the bottom of the list.
  15. If you want to re-order the buttons on the Utilities List, select a button you want to move and select the Move button on the right of the screen. Use the arrow keys to position the button where you want it to appear on the Utilities List.
  16. When Finished adding and re-ordering buttons, select OK. The first Menu Properties screen reappears.
  17. Select Finished to save the changes and return to the Billing Review screen.
