How to: Add an Uncancel Encounter Button to a Menu

Purpose
Certain users have the permission to cancel patient encounters. In the event that the wrong encounter is canceled, you can also un-cancel an encounter. You will need to add a button to the desired menu to use this function. The uncancel function is restricted and only users who have the authority to uncancel will see the button.

Before You Begin
Be sure to turn on Configuration Mode before you begin. See How to: Turn on Configuration Mode for instructions.

Details:

Steps
Follow the steps below:

  1. Access the encounterPRO screen with the menu you want to add the Chart Export button. In many cases, practices choose to add this button to the menu on the Patient Demographics screen.
  2. Select the menu name in small type that is displayed above the menu buttons. The Menu Properties screen appears.
  3. If a local copy does not already exist select the Create Local Copy button. The system prompts you to confirm that you want to create a local copy of the menu.
  4. Select Yes. A new button for the Local Version appears on the right side of the screen.
  5. Be sure the Local Version button is highlighted (darkened). Select Edit Menu.
  6. Select New to add a new button. A list of menu item types appears.
  7. The Chart Export button is a Service. Select Service from the list. A list of contexts appears, in this case General and Patient.
  8. Select Patient. A list of patient-level services appears.
  9. Scroll down the list of services and select Uncancel Encounter from the list. A preference screen appears.
  10. Select Finished. The Menu Item Properties screen reappears.
  11. Edit the button title, if desired.
  12. Select Finished. The Menu Properties screen reappears with the new button at the bottom of the list of menu items.
  13. If you want to re-order the buttons on the menu, select the bar for the menu item and then select the Move button on the right to move the item up or down on the list.
  14. When finished arranging the order of the buttons on the menu, select OK. The Menu Properties screen reappears.
  15. Select Finished. The original screen from step 1 reappears. The new button has been added to the menu. You may need to select Other Options to display all the menu buttons.
    NOTE:  As the user configuring the menu, you will see the new button immediately. Other users will need to log out of EncounterPRO and log back in to see the changes.
