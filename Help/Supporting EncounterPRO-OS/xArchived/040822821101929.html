How to: Change a folder name

Solution: First change the folder description in c_folder. It has to be changed in the 'folder' and 'description' column. Run the appropriate script.
Details:

To change a folder in 4.0 it has to be done in the two below tables:

1) First change the folder description in c_folder. It has to be changed in the 'folder' and 'description' column.

dbo_c_Folder

folder context_object context_object_type description status sort_sequence workplan_required_flag
Labs Treatment LAB Lab OK 6 Y

2) You will to run the following script:

Update p_Attachment
set attachment_folder = 'Labs- --new name
where attachment_folder = 'Lab Report' --old name
