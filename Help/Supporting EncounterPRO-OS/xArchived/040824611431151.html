How to: Find EHRI Forms

Solution:

Please create a shortcut to this folder so that if the forms are modified or updated you will always use the current form. (COMPANY Expense Form, Time Off Request Form, Shipping Form, etc.)

T:\Administration\FORMS

 

REMINDER: 
In Outlook under Public Folders there is a contact list named Company Directory which lists employee  business contact information.  I suggest choosing the view named ?Phone List?, it may be sorted by Company, File As (alphabetical by last name), Full Name, Ext #, etc. 
? If you need to know the extension of someone in the ATL office please refer to this list.
