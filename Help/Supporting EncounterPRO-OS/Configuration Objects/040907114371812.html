How to: Install Reports with the Report Installer

Solution:

This solution explains how to install reports with the Report Installer.

Notes
The report installer can be found at 4.05 ReportInstaller on JMJ FTP

Details:

 

  1. Download the installer to the practice's server to a folder like this: D:\Install\JMJ
  2. Run the installer. Enter the server name where the database resides and the database name. Next you will see a list of reports to install.
  3. For each report you wish to install: Select the report you wish to install and click Finish.
    If the report installed correctly you will get a message indicating it was successfully installed.
    If the report did not install correctly you will get a message indicating it was not successfully installed.
  4. The report will usually need further configuration. Depending on whether the report is for the Patient or Encounter or General context, to configure the report, go to Configuration, Reports, Patient/General/Encounter, select the report, Default and enter the values that are required.
