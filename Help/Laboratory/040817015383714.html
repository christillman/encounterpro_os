Description
This screen is used to track and manage send out labs and tests for practices that have a lab integration. A separate Daily Referrals screen will be available for each laboratory entity with which the practice is integrated.  encounterPRO Support can default the list to show a certain laboratory which can then be changed manually, or the user can manually select the laboratory each time.
This screen shows the status of all labs ordered on a specific date and can be used to navigate to the dashboard for that treatment. A lab or test will only appear on the appropriate list if that laboratory was selected as the consultant on the Patient Referred to or Specimen sent to fields of the Specimen Collection screen.

How to Access This Screen
Access this screen by selecting the Daily Report on the Utilities List and choose a specific laboratory.
