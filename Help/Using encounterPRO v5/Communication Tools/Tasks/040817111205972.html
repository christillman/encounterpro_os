Details: Description
This screen is used to access Inbox Tasks assigned to you or the roles assigned to you. Inbox tasks are those tasks that do not have a corresponding service bar on the Office View, probably because they do not concern a patient that is physically in the office. Inbox tasks can be:
  • Automatically generated by the system as part of a treatment workplan, such as a Task to alert a provider to review lab results when they are scanned into the patient's lab reports folder
  • Manually created by another user either from an encounter screen with an Inbox Task button or from the Task list screen

You can use this screen to monitor how long items are waiting for completion. Each Task bar displays a description of the item plus the number of minutes it has been waiting (in the green box).

Lists on the Screen
The screen has several lists that can be displayed:

  • Each user has a My Tasks tab. These are items specifically assigned to you.
  • Each user has a tab for each role he or she is assigned, such as Provider or Phone Nurse.
  • These items can be completed by anyone with that role.
  • If your practice has multiple locations, each location has a bar on the right of the screen so that you can view Tasks by location.
  • When you complete a Task assigned to you or you take ownership of a task assigned to one of your, the item moves to your Completed list.
  • Lists can be sorted by age or description. The default is by age with the newest items appearing at the top of the list.

How to Access This Screen
Access this screen by selecting the Tasks tab on the Office View.
