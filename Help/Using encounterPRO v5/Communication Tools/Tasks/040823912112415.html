Details: Description
Use this screen to create a new Task. If the Task was created from a Patient Chart, the Patient name appears. If the Task was created from a specific treatment, the treatment description appears in the Item field.

How to Access This Screen
Access this screen by selecting one of the Inbox or Inoffice Task buttons on an encounterPRO menu. Additionally, you can access this screen by selecting the New Task button on the Task list screen.
