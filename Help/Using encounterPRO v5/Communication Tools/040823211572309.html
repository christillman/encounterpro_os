How to: Set the Audible Priority Alert

Purpose

There are three Msg/Task Preferences that allows the encounterPRO Administrator or Super User to turn on an audible alert to notify users when a high or urgent priority task or message is waiting. The audio alert it not required. High and urgent priority messages and tasks will be indicated by red !! on the item bars and on the Office View tabs.

You can set a different sound for High and Urgent messages and tasks to differentiate the two levels, if desired. You can either enter the full UNC file path for the .wav file you want to use for each priority level alert or accept the default sound and differentiate the two levels by indicating the number of 'beeps' for high and urgent.

Before You Begin

Determine at what level you will be setting the preference. This preference can be set at the following levels:

  • Globally - which will affect every workstation in the entire practice
  • Office - which will affect every workstation in the office location specified
  • Computer - which will affect only the workstation specified

EHR recommends that the preference be set on a workstation by workstation basis. Determine with the practice personnel which workstations will be most effective to have an audio alert.

If you want to use a sound other than the default 'beep' you will need to know the full UNC file path of the .wav files you want to use for each of the priority levels.

Details:

Steps

Follow the steps below:

  1. From the Configuration tab, select the Preferences tab. The Preferences screen appears.
  2. Select Msg/Task Preferences on the left.
  3. You can set the audible priority alert option for the entire practice, the office location, or computer. Select the level you want to change the preference for at the bottom of the screen:
    • Global -- changes the preference for all users
    • Office -- changes the preference for all users who log in to the current office location
    • Computer -- changes the preference for all users who log in to the specific computer
  4. Select Audio Priority Sound.
  5. Select Yes. 
  6. Select Finished. The Preferences list reappears.
  7. Scroll down the list to locate the preference Priority Sound - High.
  8. Enter the complete UNC file path of the .wav file you want to use for the audible alert for High priority. If you want to use the default sound, you can select the number of 'beeps' to use for the High Priority messages and tasks. Enter a number plus 'beep'. For example, 2 beep for two sounds in rapid succession. The default is 1 beep for High priority.
  9. Select Finished. The Preferences list reappears.
  10. Scroll down the list to locate the preference Priority Sound - Urgent.
  11. Enter the complete UNC file path of the .wav file you want to use for the audible alert for Urgent priority. If you want to use the default sound, you can select the number of 'beeps' to use for the Urgent Priority messages and tasks. Enter a number plus 'beep'. For example, 2 beep for two sounds in rapid succession.  The default is 2 beep for Urgent priority.
  12. Select Finished. The Preferences list reappears.
  13. Select OK to return to the Configuration list.

NOTE: The change will not take effect immediately. encounterPRO will need to be closed and re-started on each workstation affected for the audible alerts to begin.
