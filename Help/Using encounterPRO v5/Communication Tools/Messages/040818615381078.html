How to: Move Messages into a Folder.

Purpose
Many users find it useful to further organize their messages in a series of personally created folders. Moving or saving a message into a user-created folder actually creates a copy of that message in the new folder. The original message will still appear in the Read or Sent messages folder.

Before You Begin
You will need to create your message folders before you can move the messages. See How to: Create a Messages Folder for details.

Details:

Steps
Follow the steps below:

  1. From the Office view, select the Messages tab.
  2. Locate the message you want to move on the Inbox, Read or Sent Messages list.
  3. Select the message bar to open the message.
  4. Select Move to Folder. A list of available folders appears.
  5. Select the folder in which you want to save the message.
  6. If you want the message to be saved in more than one folder, repeat steps 4 and 5.
  7. Select I'm Finished to return to the Message list.
