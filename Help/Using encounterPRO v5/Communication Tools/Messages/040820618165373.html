How to: Delete a Message

Purpose
Messages that are not related to a specific patient can be moved to a Deleted Messages folder to assist with organization. Messages are not actually deleted, they are simply moved to a Deleted Messages folder. If a "deleted" message needs to be retreived later, the user can move the message from the Deleted Messages folder to any other available folder.

Before You Begin

NOTES:
You cannot "delete" messages from the Sent Messages or Deleted Messages folder.
You cannot "delete" patient-related messages.

Details:

Steps
To delete an encounterPRO message, follow these steps:

  1. Select Messages from the Office View.
  2. Select the gray bar for the message you want to delete. The message is displayed.
  3. Select the Delete button at the bottom of the screen.
  4. The system prompts you to be sure that you want to delete the current message. Select Yes. The message is moved to your Deleted Messages folder.
