Details: Description
This screen is used to search for patients. You can search using the various criteria, including:
  • Last name or partial last name
  • First name or partial first name
  • Billing ID
  • Social Security Number
  • Date of birth
  • Telephone number
  • Employer name
  • Employee ID

The more criteria you enter, the narrower the search will be.
