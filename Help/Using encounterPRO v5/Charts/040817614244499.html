How to: Add a Portrait to a Patient's Chart

Purpose
Adding a portrait to the patient's chart is another way to ensure that you have the correct patient's chart open.

Before You Begin
You will need to use a digital camera or scan a photo so that the portrait is in a compatible file format. EncounterPROcan import digital photos directly from many digital cameras. The camera must be configured as an external source on the workstation it is connected to. See  External Sources Available by Computer.

Details:

Steps
Follow the steps below:

  1. Access the Import Patient Portraits Attachments by either: selecting the Portrait box near the top of the Patient Chart, or selecting the Portrait button at the bottom of the Patient Demographics screen.
  2. Select the New (star) button. A list of External Sources appears.
  3. Select the source of the portrait file. If you are importing directly from a digital camera, select the Digital Camera option. If you are importing a scanned file, select File Import.
  4. Locate the portrait for this patient and select it. A gray bar for the file appears on the list.
  5. Select Finished. The portrait appears on the SOAP and Patient Demographics screens.
