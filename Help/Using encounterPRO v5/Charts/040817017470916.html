Details: Description
This screen is used to select the patients whose charts you want to export and what information out of the chart you want to include in the export. You can save the information to the workstation or network or send it directly to a printer.

When exporting the charts of multiple patients, you can select the patients individually or select a primary provider and all patients with that primary provider will be included in the export.

You can export any or all of the following information from the patients' charts:
  • Assessment history
  • PFSH
  • All medications report
  • All encounter reports
  • A list of attachments
  • Actual attachments
