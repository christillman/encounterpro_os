How to: Merge Patient Records

Purpose
On rare occasions a duplicate chart will be created in the database. When this happens, you can perform a merge patient function to combine the two charts into one. This operation removes all information from one chart and moves it into the designated destination chart. Be sure you know which chart you want to keep and which can be discarded before beginning this process.

Before You Begin
Merging charts is a restricted function. The instructions assume that a Merge Patients button has been added to the Utilities list. If you have the appropriate permissions, but do not see the button you can add it. See How to: Customize Menus for details.

CAUTION: Use extreme caution! This operation cannot be undone (even by EHR Product Support). If you merge the incorrect files, you will need to re-create both original files. The system will prompt you several times to make sure that you understand the consequences of this action.

Details:

Steps
Follow the steps below:

  1. From the Office View, select Utilities. The Utilities List appears.
  2. Select Merge Patients. The Patient Merge Utility screen appears.
  3. The patient shown in the box on the left will be merged into the patient record in the box on the right. Select Search under the Merge This Patient box on the left. The Patient Search screen appears.
    1. Enter the search criteria. A list of matching patients appears.
    2. Select the patient name bar for the patient record that you want to merge into the other patient record. This patient record will no longer be available after the merge is completed.

      The Patient Merge Utility screen reappears with the patient information displayed in the left box.
  4. The patient shown in the box on the right will be the destination of the information contained in the patient record in the box on the left. Select Search under the Into This Patient box on the right. The Patient Search screen appears.
    1. Enter the search criteria. A list of matching patients appears.
    2. Select the patient name bar for the patient record that you want to merge the other patient record into. This patient record will be available after the merge is completed.

      The Patient Merge Utility screen reappears with the patient information displayed in the right box.
  5. Check that you have the correct patients in each box. Remember, the merge process will merge all data from the patient on the left into the patient record on the right.
  6. Select Merge Patients. The system displays a warning message.
  7. To proceed with the merge, select I'm Sure. The system displays a second warning message.
  8. If you are sure that you have the correct patients and your want to remove all information from the chart on the left and merge it into the chart on the right, select Yes.
  9. The charts are merged. The patient chart that was listed on the left is now empty and unavailable. All data will now appear in the patient chart that was listed on the right.
  10. Select Finished to return to the Utilities list.
