Purpose
It can be helpful to document patient relationships within your practice. This feature allows you to note on the Patient demographics screen which of your patients are related and what that relationship is.

Currently the relations that are documented are informational only. There is no sharing of data between the patient records.

Before You Begin
Both persons you want to note a relationship between must be in the patient database.
Details:


Steps
Follow the steps below:

  1. From the Office View, select the Charts tab. The Patient Search screen appears.
  2. Enter the search criteria for the patient name.
  3. Select the patient from the list of matching patients. The Patient Demographic screen appears.
  4. Select the Relations tab. If any relations have been documented, they appear in the Relations box.
  5. Select Find Relation. The Patient Search screen appears with the current patient's last name already entered and a list of matching patients.
  6. Change the search criteria as needed and select Search. A list of matching patients appears.
  7. Select the name bar of the patient you want to add as a relation to the original patient. A list of possible relationships appears.
  8. Select the relationship between the patients from the list. The patient appears in the Relations box.
  9. Repeat steps 5 - 8 to add additional relations.
  10. When all relations have been documented, select Finished. The Patient Search screen reappears.

NOTE: You will need to repeat this procedure for the patients you added to the relationship list, encounterPRO does not automatically add reciprocal relationships.
