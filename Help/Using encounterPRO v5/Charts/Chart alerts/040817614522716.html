Purpose
Chart alerts can be used for information that you want to be sure the office personnel will notice. Alerts are commonly created for items such as:

  • Clinical data, for example a latex allergy
  • Social information, for example similar or same named patients (for example, multiple John Smith's)
  • Financial or administrative, for example a patient needs a return to work form before they depart the office today or needs to see the billing department about a past due amount

Before You Begin
There are two types of chart alerts:

  • Alerts: Alerts are more permanent and can only be removed by a provider or clinical staff person.
  • Reminders: Reminders are temporary, possibly even for the duration of the current encounter. Any user can add and remove reminders as needed.

Both types of reminders are considered part of the legal medical record will be saved in the database, even after they are removed or dismissed and are no longer viewable from the encounterPRO interface.


CAUTION: Remember that patient confidentiality ultimately dictates what should and should not be included within the alert. The Chart Alert screen will display whenever any user accesses the patient's chart.

Details:

Steps
Follow the steps below:

  1. Access the Chart Alert screen by doing one of the following:
    • From the Patient Demographics screen, select the Alert button located at the bottom of the screen.
    • From within the Patient Chart, select the Other Items button, then select the Alert button.
    • From the Waiting Room, select the patient name bar, then select the Alert button.
    The Chart Alerts screen appears.
  2. If you want to create an alert (more permanent), select the New Alert button.
    If you want to create a reminder (temporary), select the New Reminder button.
  3. Enter the text of your alert, then select OK.
