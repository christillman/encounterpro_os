How to: Document a Past Encounter.

Purpose
There may be times when you are unable to chart an encounter while the patient is in the office. Additionally, you may decide to enter information from previous visits in encounterPRO in order to have a complete record.

Before You Begin
Since this visit occurred in the past, you will need to manually check-in the patient to create the past encounter.

Details:

Steps
Follow the steps below:

  1. Select the Charts tab.
  2. Search for Patient by name, (last name, first name) SS#, etc.. A list of matching patients appears.
  3. Select the Patient name bar to view the Patient Demographics screen. Confirm that this is the correct patient (birth date, billing number, etc.)
  4. Select the New Encounter button on the bottom of the screen. The Create New Encounter screen appears.
  5. Select the date of the past encounter.
  6. Select the Encounter Type bar, then select the appropriate encounter type (nurse visit, well visit, office visit, etc.)
  7. Select Finished. Since you entered a past date for this encounter, the system prompts you to select how you want to enter the information.
  8. If you select Chart Past Encounter, the system will go straight to the Patient Chart to allow you to document the visit.
    NOTE: When charting a past encounter in this method, each assessment, treatment, lab or test must be ordered individually. No workplan will be launched to step the user through the screens.
  9. If you select Check-in the Patient, the patient goes to the Waiting Room and you will follow the workplan for the encounter type chosen. The workplan for the encounter type chosen in step 6 will be followed as if the patient were physically in the office.
    1. Select the Holding/Chart Room tab.
    2. Select one of the holding rooms, then select Get Patient.
    3. Choose your patient from the Waiting Room list.
    4. Select Finished.
    The first screen in the workplan appears.
  10. Document all the histories, diagnosis, and or treatments as applicable for the patient encounter.
  11. After all information has been entered, select Exit.
  12. The billing screen appears. Select the I'm Finished button.
