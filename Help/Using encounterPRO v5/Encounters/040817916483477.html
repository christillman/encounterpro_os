How to: Document a Nurse Only Visit

Purpose
Nurse visits are those encounters during which the patient is seen by the clinical staff only and not the practitioner. Common nurse visits are for:
  • Blood pressure and weight checks
  • Allergy shots
  • Immunization only visits
  • Lab work
  • Nurse only follow-up visit
Before You Begin
An encounter MUST have a valid assessment in order to be billable. The clinical staff person needs to know which existing assessment to use for today's encounter or know which new assessment to select.
Details:

Steps
NOTE: Refer to How to: Manually Check in a Patient, if needed. The following instructions assume your patient will be in the Waiting Room.

To Document a Blood Pressure Check Visit:

  1. Select the exam room or nurse station where the patient will be seen.
  2. Select the Get Patient button at the bottom of the screen.
  3. Select the name of the patient, and then select Finished. The Vitals screen appears.
  4. If this visit is a blood pressure check, enter that information. Enter any additional vitals you determine necessary (such as temperature, weight, etc.).
  5. Select Done to exit the Vitals screen. The next step in the workplan appears. Often this is the Chief Complaint screen.
  6. Enter the reason for today's visit or select from the pick list. Select I'm Finished. The Patient Chart screen appears.
  7. Document any additional information for this visit.
  8. Select Exit. The billing screen appears. Check the billable items listed. If this is a visit you will charge for, be sure that the nurse visit code, 99211, appears on the list.
  9. Select I'm Finished to close the encounter.

To Document an Allergy Shot Visit:

  1. Select the exam room or nurse station where the patient will be seen.
  2. Select the Get Patient button at the bottom of the screen.
  3. Select the name of the patient, and then select Finished. The Vitals screen appears.
  4. Enter the vitals for today's visit, according to your practice's requirements. 
  5. Select the Chief Complaint bar. The Chief Complaint screen apears.
  6. Enter the reason for today's visit, or select from the pick list. (For example: allergy shot.) Select I'm Finished. The Vitals screen reappears.
  7. Select Done to exit the Vitals screen. The Patient Chart appears.
  8. Important: Review the patient's desensitization schedule
  9. Since this is an allergy shot visit, you need to associate today's treatment with the existing allergy assessment. Generally this will be something like Allergic rhinitis or Allergen Desensitization. Locate the existing assessment in the patient's chart.
  10. Select the icon to the left of the assessment description. The treatment list for the assessment appears.
  11. Select the appropriate allergen immunotherapy treatment (based on the total number of injections you will be administering). Select your appropriate allergy shot from the list. If it is not listed, use the different search options to locate the desired allergy shot.
  12. Select Finished. The Allergen Immunotherapy screen appears.
  13. Select the location for the injection. If desired, add a procedure note.
  14. Select Finished. The Patient Chart reappears.
  15. Select Exit. The Billing Review screen appears. Check the billable items listed. If this is a visit you will charge for, be sure that the nurse visit code, 99211, appears on the list.
  16. Select I'm Finished to close the encounter.

To Document an Immunization Only Visit:
The steps for an immunization only visit are a bit more complicated because of the billing requirements. You need to have a vaccine/disease assessment for each immunization, plus you need to select the correct vaccine administration codes. The instructions that follow assume that the patient is in the Waiting Room.

  1. Select the exam room or nurse station where the patient will be seen.
  2. Select the Get Patient button at the bottom of the screen.
  3. Select the name of the patient, and then select Finished. The Vitals screen appears.
  4. Enter the pertinent vitals for today's visit, then select Done. The Chief Complaint screen appears.
  5. In the Chief complaint text area document the reason for today's visit. (For example: immunization only.) The Patient Chart screen appears.
  6. If this is an immunization only visit, select the New Other DX button. A menu of assessment types appears.
  7. Select the Vaccine DX button. The list of vaccine/disease assessments appears.
  8. Select the disease(s) for which the patient is being immunized (for example: Hepatitis). You need to select an assessment for each immunization you plan to give today.
  9. When you have selected all the assessments, select OK. The treatment list for the first disease appears.
  10. Select the vaccines and administration codes that apply for this patient. (You need to select Vaccine Admin First and the appropriate Vaccine Admin Add'l depending on the number of vaccines ordered.) (See Screen: How to: Order Vaccine Administration Codes.) for an explanation of how to choose the appropriate codes.
  11. Select Finished. The next treatment list for the next assessment appears. Repeat step 10 for each assessment. The Patient Chart reappears. Service bars for administering the vaccines appear in the Office View.
  12. Select Exit. The billing screen appears.
  13. Select I'll Be Back. This closes the Patient Chart, but leaves the encounter open. The Office View appears.
  14. When ready to document the vaccines, select the Administer Vaccine service bar from the Office View. The Immunization screen for the first vaccine appears.
  15. Type in all the necessary information, then select Immunization Given. Repeat step 14 for each vaccine ordered.
  16. Select the Vaccine Signature service bar to have the parent or patient indicate their consent. If no signature device is installed on that workstation, have the patient or parent select Accept without Signature. If a signature device is installed, select Sign to display the signature area. Have the patient or parent use the mouse or touchpen to sign in the area provided.
  17. Select OK when the patient or parent is finished signing. The Office View reappears.
  18. Select the Examination service bar. The Patient Chart reappears.
  19. Add any additional information for this encounter, then select Exit. The Billing screen appears.
  20. Review the billing for today's encounter. You should have:
    • Nurse visit code: 99211
    • ICD-9 code for each vaccine assessment (v-code)
    • CPT code for each vaccine given
    • CPT code for first vaccine administration (by type)
    • CPT code for the additional vaccine administrations (by type). The number additional plus the 1st should equal the total number of vaccines administered today.

    NOTE: It is a common problem to have too many or too few vaccine administration codes. See How to: Modify the Billing for an Encounter if you need to make corrections.
  21. When you are satisfied with the billing, select I'm Finished. The patient goes to the Checkout Room.

To Document a Lab Work Visit:
The steps for a lab work visit are slightly different. You can tie the lab work to a previous diagnosis, if needed. The instructions that follow assume that the patient is in the Waiting Room.

  1. Select the exam room or nurse station where the patient will be seen.
  2. Select the Get Patient button at the bottom of the screen.
  3. Select the name of the patient, and then select Finished. The Vitals screen appears.
  4. Enter the vitals required for today's visit, if any. Select Done. The Chief Complaint screen appears.
  5. In the Chief Complaint text area document the type of nurse visit (for example: lab work). Select Finished. The Patient Chart appears.
  6. If the patient is coming back for lab work because of a previous assessment made by the provider, then follow the steps below:
    1. Select the open DX for which the patient is getting the lab work done.
    2. Select the Order Treatment button at the bottom of the screen. A list of common treatments for the assessment appears.
    3. Select the lab(s) from the treatment list, or select the New Treatment button, then select Labs.
    4. Select the appropriate Lab(s) from the list. If it is not on the shortlist, use the search options to locate it.
    5. Select the lab(s) for today's visit, then select OK.
    6. Select Finished.
    7. Select Finished from the treatment dashboard screen to return to the Patient Chart screen.
  7. If there is no previous assessment in the chart for today's lab work, you must order the assessment for which the patient is getting the lab work.
    1. Select the New Sick or New Other DX button as appropriate. If you select New Other DX, select the appropriate DX type (Well, Vaccine, etc.).
    2. Select the Assessment(s) from the list or use the search option to locate the appropriate assessment. (If you do not know the assessment, select Lab Visit as the assessment.)
    3. Select OK. A list of common treatments for the assessment appears.
    4. Select the lab(s)from the treatment list, or select the New Treatment button, then select Labs.
    5. Select the appropriate Lab(s) from the list. If it is not on the shortlist, use the search options to locate it.
    6. Select the Lab(s) for today's visit, then select OK.
    7. Select Finished.
    8. Select Finished from the treatment dashboard screen to return to the Patient Chart screen.
  8. Select Exit. The billing screen appears.
  9. Select I'll Be Back. This keeps the encounter open.
  10. When ready to document the lab collection and results (for in house lab only), select the Collect or Refer service bar.
    1. Select the Collect or Refer service bar.
    2. If you collected the specimen, select Collected.
    3. If you are sending the specimen our for processing, select the Specimen Sent to bar, then select the appropriate lab.
    4. Select Finished.
    5. If the lab is processed in house, the Enter Result service bar appears.
    6. Select the Enter Results service bar. The results entry screen for that lab appears.
    7. Enter all the necessary information, then select Finished.
    8. If any send out labs were ordered, they will appear on the Outstanding Labs/Tests list once the specimen has been collected.

To Document a Nurse Only Follow up
If the provider ordered treatments (injections, labs, or tests) during a previous visit, the clinical staff can complete these tasks during a nurse only visit. Any treatments performed during the nursing visit need to be attached to the original assessment ordered by the provider. The instructions below assume that patient is in the Waiting Room.

  1. Select the exam room or nurse station where the patient will be seen.
  2. Select the Get Patient button at the bottom of the screen.
  3. Select the name of the patient, and then select Finished. If the treatment was ordered as part of a follow-up, a service bar appears immediately prompting you to perform that treatment.
  4. Select Dispatch Workplan. Select Finished. The Vitals screen appears.
  5. Enter any vital signs pertinent to today's visit. Select Done. The Chief Complaint screen appears.
  6. In the Chief Complaint text area document the reason for today's visit (for example: injection).
  7. Select Finished. The Patient Chart appears with all open Assessments and Treatments displayed.
  8. Select the description of the assessment that pertains to today's encounter. The assessment dashboard screen appears. The provider probably ordered the follow-up treatment as part of the treatment plan. Review the treatment plan to make sure that you understand what treatment is required today.
  9. Select the Order Treatment button at the bottom of the screen. The treatment list for the assessment appears.
  10. Select the treatment from the list, or select New Treatment and search for the appropriate treatment.
  11. When ready to document the treatment, select the service bar. Enter all the necessary information, then select Finished. When the last service bar is completed, the patient will go to checkout.
  12. Select Exit. The Billing Review screen appears. Make sure all billable items show in the list as well as the nurse visit code 99211.
  13. Select I'm Finished. The patient goes to the Checkout Room.
