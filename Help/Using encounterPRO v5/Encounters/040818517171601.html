How to: Change the Encounter Type

Purpose
The Encounter Type determines which workplan is used for an encounter. When the patient is checked in, encounterPRO maps the appointment type from the practice management system to the appropriate encounter type. If the patient is checked in with an appointment type that does not match their reason for today's visit, the encounter type will need to be changed to the appropriate type.

Before You Begin
It is best to change the encounter type while the patient it still in the Waiting Room, changing the encounter type after the visit has begun will cancel the original workplan, delete all data entered to that point, and place the patient back in the Waiting Room.

NOTE: The encounter visit code is NOT based on the encounter type for the visit, but on the type of assessment selected: Sick or Well.

Details:

Steps
Follow the steps below:

  1. You can access the Edit Encounter screen from the menu displayed when you select the patient name bar from one of these locations:
    • The Waiting Room:  changing the encounter type here is preferable
    • Any treatment room:  changing the encounter type after the encounter has begun will delete all data entered to that point and return the patient to the Waiting Room. Not recommended.
  2. Select the Edit Encounter button. The Edit Encounter Information screen appears.
  3. Select the bar for the Encounter Type. The Select Encounter Type screen appears.
  4. Select the desired encounter type from the short list or use the search options to locate the desired encounter type.
  5. Notice that the Encounter Description did not automatically change. Select the gray bar for the Encounter Description and type in the appropriate description for this encounter.
  6. When you are finished changing the Encounter Description, select OK. The Edit Encounter screen reappears.
  7. Select Finished. If the encounter had already begun (you accessed the Edit Encounter screen from a treatment room instead of the Waiting Room), a message appears to alert you that the original encounter workplan will be cancelled. Confirm that you want to change the encounter type and cancel the original workplan by selecting Yes. The patient appears in the Waiting Room. (Notice that the elapsed time is also reset on the patient name bar.)
