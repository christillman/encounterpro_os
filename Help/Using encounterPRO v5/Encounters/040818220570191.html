How to: Correct Encounter Data

Purpose
There will be times when data has been incorrectly entered into an encounter. For example, the wrong weight or a notation in the physical that should be left ear but is showing under right ear. encounterPRO allows users (with a certain permission level) to correct this information. All changes to data are logged in the system and can be reviewed in the User Audit Log or Encounter Audit Log. If the change is made to a treatment or assessment, users can also view a log from the appropriate dashboard screen by selecting the Properties tab.

Before You Begin
If the change needs to be made to the PFSH, that requires additional steps. See below.

Details:

Steps
Follow the steps below:
To make a change to the Vitals:

  1. From the Patient Chart, select the small icon to the left of the Vitals (in blue text). The Vitals screen appears and displays the data recorded in the most recent set of vitals.
  2. If the data that needs to be changed is from a different set of vitals, select the bar for the desired set from the list of Timed Sets on the left of the screen.
  3. Change the data as needed, then select Done. The Patient Chart reappears.

To make a change to the PFSH:

  1. From the Patient Chart, select the small icon to the left of the PFSH (in blue text). The PFSH history display screen appears and displays the entire history documented for this patient. The PFSH is a cumulative history and displays all data from all encounters.
  2. Select in the large gray text area. A menu of options appears.
  3. Select Review w/Changes. The PFSH drilldown screen appears.
  4. Change the data as needed, then select Finished. The PFSH history display screen reappears.
  5. Select OK to return to the Patient Chart.

To make a change to the physical exam:

  1. From the Patient Chart, select the small icon to the left of the Physical (in blue text). The Physical Exam screen appears and displays the data documented by the provider for this encounter.
  2. Change the data as needed, then select Finished. The Patient Chart reappears.
