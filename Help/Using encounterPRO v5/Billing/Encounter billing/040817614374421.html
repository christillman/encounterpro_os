How to: Modify the Billing for an Encounter.

Purpose

The provider is responsible for checking that the billing information recorded for a visit is accurate. EncounterPRO will collect this information and attempt to calculate a visit level based on the information gathered during the encounter. The E & M coding level and billing information is ultimately up to the provider, using his or her knowledge of the medical complexity and risks for each encounter. For more information about  Screen: How encounterPRO Calculates E & M Coding.

Details:

Steps
Follow the steps below:
To Change the Visit Level:
If you do not agree with the encounter visit level calculated by encounterPRO, you can easily change it. You should first review the E & M Coding to determine why encounterPRO suggested that level. See Screen: How to: Review the Calculated Encounter Visit Level.

IMPORTANT: EncounterPRO calculates a suggested visit level based on the amount of information that was documented during the encounter. The system can only measure the work done, it cannot evaluate the necessity of the work. Ultimately it is the encounter owner or provider, not encounterPRO who is responsible for the E & M levels chosen. The system merely offers a suggested level based on the observations recorded, and gives the provider the option to view the reasons that this level is suggested, but it is he or she who chooses the encounter level that is approved and sent to billing

  1. From the Patient Chart, select Exit. The Billing Review screen appears with the visit level calculated and the visit code listed in the gray box on the right.
  2. If you want to change the visit level, select the number for Current Visit Level. The Encounter Coding Levels screen appears. You can review the levels reached for each area of the encounter, history, examination, and decision making.
  3. You can change the calculated visit level by doing one of the following:
    • Select the number that corresponds to the desired visit level for the User Selected Visit Level
    • Select a different bar for one of the areas (History, Examination, Decision Making) from the one marked by the asterisk (*). The Calculated Visit Level may change based on that difference. For example, if the system indicates the you achieved a Problem Focused Examination, select Expanded Problem Focused. Then check the Calculated Visit Level to see if it reflects the change.
  4. Select Finished to return to the Billing Review screen.
  5. Select I'm Finished to send the encounter to Checkout.

NOTE: If you return to the Patient Chart and make any changes, encounterPRO will recalculate the visit code and you will need to repeat steps 1-6.

To Associate a Treatment with the Appropriate Assessment:
If a billable treatment is not associated with the appropriate assessment, there can be a delay or rejection from the insurance carrier. If a treatment is associated with the wrong assessment on the billing screen, you can correct that in the chart or on the billing screen. See Screen: How to: Associate Treatments With a Specific Assessment for instructions for doing this from the chart.

  1. From the Patient Chart, select Exit. The Billing Review screen appears with the visit code and billable items listed in the gray box on the right.
  2. If you want to change which assessment is associated with a treatment, select anywhere in the gray area to the right of the billing screen where the charges are listed. A second billing screen appears.
  3. Select the gray bar for the treatment for which you wish to change the associated assessment. The Encounter Chart Edit screen appears. A list of associated assessments appears near the bottom of the screen. The assessment associated with the treatment appears highlighted (darkened).
  4. De-select the assessment(s) that should not be associated with the treatment. Select the assessment (to darken it) that should be associated with the treatment.
  5. Select Finished. The second billing screen reappears. The treatment now appears with the selected assessment(s) below it. A notation that the treatment billing was edited displays on the gray bar.
  6. Select the Finished button to return to the Billing Review screen.
  7. Select the I'm Finished button to send the patient to checkout and send the billing information to your billing system.

To Change a Billable Code:
If the CPT code displayed on the Billing Review screen is incorrect, you can either cancel the procedure and add the correct one from the chart, or change the procedure code on the billing screen. For example, if the wrong vaccine administration code was selected, you can either cancel it and add the correct code from the chart screen, or change it on the Encounter Charge Edit screen.

  1. From the Patient Chart, select Exit. The Billing Review screen appears with the visit code and billable items listed in the gray box on the right.
  2. If you want to change a CPT code for a specific procedure, select anywhere in the gray area to the right of the billing screen where the charges are listed. A second billing screen appears.
  3. Select the gray bar for the procedure that you want to change. The Encounter Chart Edit screen appears.
  4. Select the Change Procedure button. The Select Procedures screen appears.
  5. Select the correct procedure from the list or use the search techniques to locate the desired procedure. The Encounter Charge Edit screen reappears with the new procedure description and CPT code.
  6. Select Finished. The second billing screen reappears. The new procedure appears with a notation that the treatment billing was edited.
  7. Select the Finished button to return to the Billing Review screen.
  8. Select the I'm Finished button to send the patient to checkout and send the billing information to your billing system.

To Add a Modifier Code:
Many times you will need to add a modifier code to the encounter billing. For example, if the patient came in today for a well visit, but also had a sick assessment you can often add a -25 modifier to the encounter billing.

  1. From the Patient Chart, select Exit. The Billing Review screen appears with the visit code and billable items listed in the gray box on the right.
  2. If you want to add a modifier to the encounter billing, select anywhere in the gray area to the right of the billing screen where the charges are listed. A second billing screen appears.
  3. Select the New Secondary Visit Charge button. A list of secondary visit charges appears.
  4. Select the secondary charge or modifier code from the list.
  5. The new visit charge appears and is associated with all of the assessments from today's encounter.
  6. If you need to change the assessments associated with this new visit charge, select the gray bar for the new charge. The Encounter Charge Edit screen appears. All the assessments currently associated with this charge are highlighted (darkened).
  7. De-select any assessments that do not belong with the current charge. Select (darken) any assessments that should be associated with the current charge.
  8. Select Finished. The second billing screen reappears. The new visit charge now appears with the selected assessment(s) below it. A notation that the billing was edited displays on the gray bar.
  9. Select the Finished button to return to the Billing Review screen.
  10. Select the I'm Finished button to send the patient to checkout and send the billing information to your billing system.

To Align Treatments with an Assessment:
You might notice that a normally billable procedure does not have a green $ on the Patient Chart, or when looking over the Billing Review screen, notice that a treatment is not billable or is aligned with the wrong assessment. Follow the steps below:

  1. From the Billing Review screen, select the gray box with the visit charges. The Billing screen appears.
  2. Each billable procedure should have a large gray bar. Underneath the gray bar is the assessment with which the procedure is associated.
  3. Select the gray bar for the procedure you need to associate with a different assessment. The Encounter Charge Edit screen appears.
  4. Make sure that the assessment that goes with the selected procedure is the only one highlighted (darkened) in the Associated Assessments list.
  5. Select Finished.


To Correct the Vaccine Admin Codes:
The vaccine admin codes are based on the number and type of immunizations given during the encounter. Each vaccine delivery mechanism has it's own associated admin codes. For example, Intramuscular, Intranasal, Oral. In addition, each type will have a code for the first administration and additional codes for additional administrations of that same type. For example, if a patient received two intramuscular injections, plus an oral vaccine the correct admin codes would be:

  • IM first
  • IM 1 add
  • Oral first
  1. From the Billing Review screen, select the gray box with the visit charges. The Billing screen appears.
  2. Each billable procedure should have a large gray bar. Underneath the gray bar is the assessment with which the procedure is associated.
  3. Select the bar for the incorrect administration code. The Encounter Charge Edit screen appears.
  4. Select the Change Procedure button near the top of the screen. The Select Procedure screen appears.
  5. Make sure that the Procedure Type is Vaccine Admin.
  6. Select the correct vaccine administration procedure. The Encounter Charge Edit screen reappears with the new procedure code.
  7. Be sure that the Associated Assessments list shows the appropriate assessment for this vaccine administration code.
  8. Select Finished.
