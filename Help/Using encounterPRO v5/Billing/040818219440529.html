Purpose
Practice managers and billing supervisors want to be able to keep track of daily billing. There are several methods that you can use to review the day's billing:
  • The Checkout Room
  • General Reports:
    • Billing for Select Date by Doctor
    • Today's Bills
    • Select Date Bills
Details:

Steps
Follow the steps below:
To Review Billing Using the Checkout Room:

  1. From the Office View, select the Checkout tab. The Checkout room for today appears. The icon to the right of the encounter bar gives an indication of the billing status of the encounter:
    • Green bill - Billing accepted by PMS
    • Green  ü- Billing posted to PMS
    • Red   ü - Billing not posted
    • Red $ with X - Error posting
    • Red $ with R - Retry posting in progress
    • Red $ - Not a billable encounter
    • Envelope - Billable message
  2. If you want to locate encounters that were not rejected, but have not posted yet (red   ü), select the box for Billing Status and select Error Posting from the list. The display is filtered to show only those encounters that were unable to successfully post to the PMS system.
  3. If you want to locate encounters that have been rejected by the PMS system (red $ with X), select the box for Billing Status and select Error Posting from the list. The display is filtered to show only those encounters that were unable to successfully post to the PMS system.
  4. If an encounter has been rejected, select the billing icon (red $ with X) to view details on why this encounter was rejected. The Posting History screen appears. The reasons the encounter failed to post to the PMS system are listed in the lower portion of the screen. When finished reviewing the information select Finished to return to the Checkout Room
  5. Use the arrow buttons or the calendar to view the Checkout Room for a different day.

To Generate the Today¡¦s Billing Report:

  1. From the Office View, select the Utilities tab. The Utilities list appears.
  2. Select General Reports. The Select Reports screen appears.
  3. Select Today's Billing report from the short list. If the report does not appear on the short list, select the Description button and enter Bills, then select OK. Select Today's Bills from the list of matching reports.
  4. With the Today's Bills report highlighted (darkened), select OK. The reports menu appears.
  5. Select the output option you prefer:
    • Preview Report
    • Default Printer
    • Select a printer
    • Print from server

    The report is generated.

To Generate the Billing for Select Date by Doctor Report:

  1. From the Office View, select the Utilities tab. The Utilities list appears.
  2. Select General Reports. The Select Reports screen appears.
  3. Select Billing for Select Date by Doctor report from the short list. If the report does not appear on the short list, select the Description button and enter Billing, then select OK. Select Billing for Select Date by Doctor from the list of matching reports.
  4. With the Billing for Select Date by Doctor report highlighted (darkened), select OK. The reports parameters screen appears.
  5. Select the User bar, the Select User screen appears.
  6. Select the provider for whom you want to produce the report. The name appears in the User field.
  7. Select Next.
  8. Select the Discharge Date bar. A calendar appears to allow you to select the date for the report. This report uses the Encounter Discharge Date as the parameter.
  9. Select the date on the calendar, then select OK.
  10. Select Finish. The print menu appears.
  11. Select the output option you prefer:
    • Preview Report
    • Default Printer
    • Select a printer
    • Print from server
    The report is generated.

To Generate the Select Date's Billing Report:

  1. From the Office View, select the Utilities tab. The Utilities list appears.
  2. Select General Reports. The Select Reports screen appears.
  3. Select the Select Date's Billing report from the short list. If the report does not appear on the short list, select the Description button and enter Billing, then select OK. Select the Select Date's Billing from the list of matching reports.
  4. With the Select Date's Billing report highlighted (darkened), select OK. The reports parameters screen appears.
  5. Select the Encounter Date bar. A calendar appears to allow you to select the date for the report. This report uses the Encounter Start Date as the parameter.
  6. Select the date on the calendar, then select OK.
  7. Select Finish. The print menu appears.
  8. Select the output option you prefer:
    • Preview Report
    • Default Printer
    • Select a printer
    • Print from server
    The report is generated.
