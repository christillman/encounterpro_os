How to: Cancel a Prescription Medication

Purpose
At each visit, the clinical staff needs to review the patient's medications and make sure the list is accurate. Canceling a medication indicates that the medication was entered by mistake and should be removed from the patient's chart. If the patient was taking the medication but has discontinued it, you should close it. See Screen: How to: Close a Discontinued Medication for instructions

Details:

Steps
Follow the steps below:

  1. Access the Current Medications screen by doing one of the following:
    • From the Patient Chart, select the Meds box at the top of the screen
    • From the Enter Current Meds service bar
    • From the Meds box on the left side of the Vitals screen
    • As part of the workflow, the Current Medications screen appears automatically
    The open medications for this patient are listed. Medications that are not associated with a specific assessment are shown at the top of the list.
  2. Select the medication that is in error. A Treatment dashboard appears.
  3. Select the Other Options button, then select the Cancel RX button to cancel this medication. You must enter a reason for canceling the treatment. For example: Error, patient was never taking.
  4. Select Cancel Treatment, then Finished to return to the Current Medications screen.
  5. Repeat steps 2 - 4 to cancel any other medications that are in error.
  6. Select Finished again to proceed in the workflow.
