How to: Set the Preference to Skip the Who Ordered This Medication Screen

Purpose

When documenting current medications, a screen will automatically display that allows the clinical staffer to select a provider from the patient's care team as the ordering provider for that medication. This screen can be suppressed by selecting a Workflow Preference, if the practice does not wish to document the ordering provider.

NOTE: This applies only to medications added through the Current Medications screen and not to medications ordered by an encounterPRO user as part of a treatment plan.



Details: Steps
Follow these steps:
  1. From the Office View, select the Configuration tab. The Configuration List appears.
  2. Select Preferences. The Preferences screen appears.
  3. Select Workplan Preferences on the left and a list of available preferences is displayed.
  4. Select Global as the preference level at the bottom of the screen.
  5. Select Past Med Prompt Who Ordered from the available list. A preference parameters screen appears.
  6. Select 'No' to suppress the 'Who Ordered This Medication' screen. The list of preferences reappears.
  7. Select Finished. The Preferences screen reappears.
NOTE:  Each encounterPRO users will need to re-start the application for this preference to take effect on their workstation.
