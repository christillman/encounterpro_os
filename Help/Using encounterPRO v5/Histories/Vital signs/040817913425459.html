Purpose
Recording Vital signs with encounterPRO is quick and easy. The Vital Signs screen is designed to display the appropriate fields depending on the encounter type, patient age, and the practice specialty. For example, the Vitals screen for a Pediatric office visit will differ from the Vitals screen for an Internal Medicine office visit.

Regardless of specialty, certain elements of the Vitals screen are consistent. The Vitals screen allows the documentation of the following information:
  • Weight
  • Height
  • Temperature
  • Blood pressure
  • Respiratory rate

Other options are also available such as Orthostatic or Postural blood pressures, Pulse Oximetry, BMI, Fundal Height, Respiratory rate, etc; depending on the practice specialty and preferences.

Before You Begin
If the Vitals screen did not automatically appear when you got the patient from the waiting room, you can select the room or station where the patient is located, then select the Vitals service bar. On the Vitals screen (and several other encounterPRO screens) you will see a    button for several of the fields. This button allows you to display a number pad for easy touchscreen entry.

Details:

Steps
To Enter Height,  Weight, Temperature, and BP:
From the Vitals screen, you can enter the height and weight in one of three ways:

  1.  From the Vitals screen, enter the patient's height using of the following methods:
    • Select the Height field and type the numbers
    • Select the  button next to the Height field, then select the numbers from the number pad
    • Select the   button next to the Height field, then select from the list of recently used numbers on the right.
  2. Enter the patient's weight using one of the following methods:
    • Select the Weight field and type the numbers
    • Select the    button next to the Weight field, then select the numbers from the number pad
    • Select the   button next to the Weight field, then select from the list of recently used numbers on the right.
  3. If you want to enter weight in pounds and ounces, simply:
    1. Select the Weight field, type the number of pounds.
    2. Press the Spacebar to insert one (1) space.
    3. Enter the number of ounces.

      Or
    4. Select the    button for the Weight field, select the number of pounds.
    5. Select Space to insert one (1) space.
    6. Select the number of ounces. When you press tab, or touch another field, encounterPRO converts the information to pounds and ounces.
  4. Enter the patient's temperature in degrees of Fahrenheit or Celsius (see using Screen: English/Metric) in one of three ways:
    • Type the numbers in the Temperature field
    • Select the    button next to the Temperature field, then select the numbers from the number pad
    • Select the    button next to the Temperature field, then select from the list of recently used numbers on the right
    • After you enter the temperature, select the method used to obtain the temperature: Ear, Oral, Rectal, Axillary, Temporal
  5. From the Vitals screen, you can enter the blood pressure in three main steps.
    1. Enter the Systolic pressure first using one of the following methods:
      • Typing the numbers in the left-most BP field
      • Selecting the    button next to the BP field, then selecting the numbers from the number pad and selecting OK
      • Selecting the    button next to the BP field, then selecting from the last used numbers on the list and selecting OK. This list holds Systolic pressures entered by the user.

    2. Immediately following the entry of the Systolic pressure and selecting OK a number pad will appear to enter the Diastolic pressure. Enter the Diastolic values using one of the following methods:
      • Selecting from the number pad
      • Selecting a number from the last used numbers list. This list holds Diastolic pressures entered by the user.

    3. Select the location where the pressure was taken from the list.
  6. Enter the remaining vital signs required by your practice. At this point, you have several other options:
    • Adding a new set of vitals
    • Viewing the patient's chart
    • Ordering labs and tests
    • Ordering treatments
    • Entering Current Medications
    • Documenting Allergies
  7. If you do not need to perform any of the above functions, you have two other options:
    • Done -- the Vitals screen closes and the Vitals service is complete. The next service in the workplan for the nurse appears on the Office view, or the screens will automatically come up in sequence.
    • I'll Be Back -- the Vitals screen closes but a service bar is created for quick access back into the screen for completion. This service can also be accessed through the user's Tasks list. Choosing I'll Be Back halts the workplan and it will not move on to the next step until the Vitals service is completed. If a user needs to access Vitals after this service is closed, he/she can do so by selecting the Vitals text on the SOAP screen (View Chart in the name bar option menu) and choosing Enter/Edit from the Vitals dashboard.
