How to: Document Patient-reported Allergies

Purpose
The Allergies screen can be configured into the nurse's workflow to ensure that allergies are updated for each visit. It can also be accessed from the Vitals screen and the Patient Chart. Allergies noted in previous visits appear automatically on the Allergies screen. You can add new allergies as needed.

Before You Begin
If a patient has had allergies noted in his or her chart in previous encounters, the Allergies button on the Vitals and the Patient Chart screens will have red lettering (Allergies). If no allergies have been noted, the button lettering is black and reads No Allergies.

Details:

Steps
Follow the steps below:

  1. From the Vitals screen, select the Allergies/No Allergies button. The Allergies Assessments screen appears listing the currently documented allergies for this patient.
  2. If the patient reports no known allergies, select the One Touch NKA button to document that you asked during today's visit. If the patient reports a new allergy, select the New Allergy button. The Select Allergy Assessment screen appears.
  3. Select the appropriate Allergy from the Short list, or locate it using the search options available. You can select more than one item.
  4. When you are finished selecting items, select OK. You will need to select a severity and enter a reaction for each allergy you select in step 3. The Reaction title list appears for the first allergy you selected.
  5. Select a reaction title (severity) for the allergy or select Cancel if the reaction is unknown.
  6. Select reaction symptom(s) from the pick list. If the needed symptom is not listed, type it in the white box.
  7. Select the OK button. The reaction severity and symptoms will be displayed on the Allergy Assessments screen. If you selected more than one allergy in step 3, you will need to repeat steps 4-6 for each allergy.
  8. Select Finished to return to the Vitals screen. The text on the Allergies button will appear in red.
