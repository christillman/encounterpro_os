Details: Description
This type of screen is used for several types of histories, including:
  • PFSH
  • Development
Review the previously entered information in the gray text area. Select a bar on the left to view the information for that specific category or observation.
You have several options at this point:
  • If you want to note that you have reviewed the information, select the gray area then select the Reviewed with No Changes button.
  • If you want to add or make changes to the information already entered, select the gray area then select the Reviewed with Changes button. The drilldown screen appears to allow you to alter or add information.
  • If there is no existing information, select the gray area then select New Results. The drilldown screen appears to allow you to enter new information.
How to Access This Screen
Access this screen by:
  • Selecting PFSH from the Add History menu on the Patient Chart
  • Selecting Development from the Growth Chart tab on the Patient Chart
  • It can be automatically displayed as a step in the encounter workplan
