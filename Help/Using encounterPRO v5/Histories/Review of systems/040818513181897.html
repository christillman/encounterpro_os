Purpose
The Review of Systems (ROS) screens come in two formats: Yes/No and Drilldown. The format you see will depend on your practice's initial preferences, but can be changed for individual providers.

Before You Begin
The ROS is counted towards the E & M coding level for patient history. If there is no ROS documented, that can limit the encounter level that can be achieved.
Details:

Steps
Follow the steps below:

  1. From the Patient Chart, select the Pick H_and_P button on the right. The menu of histories appears.
  2. Select the ROS button. The preferred ROS screen appears.
  3. If your preferred ROS is a Yes/No screen, then do any combination of the following:
    • Use the Set All button below the No or Yes column to select all buttons in that column.
    • Select Yes and No individually for each body system.
    • Select the Comment bar to enter a comment for a specific body system.
  4. If your preferred ROS is a drilldown screen, then do any combination of the following:
    • Select the right side of the bar labeled "Drill Down." The next level of observations appears. Either select again, select a result, or enter a result. Use the navigation buttons on the bottom right to go Up one level, to the Next observation or to the Top level of the ROS. 
    • Select the left side of the bar where the body system description is located. The defaults set for that system are populated.
    • Select one of the Standard Exams on the right side of the screen. The results for that standard exam are populated.
    • Select the Previous Exam button at the top right of the screen (if one appears). A list of previous examinations appears. Select the desired examination. The results from that instance of the exam are populated. (You may want to select Review Physicals at the bottom of the screen to review the physical examination from previous encounters to be sure you are selecting the correct one.)
  5. Select Finished to return to the Patient Chart.
