Troubleshooting: encounterPRO Locks Up or Freezes

Triage Protocol: Troubleshooting: encounterPRO Locks Up or Freezes
Overview: There can be several different reasons that the encounterPRO screens will appear to freeze or "lock-up." Please try the solutions below.
Symptoms:
Resolution:

  1. The active window is not displayed on top.
    Sometimes when several windows are open at the same time, the window that is active or "has focus" is hidden beneath another window. Use Alt+Tab on the keyboard to page through all the current windows until you find the one that is active and is generally awaiting a user input such as selecting OK or Cancel.

  2. Screen saver settings on the server.
    Access the server and go to the Control Panel. Select Power Save and make sure the power save setting is OFF.
