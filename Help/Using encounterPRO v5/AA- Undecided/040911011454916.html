Overview:

Pre-Implementation Configuration - Basic Configuration

This course is intended for the implementation team and all users who will be involved in configuring encounterPRO prior to implementation.  The implementation team from the practice will collaborate with the PSI on how to best customize certain areas of the encounterPRO system.
This is the first in a series of concentrated pre-implementation configuration courses which include:

-- Basic Configuration (2 hours)
-- Assessments and Treatments (1 hour)
-- Medications and Vaccines (2 hours)
-- Default Treatment Modes (1 hour)
-- Consultation on Customizing encounterPRO (min. 2 hours)

Audience: At least one user per office location
Prerequisites: 
-- Clinical Staff or Provider
-- epro Administrator

Agenda:

  • The Role of the Advanced User
  • Maintaining Health Maintenance Rules
  • Maintaining Consultants
  • Maintaining the Folders List
  • Customizing encounterPRO Using Preferences
  • Creating and Maintaining Standard Exams

Pre-Implementation Configuration - Assessments & Treatments

This course requires an enrolment key
This course covers how to add and maintain assessments and treatments in encounterPRO. At least one user at each practice location needs to complete this course.

Prerequisite:
-- Pre-Implementation Configuration - Basic Configuration

Agenda:

  • Configuring Assessments & Treatments
  • Adding and Maintaining Assessments
  • Adding and Maintaining Allergy Assessments
  • Adding and Maintaining Simple Treatments
  • Troubleshooting Billing for Treatments
  • Maintaining Treatment Lists by Assessment

Pre-Implementation Configuration - Medications and Vaccines

This course covers how to add and maintain prescription medications, in-office drugs, and vaccines in encounterPRO. At least one user at each practice location needs to complete this course.

Prerequisite:
-- Pre-Implementation Configuration - Basic Configuration

Agenda:

  • Configuring Medications and Vaccines
  • Maintaining Prescription Medications
  • Advanced Prescription Medication Configurations
  • Configuring In-office Medications
  • Configuring Vaccines

Pre-Implementation Configuration - Treatment Modes

This course requires an enrolment key
This course covers how to understand and use default treatment modes in encounterPRO. At least one user at each practice location needs to complete this course.

Prerequisite:
-- Pre-Implementation Configuration - Basic Configuration

Agenda:

  • Understanding and Using Treatment Modes
  • Default Treatment Modes
