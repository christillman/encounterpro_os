encounterPRO University Course Catalog

Solution Title: encounterPRO University Course Catalog
Overview:

In ePRO U individual courses are grouped into categories based on the practice and user's experience level with the encounterPRO EHR. There are several course categories at this time:

encounterPRO New Client Implementation

These courses are intended for new clients. The course materials in ePRO U are intended to accompany and support a webinar or on-site education session with the PSI.  We do not recommend that the users attempt to use the course materials in ePRO U on their own. The courses should be completed and all users must pass the Readiness Quiz for their role(s) before Go Live.

Select the course title to view details about each course:

  • Introduction to encounterPRO version 5
  • encounterPRO Basics
  • e-Records Manager
  • Clinical Staff
  • Providers
  • Billing and Coding for Providers
  • Practice Manager
  • Billing Specialist
  • Front Desk Administration
  • Pre-Implementation Configuration - Basic Configuration
  • Pre-Implementation Configuration - Medications and Vaccines
  • Pre-Implementation Configuration - Assessments and Treatments
  • Pre-Implementation Configuration - Treatment Modes

Migrating from version 2 to encounterPRO version 5

These courses are intended for version 2 clients migrating to version 5. The course materials in ePRO U are intended to accompany and support a webinar or on-site education session with the PSI.  We do not recommend that the users attempt to use the course materials in ePRO U on their own.  The courses should be completed and all users must pass the Readiness Quiz for their role(s) before Go Live.

  • Pre-Migration Configuration - Medications and Vaccines
  • Pre-Migration Configuration - Assessments and Treatments
  • Pre-Migration Configuration - Treatment Modes

Upgrading from version 4 to encounterPRO version 5

These courses are intended for version 4 clients upgrading to version 5. Because version 4 users will have a great deal of familiarity with the encounterPRO system, the majority of course materials in ePRO U are intended for the users to complete on their own. This is, in effect, a "Self-Service" option. (An instructor-led option can be obtained for an additional cost, similar to the migrating from 2 to 5 model.) The only courses that require a PSI are the What's New in Version 5 and the Pre-Upgrade Configuration course. The courses should be completed and all users must pass the Readiness Quiz for their role(s) before Go Live.

Select the course title to view details about each course:

  • New Features in Version 5
  • Changes Required for Version 5 - All Roles
  • New Features - Clinical and Provider
  • Configuration Changes
  • Configuring New Features

Advanced encounterPRO Topics (Coming Soon)

These courses will become the Super User curriculum, offering courses on Advanced Configuration, Creating Reports, etc. Users must have passed the Readiness Quizzes for all the Pre-Implementation Configuration courses and should be proficient ePRO users prior to enrolling in these courses.
