How to: Reprint or Re-send a Document Using Document Manager.

Purpose
Document Manager allows the user to select a recipient and route for each document. Some documents will be printed and some can be sent electronically. The available routes will depend upon the document Purpose, and the services that the practice has subscribed to, such as SureScripts-RxHub e-prescribing.

To view a list of available routes and to see why a specific route is not available for a specific document, select the Other button on the Document Manager screen, then select Show Routes. A screen appears detailing which routes are available for that document and if any routes are unavailable, why that route is unavailable.

Before You Begin
You will need to understand the basics of the Document Manager function. See Concept: Document Manager and Screen: Document Manager for detailed information.

Details:

Steps
When documents have been ordered:

  1. Select the Documents ordered icon on the Patient Chart or Office View or select the Document Management button on any menu. The Document Management screen appears.
  2. If the document status is Ordered, you have several options:
    1. Select View to preview the document
    2. Select Other, then Edit to access the dashboard for the document

    3. Note: Changes can be made to prescriptions even after they appear on  the Document Manager screen. However, the user making the changes must be RX-certified as that user will be considered the ordering provider for the purposes of the signature on the prescription and eligibility for sending via SureScripts.

  3. When you are satisfied that the document is ready to send/print confirm the recipient. If the recipient that defaulted is not the method you want to use for this document, select the recipient bar. A list of available recipients for the document appears.
  4. Select the desired recipient. A preferred route for this recipient appears.
  5. If more than one route is available for the recipient selected in step 4, select the Via Route bar. A list of available routes appears.
  6. Select the desired route for this document. A primary address for this route appears.
  7. If more than one address is available for the route you selected in step 6, select the Address bar.
  8. Select the desired address.
  9. Select the Send/Print button (the name of the button is determined by the recipient selected in step 4).
  10. Repeat steps 2 through 9 for all documents on the Document Management screen that have not yet been sent.
  11. When all documents have been printed, sent, or completed, select OK.
