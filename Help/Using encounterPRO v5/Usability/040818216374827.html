Purpose
You can use any one of several different search options to find an item that is not on the short list:
  • Description
  • (Code)
  • Trade name (drug)
  • Generic name (drug)
  • Category
  • Specialty/All
Before You Begin
You can set the preference to change the default search from Begins With to Contains. It is often easier to search using the Contains option.  Go to General Preferences and set the Default Search String Mode to the desired setting.

Items such as drugs, assessments, labs, and certain other treatments are designated as belonging to specific specialties in the database. This is in an effort to make searching more targeted. The Specialty/All button defaults to display the matching search results that are designated for your specialty. Select the button labeled Specialty to display all matches in the entire database.
Details:

Steps
Follow the steps below:

  1. Display the desired selection screen (Select Prescription Drug, Select Assessment, etc.). The short list for that item appears.
  2. Select one of the search options:
    • Description
    • (Code)
    • Trade name (drug)
    • Generic name (drug)
    • Category

    An entry keypad appears.
  3. Select the Begins With or Contains option for this search.
  4. Use the keyboard (either on the touchscreen or the regular keyboard) to enter a partial item name. The system uses an exact matching algorithm, so a partial name is going to match more items than a longer name. If you selected Category in step 2, a list of available categories appears. Select the desired category.
  5. Select OK. A list of matching items appears. Note that the Specialty button now appears above the Category option.
  6. If the item you need does not appear in the list of matching items, select the Specialty button to toggle to All. This will show all the matching items in the entire database.
  7. If the item is still not found in the list of matches, repeat the search using either a shorter partial name or select a different search option.
