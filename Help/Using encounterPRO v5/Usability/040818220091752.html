How to: Create a Personal Treatment List

Purpose
Each user can create his or her own treatment list for any, or all, assessments. Remember that Personal lists are only viewable and usable by the user who creates them.

Before You Begin
Once you have added even a single treatment to your personal treatment list, that list is the default for that assessment. You can toggle between the Personal and Common lists as needed.

Details:

Steps
Follow the steps below:

  1. After you have selected an assessment, a treatment list appears.
  2. Select the Personal tab at the upper left of the screen.
  3. Select the Copy From tab, the options are:
    • Common List.
    • Default List.
    • Other Assessment Common List.
    • Other Personal List. 

    Once you have made your option selection your personal treatment list is created.
  4. If a treatment you commonly order is not already on the list, select the New Treatment button. A menu of treatment types appears.
  5. Select the treatment type for the treatment you want to add to the treatment list. The Select Treatment screen appears.
  6. Select the treatment(s) from the short list or use the search options to locate the desired treatments. You can select multiple treatments and each will appear in the Selected Items list.
  7. Select OK. The treatment list reappears with the new treatment(s) at the bottom of the list.
  8. The text of the new treatment appears in light (not bold) print, indicating that it has been ordered for this patient, but has not been permanently added to the treatment list for this assessment.
  9. To add the item to the treatment list, select the icon to the left of the treatment. The treatment item screen appears.
    NOTE: If the treatment you want to add is a referral or follow-up, when you select the icon on the left, a screen similar to the above appears. There is no Update List option, however. If you simply select OK, the follow-up or referral will be added to the treatment list.
  10. Select the Yes button under Update List then select OK. The treatment list reappears. The description is now bold, indicating that it has been added to the treatment list for this assessment.
  11. Repeat steps 3 through 9 to add additional treatment items to the treatment list for this assessment.

NOTE: To remove unwanted treatments from the Personal List, see How to: Update a Treatment List.
