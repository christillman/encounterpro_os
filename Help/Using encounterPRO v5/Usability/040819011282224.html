Description
Short lists are subsets of items that help you locate and select your most used items quickly. Several different types of items in encounterPRO offer short lists such as:
  • Users (to select a recipient for a Message or Task)
  • Assessments (there is a short list for each assessment type: sick, well, OB, etc.)
  • Treatments (Each treatment type has its own short list)

Short lists can be Common or Personal. The Common short list for a type of item is accessible and shared by all uses in the practice. In addition, each user can have a Personal Short list that is accessible only to him or herself.

TIP: Short lists can contain any number of the items in question. However, the most effective short lists are those that are four pages or fewer. If the short list requires the user to page up and down many times, it will be faster to simply search by the description. You can arrange a short list alphabetically or manually sort up and down to place the most often used items at the top of the list.
