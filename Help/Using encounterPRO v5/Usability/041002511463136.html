How to: Display the Hotkey Assignments

Solution:

There are several functions that can be activated and deactivated using the Function keys on the keyboard. These are referred to as Hotkeys or F keys.

You can display the current function key assignments by holding down the Shift key and pressing F1 (Shift+F1).
