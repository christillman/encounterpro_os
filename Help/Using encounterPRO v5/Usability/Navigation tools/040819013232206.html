Description

  • Icons are pictures that should give you some indication as to their functionality. The instructions in the chapters that follow should allow you to identify with ease the appearance and function of the icons that appear on the various screens. Some icons appear on menu bars.
  • Buttons function similarly to icons. They have words or symbols on them that describe their function.
  • Menus are simply a collection of icons or buttons. For ease of use, whenever an icon is located on a Menu, there will be a word description beneath it to explain its function since some icons can have more than one function.
  • Bars and buttons function in an identical manner, the only difference being that bars are usually longer and display more text.
  • Radio Buttons are small, round buttons usually with a choice written beside them. Touching on the written choice is the same as touching on the button. You can select only one choice from the list.
  • Tabs function in an identical manner to bars and buttons. The slight difference is that tabs are generally lined up at the edge of the screen whereas buttons and bars can be solitary and placed anywhere on the screen.
  • A Box is any rectangular area on the screen with information inside it. Some boxes allow you to type text directly by clicking inside the box. Others offer choices when you click inside the box.
