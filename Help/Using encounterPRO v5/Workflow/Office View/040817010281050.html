Details: Description
This screen allows users to view and select all service bars currently ordered for the patient in the specific treatment room. Each examination room, telephone room, chart/holding room is considered a Treatment Room in encounterPRO.
A user must display the treatment room screen before he or she can select the patient name bar or any service bars for that patient.

When you select the patient name bar from the treatment room screen, a menu of options appears. There are a number of useful functions that can be performed from this menu, including:
  • Viewing the Patient Chart
  • Taking over the encounter from the original provider
  • Printing reports
  • Ordering labs and tests

Starting the Encounter
This screen also has the Get Patient button at the bottom of the screen to allow the clinical staff to start the encounter by bringing the patient into the treatment room.
Some specialized treatment rooms, such as the Telephone Rooms or Entry of Records room, will also have a Check-in button at the bottom of the screen. This allows a user to manually create a new encounter of the desired encounter type and bring the patient into the current treatment room with fewer steps than going through the Charts tab.

How to Access This Screen
These screens can be accessed by selecting anywhere in the specific treatment room area on the Office View.
