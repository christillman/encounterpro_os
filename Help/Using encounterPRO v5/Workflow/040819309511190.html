Description
This solution defines the basic user roles within encounterPRO and lists the functions that are typically performed by users assigned to that role. Based on the practice's specialty, some functions listed may not be needed.
The common user roles in encounterPRO:

  • Practice manager
  • Front Desk Administration
  • e-Records Manager (Scanning)
  • Clinical Staff
  • Provider
  • encounterPRO Administrator
  • User Configuration Level I (Basic Configuration)
  • User Configuration Level 2 ('Super Users')

NOTE: Users are often assigned more than one role in a practice

Additional Roles
Some larger practices may also separate duties into the following roles:

  • Referral specialist
  • Telephone Triage
  • Billing specialist

Users with these kinds of roles will need to perform a subset of the functions assigned to the other roles, for example the referral specialist will perform several of the functions that are listed below Front Desk Administration.
