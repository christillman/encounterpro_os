Purpose

Scribing is a technique for documenting encounters which allows another user, the scribe, to accompany the provider during patient examinations transcribe a patient's history and serve as the provider's documenter. When a scribing user is logged in the blue information line at the top of the screen will indicate that the logged on user is the selected user, not the scribe. All information documented during the current session will be noted as being documented by the scribe under the direction of the selected user. There are several steps that need to be performed in order to act as a scribe during an encounter.

Enable Scribing for the Practice

EncounterPRO allows the practice to enable scribing through a General Preference. This preference can be set at all levels:

  • Global
  • Office
  • Computer
  • Specialty
  • User

Set User Permission to Scribe for Other Users

The permissions for any user who will be scribing for others need to be set on the User Configuration screen.

Log In as the Scribe

When  user who has permissions to scribe for others logs in to encounterPRO, the system will prompt the user to select which provider for whom he or she will be scribing.

IMPORTANT:  Each user who logs in to scribe for another user must agree to abide by the following:
'By logging into the EncounterPRO® Electronic Health Record as a scribe for a licensed provider, I hereby affirm that the provider I designate has specifically asked me to do so.  I further affirm that when I am logged into the EncounterPRO EHR as a scribe, I am only recording observations and orders made and communicated to me by such provider.  I am not recording my own observations, orders or other information.'

Details:

Steps
Follow these steps:
To enable the scribing preference:

  1. From the Office View, select the Configuration tab. The Configuration list appears.
  2. Select Preferences. The Preferences list appears.
  3. Select General as the Preference Type on the left.
  4. Select the level you want to set the preference at the bottom of the screen:
    • Global
    • Office
    • Computer
    • Specialty
    • User
  5. Locate the preference for Enable Scribing.
  6. Select Yes, then Finished. The Preference list reappears.
  7. Select Finished to return to the Configuration list.

To update the user permissions for the scribe:

  1. From the Configuration list select Users. The Select User screen appears.
  2. Select the user who needs to have permissions to scribe for other users. The User Configuration screen appears.
  3. Select the Permissions tab.
  4. Locate the Office Privilege called, 'User may scribe for others.'
  5. Select Yes, OK. The Select User screen reappears.
  6. Repeat steps 2 through 5 to set the privilege for other users.
  7. Select OK to return to the Configuration list.

NOTE: The user will need to log out of encounterPRO and log back in for the new permissions to take effect.

To scribe for another user:

  1. When the scribing user logs in the system will prompt him/her to select the user they will be scribing for during this session.
  2. If you are not scribing for another user during this session, select Cancel.
  3. If you will be scribing for another user, select that user from the list. The Office View appears. The blue information line at the top of the screen will indicate that the logged on user is the selected user, not the scribe. All information documented during the current session will be noted as being documented by the scribe under the direction of the selected user.
