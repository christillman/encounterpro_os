Description

This list shows the configuration functions that the Advanced Users are often asked to perform in encounterPRO. Each function is linked to a solution that gives step-by-step instructions to accomplish that function.
  • Adding and maintaining the folder list
  • Adding new assessments
  • Adding new drugs; adding packages to existing drugs
  • Adding new Health Maintenance Rules
  • Adding new patient education materials
  • Adding, updating, removing users
  • Changing auto-close rules for existing assessments
  • Changing specific preferences
  • Creating and updating standard exams (both common and personal)
  • Maintaining the consultants list
  • Selecting the appropriate treatment modes for in-office vs. send out labs, tests, etc.
  • Setting up external hardware such as scanners, printers, signature devices, etc.
  • Updating an existing treatment list or creating a personal treatment list by assessment
