How to: Modify an Existing Standard Exam.

Purpose
Standard exams can be a great time saver when entering physicals, ROS, or lab results. They allow one-touch population of results on a drill-down screen. After selecting the standard exam button to populate the results, the provider or nurse can change any of those results which do not apply to the current patient as needed. Your system was delivered with several standard exams, and you may have created others. From time to time, you may need to modify the results for those exams.

Before You Begin
You must have the specific permissions to be able to modify common standard exams which can be used by the entire office. All clinical users have the ability to create and modify their own personal standard exams that are visible only to them.

TIPS:
Changing the Common Standard Exams should only be done with the agreement of all users who will be using that exam.

If you decide to create your own Personal Standard Exams, choose a name that is NOT the same as the Common Standard Exam to reduce confusion. Many users prefix the exam name with their initials to clearly identify Personal Standard Exams.

Details:

Steps
Follow the steps below:

  1. Access the drill-down screen with the standard exam you want to modify. The existing standard exams are listed on the right of the screen.
  2. Select the button for the standard exam you want to modify. If you have created any Personal Standard Exams, be sure which exam (Personal or Common) you are modifying. The previously saved results are populated.
  3. Change the results for all the observations to the results you want to set for this standard exam.
  4. When you are satisfied with your results, make sure the standard exam button is still highlighted (darkened) then select the Set button. If you want this to be a common standard exam to be shared by the office, select Set Common Results. If you want this to be a personal standard exam to be used by only yourself, select Set Personal Results.
  5. The system prompts you to confirm that you wish to set the results for the new standard exam to the selected items. Select OK.
  6. If you want to modify another standard exam for this screen, select Clear All, then repeat steps 2 - 5.
  7. Select Finished to close the drill-down screen.
