Description
Configuring users consists of adding users to the system and setting up their roles and access rights so that they can perform the necessary tasks within encounterPRO.

There are two main areas where you will accomplish this:
  • The User Definition screen under the Configuration tab
  • The Authorize Services screen under Preferences

User Roles
Each user will be assigned to at least one role, though most will be assigned multiple roles. Roles influence functions like workflow, Tasks, and permissions. For example, steps in a workflow can be assigned to a role, such as Provider, Front desk, or Clinical Staff.

NOTE: All nurses and providers should be assigned the role of Clinical Staff in addition to their other roles, as many steps in typical workflows are assigned to Clinical Staff.

Access Privileges
Access privileges allow users to perform functions within encounterPRO. These are separate from roles. You set the high-level access privileges for each user from the Users screen under the Configuration tab. You can further refine a user's access privileges by either granting or revoking access to specific services using the Authorize Services screen (also under the Configuration tab). Settings on the Authorize Services screen override the access privileges set on the User Definition screen.
