Purpose

In encounterPRO the providers have the ability to flag objects as confidential to limit who can view these items in the patient's chart and prevent them from appearing on reports. Objects that can be flagged as confidential include:

·         A specific assessment, such as Pregnancy

·         A specific treatment, such as a prescription or lab test

·         An encounter

NOTE: Flagging an assessment does NOT automatically flag any associated treatments as confidential. For reports that display Assessments with Treatments (ATAT), if the assessment is confidential the assessment will not appear and the associated treatments will not appear either. On reports that list treatments separate from their associated assessments (AATT), the treatments will appear unless they are also individually flagged as confidential.

If the user wants to make an item confidential, select the Make Private button to add a progress note that Denies access to that item. (These buttons must be configured by EHR Product Support or a Super User at the practice.)

Confidential Objects Can Be Viewed By:

  • All Super Users
  • The Primary provider for the patient
  • The Encounter owner for any encounter in which the object is open

Printing Reports for Patients With Confidential Objects
IMPORTANT: When printing, the user must use the SELECT READER option from the Print Menu.
 
Choose Select Reader, then select who this report is being printed for.  To be sure that no confidential information appears on the report, select a Special User, such as Patient or Unknown. Special Users follow the same rules as for viewing. Which means that none of the Special Users would be able to view confidential objects.

Before You Begin

Your practice may want to have some indicator that there are confidential items in the patient's record. You may want to always add a Chart Alert to print using Select Reader or some standard verbage so that reports are not unintentionally printed with confidential information.

Details: Steps
Follow these steps:

To make an assessment confidential:

  1. From the Patient's Chart, select the description of the assessment to display the assessment dashboard.
  2. Select the button for Make Private. A progress note is added that the item has been flagged as Confidential.

NOTE: Flagging an assessment does NOT automatically flag any associated treatments as confidential. For reports that display Assessments with Treatments (ATAT), if the assessment is confidential the assessment will not appear and the associated treatments will not appear either. On reports that list treatments separate from their associated assessments (AATT), the treatments will appear unless they are also individually flagged as confidential.

To make an entire encounter confidential:

  1. From the Patient Demographics screen, select the All Encounters tab. A list of open encounters appears.
  2. If the encounter that you want to make private does not appear on this list, select All Encounters. A list of all encounters for the current patient appears.
  3. Select the bar for the desired encounter. A menu of options appears.
  4. Select the button for Make Private. A progress note is added that the item has been flagged as Confidential.

To make a treatment confidential:

  1. From the Patient's Chart, select the description of the treatment to display the treatment dashboard.
  2. Select the button for Make Private. A progress note is added that the item has been flagged as Confidential.
