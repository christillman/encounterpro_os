How to: Update a Patient's Treatment Plan.

Purpose
If a patient is not responding to the current treatments, you may want to update the treatment plan. It is important that any discontinued treatments are closed (not canceled). New treatments can be ordered, and progress notes can be added to indicate the status of the patient.

Before You Begin
It is a good idea to review the treatment plan for a diagnosis before updating it. Access the assessment dashboard screen by selecting the description of the assessment on the Patient Chart screen.

Details:

Steps
Follow the steps below:

  1. From the Patient Chart, select the bold description of the assessment for which you want to review the treatment plan. If a menu of options appears, select Dashboard. The dashboard for the assessment appears.
  2. Optional. To review the entire treatment history for this assessment, select the DX History button at the bottom of the screen. The Assessment History screen appears listing all instances of this assessment and the treatments ordered. When finished reviewing the history, select OK. The dashboard screen reappears.
  3. If there are treatments that you want to discontinue, follow these steps:
    1. Select the description of the treatment on the dashboard screen. A new dashboard screen for that treatment appears.
    2. Select at the top of the screen or the Close Treatment button at the bottom of the screen.
    3. Select Finished at the bottom of the screen. The assessment dashboard reappears.
  4. If you want to add a new treatment to the treatment plan for this assessment, follow these steps:
    1. Select the Order Treatment button at the bottom of the screen. The treatment list for the assessment appears.
    2. Select a treatment from the list or select New Treatment to search for a treatment that is not on the list.
    3. When all new treatments have been selected, select Finished. The assessment dashboard screen reappears.
  5. If you want to add a progress note to indicate the status of the patient, follow these steps:
    1. Select the link at the top of the screen or the DX Note Menu button at the bottom of the screen (you may need to select Other Options first.)  A menu of progress notes appears.
    2. Select one of the OneTouch progress notes (Resolved/Close, Resolved, Resolving, Improved, Stable)
    3. Or select Progress Note to be able to enter your own note text. Enter the note text and select OK.
  6. Select Finished to return to the Patient Chart.
