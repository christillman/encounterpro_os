How to: Rediagnose an Assessment

Purpose
On occasion, a provider may order a preliminary assessment and then need to change that assessment as more information, such as lab results, become available. encounterPRO allows the provider to re-diagnose the patient and keep the treatment plan intact, reassigning all the treatments to the new assessment. This way, the original assessment does not have to be cancelled and all the treatments reordered under the new assessment.

Before You Begin
Be sure to review the existing treatment plan to make sure that all elements still apply to the new assessment. If you need to make changes, see How to: Update a Patient's Treatment Plan for details.

Details:

Steps
Follow the steps below:

  1. From the Patient Chart, select the description for the assessment that needs to be changed.
  2. If a menu of options appears, select Rediagnose. If the dashboard appears, select Rediagnose from the bottom of the screen (may be under Other Options). The Select Assessment screen appears.
  3. Select the new assessment from the short list or search for the desired assessment.
  4. Select OK. The Patient Chart reappears. All treatments formerly attached to the original assessment now appear below the new assessment. The original assessment appears on the chart with (Rediagnosed) next to the description.
