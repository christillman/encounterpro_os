How to: Add an Allergy Assessment

Purpose
In encounterPRO a patient's allergies are documented as assessments. Allergy assessments require a different set up than other assessment types, such as sick and well assessments.

Before You Begin
If you are setting up an allergy assessment for a drug allergy, you need the list of reactive drugs in your database.

NOTE: Automatic Allergy Alerts will only appear when the selected Drug being prescribed is identified on the Reactive Drug list for the allergy assessment. For example, the drug Amoxicillin needs to be on the Reactive drug list along with any of the other Penicillin derivatives for the Penicillin allergy assessment for the system to flag Amoxicillin as contraindicated. A periodic review of the drug allergy list should be completed and the required Reactive Drugs added. Never rely solely on an Automatic Prescription Alert. Always check the documented Allergies for the patient.

Details:

Steps
Follow the steps below:

  1. From the Configuration List, select Diagnoses. The Select Assessment to Configure screen appears.
  2. Select the Assessment Type bar then select Allergy. The Select Assessment to Configure screen appears. (The default Assessment Type is Sick.)
  3. Select the New Assessment button.
  4. Enter the name of the Allergy in the Description field, then OK. The Assessment Definition screen appears.

    NOTE:  An ICD-9 code is not required in the ICD-9 box since this assessment will not be used for billing purposes.

  5. Select the Category for this allergy.
    NOTE: If you select Drug as the category, the Reactive Drug tab appears after you save the assessment. Leave the Risk Level and Complexity as N/A. This type of assessment will not be used towards E & M coding. If desired, add a Patient Reference or Provider Reference Materials.

  6. Select the appropriate button and select the reference material you want to use for this assessment.
  7. Select Specialty, then select the specialties that you want to be able to use this assessment. Be sure to select your own specialty. Then select OK.
  8. Select the Acuteness/Auto Close tab. Select:
    Chronic - allergies are considered chronic assessments
    None - do not close the assessment automatically
  9. Select OK. The Select Assessment to Configure screen reappears.
  10. If this was a Drug allergy assessment, you still need to set up the reactive drug list. Select the newly created assessment from the list, then select Edit Assessment. A new tab for Reactive Drugs appears on the Assessment Definition screen.
    NOTE: If this is a 'new drug' it must be configured in the encounterPRO Prescription Medications database before it can be added to the Reactive Drugs list. See Adding a Simple Drug.

  11. Select the Reactive Drugs tab.
  12. Select Add Drug. The Select Prescription Drug screen appears.
  13. Select all the drugs that react for this allergy then select OK to return to the Assessment Definition screen.
  14. The list of reactive drugs selected appears. Be sure that you have included all the drugs related to the drug allergy, then select OK. The Select Assessment to Configure screen reappears.
  15. Select OK to return to the Configuration List.
