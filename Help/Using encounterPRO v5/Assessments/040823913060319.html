Purpose
Assessments and treatments can be canceled if they are ordered in error. If an assessment or treatment has been canceled inadvertently, it can be uncanceled. See How to: Uncancel Treatments and Assessments.


Before You Begin
When you cancel an assessment, all associated treatments are also canceled.

Details:

Steps
Follow these steps:

  1. Display the Patient Chart for the patient for which you want to cancel an assessment.
  2. Select the description of the assessment you want to cancel. The dashboard screen appears.
  3. Select the Cancel Diagnosis button. The system will prompt you to enter a reason to cancel the assessment.
  4. Enter the reason you want to cancel the assessment and select OK.
  5. The system prompts you to be sure that you want to cancel the assessment. Select Yes. The status is changed to Canceled.
  6. Select Finished to return to the Patient Chart screen. The assessment will appear without the red background.
