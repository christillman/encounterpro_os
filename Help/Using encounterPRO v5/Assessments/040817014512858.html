Description
This screen is used to order the assessments for the patient. Assessments are grouped by type:

  • Sick
  • Well
  • OB
  • Allergy - used to document patient-reported allergies Vaccines (V codes) - used most often for immunization only visits

How to Access This Screen
Access this screen by selecting the New Sick DX or New Other DX buttons on the Patient Chart.
