Purpose
From time to time you may need to update the configuration of an assessment. In fact, when encounterPRO is installed, the sick assessments are not set to auto close. This allows each practice to determine the auto close interval it believes to be appropriate. Modify those assessments that should auto close, by setting the auto close interval.

Before You Begin
If you want to set auto close rules for your acute, sick assessment you should have a consensus among the practice providers on the length of time that each assessment should stay open.

Details:

Steps
Follow the steps below:

  1. From the Configuration tab, select the Diagnoses button. The Select Assessment to Configure screen appears.
  2. Select the assessment you want to modify. You may have to use the search options to locate the desired assessment. A menu of options appears.
  3. Select Edit Assessment from the menu. The Assessment Definition screen appears.
  4. Make the necessary changes. See Screen: Configure Assessments screen for detailed information on the fields on this screen.
  5. If you want to change the auto close rule, select the Acuteness/Auto Close tab. Select one of the following:
    • None - do not close the assessment automatically. It is the Provider's/Nurse's responsibility to manually close all assessments that are set to Auto Close as NONE.
    • End of Encounter- the assessment will automatically close at the end of the encounter.
    • After Interval- select a specific time interval after which the assessment will automatically close.
  6. Select OK. The Assessment Configuration screen reappears.
