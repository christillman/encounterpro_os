How to: Monitor Recurring Acute Problems.

Purpose
EncounterPRO can assist you in monitoring patients who have recurring acute problems, such as sinusitis or otitis media. You can see which treatments have been used for this patient in the past to help you determine today's treatment plan. If needed, you can generate a report of all the occurrences of a particular assessment to send with a referral.

Recurring problems are noted on the Patient Chart by a number in parentheses such as (2X), (3X), etc. This notation appears next to the assessment description and denotes the number of times that the patient has been diagnosed with that problem.

Details:

Steps
Follow the steps below:
To View the Assessment History:
You can view all the dates on which a patient was diagnosed with a specific assessment and the treatments that were ordered at each encounter. 

  1. From the Patient Chart or the Quick View screen, locate the assessment you want to review. Assessments that have been ordered multiple times will have a designation next to the description to indicate the number of times the assessment was ordered, such as (3x).
  2. Select the description for the assessment. The assessment dashboard screen appears.
  3. Select the DX History button. The Assessment History screen appears. The screen will display all assessments matching the first three digits of the ICD-9 code. You can select the button to display assessments matching the three digits plus first decimal point if desired.
  4. When finished reviewing the information, select OK. The dashboard screen reappears. Select Finished to return to the Patient Chart screen.

To Generate a Report on the Assessment History:
You can produce a report on the assessment history to send with a referral or give to the patient.

  1. From the Patient Chart select Other Items. A menu of options appears.
  2. Select Print Reports. A menu of reports appears.
  3. Select Pick Report. The Select Report screen appears.
  4. Select Patient as the Context for the report. The short list of patient-level reports appears.
  5. Select Assessment Histories from the list. (There is another report called Assessment History which lists all assessments for the patient.)
  6. Select the gray box to choose the assessment for this report. A list of all the assessments, both closed and open, ordered for this patient appears.
  7. Select the assessment for this report, then select OK. The system will generate the report using the first three digits of the ICD-9 code. This means that if you select Otitis Media, the report will include all otitis media assessments, including bilateral, left, and right.
  8. Select Finish. The report output menu appears.
  9. Select the option you want to use for this report:
    • Preview
    • Default printer
    • Select printer
    • Print from server
  10. The report is produced.
