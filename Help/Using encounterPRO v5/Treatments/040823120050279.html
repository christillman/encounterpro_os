Description

The Default Treatment List is available as one of the tabs on every treatment list for every assessment at the practice. The Default Treatment List allows the practice to create a list of commonly used treatments that can be accessed from any assessment treatment list.

Uses

Many practices like to use the Treatment Group function to create treatment groups on the Default Treatment List such as :

  • Follow-ups for 2 days, 1 week, 2 weeks, etc.
  • Referrals to commonly referred to specialities, such as ENT, Allergist, Dentist for Pediatrics
  • Commonly ordered x-rays
  • In-house labs
  • Commongly ordered in-office medications

It is very simple to copy the treatments from the Default Treatment list to create a new Common or Personal treatment list for a specific assessment.
