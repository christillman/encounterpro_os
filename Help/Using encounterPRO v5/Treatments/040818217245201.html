How to: Create a Treatment Group

Purpose
If your treatment list for a particular assessment is lengthy, you can create groups of treatments to shorten the list. Treatment groups are simply lists within the treatment list. The grouping may be like items, such as antibiotics, or a set of items that you would order together, such as all vaccines typically given at 6 months.

Before You Begin
You should determine if you will be creating a treatment group on the common list or your personal list.

NOTE: In previous versions, treatment groups were called Composite Treatments.

Details:

Steps
Follow the steps below:

  1. After you have selected an assessment, a treatment list appears.
  2. Select the New Group button.
  3. Enter a name for this treatment group then select OK. The new group is added to the bottom of the treatment list.
  4. Select the icon to the left of the group name. A menu of options appears.
  5. Select New Treatment. A menu of treatment types appears.
  6. Select the treatment type for the item you want to add to the group. The selection screen for the type of treatments chosen appears.
  7. Locate the desired treatment(s) and select the item. You can select multiple items of this type and each will appear in the Selected Items list.
  8. Select OK. The treatment(s) selected appears on the treatment group list.
  9. Repeat steps 5 - 8 to add additional treatments to this group.
    NOTE: You do not have to perform an extra step to permanently add the treatments to the treatment group list, they are automatically saved to the list and will be available when selecting treatments for this assessment.
  10. When finished adding treatments, select Finished. The Select Treatment screen reappears.
