How to: Review Lab/Test Results

Purpose
Lab and Test results can be reviewed from many different locations in encounterPRO. In the case of in house labs, the results will appear on the treatment dashboard, accessible from the patient's chart. When the labwork is done off site, then there are several options for locating and reviewing the results.

  • From the Treatment HX tab of the Patient Chart
  • From the Lab History screen
  • From the Daily Screen From the Outstanding Labs/Tests screen


Before You Begin
If your practice is integrated with a lab, then you will want to use the method, "To review results from the Daily screen."
It is important to know whether the lab you are searching for is open or closed. Certain displays are restricted to only open treatments, while others will show both open and closed treatments.

Details:

Steps
Follow the steps below:
To review the results from the Patient Chart (Open treatments only):

  1. From the Patient Chart, select the Treatment HX tab.
  2. Select the Labs and Tests button on the right. A list of Outstanding Labs/Tests appears. This list includes only Open labs and tests.
  3. Select the lab you want to review. The treatment dashboard appears.
  4. Select the link at the top of the dashboard screen to indicate that you reviewed the results.
  5. If you want to add a review note or order additional treatments, use the buttons on the dashboard.
  6. If you want to close this treatment, select the link or Close Treatment button.
  7. When finished, select the Finished button. The Treatment HS screen reappears.

To review the results from the Lab History screen (Open and Closed treatments):

  1. From the Patient Chart, select the Review Menu button. (You may need to select Other Items first to display the entire menu.) A menu of review options appears.
  2. Select the Review Labs or Review Tests button. The History Display screen appears with all Labs (or Tests) displayed along the left. Select the bar for the desired treatment to review the results.
  3. To note that you reviewed the results, select in the white text area. A menu of options appears.
  4. Select Reviewed. An "R" appears on the bar for that treatment on the left.
  5. If you want to make a comment about the treatment, select the white text area again. Select the Enter Comments button. Enter the text of your note, or select from the pick list on the left. When the note is complete, select OK. The History Display screen reappears.
  6. When you have finished reviewing all treatments, select OK. The Patient Chart reappears.

To review the results from the Daily screen (Lab Integration):

  1. From the Office View, select the Utilities tab. The Utilities list appears.
  2. Select Daily (Labname). The Daily (Labname) Results screen appears.
  3. Select the date the lab was ordered at the top of the screen. A list of labs appears. For each lab ordered on that date, the status of the lab results is listed and the user who is responsible for the next step in the workplan is listed.
  4. Select the lab you want to review. The treatment dashboard appears.
  5. Select the link at the top of the dashboard screen to indicate that you reviewed the results.
  6. If you want to add a review note or order additional treatments, use the buttons on the dashboard.
  7. If you want to close this treatment, select the link or Close Treatment button.
  8. When finished, select the Finished button. The Daily (Labname) Results screen reappears.

To review the results from the Outstanding Labs & Tests screen (Open treatments only):

  1. From the Office View, select the Utilities tab. The Utilities list appears.
  2. Select the Outstanding Labs/Tests button. The Outstanding Labs/Tests screen reappears. The list is sorted by date ordered with the newest at the top.
  3. To filter the list, you can do any of the following:
    • To filter by ordering provider, select the bar that reads to the right of Ordering Provider. The Select User screen appears. Select the provider you want to filter the list for.
    • To filter by patient name, select the bar below By Name. A keyboard popup appears. Type the patient's last name (or partial last name) and select OK.
    • To filter by office location, select the bar below by Office. A list of office locations appears. Select the office you want to filter the list for.
  4. To change the filter, select the Clear filter button on the right of the screen. Repeat step 3 to filter the list again.
  5. Locate the lab/test that you want to review. Select the bar for the test description. The treatment dashboard for that treatment appears.
  6. If you want to add a review note, select one of the links at the top of the screen. Enter the text of the note and select OK.
  7. If you want to close this treatment, select at the top of the screen or the Close Treatment button at the bottom of the screen.
  8. To return to the Outstanding Labs/Test list, select Finished.
