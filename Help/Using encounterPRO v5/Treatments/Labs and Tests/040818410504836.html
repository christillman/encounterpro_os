Purpose
The Daily Lab Results screen has been added to the Utilities list to help your practice track the labs and results. (The button will have the name of the lab facility you are integrated with, for example, Daily Labcorp.) If you are integrated with more than one external lab facility, you will have a button for each lab facility.

Your office can track and manage the labs by reviewing the Daily Lab Results screen to check for labs with no results and labs that have not been reviewed in a timely manner.

Before You Begin
Labs that have been ordered must have the lab facility entered in the Sent to or Referred to field to appear on the list. See How to: Order Labs (Lab Integration) for detailed instructions.

Details:

Steps
Follow the steps below:
To Review Lab Results:

  1. From the Utilities tab, select Daily Lab Results. The Lab Results screen will appear. If you are integrated with more than one external lab facility, you will have a button for each lab facility.
  2. For the date listed at the top of the screen, the display shows:
    • Lab ordered
    • Patient name and billing number
    • User who ordered the lab
    • Date results were received
    • Date results were reviewed
    • User who approved the results
    • If not reviewed, user who received the Tasks to review the lab
  3. To view results for a previous date, select the forward and backward button to scroll through dates one at a time. To select a specific date, select the date bar and select the desired date from the calendar.
  4. If you want to review the lab results, select the description of the lab ordered. The treatment dashboard appears. You can review the results and select to make a notation.
  5. When finished, select OK to return to the Utilities list.

To Manage  Lab Results:

  1. From the Utilities tab, select Daily Lab Results. The Lab Results screen will appear. If you are integrated with more than one external lab facility, you will have a button for each lab facility.
  2. For the date listed at the top of the screen, the display shows:
    • Lab ordered
    • Patient name and billing number
    • User who ordered the lab
    • Date results were received
    • Date results were reviewed
    • User who approved the results
    • If not reviewed, user who received the Tasks to review the lab
  3. To view results for a previous date, select the forward and backward button to scroll through dates one at a time. To select a specific date, select the date bar and select the desired date from the calendar.
  4. If the lab results have been received, but have not yet been approved, check which user has the task pending. To send a reminder Task:
    1. Select the description of the lab. The treatment dashboard appears.
    2. Select the Task (ToDo) button. Send the provider who ordered the lab a reminder that the results for the lab have been received and are awaiting his/her review.
    3. Select Send.
  5. When finished, select OK to return to the Utilities list.
