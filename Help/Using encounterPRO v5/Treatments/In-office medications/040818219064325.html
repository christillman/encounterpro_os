How to: Document Administration of In-office Prescription Medications.

Purpose
Documenting the administration of in-office drugs is much like documenting the administration of a vaccine. A service bar will appear in the Office View for the in-office drug after it is ordered.

Before You Begin
Some in-office medications will allow you to select the HCPCS administration code. You will need to know the appropriate code for this specific instance - generally it is based on the number of units of the medication you are administering.

Details:

Steps
Follow the steps below:

  1. From the Office View, select the in-office administration service bar. The Office Medication screen appears.
  2. Select the drug maker.
  3. Enter the Lot Number. The lot number will default from the last number entered. Lot numbers default by office location.
  4. Select the Expiration Date.
  5. Select the location on the patient's body where you applied the medication.
  6. Select Medication Given.
