How to: Order In-office Medications

Purpose
In office drugs are those drugs that are administered during the encounter, like a cortisone injection or a breathing treatment. Ordering an in office drug will create a service bar to notify the clinical staff to prepare and/or perform the treatment.

Before You Begin
If you do not order the treatment from the treatment list for a specific assessment, you will need to associate the treatment to the appropriate assessment later. See How to: Modify Billing for an Encounter or How to: Associate Treatments With a Specific Assessment for instructions

Details:

Steps
Follow the steps below:

  1. From the Select Treatment screen for a particular assessment, select the desired in office drug from the treatment list.
  2. If the in office drug you want to order does not appear on the treatment list, select New Treatment. A menu of treatment types appears.
  3. Select the Office Drugs button. The Select Prescription Drug screen appears.
  4. Select the desired drug from the list. If the drug does not appear on the list, use the search options to locate the desired drug. A pop-up menu allows you to select the appropriate package for the drug.
  5. Select OK. A dosage screen appears.
  6. Confirm or change the package and dosage for this drug.
  7. If you want to charge for the procedure, select the Administration Procedure bar and select from the list of applicable procedures.
  8. When the information is correct, select the Finished button. A service bar is created for each in office drug ordered.
