Description
This screen is used by the provider to confirm the dosage and instructions for administering an in-office medication.

How to Access This Screen
Access this screen by:

  • Selecting an in-office medication from the Select In-Office Medications screen
  • Selecting the small icon to the left of the in-office medication on the Patient Chart
  • Selecting the description of the in-office medication on the Patient Chart, then selecting the description again on the dashboard
