How to: Uncancel Treatments and Assessments

Purpose
Assessments and treatments can be canceled if they are ordered in error. If an assessment or treatment has been canceled inadvertently, it can be uncanceled.

NOTE: When an assessment or treatment is un-canceled, it will be in a 'Open' status.


Before You Begin
When you cancel an assessment, all associated treatments are also canceled. When you uncancel an assessment that has canceled treatments associated with it, those treatments are not automatically uncanceled. If those treatments are valid, they need to be individually uncanceled. 

Details:

Steps
Follow these steps:
To uncancel a treatment:

  1. Display the Patient Chart for the patient for which you want to uncancel a treatment.
  2. Select the Incl Canceled button near the center of the bottom of the screen (just above the lower tabs). Any canceled assessments and treatments will display with a red background. (Remember the difference between closed and canceled.)
  3. Select the description of the canceled treatment. The dashboard screen appears.
  4. Select the Status button on the right, which reads 'Canceled.'
  5. The system prompts you to be sure that you want to un-cancel the treatment. Select Yes. The status is changed to Closed.
  6. Select Finished to return to the Patient Chart screen. The treatment will appear without the red background.

To uncancel an assessment:

  1. Display the Patient Chart for the patient for which you want to uncancel an assessment.
  2. Select the Incl Canceled button near the center of the bottom of the screen (just above the lower tabs). Any canceled assessments and treatments will display with a red background. (Remember the difference between closed and canceled.)
  3. Select the description of the canceled assessment. The dashboard screen appears.
  4. Select the Status button on the right, which reads 'Canceled.'  (Note: Selecting 'Cancelled from the dashboard does not give a prompt & opens to either the treatment i.e F/U screen or the Assessment treatment screen you have cancelled)
  5. The system prompts you to be sure that you want to un-cancel the assessment. Select Yes. The status is changed to Closed.
  6. Select Finished to return to the Patient Chart screen. The assessment will appear without the red background.
