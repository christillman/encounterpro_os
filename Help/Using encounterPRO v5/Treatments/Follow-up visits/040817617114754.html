How to: Order a Follow-up Visit.

Purpose
Follow-up visits can be ordered as part of the treatment plan for a specific assessment, as part of a general preventative visit schedule, etc. After the provider orders the follow-up and specifies the timeframe and purpose, the treatment workplan displays a service bar for the front desk staff. If no service bar is desired, that is a configuration option.

Before You Begin

If you want to order specific treatments as part of the follow-up treatment, you can do so. It is important that the clinical staff know that you will be ordering follow-ups with these associated treatments so that when the patient returns for that follow-up visit the proper workplan can be dispatched. Please review the solution Concept: Follow-ups with Associated Treatments before you order your follow-up.

Details:

Steps
Follow the steps below:

  1. If the desired follow-up appears on the Treatment List screen for the appropriate assessment, select it from the list. Go to step 6.
  2. If the follow-up you want to order does not appear on the treatment list, select New Treatment. A menu of treatment types appears.
  3. Select the Follow-up button. The Follow-up Visit screen appears.
  4. For each follow-up, you can enter a great deal of detail. This will help the front desk staff make the appropriate appointment for the follow-up visit. Select as much detail as you desire:
    • Timeframe: When to schedule the follow-up visit: specific date, first available, ASAP, or emergency
    • Goal for the Follow-up: Select the gray bar under Follow-up Goal, when there is a specific reason for the follow-up visit (for example, OB ultrasound)
    • Provider: If you want the patient to follow-up with a different provider, select the gray bar then select the desired provider from the list
    • Encounter Type: If the follow-up visit needs to be a specific encounter type, select the gray bar then select the desired encounter type from the list. For example, if a patient needs to return for a nurse-only visit for a vaccination or injection, select Nurse Visit.
    • Description: Accept the automatically generated description, or enter a new description for the follow-up visit (this will appear on the service bar for the admin/front desk user)
  5. When finished, select Finished. The Select Treatment screen reappears.
  6. Select additional treatments for this assessment, if needed. When done, select Finished. The Edit Follow-up screen appears.
  7. At this point, you can change any of the information entered on the first Follow-up screen, if needed. You can select the Add Treatment button to specify the exact treatment for which the patient is to return (this will create a follow-up workplan for the return visit). When satisfied with the follow-up instructions, select Finished. A service bar appears to allow the staff to document scheduling information for the follow-up.
