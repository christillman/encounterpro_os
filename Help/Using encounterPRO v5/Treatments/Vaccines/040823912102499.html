Description

The Immunization Record screen displayed the detailed record of the patient's past immunizations. These immunizations can be either captured from vaccines ordered in encounterPRO or from past vaccinations manually entered.

How to Access This Screen
Access this screen by selecting the Vacc tab on the Patient Chart, then selecting View Detail.
