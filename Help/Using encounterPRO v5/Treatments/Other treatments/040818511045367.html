Purpose
Patient instructions are considered treatments in encounterPRO. The provider can 'order' patient instructions from the treatment list of a particular assessment so that those instructions appear as part of the treatment plan for that assessment. If desired, the patient instructions can be ordered outside of an assessment and will appear on the Patient Chart for the encounter in which they were ordered.

Before You Begin
There are two types of instructions in encounterPRO:

  • Encounter instructions : which appear on the Patient Chart and assessment dashboard only for the encounter they were ordered.
  • Ongoing Instructions : which appear on the Patient Chart and assessment dashboard for all encounters going forward from the encounter in which they were ordered. Ongoing instructions will appear until the user either closes the instruction from the treatment dashboard screen or closes the assessment with which the instruction was originally associated.
    Details:

    Steps
    Follow the steps below:
    To order patient instructions associated with a specific assessment:

    1. From the Patient Chart, access the treatment list for the desired assessment by doing one of the following: 1)Select the New Sick or New Other DX button on the right of the chart and select the desired assessment. 2)Select the small icon next to the description of an existing assessment (open assessment) on the Patient Chart. 3)Select the description of the existing assessment (open assessment) then select the blue description from the dashboard screen The treatment list for the selected assessment appears.
    2. If the instruction you want to order appears on the treatment list, select it. You can select as many treatments from the list as needed.
    3. If the instruction you want to order does not appear on the treatment list, select the New Treatment button on the right of the list screen.
    4. If you want the instruction to appear only for the current encounter, select Encounter Instruction. If you want the instruction to continue to appear until the assessment is closed, select Ongoing Instruction. An observation comment screen (with pick list) appears.
    5. Select the instruction from the pick list, or type the instruction in the white text box.
    6. Select OK. The treatment list reappears with the new instruction selected (highlighted).
    7. If you want to save this instruction to the treatment list for future use, select the small icon to the left of the description.
    8. Select the Yes button below Update List, then OK. The treatment list may automatically sort the list to place the new treatment at the top of the list.
    9. Finish selecting or ordering treatments for this assessment. When the treatment plan is complete, select Finished. The Patient Chart reappears.

    To order patient instructions not associated with a specific assessment:

    1. From the Patient Chart, select the Order Treatments button from the right side of the chart. (You may have to select the Other Items button first to display the entire menu.)
    2. If you want the instruction to appear only for the current encounter, select Encounter Instruction. If you want the instruction to continue to appear until the assessment is closed, select Ongoing Instruction. An observation comment screen (with pick list) appears.
    3. Select the instruction from the pick list, or type the instruction in the white text box.
    4. Select OK. The Patient Chart reappears. Most practices are configured to have a service bar appear for the clinical staff to print the patient materials.
