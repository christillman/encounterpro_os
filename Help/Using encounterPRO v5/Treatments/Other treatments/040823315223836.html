Purpose

At Allergy Specialties the allergy shot module is used to administer allergy shots efficiently. The procedure that follows describes a commonly used process for administering allergy shots.
Details:

Steps
Follow these steps:

  1. Check in the patient for an Allergy Shot Visit.
  2. Go to the shot room/procedure room and select Get Patient. The Patients Waiting screen appears.
  3. Select the patient from the list of patients waiting. The Allergy Reaction History screen appears.
  4. Review the immunotherapy regimens and shot reaction history. When finished, select Finished. The Allergy Shot Administration screen appears. The details for this injection default from the schedule selected for this patient, increasing by one, from the previous dosage.
  5. Make any changes needed to this injection, such as amount or location, then select Shot Given. The system prompts you to confirm the dose administered. Select Yes.
  6. If there are multiple regimens for this patient, select the tab for the next regimen/shot. The screen will update to show the shot history for this regiment and the information for today's injection.
  7. Make any changes needed for today's injection, then select Shot Given. The system will prompt you to confirm that the correct dose was administered. Select Yes.
  8. If you are finished administering injections for this patient, select Finished. A service bar, labeled Wait 30 Minutes appears. (number of minutes is configurable.) When the number of minutes to wait has been reached, the Wait service bar closes and the Allergy Reaction Check service bar appears.
  9. Select the Allergy Reaction Check service bar. The screen will show the injections given today.
  10. Select the Reaction bar to document the reaction, if any. If there is no reaction, select No Reaction. If there is a reaction, select Reaction and enter the details of this reaction.
  11. When finished documenting reactions for today's injections, select Finished. The patient moves to the Checkout room.
