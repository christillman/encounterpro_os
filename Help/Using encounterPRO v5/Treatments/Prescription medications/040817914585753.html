How to: Print or Reprint Prescriptions

Purpose

There are several methods used to print prescriptions in encounterPRO. Many practices choose to configure their prescriptions to automatically print when ordered or automatically at the end of the encounter. If automatic printing is not turned on for your practice, then what happens when you have prescriptions pending depends on whether or not your practice is using Document Manager for your prescriptions.

  • If you are using Document Manager for your prescriptions, the NewRX or Refill will appear on the Document Manager screen. See How to: Print or Send a Document Using Document Manager for details.
  • If you are not using Document Manager, a service bar will appear to print the prescription.

If you normally automatically print prescriptions, but there was a problem and you need to re-print the prescription follow the procedure below.

Before You Begin

If you need to make changes to the prescription or you want to order a refill of the medication, see How to: Edit a Prescription or How to: Order Prescription Refills respectively. This procedure is intended to print or re-print the original prescription.

Details:

Steps

Follow these steps:

  1. Access the dashboard screen for the prescription treatment by doing one of the following:
    • Select the Current Meds box on the Vitals or Patient Chart screen, then selecting the desired medication from the list.
    • Select the prescription treatment description on the Patient Chart screen.
  2. Select the link at the top of the dashboard. A progress note will appear indicating that the prescription was ordered to print.
