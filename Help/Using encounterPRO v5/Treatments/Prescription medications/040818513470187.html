How to: Order Prescription Refills

Purpose
Prescription refills are quickly accomplished from the treatment dashboard. You can make a restricted set of changes to a prescription and still call it a refill. If you need to change the drug, you will need to close the original treatment and order a new prescription treatment.

Details:

Steps
Follow the steps below:

  1. From the Patient Chart, do one of the following:
    • Select the description of the prescription drug treatment
    • Select the Current Meds box at the top of the screen, then select the desired drug from the list. The prescription drug treatment dashboard appears.
  2. If you do not need to make any changes to the prescription, follow these steps:
    1. Select the Refill RX button at the bottom of the screen. The Refill screen appears.
    2. Enter a comment for this refill. This comment will appear in the patient's chart and on the printed prescription.
    3. If you use Document Manager for your prescriptions, select the following options:
      • Order Prescription - Select Yes to order a prescription that will either print or appear on the Document Manager screen. Select No if you will be calling in the prescription or handwriting a prescription.
      • Send Now - Select Yes to immediately send the prescription to the default printer. Select No to create a service bar (if you are printing prescriptions) or if you want a user to select the recipient and dispatch the prescription document from the Document Manager screen.
      If you do not use Document Manager for your prescriptions, select from the following options:
      • Print Now
      • Do Not Print
      • Print at Checkout
       
    4. If you want to manually sign the refill prescription, select the Sign for Refill button. Complete the signature and select OK.
    5. Select Finished. If you selected Print Now, the print menu appears to allow you to select a specific printer.
  3. If you need to make any changes to the prescription, follow these steps:
    1. Select the Refill RX button at the bottom of the screen. The Refill screen appears.
    2. Select the Review RX button at the bottom of the screen. The Prescription ordering screen appears.
    3. Make changes as needed (you cannot change the medication - if you need to select a new medication, close the old prescription and order the a new prescription) .
    4. Enter a comment for this refill describing the change made. This comment will appear in the patient's chart and on the printed prescription.
    5. If you use Document Manager for your prescriptions, select the following options:
      • Order Prescription - Select Yes to order a prescription that will either print or appear on the Document Manager screen. Select No if you will be calling in the prescription or handwriting a prescription.
      • Send Now - Select Yes to immediately send the prescription to the default printer. Select No to create a service bar (if you are printing prescriptions) or if you want a user to select the recipient and dispatch the prescription document from the Document Manager screen.
      If you do not use Document Manager for your prescriptions, select from the following options:
      • Print Now
      • Do Not Print
      • Print at Checkout
       
    6. Select Finished. If you selected Print Now, the print menu appears to allow you to select a specific printer. The original prescription will be closed and a progress note attached to it noting that it was Modified.
  4. Select Finished to return to the Patient Chart.
