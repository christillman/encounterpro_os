How to: Register for SureScripts e-Prescribing.

Purpose
In order to use e-Prescribing, both the provider ordering the prescription medication and the pharmacy must be registered with SureScripts-RxHub. RX-certified users can register with SureScripts-RxHub using the User Definition screen in encounterPRO.

NOTE: Each user who wants to register with SureScripts must have a unique NPI number. A DEA number is no longer required.

Before You Begin

EncounterPRO determines the ordering provider using the following logic:

  1. The last provider to order a refill will be the 'ordered by' provider.
  2. If no refills have been ordered then the last provider to sign for the prescription will be the 'ordered by' provider.
  3. If no one has signed for the prescription then the provider who originally ordered the prescription will be the 'ordered by' provider.

NOTE: Users will need to be able to access the User Definition screen in order to register for SureScripts-RxHub. If an encounterPRO user is not normally permitted to access that screen, the encounterPRO Administrator can assist.

In states where Nurse Practitioners can prescribe medications, they should register with SureScripts-RxHub. If a user is not RX-certified and would like to be able to order refills that can be sent via SureScripts-RxHub, they can do so if acting as a 'scribe' for a SureScripts-RxHub registered provider. See How to: Set Up and Use Scribing for details.

IMPORTANT: When a user logs in as a scribe for another user, he or she must abide by the following: 'By logging into the EncounterPRO Electronic Health Record as a scribe for a licensed provider, I hereby affirm that the provider I designate has specifically asked me to do so. I further affirm that when I am logged into the EncounterPRO EHR as a scribe, I am only recording observations and orders made and communicated to me by such provider. I am not recording my own observations, orders or other information.'
Details:

Steps
Follow the steps below:

  1. From the Configuration tab, select the Users tab. The Select User to Edit screen appears. The display defaults to Active users.
  2. Select the name bar of the user you want to modify. You may need to use the Search options to locate the desired user. The User Definition screen appears.
  3. Select the User tab.
  4. Select the Not Registered button for E-RX Status.The SureScripts Prescriber Registration screen appears.
  5. Complete the information on the registration screen. Confirm that the NPI number is correct.
  6. Select both NewRX and RefReq/RefRes in the Service Level box.
  7. Select Register. The system will display a message that the registration was successful. The User Definition screen reappears.
  8. When finished making changes, select OK. The Select User to Edit screen reappears.
  9. Select OK to return to the Configuration list.
