Problem: Unable to View Spirometery Results Files

Cause:

Midmark does not yet have an sdk that will properly install on a 64 bit machine.

Solution:

Midmark has a viewer utility that does work on a 64 bit machine, it only allows you to view .car files ( past spirometery results) this should be sufficient because we do not advise performing spirometery exams directly on the server itself.

The viewer utility is located on the mid mark website under the following:

Support, documents and downloads, medical, diagnostics, IQMark digital spirometers, viewers

Details: EncounterPRO is unable to view the .CAR files with the Midmark Spirometery results when installed on a 64-bit machine.
