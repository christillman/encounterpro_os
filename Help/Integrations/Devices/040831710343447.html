Triage Protocol: Welch Allyn Vitals Monitor

Triage Protocol: Triage Protocol: Welch Allyn Vitals Monitor
Overview:

The Welch Allyn Vitals Monitor external source component allows users to collect vital signs from the appropriate Welch Allyn devices and load them directly into the vitals screen.  If the vitals measurements do not appear to be automatically entered into the vitals screen, there are some steps to try to identify and resolve the problem.

Symptoms:

When the vitals monitor external source is enabled, the appropriate data entry fields on the vitals screen are grayed out and a colored bar appears under each one.  The color of the bar under the data entry field indicates the apparent status of the Vitals Monitor as follows:

 

Bar Color

Meaning

Green

The monitor appears to be operating normally and new readings at the monitor should automatically populate the appropriate data entry field.

Red

The monitor appears unavailable.  This could be because it is not turned on, or because it is not connected to the computer.  For the monitors using an infra-red (IR) connection, the IR device may not have a clear path to the vitals monitor.

Black

When the bar is black it means that the particular data field already had data in it when the vitals screen came up.  In this case EncounterPRO does not let the external device automatically update the field.  To tell EncounterPRO to start getting values from the device, click on the ellipses (?) next to the field.

Resolution:

The vitals screen comes up with no grayed fields and no colored bars.

·         Make sure the 'Welch Allyn Vitals Monitor' is enabled in Configuration/External Sources

   

The vitals screen comes up with grayed fields and red bars.

·         Make sure the vitals monitor is connected to the computer, is plugged in, and turned on.  For the monitors using an infra-red (IR) connection, make sure there is a clear path from the IR device connected to the computer and the IR panel on the vitals monitor.

·         To rule out computer hardware problems, try connecting the vitals monitor to a different EncounterPRO computer.

 

 

The vitals screen comes up with white fields and black bars.

·         The vitals screen will not allow the external device to automatically overwrite previously entered values.  Click the ellipses button (?) and select the 'Get Result(s) from Welch Allyn Vitals Monitor' option.  This should turn the field gray and the colored bar green.
