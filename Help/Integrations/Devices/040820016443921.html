How to: Set up a Ricoh Printer for Prescriptions

Solution: In order for the prescriptions to properly work you have to have the printers added as two separate printers
Details:

If a virtual printer is not created you will need to create one or have the hardware team create one

  1. Right-Click the Printer and select Properties
  2. Select the General Tab
  3. Click the Printing Preferences Button
  4. Select the Paper Tab
  5. Change the Input Tray to Bypass Tray
  6. Click the Custom Paper Size Button
  7. Change the Unit of Measure to Inch
  8. Change Width to 4.25
  9. Change Length to 5.5
  10. Navigate back to Printer Properties
  11. Select the Advanced Tab
  12. Click the Printing Defaults Button
  13. Select the Paper Tab
  14. Change the Input Tray to Bypass Tray
  15. Click the Custom Paper Size Button
  16. Change the Unit of Measure to Inch
  17. Change Width to 4.25
  18. Change Length to 5.5
  19. Navigate back to Printer Properties
