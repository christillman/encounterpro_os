Solution Title: Checklist: Implementing a New SureScripts Client
Overview:

A copy of this checklist should be attached to the client profile in Outlook as soon as the client expresses interest in e-prescribing.

There are several steps in this process that require additional explanation and instruction. Links to the appropriate solution are provided in the Comments/Resources column.

Stage 2: Installation and Education

  • ePRO Client 5.1.11 or higher
  • Mod level 141 or higher
  • Server 7.0.13 or higher
  • PDF distiller application (GhostScript)
  • JMJDocument version 1.0.5.4
