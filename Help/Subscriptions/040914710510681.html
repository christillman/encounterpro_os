Troubleshooting: e-Prescribing With SureScripts

Triage Protocol: Troubleshooting: e-Prescribing With SureScripts
Overview:
There are several common situations that you may experience while using EncounterPRO to e-Prescribe medications using the SureScripts system. Several problems can result from incomplete prescriptions or other user errors and can be quickly corrected.
Please review this list and attempt to correct the problem before contacting EHRI Product Support. If you cannot correct the problem, be sure to have a patient and treatment example so that the Support Analyst can trace the document in question.
 
Symptoms:
Click on the description of the problem to expand and view a description of probable causes and actions to take
  •   Users cannot register for SureScripts on the User Definition screen
  •   Users get an 'Error ordering document' message when ordering any medication
  •   The prescription did not appear on the Document Manager screen and did not print
  •   The prescription went directly to the printer but the user wanted to send via SureScripts
  •   The prescription went to Document Manager and was sent to the wrong pharmacy - I didnt get a chance to select a different pharmacy or click Send
  •   I want a printed copy of the prescription, but when I print or preview it I see a lot of codes
  •   The prescription ordered appears on the Document Manager screen but SureScripts is not listed on the drop-down list of routes
  •   Cannot print or preview prescriptions from Document Manager
