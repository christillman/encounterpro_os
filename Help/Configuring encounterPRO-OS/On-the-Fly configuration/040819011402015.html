Description
Standard exams are templates used by providers for physical exams and other drilldown observations to quickly fill in results. Generally, a standard exam will document a ?normal? result set. They can also be set up to quickly fill in the expected results for specific conditions, such as Asthma or Otitis Media.

After the standard exam is applied, the provider can make any changes needed to document today?s examination of the specific patient.

Criteria
Standard exams can have specific criteria so that they only appear for use with patients who meet the criteria. You can set the following criteria for a standard exam:

  • Age ? select an age range or stage
  • Gender ? select male or female only
  • Race ? select which race this template applies to
