How to: Set Defaults for Drilldown Observations.

Purpose
Default results allow the user to quickly enter results for a physical or other drilldown screen. If a default result is set for an observation, the user can select the left side of the drilldown observation to quickly populate the default results. The default is usually set to the normal or expected results for the observation. After the default is populated, the user can make any changes needed by drilling down to the entry level.

Before You Begin
Default results are practice-wide. You should be sure to get consensus among the practitioners on what the default results should be before setting the defaults.

Details:

Steps
Follow the steps below:

  1. Go to the desired observation and drill down to the level for which you want to set default results. Common drilldown screens are the Physical Examination, Histories, Lab Results, etc
  2. Select the results that will be the default for the observations on this level. You do not have to set a default for every observation.
  3. Be sure you are displaying the level of observations that you want to set the default for, then select the Set button.
  4. Select the Set Common Results option. (This will establish these results as the default across the practice.) The system asks if you want to set the common results for the default exam.
  5. Select OK.
  6. Repeat steps 1 through 5 to set defaults for any remaining observations.
