How to: Modify a Short List.

Purpose
You can set up short lists to aid in locating items while documenting an encounter. Short lists are available for assessments, drugs, treatments (by type), etc. There are two short lists for each of these categories of items: the Common Short List and Personal Short Lists. The Common Short List is displayed for all users in the practice. Only users with privileges to update common short lists will be able to add or delete items from those lists. Conversely, each user has a Personal List for each item that they alone can view and modify.

Details:

Steps
Follow the steps below:
To Add an Item to the Short List:

  1. Access the screen for the type of item for which you want to modify the short list, for example, the Select Assessment screen or the Select Lab Observation screen. The system defaults to display the short List. If there is a Personal Short List, that will display first, if no Personal Short List exists, the Common Short List appears.
  2. To toggle the display between the Common and Personal list, select the Short List button. The list currently displayed is noted below the search options buttons.
  3. To add items to the short list, use the search options to locate the item you want to add.
  4. Select the icon to the left of the item, a menu of options appears.
  5. If you want to add this item to the Common Short List, select the Common Short List button. If you want to add this item to your Personal Short List, select the Personal Short List button.
  6. The new item will be added to the bottom of the selected short list. Repeat steps 2 - 5 to add additional items to the short list.
  7. When finished, if you want to re-order the list, select the Short List button from the Search Options. This button will toggled between the Personal and Common Short Lists, be sure you have the desired list displayed.
  8. Select the icon to the left of any item on the list. A menu of options appears.
    • If you want to alphabetize the entire list, select the Sort button (ABC)
    • If you want to move individual items, select the Move button and move the selected item up or down in the list
To Remove an Item from the Short List:
You can remove an item from a short list, but leave it in the database to select as needed.

  1. Access the screen for the type of item for which you want to modify the short list, for example, the Select Assessment screen or the Select Lab Observation screen. The system defaults to display the short List. If there is a Personal Short List, that will display first, if no Personal Short List exists, the Common Short List appears.
  2. If you want to remove items from the short list, select the Short List button from the Search Options. This button will toggled between the Personal and Common Short Lists, be sure you have the desired list displayed.
  3. Select the icon to the left of any item on the list. A menu of options appears.
  4. Select the Remove from Short List button (trash can on the left). The item is removed from the list.
  5. Repeat steps 2 - 4 to remove additional items from the short list.
