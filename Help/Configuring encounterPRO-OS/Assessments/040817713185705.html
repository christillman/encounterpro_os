How to: Add a New Assessment

Purpose
Configuring a new assessment will depend on the type of assessment. Allergy assessments are configured differently than the other assessment types. See Screen: How to: Add an Allergy Assessment for detailed instructions.
This solution is for non-allergy assessments.

Before You Begin
Minimal risk and complexity levels are defaulted for all assessments. You may want to set the levels according to your practice's preferences. For details on how encounterPRO calculates E & M levels, refer to Screen: Concept: How encounterPRO Calculates E & M Coding for details.

NOTE: The ICD-9 codes are required if the assessment is to be a billed item.

Details:

Steps
Follow the steps below:

  1. From the Configuration List, select Diagnoses. The Select Assessment to Configure screen appears.
  2. Select the Assessment Type bar then select the desired assessment type from the list. (The default Assessment Type is Sick.)

    NOTE:  You will not be able to select the assessment type on the Assessment Definition screen.
  3. Select the New Assessment button. The New Assessment window appears.
  4. Enter the Description of the assessment and the ICD-9 code, then select OK. The Assessment Definition screen appears.

    NOTE: If you do not enter an ICD-9 code, the billing will not calculate correctly.
  5. Enter the remaining information for the assessment on the Definition tab:
    • Category - If you know the correct Category, select it from the list. Select New Code, Needs Configuration if you do not know all the information required for the new assessment. Categories are used to aid in searching for an assessment
    • Risk Level - Change the risk level if you want to increase the E & M coding level for this assessment
    • Complexity - Change the complexity level if you want to increase the E & M coding level for this assessment
    • Status - select Active to make the assessment available for use
    • Specialty List - Select this button to select the specialties that will use this assessment (always add your own specialty)
    • Patient Reference Materials - Select the bar if you want to associate a patient reference document with this assessment
    • Provider Reference Materials - Select the bar if you want to associate a provider reference document with this assessment
  6. Select the Long Description tab. The description you enter here will appear when you select the assessment on the Select Assessment to Configure screen.
  7. Select the Acuteness/Auto Close tab. Select Acute or Chronic as the type of assessment.
  8. Select the appropriate Auto Close rule for this assessment:
    • None - do not automatically close the assessment. It is the Provider's responsibility to close all Assessments that are set to Auto Close as NONE.
    • End of Encounter - the assessment will close automatically at the end of the encounter. This is useful for well visit assessments.
    • After Interval - select a specific time interval after which the assessment will automatically close. This is useful for acute problems, such as otitis media.
      Note - Closing assessments, whether manually or automatic, is very important for system performance.
  9. When finished entering the information, select OK. The Select Assessment to Configure screen reappears with the new assessment at the top of the list.
  10. If you want to add this assessment to the short list for this assessment type, select the icon to the left of the description. A menu appears. Select the Common or Personal Short list button to add it to that list.
