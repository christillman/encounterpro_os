How to: Set the Default Search String Preference

Purpose
The search screens in encounterPRO allow users to enter search criteria, such as a name or partial name of an item. The system can search for matching items that contain the search criteria or begin with the search criteria. There is a General Preference in encounterPRO that determines whether the system will default to 'Begins with' or 'Contains.'

This preference can be set at the following levels:

  • Global
  • Office
  • Computer
  • Specialty
  • User
Details:

Steps
Follow these steps:  

  1. From the Office View, select the Configuration tab. The Configuration List appears.
  2. Select Preferences. The Preferences screen appears.
  3. Select General Preferences on the left and a list of available preferences is displayed.
  4. Select the level at which you want to apply the preference at the bottom of the screen.
  5. Select Default Search String Mode from the available list. (If the preference name appears in italics, it is not available at the level selected. Change the preference level at the bottom of the screen and re-try.) A preference parameters screen appears.
  6. Select the preferred search mode: Begins with or Contains.
  7. Select Finished. The Preferences screen reappears.

NOTE: Some preferences will be apparent to the user making the change immediately, others require you to restart the EncounterPRO EHR. Other users in the practice will generally need to restart the EncounterPRO EHR.
