How to: Configure the Vaccine Consent Statement

Purpose
When an immunization is ordered, the treatment workplan orders a Vaccine Consent service. This service displays a screen iwith a list of the immunizations that have been ordered in the current encounter and the consent statement that the parent or guardian will review and sign. The wording of the consent statement is configurable in two ways.

Using a Preference

If the consent statement is brief (255 characters or less), then the statement can be entered in the Preferences screen. This Treatment Preference can be set at the following levels:

  • Global
  • Office
  • Specialty
  • User

So you could set up several different consent statements to use in the practice as needed.

Using a Service Attribute

If the statement needs to be longer than 255 characters, a Super User or EHR Product Support can set it up as a service attribute in the Authorize Services function (this allows up to 1024 characters). If a service attribute does not exist, then the Vaccine Consent Statement entered in the preference is used. The consent statement configured using a service attribute is assigned by specialty only. So unless your pratice has multiple specialties, all encounters will see the same consent statement.

Details:

Steps
Follow these steps:
To set a shorter consent statement using the Vaccine Consent Statement preference:

  1. From the Office View, select the Configuration tab. The Configuration List appears.
  2. Select Preferences. The Preferences screen appears.
  3. Select the Treatment Preferences on the left and a list of available preferences is displayed.
  4. Select the level at which you want to apply the preference at the bottom of the screen. This preference is available at the Global, Office, Specialty and User levels. It is not available at the Computer level.
  5. Select the Vaccine Consent Statement preference from the available list. (If the preference name appears in italics, it is not available at the level selected. Change the preference level at the bottom of the screen and re-try.) A preference parameters screen appears.
  6. Enter the consent statement that you want to appear on the Vaccine Signature screen. (Maxiumum of 255 characters.)
  7. Select Finished. The Preferences screen reappears.
  8. Select OK. The Configuration list reappears.

NOTE: Some preferences will be apparent to the user making the change immediately, others require you to restart the EncounterPRO EHR. Other users in the practice will generally need to restart the EncounterPRO EHR.

To set a longer consent statement using the service attribute (this procedure is restricted to Super Users and EHR Product Support):

  1. From the Office View, select the Configuration tab. The Configuration list appears.
  2. Select Authorize Services. The Authorize Services screen appears.
  3. Select Treatment as the Service Type at the top of the screen.
  4. Scroll down and select Vaccine Signature.
  5. Select Configure Service. A list of specialties appears.
  6. Select the specialty that you want this consent statement to apply to or select Default Settings at the bottom of the list to select the current user's specialty. The parameters screen appears.
  7. Enter the consent statement in the text box.
  8. Select Finish. The Authorize Services screen reappears.
  9. If you want to enter a different consent statement for another specialty, repeat steps 5 through 8.
  10. Select Exit to return to the Configuration list.
