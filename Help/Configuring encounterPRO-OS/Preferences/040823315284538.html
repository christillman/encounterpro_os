How to: Set the Computer Secure Preference

Purpose
The Computer Secure preference allows the encounterPRO Administrator or Super User to set the security for a specific computer so that the users can view the Office View without logging in. This can be very useful in a central area such as the nurse's station or lab so that users can see at a glance which patients are in which treatment rooms awaiting which services. In order to perform any function in encounterPRO, the user will be required to log in to the system.

Details:

Steps
Follow these steps:

  1. From the Office View, select Configuration. The Configuration List appears.
  2. Select Preferences. The Preferences screen appears.
  3. Select Security Preferences as the Preference Type on the left.
  4. Select the level at which to set this preference: Global, Office, Computer, Specialty.
  5. Select Computer Secure. The parameters screen appears.
  6. Select Yes then select Finished. The Preferences screen reappears. 
  7. Select OK to return to the Configuration list.

NOTE: You will need to log out of encounterPRO and restart the system in order for this change to take effect. Other users will not see the effect of this change until they also restart encounterPRO.
