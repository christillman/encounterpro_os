Purpose

This solution gives the basic instructions for chaning a preference in EncounterPRO. Only users with specific permissions will be able to change preferences.

Preference Types
There are several Preference Types used to categorize the preferences:

  • Attachment Preferences
  • Billing Preferences
  • Color Preferences
  • Computer Preferences
  • EpIE Preferences
  • For Future Use
  • General Preferences
  • Maintenance Preferences
  • Msg/Task Preferences
  • Reports/Documents Preferences
  • Prescription Preferences
  • Security Preferences
  • Server Configuration Preferences
  • Shell Preferences
  • Subscription Preferences
  • Version 2,4 Only Preferences
  • User Favorites Properties
  • Workflow Configuration

Preference Levels
Each of the preference types can be applied to up to five different levels. The level controls the scope of the configuration change being made.

Details: Steps
Follow these steps:
  1. From the Office View, select the Configuration tab. The Configuration List appears.
  2. Select Preferences. The Preferences screen appears.
  3. Select the Preference Type on the left, a list of available preferences is displayed.
  4. Select the preference level at the bottom of the screen.
  5. Select the preference you want to change from the available list. A preference parameters screen appears.
  6. Make the needed changes and select Finished. The list of preferences reappears.
  7. Select Finished. The Preferences screen reappears.
