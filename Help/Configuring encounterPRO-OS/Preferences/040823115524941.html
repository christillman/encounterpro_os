How to: Set a Default URL for the encounterPRO Logon Screen

Purpose
The encounterPRO logon screen (also known as a 'splash screen') can be modified to display a specific web page. This is set as a preference and can be set at the following levels:

  • Global
  • Office
  • Computer
  • Specialty

The preference cannot be set at the individual user level.

Details:

Steps
Follow these steps:

  1. From the Office View, select the Configuration tab. The Configuration List appears.
  2. Select Preferences. The Preferences screen appears.
  3. Select the General Preferences on the left and a list of available preferences is displayed.
  4. Select the level at which you want to apply the preference at the bottom of the screen. The available levels are:
    • Global
    • Office
    • Computer
    • Specialty
  5. Select the Logon Screen Browser URL preference from the available list. (If the preference name appears in italics, it is not available at the level selected. Change the preference level at the bottom of the screen and re-try.) A preference parameters screen appears.
  6. Enter the URL for the web page you want to appear on the logon screen.
  7. Select Finished. The Preferences screen reappears.
  8. Select OK. The Configuration list reappears.

NOTE: Some preferences will be apparent to the user making the change immediately, others require you to restart the encounterPRO EHR. Other users in the practice will generally need to restart the encounterPRO EHR.
