How to: Restrict Multiple Simultaneous User Logins

Purpose

Your practice may want to restrict the way that users log in to encounterPRO. A Security Preference can be set to allow or restrict individual users from logging into multiple workstations simultaneously. If the preference is set to 'No', then the user will be offered an 'Override' option only if no other sessions for the same user are active.

Details:

Steps
Follow these steps:

  1. From the Office View, select the Configuration tab. The Configuration List appears.
  2. Select Preferences. The Preferences screen appears.
  3. Select the Security Preferences on the left and a list of available preferences is displayed.
  4. Select the level at which you want to apply the preference at the bottom of the screen. This preference can only be set at the Global or Office level.
  5. Select the Allow Multiple Logons preference from the available list. (If the preference name appears in italics, it is not available at the level selected. Change the preference level at the bottom of the screen and re-try.) A preference parameters screen appears.
  6. If you want to allow an individual user to log in to encounterPRO on multiple workstations simulateneously, select Yes.

    If you want to restrict individual users to a single active login, select No.
  7. Select Finished. The Preferences screen reappears.
  8. Select OK. The Configuration list reappears.

NOTE: Some preferences will be apparent to the user making the change immediately, others require you to restart the EncounterPRO EHR. Other users in the practice will generally need to restart the EncounterPRO EHR.
