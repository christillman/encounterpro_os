How to: Set the Sent Message Date Range

Purpose
You can specify how long sent messages should remain on your Sent Messages folder. Messages older than the specified time are archived. They are never deleted from the system.

Before You Begin
You can set this Msg/Task Preference globally (if you have sufficient permissions) which will change the setting for all users in the practice. Be sure to consult with the other providers in the practice before making global preference changes. Alternatively, you can set this preference for the office location, specialty, or an individual user.

Details:

Steps
Follow the steps below:

  1. From the Configuration tab, select the Preferences tab. The Preferences screen appears.
  2. Select Msg/Task Preferences on the left.
  3. You can set this default for the entire practice, the office location, the specialty, or the individual user. Select the level you want to change the preference for at the bottom of the screen:
    • Global - changes the preference for all users
    • Office - changes the preference for all users who log in to the current office location
    • Specialty - changes the preference for all users with the selected Specialty
    • User - changes the preference for the current user only, is effective on all workstations
  4. Select Sent Items Display Date Range.
  5. Enter the desired time frame in the format:
    • # Day
    • # Month
    • # Year
  6. Select Finished. The Preferences screen reappears.
  7. Select OK to return to the Configuration list. You will see the change immediately. Other users will need to log out of encounterPRO in order for the change to take effect for them.
