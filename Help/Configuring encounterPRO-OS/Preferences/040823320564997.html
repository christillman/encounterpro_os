How to: Set the Attachment Folder Preference

Purpose
The practice can set default folders which will be used to store the documents created by the Document Manager. A default folder can be set for each different report context: general, patient, encounter, and treatment. When a document is created (printed or sent electronically) a copy of that document is saved to the default folder defined in the Attachment Preference for that report context.

Details:

Steps
Follow these steps:

  1. From the Office View, select the Configuration tab. The Configuration List appears.
  2. Select Preferences. The Preferences screen appears.
  3. Select the Attachment Preferences on the left and a list of available preferences is displayed.
  4. Select the level at which you want to apply the preference at the bottom of the screen. These preferences are only available at the Global level (the entire practice) and the Office level (the office location).
  5. Select the desired preference from the available list:
    • Attachment Default Folder - Assessment
    • Attachment Default Folder - Encounter
    • Attachment Default Folder - Patient
    • Attachment Default Folder - Treatment

    A preference parameters screen appears.

  6. Select the folder where you want reports for the context selected in step 5 to be saved.
  7. Select Finished. The Preferences screen reappears.
  8. Repeat steps 5 through 7 for each preference for which you want to set the default folder.

NOTE: Some preferences will be apparent to the user making the change immediately, others require you to restart the encounterPRO EHR. Other users in the practice will generally need to restart the encounterPRO EHR.
