How to: Set the Favorite Chart Display Mode

Purpose
The Patient Chart can be defaulted to several different display options:

  • Assessments matched with associated treatments
  • Assessments only
  • Treatments only
  • Assessments on top and treatments on the bottom

Before You Begin
You can set this Preference globally (if you have sufficient permissions) which will change the setting for all users in the practice. Be sure to consult with the other providers in the practice before making global preference changes. Alternatively, you can set this preference for the office location, specialty, or an individual user.

Details:

Steps
Follow the steps below:

  1. From the Configuration tab, select the Preferences tab. The Preferences screen appears.
  2. Select General Preferences on the left.
  3. You can set this default for the entire practice, the office location, the specialty, or the individual user. Select the level you want to change the preference for at the bottom of the screen:
    • Global -- changes the preference for all users
    • Office -- changes the preference for all users who log in to the current office location
    • Specialty -- changes the preference for all users with the selected Specialty
    • User -- changes the preference for the current user only, is effective on all workstations
  4. Select Chart Display Mode.
  5. Enter one of the display options:
    • ATAT -- displays assessments with associated treatments
    • AATT -- displays assessments at the top and treatments at the bottom
    • AA -- displays only assessments
    • TT -- displays only treatments
  6. Select Finished. The Preferences screen reappears.
  7. Select OK to return to the Configuration list. You will see the change immediately. Other users will need to log out of encounterPRO in order for the change to take effect for them.
