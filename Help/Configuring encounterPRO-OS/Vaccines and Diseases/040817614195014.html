How to: Add a New Vaccine

Purpose
From time to time new vaccines will come to market and must be configured in encounterPRO so that they can be ordered and billed for properly.

Before You Begin
Configuring a new vaccine is a multi-stage process. You must:

  1. Add the vaccine. The vaccine must be configured using the Vaccines & Diseases button under the Configuration tab.
    NOTE: After the vaccine has been saved, the name of the vaccine cannot be changed in the ePRO interface. Contact EHRI Product Support if you need to have a vaccine name changed.
     
  2. Create a Vaccine Procedure.
  3. Link the vaccine to the vaccine procedure. You will link the new vaccine procedure to the vaccine in order for the billing to be properly calculated.
  4. Link the vaccine to the diseases it immunizes against.
  5. Select the Vaccine Makers for the new vaccine. When administering a vaccine, the nurse will select the vaccine maker and enter the lot number, for tracking purposes. You need to link the vaccine makers to the new vaccine so that they will be available on the Immunization screen for the nurse.
  6. Enter the NDC (National Drug Code) code for this vaccine. If the NDC code is not listed for a specific Vaccine on the Definition screen, the clinical staff person can enter it on the Vaccine Administration screen for each patient.

NOTE: If the vaccine prevents a disease that is not currently listed on the ProVacs, you will need to contact Support and request that the vaccine and disease be added to the immunization schedule.

Details:

Steps
Follow the steps below:
To Add the Vaccine:

  1. From the Configuration tab select the Vaccines and Diseases tab. The Vaccines and Diseases screen appears.
  2. Select the New Vaccine button.
  3. Enter the name of the new vaccine. The Vaccine Definition screen appears.
  4. If desired, you can link a patient or provider reference material to this vaccine. Select the appropriate N/A button and select the document you want to use as a reference material.
  5. Proceed to the next stage, creating the procedure for the vaccine.

To Add the New Vaccine Procedure:
Note: Before you add a vaccine procedure to the database, you should perform a search of the database to confirm that it does not already exist.

  1. From the Vaccine Definition screen, select the Procedures tab.
  2. Select Add Procedure. The Select Procedure screen appears with the short list of vaccine procedures.
  3. If the vaccine procedure already exists, select it from the list and go to Step 11.
  4. If, after searching, you confirm the vaccine procedure does not exist, select the New Procedure button. The Vaccine Procedure screen appears.
  5. Enter a description for the new procedure (required).
  6. Enter the CPT Code, Units, and CPT Modifier for the new procedure.
  7. Select the Category bar then select Vaccines.
  8. If you want encounterPRO to calculate E & M coding for this procedure, select a Risk Level for the new procedure. If this is a billable procedure, make sure the Bill Flag is set to Bill or Bill 1st.
  9. Enter the Charge and Billing Code for the new procedure, if applicable.
  10. When finished entering information for the new procedure, select OK. The new procedure is added to the master procedure list in the database.
  11. Search for the newly added procedure, select it. The Vaccine Definition screen reappears with the selected procedures.
  12. Proceed to the next stage, linking the vaccine to the diseases it prevents.

To Link a Vaccine to a Disease:

  1. From the Vaccines Definition screen, select the Diseases tab.
  2. Select the Add Disease button. A list of diseases appears.
  3. Select the appropriate disease(s) from the list then select OK. The diseases selected appear on the Diseases tab.
  4. Proceed to the next stage, adding the vaccine maker.

To Add Vaccine Makers to a Vaccine:

  1. From the Vaccines Definition screen, select the Makers tab.
  2. Select the Add Maker button. The Makers list appears.
  3. Select all appropriate vaccine makers for the current vaccine then select OK.
  4. The Vaccines Definition screen reappears.
  5. Proceed to the next stage, entering the NDC code.

To Add the NDC (National Drug Code) Code for the Vaccine:

  1. From the Vaccines Definition screen, select the Drug Codes tab.
  2. Enter the NDC Code.
  3. Select OK to save the changes to the new vaccine.

NOTE: To Configure the Vaccine Schedule: there is no interface, the practice must ask EHR Product Support to add the new vaccine to ProVacs. The new vaccine is automatically added to your specialty. If you need to have the vaccine accessible by multiple specialties in your practice, go the the Specialties tab and add the needed specialties to the list.

Troubleshooting
Setting up a new vaccine is a complex process. Listed below are some of the common problems and solutions

The nurse is trying to administer the vaccine and the maker needed is not in the list.
Go back to the Vaccine Definition screen and be sure that all the vaccine makers needed are listed on the Makers tab.

The vaccine billing code is not showing up on the Billing Review screen.
Check to make sure the vaccine is associated with an appropriate, billable assessment. If it is, then go to the Vaccine Definition screen and check the Procedures tab. There must be a billable procedure associated with the vaccine in order to bill for it.
