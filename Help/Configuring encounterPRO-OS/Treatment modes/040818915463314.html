Description
To understand what a treatment mode is, we must first understand what a treatment workplan is.  A treatment workplan is a defined set of workplan steps that control the behavior of a treatment. 
A treatment mode attaches a treatment workplan to a specific treatment to make it behave according to the treatment workplan steps.

Why Do We Use Treatment Modes?
In EncounterPRO, each treatment type is associated with a treatment type default workplan.  This treatment type default workplan controls the behavior of all treatments within a treatment type, unless a treatment mode dictates otherwise for a specific treatment.
Treatment modes are used whenever there is a need to deviate from the treatment type default workplan. For example:

  • The practice may need certain treatments to behave differently from other treatments within a treatment type. The urinalysis and rapid strep lab tests may be performed in-house, while all other lab specimens are sent out.
  • An office location may need certain treatments to behave differently than they do in other office locations. One office location may have the equipment to perform x-rays on site, while the other locations need to send the patients out to a radiology consultant.
  • An individual may need certain treatments to behave differently than they do for others in the office or in the practice.

Setting the Default Treatment Mode
When you set a default treatment mode, the treatment behaves the same way for all users. For example, if your practice sends all patients out for lab work, you should set the default treatment mode for all labs and tests to Ongoing Send or Refer Out. Default treatment modes are set on the Treatment Type Configuration screen.

Selecting a Treatment Mode On the Fly
While in an encounter-context the treatment mode can be changed for a specific treatment or for a specific instance of a treatment.

If there are a few labs that are performed in the office, you should set the default treatment mode for all labs to something like Ongoing Send or Refer out. Then, from an encounter-context (probably a test patient), select those labs that are performed in-house. After you select the labs, you can change the treatment mode for that specific treatment -- setting either a default for that particular treatment or just for that specific instance of the treatment. See How to: Set the Default Treatment Mode for a Specific Treatment for detailed instructions on selecting treatment modes.

Details:

Available Treatment Modes
Treatment modes with "Enc" (encounter) included in the description are used for labs and tests which will be closed at the end of the encounter. Treatment modes with "Ong" (ongoing) included in the description are used for labs and tests which will remain open after the encounter, as they will appear on the Outstanding Labs and Tests screen or Outstanding Other Treatments screen and must be manually closed.

A treatment mode must be configured as one of the options for a specific treatment type in order to be available for use with treatments of that type. The table below lists some of the common treatment types and the treatments modes that are available for that type.

Labs

  • Enc Coll and Perform -- used for in house labs that will be both collected and performed during the encounter.
  • Enc Coll Perform Rev -- used for in house labs that will be both collected and performed during the encounter. A Lab Review service bar appears to allow the provider to quickly review all labs ordered during the encounter.
  • Enc STAT Coll Perf Rev -- used for in house labs that will be both collected and performed during the encounter. A Lab Review service bar appears to allow the provider to quickly review all labs ordered during the encounter.
  • Ong Coll Perform Later -- used for in-house labs that will be collected during the encounter, but the lab remains open to allow results to be entered later.
  • Ong Send or Refer Out -- used for treatments that will be sent to an outside lab or referred outside the office. This lab will remain open after the end of the encounter.
  • Ong Send or Refer Int (for practices with Lab Integration) used for treatments that will be sent to an outside lab or referred outside the office. This lab will remain open after the end of the encounter.

Tests

  • Enc Dashboard Now -- used for tests for which you want a dashboard screen to appear immediately after ordering the test.
  • Enc Drill Down -- used for tests for which you want a service bar to be created when the test is ordered. The results screen will appear in the drill down format.
  • Enc Drilldown by Prov -- used for tests for which a drill down screen is ordered for the provider after ordering the test.
  • Enc Timed Grid Now - used for tests for which a timed grid screen appears immediately after ordering the test.
  • Enc Vitals Format - used for tests for which a service bar appeasr when the test is ordered. The results screen will appear in the vitals format.
  • Ong Dashboard (Consent) - used for tests for which a dashboard screen with a patient consent form appears immediately after ordering the test. This test will remain open after the end of the encounter.
  • Ong Drilldown - used for tests for which a drilldown screen appears immediately after ordering the test. This test will remain open after the end of the encounter.
  • PPD -- used for the PPD skin test.

In-house Drugs

  • Administer Now -- used for in house drugs for which the administration screen appears immediately after ordering the drug.
  • Dilating Drops -- used for dilating drops.
  • With Reaction Check-- used for in house drugs for which a reaction check is performed after a specified time period.

Prescription Medications

  • Medication (Consent) -- used for medications ordered for which you want a prescription to print.
  • Medication No Print -- used when a prescription should not print.
  • Medication Print Last -- used for prescription medications ordered for which the prescription should be printed as the last prescription (when multiple prescriptions will be printed).
  • Medication Print Now -- used for prescription medications ordered for which a prescription should print immediately.
  • OTC No Print -- used for over the counter medications ordered, but a prescription should not print.
  • OTC Print Last -- used for over the counter medications ordered for which a prescription should be printed as the last prescription (when multiple prescriptions will be printed).

Procedures

  • Close Now -- used for procedures that should close automatically as soon as they are ordered.
  • Enc Dashboard -- used for procedures for which a service bar is created when ordered. The results screen appears as a dashboard screen.
  • Enc Procedure Now -- used for procedures for which a dashboard screen appears immediately after ordering the procedure.
  • Nurse Enc Procedure -- used for procedures performed by the nurse during the encounter.
  • Ong Dashboard -- used for procedures for which a service bar is created when ordered. This procedure will remain open after the end of the encounter.
  • Ong Refer Out -- used for procedures for which the patient will be referred to an outside consultant. This test will remain open after the end of the encounter.
