Purpose
The Workflow Status screen allows a Super User to troubleshoot workplans, turn on tracing and config mode, and review the service attributes.

Details:

Steps
Follow the steps below:

  1. From most encounter screens, press F3. The Workflow Status screen appears.
  2. When finished, select OK. The original screen reappears.

NOTE: If the Workflow Status screen does not appear, try to 'get focus' on the screen by selecting an item or tabbing.
