How to: Configure One-Touch Progress Notes

Purpose
Pre-programmed or one-touch progress notes can be created for use on the Assessment Dashboard to quickly document patient progress. EncounterPRO is delivered with several one-touch progress notes. If these notes do not meet the needs of your practice additional one-touch progress notes can be created using this procedure.

Before You Begin
You need the text of the progress notes that you want to add. Also, you will need to Turn on Configuration Mode.

Details:

Steps
Follow the steps below:

  1. With Configuration Mode turned on, access the assessment dashboard (select an existing assessment on the Patient Chart). (See How to: Turn On Configuration Mode for detailed instructions.)
  2. Select the link at the top of the dashboard. A menu of one-touch progress notes appears.
  3. Select the small text menu name. The Menu Properties screen appears.
  4. If a local copy of the menu does not already exist, select Create Local Copy.
  5. Select Edit Menu.
  6. Select the New button. The Menu Item Properties screen appears.
  7. Select Service, then select Assessment. A list of assessment services appears.
  8. Select New Note. The parameters screen appears.
  9. Select the Context Object bar.
  10. Select Assessment from the list, and then, select Next.
  11. Select the Standard Note Type bar then select Progress Notes and then, select Next.
  12. The Special Note Type defaults to Progress Notes, select Next.
  13. The Preconfigured Treatment Titles or Keys parameter appears. Leave this setting blank then select Next.
  14. Type the title for the progress note in the Other Note Title or Key field, typically this is "Progress Note." Select Next.
  15. Type the text of your progress note in the Note Text box. This is the text that appears in the Patient Chart.
  16. Select Finish. The Menu Item Properties screen reappears.
  17. Change the text of the Button Title. This is the text that will appear on the DX Note Menu.
  18. Select Finished. The Menu Properties screen appears with the new button at the bottom of the list.
  19. Change the order of the buttons, if desired.
  20. Select OK then Finished. The DX Note Menu reappears.
  21. Select Cancel to close the menu. Re-select the link and the menu appears with the new button.

NOTE:  Other encounterPRO users will not see the new progress note menu button until they close and re-start encounterPRO.
