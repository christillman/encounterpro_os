Requirements for Deleting Specialties from the Database

Solution Title: Requirements for Deleting Specialties from the Database
Overview: per Kathleen, specialties can be removed / deleted from the database only if they do not exist in the following tables:
  • u_assessment_treat_def_attrib.value
  • c_user.specialty_id
  • p_treatment_item.specialty_id
  • u_assessment_treat_definition.user_id
  • u_top_20.user_id
