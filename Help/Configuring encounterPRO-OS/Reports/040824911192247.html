How to: Determine which signature is printed on a prescription

Solution:

EncounterPRO first determines the effective ordered-by provider using the following logic:

1) The last provider to order a refill will be the ordered_by provider. 
2) If no refills have been ordered then the last provider to sign for the prescription will be the ordered_by provider.
3) If no one has signed for the prescription then the provider who originally ordered the prescription will be the ordered_by provider.

Once the ordered-by provider has been determined, then the following logic is used to determine which signature to display:

The ordered-by provider's signature is used if any of the following are true
1) the ordered-by provider does not have a supervisor and is certified
2) the ordered-by provider has a supervisor and the use_supervisor_signature_stamp is false

If the ordered-by provider has a supervisor then the supervisor's signature is used if either of the following are true:
1) the ordered-by provider is not certified
2) the ordered-by provider is certified and the use_supervisor_signature_stamp is true

Finally, if an actual signature is available from the prescription, then the actual signature is used.  Otherwise, the appropriate signature stamp is used.  Note that whether or not a signature stamp is available or used is subject to two preferences, namely rx_use_signature_stamp and rx_always_use_signature_stamp.

Other attributes controlling signature printing:

  • print_signature_for_controlled_substance

    • 'N' = Do not print the signature if the drug's controlled_substance_flag = 'Y'.

    • 'Y' = Print signature regardless of controlled_substance_flag

    • Default is 'N'

  • generic_signature
    • If this prescription has two signature lines at the bottom, then the generic_signature attribute controls which side to print the signature on.
    • 'L' = Print the signature on the left side for generic-allowed and the right side for brand-name-required
    • 'R' = Print the signature on the right side for generic-allowed and the left side for brand-name-required.
    • Default is 'L'

  • use_supervisor_signature_stamp

    • Controls whether to use signature stamp of the physician's extender who wrote the prescription or their supervisor.

    • 'Y' = Use the supervisor's signature stamp

    • 'N' = Use the extender's signature stamp
