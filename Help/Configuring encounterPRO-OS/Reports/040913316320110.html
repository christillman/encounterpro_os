How to: Change Refill Buttons and Links on RX Treatment Dashboard to Use Doc Mgr

Solution:

In most databases, the configuration for the Prescription Treatment Dashboard is not set up to use the Document Manager for refills. If the practice is using e-Prescribing, you will need to update the links at the top of the screen and the buttons at the bottom of the screen to send the refills to Document Manager.

Suggested links at the top of the dashboard:

The items in blue above will both order a NewRX document and send it to Document Manager. The Print Now link has the flag to Print Now to the default printer.

Suggested buttons at the bottom (Services on the Treatment type):

[Refill Print Now] [Refill w/DocMgr]

These are set up to use Document Manager and to add a progress note that a refill was ordered and either Printed or sent to Doc Mgr the Print Now buttons use Doc Mgr, but are set to Print Now and should print at the default printer.

This is assuming that the default treatment mode for Prescriptions is the E-RX treatment mode.



Details:

Configuring the Refill Links on the Treatment Dashboard

For the  link:

  1. Turn on Configuration Mode.
  2. Display the Treatment Dashboard. Right-click in the display area to display the Display Script.
  3. Add a new line - General > Print. Enter the text for the link, for example .
  4. Select Next.
  5. Add a Standard Quick View menu: Standard Quick View Refill Prescription Drug and Print Now to Default Printer. This menu has the following configuration:
  • One button: Order Workplan - Standard Refill Medication and Print Now OoO

Save the changes and return to the Display script.

For the  link:

  1. Turn on Configuration Mode.
  2. Display the Treatment Dashboard. Right-click in the display area to display the Display Script.
  3. Add a new line - General > Print. Enter the text for the link, for example .
  4. Select Next.
  5. Add a Standard Quick View menu: Standard Quick View Refill Prescription Drug. This menu has the following configuration:
  • One button: Order Workplan - Standard Refill Medication and Send to Document Manager

Save the changes and return to the Display script.

The Workplan for the Refill w/Doc button:

This workplan has two steps:

Step 1: Refill Medication Progress Note

  • Orders a Service - New Note
  • Context - Treatment
  • Standard Note Type = Refill
  • Note Text - Enter your preferred note text
  • Flags - In Office, Counts Toward Completion = Yes, Perform Immediately = Yes

Step 2: EPRO Report - Drug Prescription

  • Orders a Service - EPRO Report
  • Document Creation = Order Document
  • Select Report - Drug Prescription
  • Print Now = No
  • Context = Treatment
  • Document Purpose = NewRX
  • Flags - Out of Office, Counts Toward Completion = Yes

For the buttons on the bottom of the dashboard:

  1. Go to the Configuration Tab, select Treatment Types.
  2. Locate Prescription Medication and select it.
  3. Select the Services tab.
  4. Add a new service: Refill Print Now (Yes, Yes, No) that orders the workplan: Standard Refill Medication and Print Now OoO
  5. Add another new service: Refill w/DocMgr (Yes, Yes, No) that orders the workplan: Standard Refill Medication and Send to Document Manager
