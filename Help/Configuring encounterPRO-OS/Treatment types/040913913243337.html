How to: Change the Default Treatment Mode for a Treatment Type

Solution: There are times when you will need to change the default treatment mode for all treatments of a specific treatment type. For example, when activating e-Prescribing and Document Manager, Product Support will change the default treatment mode for the Prescription Drug treatment type to e-RX.
Details:
  1. Select the Configuration tab. The Configuration list appears.
  2. Select Treatment Types. The Treatment Type screen appears.
  3. Scroll down in the list to locate the desired treatment type. Select the bar for that treatment type. The Treatment Type definition screen appears.
  4. Select the Default Treatment Mode tab. The current default appears. If the practice has more than one location, you can set a different default treatment mode for each location.
  5. To change the current default, select the bar for the default treatment mode. A list of available treatment modes for this treatment type appears.
  6. Select the desired treatment mode. The Default Treatment Mode tab reappears with the new default.

NOTE: Only treatments of this treatment type that are ordered AFTER the change will have the new default treatment mode. Any specific treatments that had a different default manually set will retain that setting. This can mean that if you are activating e-prescribing at a practice and someone changed the default for several individual medications to something other than the treatment type default treatment mode that you will need to manually reset the default for those medications to e-RX. If a large number of changes were made , you may want to do this in the tables.
