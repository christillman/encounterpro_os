Description

There is a special class of users in encounterPRO that are referred to as 'Super Users.' These users have been granted permissions to access all functions in the encounterPRO system. EHR recommends that each practice have at least one person certified and designated as a Super User.

Super Users are called upon to perform certain maintenance and configuration functions in the encounterPRO system.

NOTE: Any user with 'Super User' designation on the User Configuration/Permissions tab will have access to ALL encounterPRO functions. You cannot restrict a Super User from individual functions using the Authorize Services feature, as you can with all other encounterPRO users.

Super Users at the Practice

It is recommended that each practice have at least one certified Super User on staff. If the practice has multiple locations, each location should have its own Super User.

Functions Typically Performed by Super Users

  • Maintain encounterPRO users
  • Understand and implement security on the encounterPRO system
  • Set preferences to tailor the system to the practice's methodology and liking
  • Maintain the prescription drug lists
  • Configure assessments
  • Configure treatments
  • Create and modify history observations (PFSH, HPI, etc.)
  • Create and modify physical examinations
  • Work with EHR Product Support to troubleshoot and resolve issues
