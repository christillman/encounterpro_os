How to: Add Patient Education Materials

Purpose
Patient materials can be used to store handouts that are scanned or imported. They can be included as part of a treatment plan and ordered from Treatment lists. Additionally, you can select a specific webpage as a patient material, linking to it via the unique URL.

Before You Begin
Determine the location on the workstation or network of the file containing the desired materials.

Details:

Steps
Follow the steps below:
To Add Patient Materials from a File:

  1. From the Configuration list, select Patient Materials. The Patient Education Material Configuration screen appears.
  2. Select New Material from File. The Select New Patient Material File screen appears.
  3. Locate the file you want to import, select it and select Open. If you want to import multiple files in the same directory, hold down the Ctrl key and select each file, then select Open. The Selected Files screen appears with the files you selected.
  4. The title of each file defaults to the filename. If you want to change the title, select the file to rename and enter the desired name in the Title field.
  5. To assign the file to a category, select the file from the list on the left, then select the Category bar. A list of materials categories appears. Select the category for the highlighted (darkened) file.
  6. Repeat steps 4 and 5 for each file you imported, then select Finished. The Configuration List reappears.
  7. Repeat steps 2 - 6 to import additional materials.

To Add Patient Materials from a Webpage:

  1. From the Configuration list, select Patient Materials. The Patient Education Material Configuration screen appears.
  2. Select New Material from URL.
  3. Enter the URL for the webpage. You should use a cut & paste technique to be sure that the web address is correct.
  4. Select the Test button to make sure that the URL is correct. A browser will open and attempt to navigate to the URL entered.
  5. Enter the desired name for the patient material in the Material Title field.
  6. To assign the file to a category, select the Category bar. A list of materials categories appears. Select the category for the highlighted (darkened) file.
  7. Repeat steps 2 - 6 to create additional materials. When finished, select OK to return to the Configuration List.
