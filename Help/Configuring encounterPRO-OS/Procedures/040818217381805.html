Purpose
On occasion, you may need to delete a procedure. For example, if a procedure was added and later found to already exist in the database, you would delete one of the two procedures. If you delete a procedure, it is removed from the database. You cannot re-activate it at a later date.

Before You Begin
Procedures are organized in the database by procedure type:

  • Primary Encounter
  • Secondary Encounter
  • Procedures
  • Drug/Item Administration
  • HCPCS
  • Perform Procedures
  • Collect Procedures
  • Vaccines
  • Vaccine Administration
  • Extra Charge
Details:

Steps
Follow the steps below:

  1. From the Configuration tab, select Procedures. The Procedure Configuration screen appears.
  2. Search for the procedure you wish to delete. You can select as the procedure type to be sure that the system searches the entire procedures database. The matching procedures are displayed.
  3. Select the procedure you want to delete. A menu of options appears.
  4. Select the Delete button. At the prompt, confirm deletion by choosing OK.
  5. Select OK to return to the Configuration list.
