How to: Modify Billing for a Lab or Test

Purpose
If a lab or test is not billing with the appropriate CPT code or the number of billed items is incorrect, you can correct that by modifying the configuration of the observations that make up that lab.

There are two places to check when you are having billing problems. The collect procedure and the perform procedure. Collect procedures are items such as venipuncture, specimen handling, etc. Perform procedures use the CPT code for the specific lab or test. You can access both of these from the treatment observation.

Before You Begin
This will change the billing for the selected procedure(s) and will affect the billing for ALL treatments which use the procedure(s).

Details:

Steps
Follow the steps below:
To Modify the Observation from a Patient Context:

  1. Check a test patient into the office. See How to: Manually Check in a Patient for detailed instructions.
  2. Go to the Patient Chart and select Other Items. A menu of additional options appears.
  3. Select Any Treatment. A menu of treatment types appears.
  4. Select the treatment type for this item. For example, Lab, Test, or X-ray. The selection screen for the treatment type you chose appears.
  5. Select the lab or treatment that needs to have the billing modified from the shortlist, if the treatment is not listed there use your search options to locate the treatment to be modified. You can select multiple treatments of this type if needed. Each item selected appears in the Selected Items list.
  6. Select OK. The Patient Chart reappears.
  7. Locate the item ordered on the Patient Chart and select the small icon to the left of the description. The entry screen appears. If you select the description, the dashboard screen appears. Select the description of the item on the dashboard and the entry screen appears.
  8. Select Other Options from the bottom of the screen. A menu of options appears.
  9. Select Configure. The Observation Definition or the Composite Observation Definition screen appears, depending on the treatment you chose.
  10. If the billing problem concerns the specimen collection billing, select the Collection Procedure button on the right. If the billing problem concerns the perform procedure, go to step 14.
  11. If the collection procedure is already listed, but is still billing incorrectly, select Edit Procedure. The Collect Procedures configuration screen appears. If there is no collection procedure listed, select Change Procedure. The Select Collect Procedures screen appears.

    1. Select the correct collection procedure from the list. You may need to use the search options to locate the desired procedure. The Observation Definition screen reappears with the collection procedure displayed.
    2. Select the collection procedure, then select Edit Procedure from the menu. The Collect Procedures configuration screen appears.
  12. Review the billing information for the collect procedure. If needed you can change the CPT code, the units, the CPT Modifier and the Billing Rule. The available Billing Rules are as follows:
    • Bill - create a billing entry for every time this procedure is ordered during the encounter
    • Don't Bill - never bill for this collect procedure
    • Bill 1st charge only - used when you only want to bill this procedure code one time during an encounter, no matter how many times it is used. For example, venipuncture codes are generally billed only once no matter how many specimens are collected
    • Bill if 2 or more - Not used
    • Accumulate Charges - used to configure Allergy Test procedures.
  13. If you do not need to modify the perform procedure, select OK. The entry screen reappears.
  14. If the perform procedure is already listed, but is still billing incorrectly, select Edit Procedure. The Perform Procedures configuration screen appears. If there is no perform procedure listed, select Change Procedure. The Select Collect Procedures screen appears.

    1. Select the correct perform procedure from the list. You may need to use the search options to locate the desired procedure. The Observation Definition screen reappears with the perform procedure displayed.
    2. Select the perform procedure, then select Edit Procedure from the menu. The Perform Procedures configuration screen appears.
  15. Review the billing information for the perform procedure. If needed you can change the CPT code, the units, the CPT Modifier and the Billing Rule. The available Billing Rules are as follows:
    • Bill - create a billing entry for every time this procedure is ordered during the encounter
    • Don't Bill - never bill for this collect procedure
    • Bill 1st charge only - used when you only want to bill this procedure code one time during an encounter, no matter how many times it is used. For example, venipuncture codes are generally billed only once no matter how many specimens are collected
    • Bill if 2 or more - Not used
    • Accumulate Charges - used to configure Allergy Test procedures.
  16. Select OK. The entry screen reappears.
  17. Perform the Collect and Enter Results steps for this treatment.
  18. Check the billing review screen to determine if the changes have corrected the billing problem.
