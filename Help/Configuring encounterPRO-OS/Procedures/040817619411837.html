How to: Add a New Procedure

Purpose
While there are several types of procedures available in EncounterPRO, we recommend that you only add the following types:
  • Perform Procedures
  • Collect Procedures

These will cover most of your labs, tests, and radiology procedures. Contact EHR Product Support if you need to add or update the other procedure types.

Before You Begin
Before adding the procedure, do a thorough search to make sure it is not already in the database.
Each Perform Procedure should have a corresponding Collect Procedure. After you have added the perform and collect procedures, you must link them together on the Observation screen. See How to: Modify Billing for a Lab or Test for detailed instruction.

Details: Steps
To Add a Collect Procedure:
  1. From the Configuration tab, select Procedures. The Procedure Configuration screen appears.
  2. Select the Procedure Type bar and select Collect Procedures. The short list for Collect Procedures appears.
  3. Select the New Procedure button. The Collect Procedures Procedure screen appears. 
  4. Confirm that this is the correct procedure type for the new procedure. If not, select the Cancel button to return to the Procedure Configuration screen and repeat step 2.
  5. Enter a description for the new procedure (required).
  6. Enter the CPT Code, Units, and CPT Modifier for the new procedure, if needed. Generally the CPT code is attached to the perform procedure, so that the item is billed only when performed, not collected.
  7. Select the Category for the procedure: Handling/Collection Codes. 
  8. If you want EncounterPRO to calculate E & M coding for this procedure, select a Risk Level for the new procedure. (Generally, this is left as N/A.)
  9. If this is a billable procedure, make sure the Bill Flag is set to Bill or Bill 1st. For example, a venipuncture collect procedure is generally Bill 1st, since you would only bill for one venipuncture regardless of how many labs you are collecting for during that visit.
  10. Enter the Charge and Billing Code for the new procedure, if applicable.
  11. Auto-billed Assessment
  12. Well Encounter Affinity
  13. When finished entering information for the new collect procedure, select OK. The new procedure is added to the master procedure list.
To Add a Perform Procedure:
  1. From the Procedure Configuration tab, Select the Procedure Type bar and select Perform Procedures. The short list for Perform Procedures appears.
  2. Select the New Procedure button. The Perform Procedures Procedure screen appears.
  3. Confirm that this is the correct procedure type for the new procedure. If not, select the Cancel button to return to the Procedure Configuration screen and repeat step 2.
  4. Enter a description for the new procedure (required).
  5. Enter the CPT Code, Units, and CPT Modifier for the new procedure, if needed. Generally the CPT code is attached to the perform procedure, so that the item is billed only when performed, not collected.
  6. Select a Category for the procedure.
  7. If you want EncounterPRO to calculate E & M coding for this procedure, select a Risk Level for the new procedure. (Generally, this is left as N/A.)
  8. If this is a billable procedure, make sure the Bill Flag is set appropriately. Choices are:
    • Bill: add a charge to the encounter billing for each time this procedure is performed
    • Don't Bill: do not bill for this procedure (used for send-out labs and tests)
    • Bill 1st Charge Only: only charge for the first occurrence of this procedure in an encounter
    • Bill if 2 or more: only charge if there are two or more occurrences of this procedure in an encounter
    • Accumulate Charges: count the number of times this procedure code is performed and send a single line item to the billing system with the units accumulating for each occurrence.

    For example, a lab perform procedure is generally Bill since you want to charge for each time that lab is performed.

  9. Enter the Charge and Billing Code for the new procedure, if applicable.
  10. Auto-billed Assessment
  11. Well Encounter Affinity
  12. When finished entering information for the new perform procedure, select OK. The new procedure is added to the master procedure list.
