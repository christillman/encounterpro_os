How to: Export a Report

Purpose

EncounterPRO version 5 allows user to share their custom reports with others in the encounterPRO community. Reports can be exported to the Reports Library or saved as a special file (.jmjcfg) that can be imported to create a new report at another practice.

Shareware Reports

EncounterPRO users who have developed their own reports can choose to share those reports with the encounterPRO community by posting their exported report files in an accessible location. It is preferred that the reports are shared via the Reports Library.

CAUTION: Shareware reports are not warranted or maintained by EHR Product Support. If there is a defect in a shareware report, there will be a fee charged if you wish EHR Product Support to troubleshoot or repair the report.

Details:

Steps
Follow these steps:
To export a report to the Reports Library:

  1. From the Office View, select the Configuration tab. The Configuration list appears.
  2. Select Reports. The Select Report screen appears.
  3. Select the context of the report you want to export.
  4. Search for the report using the Search Options.
  5. Select the small icon to the left of the report. A menu of options appears.
  6. Select Export.
  7. If the report is checked out, you will be prompted to check the file in before exporting.
  8. Select Upload to Library. The system will display a message noting that the export function was successful or not.
  9. Repeat steps 3 through 8 to export additional reports to the Reports Library.
  10. When finished, select Cancel to return to the Configuration list.


Follow these steps:
To export a report to a file:

  1. From the Office View, select the Configuration tab. The Configuration list appears.
  2. Select Reports. The Select Report screen appears.
  3. Select the context of the report you want to export.
  4. Search for the report using the Search Options.
  5. Select the small icon to the left of the report. A menu of options appears.
  6. Select Export. If the report is checked out, you will be prompted to check the file in before exporting.
  7. Select Save to File. The Select File screen appears.
  8. Navigate to the location where you want to save the file and enter the filename.
  9. Select Save. The system will display a message noting that the export function was successful or not.
  10. Repeat steps 3 through 10 to export additional reports to files.
  11. When finished, select Cancel to return to the Configuration list.
