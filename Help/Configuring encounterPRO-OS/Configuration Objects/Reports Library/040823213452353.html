How to: Import a Report from the Reports Library

Purpose

EncounterPRO version 5 offers a new method of adding reports to your system. The Reports Library is an ever-expanding repository of reports developed by encounterPRO users as well as EHR, Inc. This list of reports will be refreshed every night during the nightly sync process.

Shareware Reports

EncounterPRO users who have developed their own reports can choose to share those reports with the encounterPRO community by uploading the reports to the Reports Library. Those reports will be labeled 'Shareware' in the Reports Library. See How to: Export a Report.

CAUTION: Shareware reports are not warranted or maintained by EHR Product Support. If there is a defect in a shareware report, there will be a fee charged if you wish EHR Product Support to troubleshoot or repair the report.

Details:

Steps
Follow these steps:
To import reports from the Configuration tab:

  1. From the Office View, select the Configuration tab. The Configuration list appears.
  2. Select Reports. The Select Report screen appears.
  3. Select the context of the report you want to import:
    • General
    • Patient
    • Encounter
    • Assessment
    • Treatment
    • Observation
    • Attachment
  4. Select Library under the Search Options. A keyboard screen appears.
  5. Enter a report name or partial name. If you want to view all reports in the Reports Library for the context chosen in step 3, leave the field blank.
  6. Select OK. A list of matching reports appears.
    NOTE: If the report has been created by an encounterPRO user rather than EHR, Inc. the report name bar will display 'Shareware.' These reports are not warranted or maintained by EHR Product Support.
  7. Select the report you want to import from the list. A menu of options appears.
  8. If you want to learn more about the report, select Properties. The Properties screen appears.
  9. If you want to import the report, select Import. The system will display a message when the import function is complete to confirm that the report was imported.
  10. If you want to add the new report to the Common short list or your Personal short list you will need to search for the report using the Description or Category Search Options, then select the small icon to the left of the report name and select either Common or Personal Short List button.
  11. Repeat steps 3 through 10 to import additional reports.
  12. When finished, select Cancel to return to the Configuration list.
