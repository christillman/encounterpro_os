Solution Path: Configuring encounterPRO-OS > Configuration Objects > Reports Library

How to: Default the SOAP screen to New Only

Solution: c_chart_page_attribute 'new_data', defaults to 'False'. Set to 'True' or 'Yes' to have it clicked by default.
  • In version 4, the page_class is u_soap_page_assessments
  • In version 5, the page_class is u_soap_page_problem_list
