How to: Add Asterisks to Dispense Amounts on Prescriptions

Purpose
Certain states require the addition of asterisk characters (**) before and after the dispense amount on a presription as an added security measure. This is controled by a Prescription Preference that is set globally for the entire practice.


Before You Begin
You must set this Preference globally (if you have sufficient permissions) which will change the setting for all users in the practice. Be sure to consult with the other providers in the practice before making global preference changes. Preferences that affect the prescription formatting will not take effect until encounterPRO is restarted. These preferences will only affect new or modified prescriptions -- current prescriptions (including normal refills) will use the prescription formatting that was in effect when the original prescription was ordered.

Details:  Steps
Follow the steps below:
  1. From the Configuration tab, select the Preferences tab. The Preferences screen appears.
  2. Select Prescription Preferences on the left.
  3. You can only set this default for the entire office. Make sure that Global is selected at the bottom of the screen.
  4. Select Add Asterisks to RX Dispense Amount.
  5. Select Yes to add the asterisk characters before and after all dispense amounts on all prescriptions.
  6. Select Finished. The Preferences list appears.
  7. Select OK to return to the Configuration list.

NOTES: 

1) Changes to this preference will not take effect until encounterPRO is restarted.
2) Only new or modified prescription treatments will show the asterisks on the dispense amounts.  Reprinting the prescription or refilling without modifying the prescription will use the preference that was in effect when the prescription was originally ordered.
3)The Order Medication screens will appear the same regardless of how the preference is set, but the printed prescription will display the preference changes.
