How to: Remove a Prescription Medication from the Database

Purpose
If there is a prescription drug that you want to be sure no user in the practice can order, you can remove it from the database. For example, if a drug is recalled, you would want to remove it from the database.

Before You Begin
Caution:  This procedure will remove the drug from the database entirely, not from a single specialty list or common list. You will not be able to re-activate the drug at a later date.

Details:

Steps
Follow the steps below:

  1. From the Office View, select the Configuration tab. The encounterPRO Configuration list appears.
  2. Select Drugs. The Drug Configuration screen appears.
  3. Select one of the search option buttons.
    • If you know the Trade Name of the drug, select that button, enter the trade name or a partial name and select OK. Note the option to search by Begins With or Contains.
    • If you know the Generic Name, select that button, enter the generic name or a partial name and select OK. Note the option to search by Begins With or Contains.
    • If you want to search by type of drug, select the Category button then select a category from the list.
  4. The matching drugs are shown. If no matches were found under your specialty, select the Specialty/All button above the Category button to search the entire database. Select the drug you want to delete. A menu of options appears.
  5. Select Delete. The system prompts you to confirm that you want to remove the drug completely from the database. Choose OK when prompted. The drug can no longer be found in the database.
  6. Select Finished to return to the Configuration list.
