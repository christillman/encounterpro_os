How to: Add a New Simple Prescription Drug

Purpose
Use this screen to create a new prescription drug. There are several phases to adding a new prescription drug:
  1. Set up drug properties
  2. Create drug packages
  3. Select administration rules
  4. Determine drug categories (for searching)
  5. Select the Specialties that will use this drug
  6. Save the new drug and add to the short list, if desired
Before You Begin
Before you add a new drug to the database, you should perform a thorough search of the database to confirm that is does not already exist. Search the drugs for your specialty as well as All Specialties to be sure the drug is not already in the database.
Details:

Steps
Follow the steps below:
To Add a New Single Drug:

  1. From the Configuration List, select Drugs. The Drug Configuration screen appears.
  2. Select the New Drug button. A list appears with the different types of drugs you can configure.
  3. Select Single Drug. A window appears to allow you to enter the drug name.
  4. Enter the Trade name for the drug and select OK.
  5. Enter the Generic name for the drug and select OK.
  6. If the new drug is a controlled substance, select Yes. Otherwise, select No. The Single Drug Definition screen appears.
  7. Select the Default Duration bar and select the number and unit for the duration.
  8. If you want to associate reference materials with this drug for either the patient or provider, select the bar for that level and select the reference material.
  9. Enter the information for the Maximum Calculated Dose. Enter the Amount and Unit. The Maximum Calculated Dose is the maximum daily dose that the system will calculate when the dosage is being figured by the patient's weight

To Configure the Drug Packages:

  1. From the Single Drug Definition screen, select the Packages tab.
  2. Select Add Package. The Select Dosage Form list appears.
  3. Select the dosage form you want to use for this drug. (The forms with ** are used for compound drugs). The Select Package screen appears. This screen lists the many packages of this form that have been created previously.
  4. Use the Up and Down buttons to scroll through the list of packages. The list is in alphabetical order, not numerical order.
  5. If the required form/dosage package appears on the list, select it, then select OK. If the required form/dosage package does not appear on the list:
    1. Select the New Package button.
    2. Enter the dosage and strength.
    3. Select the administer method. The Package Description is automatically generated.
    4. Edit the description, if needed, then select OK. The Package screen appears.
  6. You must enter a default dispense amount. This is the amount that defaults on the drug prescribing screen.
  7. You must enter at least one dispense choice. Select the gray box for Dispense Choices.
  8. Select Add Dispense. A list of drug forms appears.
  9. Select the form from the list. (This does not have to be the same form as the drug package. For example, the drug package can be ampules and the dispense choice can be a box.)
  10. Repeat steps 8 and 9 to add multiple dispense choices. These choices will appear in a drop-down list for the Total Dispense Amount on the drug prescribing screen.
  11. Select OK. The Single Drug Definition screen reappears.
  12. Repeat steps 2 -- 11 to add additional packages for this drug.

To Configure the Administration Rules:

  1. From the Single Drug Definition screen, select the Administration tab.
  2. Select Add Administration. A list of administration rules appears.

    NOTE: You must add an administration rule that is compatible for each drug package previously added.

  3. Select the appropriate rule (BID, etc.). The Administration screen for the new drug appears.
  4. Enter the remaining information needed for the administration rule:
    • Enter the Amount.
    • Select the Unit. (The administration unit for the Administration Rule and Package must be the same to be compatible.)
    • Indicate whether the amount is to be multiplied by the patient's weight.
    • Accept the generated description or edit the description to meet your needs
  5. Select OK. The Drug Configuration screen reappears with the administration rule displayed in the Administration block.
  6. Repeat steps 2 -- 5 until all Administration Rules needed for this drug have been entered.

To Select the Categories and Specialties for this Drug:

  1. From the Single Drug Definition screen, select the Categories tab.
  2. Select Add Category. A list of drug categories appears. Drug Categories are used to help users search for the required drug.
  3. Select all categories that apply to this drug, and then select OK. The Drug Configuration screen reappears. The Categories tab will show the categories selected.
  4. Select the Specialties tab. The specialty of the current user should already appear on the list of specialties. You should add this drug to all the specialties in your office, to make searching for the drug easier.
  5. Select Add Specialty. A list of specialties appears.
  6. Select all specialties for which you want the drug to appear, then select OK. The Specialty tab shows the specialties selected.

To Complete the Drug Configuration:

  1. From the Single Drug Definition screen, select the Properties tab. Review the information you entered and complete any fields left blank.
  2. Select OK. The new drug is added to the master drug list and to the drug lists for the specialties selected in step 6. The Drug Configuration screen appears with the new drug listed on the left.
  3. If you want to add this drug to your short list, select the icon to the left of the description.
  4. Select the Common or Personal Short list button to add the new drug to that list. Select Finished to return to the Configuration List.
