How to: Add a New Prescription Medication Package

Purpose
To order a prescription medication, you must always choose a drug package. The package tells the pharmacist the:
  • Form of the drug (tablets, vials, suspension)
  • Amount of drug per unit
  • Administration method
Before You Begin
You need to know the packaging information for the drug you are configuring including the form, strength, and appropriate administration rules.
Details:

Steps
Follow the steps below:

  1. From the Configuration List, select Drugs. The Drug Configuration screen appears.
  2. Select the drug for which you want to add a new package. A menu of options appears.
  3. Select Edit Config. The Single Drug Definition screen appears.
  4. Select the Packages tab. The list of current packages for the selected drug appears.
  5. Select Add Package. The list of drug forms appears.
  6. Select the form for this package from the list. The Select Package screen appears. This screen lists the many packages of this form that have been created previously.
  7. Use the Up and Down buttons to scroll through the list of packages. (The list is in alphabetic order, not numeric order.)
  8. If the required form/dosage package appears on the list, select it, then select OK.

    If the required form/dosage package does not appear on the list, select the New Package button. Enter the dosage and strength. Select the administration method. The Package Description is automatically generated. Edit the description, if needed, then select OK.

    The Package screen appears.
  9. You must enter a default dispense amount. This is the amount that defaults on the drug prescribing screen.
  10. You must enter at least one dispense choice. Select the gray box for Dispense Choices.
  11. Select Add Dispense. A list of drug forms appears.
  12. Select the form from the list. (This does not have to be the same form as the drug package. For example, the drug package can be ampules and the dispense choice can be a box.)
  13. Repeat steps 10 - 12 to add multiple dispense choices. These choices will appear in a drop-down list for the Total Dispense Amount on the drug prescribing screen.
  14. Select OK. The Single Drug Definition screen reappears.
  15. Repeat steps 5 - 14 to add additional packages for this drug.
  16. Select OK, then Finished.
