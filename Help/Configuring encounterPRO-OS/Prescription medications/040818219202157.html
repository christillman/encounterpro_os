How to: Create In-office Prescription Medications

Purpose
In-office medications are configured similarly to prescription medications. In addition, you need to set up the drug makers and HCPCS codes used for billing information.
There are several phases to adding a new in-office medication:
  1. Set up drug properties
  2. Create drug packages (at least one package is required)
  3. Select administration rules (at least one administration rule is required)
  4. Determine drug categories (for searching)
  5. Select the Specialties that will use this drug
  6. Select the HCPCS codes that can be used with this drug (if you want to bill for the drug administered, a HCPCS code is required)
  7. Select the makers that manufacture this drug (all makers of the drug which might be used by your practice must be entered)
  8. Save the new drug and add to the short list, if desired

Before You Begin
Before you add a new in office medication to the database, you should perform a thorough search of the database to confirm that it does not already exist. Search the drugs for your specialty as well as All Specialties to be sure the drug is not already in the database.

If the HCPCS procedure code needed are not available when you are configuring the medication, you will need to add those procedure codes on the Procedure Configuration screen. See Screen: How to: Add a New Procedure.


 

Details:

Steps
Follow the steps below:
To Add a New In Office Drug:

  1. From the Configuration List, select Drugs. The Drug Configuration screen appears.
  2. Select the New Drug button. A list appears with the different types of drugs you can configure.
  3. Select Single Drug. A window appears to allow you to enter the drug name.
  4. Enter the Trade name for the drug and select OK. (This is a required field.)
  5. Enter the Generic name for the drug and select OK. (This is a required field. If the generic name is not known, enter the trade name in this field.)
  6. The controlled substance determination screen will appear. If the new drug is a controlled substance, select Yes. Otherwise, select No. The Single Drug Definition screen appears.
  7. Select the Default Duration bar and select the number and unit for the duration. This number will default to the Duration field of the prescription and will be used to calculate the Total Dispense Amount.
  8. If you want to associate reference materials with this drug for either the patient or provider, select the bar for that level and select the reference material.
  9. Enter the information for the Maximum Calculated Dose. Enter the Amount and Unit. The Maximum Calculated Dose is the maximum daily dose that the system will calculate when the dosage is being figured by the patient's weight.

To Configure the In Office Drug Packages:

  1. From the Single Drug Definition screen, select the Packages tab.
  2. Select Add Package. The Select Dosage Form list appears.
  3. Select the dosage form you want to use for this drug. (The forms with ** are used for compound drugs). The Select Package screen appears. This screen lists the many packages of this form that have been created previously.
  4. Use the Up and Down buttons to scroll through the list of packages. (The list is in alphabetic order, not numeric order.)
  5. If the required form/dosage package appears on the list, select it, then select OK.

    If the required form/dosage package does not appear on the list, select the New Package button. Enter the dosage and strength. Select the administer method. The Package Description is automatically generated. Edit the description, if needed, then select OK.

    The Package screen appears.
  6. You must enter a default dispense amount. This is the amount that defaults on the drug prescribing screen.
  7. You must enter at least one dispense choice. Select the gray box for Dispense Choices.
  8. Select Add Dispense. A list of drug forms appears.
  9. Select the form from the list. (This does not have to be the same form as the drug package. For example, the drug package can be ampoules and the dispense choice can be a box.)
  10. Repeat steps 8 and 9 to add multiple dispense choices. These choices will appear in a drop-down list for the Total Dispense Amount on the drug prescribing screen.
  11. Select OK. The Single Drug Definition screen reappears.
  12. Repeat steps 2 - 11 to add additional packages for this drug.

To Configure the Administration Rules:

  1. From the Single Drug Definition screen, select the Administration tab.
  2. Select Add Administration. A list of administration rules appears.
    Note: You must add an administration rule that is compatible for each drug package previously added.
  3. Select the appropriate rule (DOSE, etc.). The Administration screen for the new drug appears.
  4. Enter the remaining information needed for the administration rule:
    • Enter the Amount. Select the Unit. (The administration unit for the Administration Rule and Package must be the same to be compatible.)
    • Indicate whether the amount is to be multiplied by the patient's weight.
    • Accept the generated description or edit the description to meet your needs
  5. Select OK. The Drug Configuration screen reappears with the administration rule displayed in the Administration block.
  6. Repeat steps 2 through 5 until all Administration Rules needed for this drug have been entered.

To Select the Categories and Specialties for this In Office Drug:

  1. From the Single Drug Definition screen, select the Categories tab.
  2. Select Add Category. A list of drug categories appears. Drug Categories are used to help users search for the required drug.
  3. Select all categories that apply to this drug, and then select OK. The Drug Configuration screen reappears. The Categories tab will show the categories selected.
  4. Select the Specialties tab. The specialty of the current user should already appear on the list of specialties. You should add this drug to all the specialties in your office, to make searching for the drug easier.
  5. Select Add Specialty. A list of specialties appears.
  6. Select all specialties for which you want the drug to appear then select OK. The Specialty tab shows the specialties selected.

To Select The HCPCS Codes for this Drug:
All HCPCS codes that you wish to be able to select when administering the medication must be selected on the HCPCS tab. There is no way to manually enter the HCPCS code on the medication adminstration screen.

  1. From the Single Drug Definition screen, select the HCPCS tab.
  2. Select the Add HCPCS button. The Select HCPCS Procedure screen appears.
  3. Select the desired HCPCS code from the short list or use the search options to locate the HCPCS code. HCPCS codes generally have the drug name and a number of units of the drug. The HCPCS for New In Office Drug screen appears.
  4. Enter the Amount and select the Unit that will be administered when you want to bill the selected HCPCS procedure.
  5. Select OK. The HCPCS procedure appears on the list of HCPCS Procedures.
  6. Repeat steps 2- 5 to add additional HCPCS procedures for this drug. Each HCPCS procedure that you select will be available to the clinical staff when they administer the medication. Only those HCPCS procedures that appear here will be available for use when documenting the medication administration.

To Select The Makers for this Drug:
All manufacturers that you wish to be able to select when administering the medication must be selected on the Makers tab. There is no way to manually enter the Maker on the medication adminstration screen.

  1. From the Single Drug Definition screen, select the Makers tab.
  2. Select the Add Maker button. A list of drug makers appears..
  3. Select the desired Maker from the list. You can select multiple makers, if needed.
  4. Select OK. The Drug Definition screen reappears with the makers selected listed. Only those drug makers that appear here will be available for use when documenting the medication administration.

To Complete the Drug Configuration:

  1. From the Single Drug Definition screen, select the Properties tab. Review the information you entered and complete any fields left blank.
  2. Select OK. The new drug is added to the master drug list and to the drug lists for the specialties selected in step 6. The Drug Configuration screen appears with the new drug listed on the left.
  3. If you want to add this drug to your short list, select the icon to the right of the description.
  4. Select the Common or Personal Short list button to add the new drug to that list. Select Finished to return to the Configuration List.
