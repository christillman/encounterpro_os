How to: Add a Signature Stamp for a User

Purpose
If the providers in your office do not use signature pads to sign each prescription and encounter, you will need to set up a signature stamp for each provider so that the electronic signature can be attached to each prescription ordered and to each patient's chart.

Before You Begin
In order to perform the following task you must have the necessary privileges set up by your encounterPRO Administrator or a Super-user.

Details:

Steps
Follow the steps below:

  1. From the Configuration tab, select the Users tab. The Select User to Edit screen appears. The display defaults to Active users.
  2. Select the provider's name from the list. The User Definition screen appears.
  3. Select the Manage Signature Stamp button. The User Signature Stamp screen appears.
  4. At this point you have two options:
    • Have the provider use the mouse or touchpen to draw the signature in the space provided. When satisfied with the signature, select the Use This button to save the signature.
    • Import a bitmap file of a previously scanned signature. Select the Import Bitmap File button, locate your file, then select OK.
  5. Select Finished to save the signature. The User Definition screen reappears.
  6. Select OK to save your changes and exit.
