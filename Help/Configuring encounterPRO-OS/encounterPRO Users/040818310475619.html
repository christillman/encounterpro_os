How to: Modify Existing encounterPRO Users

Purpose
From time to time you will need to change a user's supervisor, add or remove a role, or change other information.

Before You Begin
What do you need to know before starting?

Details:

Steps
Follow the steps below:

  1. From the Configuration tab, select the Users tab. The Select User to Edit screen appears. The display defaults to Active users.
  2. Select the name bar of the user you want to modify. You may need to use the Search options to locate the desired user. The User Definition screen appears.
  3. Make the changes as needed.
    • If you need to change the user's access rights, select the Permissions tab
    • If you need to add or remove a role, select the Roles tab
    • To change the user's access code, select the Reset Password button on the User tab. (Note, you do not need to know the existing password in order to change it.)
    • If you want to make other changes, modify the user information as needed.
  4. When finished making changes, select OK. The Select User to Edit screen reappears.
  5. Select OK to return to the Configuration list.
