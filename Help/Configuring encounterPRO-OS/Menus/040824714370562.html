How to: Configure a Report to Display the Print Menu or Send Directly to the Printer

Purpose

Whether or not a report is routed to the printer or sent to Document Manager is determined by how the report is ordered.  When Document Manager is 'turned on' the default for all EPRO Report services is to send documents to Document Manager rather than directly to a printer. If the practice would like to set up certain reports to use the simple printer option, that can be accomplished as well.

Reports can be ordered using several methods and the configuration of the specific method used determines if the report will be output to the printer or use Document Manager. The most common methods include:

  • a button on a menu
  • a button on the bottom of a dashboard
  • a hotspot on the top of a dashboard
  • a workplan item
Details:

Steps
Follow these steps:
To change a button on a menu to display the print menu or send directly to the printer:

  1. Turn on Config mode. (See How to: Turn on Configuration Mode.)
  2. Navigate to the screen displaying the menu you want to change. Notice the small text next to the menu buttons, this is the system name of the menu.
  3. Select the small text with the name of the menu to access the menu configuration screen. All menus have a JMJ Version that you cannot edit. If you want to change a menu, you must first create a local copy.
  4. Select the Create Local Copy button.

    NOTE: If you later decide that you want to revert to the JMJ Version, select the Local Copy and select Inactivate, then select the JMJ Version and select Activate.

  5. If you already have a Local Copy, be sure that bar is highlighted (darkened), then select the Edit Menu button. The Menu Properties screen appears.
  6. Select New. A selection box appears for you to choose the type of button function.
  7. EncounterPRO reports are a service, select Service. A list of service types appears.
  8. Select the context for the report:
    • General
    • Patient
    • Encounter
    • Treatment

    A list of services for the level chosen appears.
  9. Select the EPRO Report service.
  10. If there are further parameters to define for this service a screen appears to allow you to enter the desired parameters. Select Finish. The Menu Item Properties screen reappears.
  11. Select Report Only, then Next.
  12. Enter a destination, if desired: SCREEN or PRINTER. Select Next.
  13. Enter a printer id or leave the Printer parameter blank to use the default printer. Select Next.
  14. Select the report, then select Next.
  15. Select Next three times until the Print/Send Now parameter appears. If you want the document to automatically print to the selected or default printer, select Yes. If you want to be able to choose the printer, leave the parameter as No.
  16. Select Next until all remaining parameters have been bypassed. Select Finished. The Menu Item Properties screen reappears.
  17. At this point, you may want to change the Button Title (this appears on the menu) to indicate the report name and that it will go to the printer. For example, Take Home-Prt. When done making changes, select Finished. Your new button appears as the last button on the menu.
  18. To reorder the buttons on the menu, select the button, then select the Move button. Use the arrow buttons to move the button into the desired position. When finished moving buttons, select OK, then Finished. The screen reappears with the new button.

Changing the Button Icons

With the Menu Properties displayed, select the button for which you want to change the icon (picture).

  1. Select the Edit button. The Menu Item Properties screen appears.
  2. Type the filename for the new button icon in the Button field.
  3. Select Finished. The Menu Properties screen reappears. A small thumbnail version of the new button icon appears.
  4. Select OK, then Finished to return to the original screen. The new button icon is displayed.


TIP: You need to know the exact system filename of the button icon. If there is another menu that uses the icon you want to use, display the Menu Properties of that menu, then select the bar for the button with the icon you desire and select Edit. The button filename is displayed in the Button field.

To add a Print option to the Display script:
Adding a Print option

  1. Turn on Config mode. (See How to: Turn on Configuration Mode.)
  2. Display the treatment dashboard, right click to view the display script. Note the name of this Display Script! You will need it in a future step.
  3. If needed, create the local copy of the display script, then select Edit Display Script.
  4. Add the following commands (lines) to the display script where you want the print button to appear:

    1. Highlight the first 'print' command in the list and select 'Insert New Command'. This places the new command above the highlighted one.
    2. Select 'General' at the top of the next screen. Select 'print' from the list. A menu will appear prompting you to enter the text you want to appear. Everything within the white typing area will appear on the screen.
    3. For the 'Print' command, you will need to add a 'Quick View' menu. If there is not an appropriate 'Quick View' menu already available, you can create one. (This step is needed so that you can attach the 'Display Scrip' command to the 'menu' button.)
    4. Copy one of the 'Quick View' menus (pick one that just has 1 button, preferably for a report). Give the new menu a descriptive name (you will need to remember this name!).
    5. The menu config screen appears. Delete any buttons that already exist. Add 'New' button -- this will be a 'Service > Patient >EPRO Report'. When you configure the report button, do not select a report from the reports list. Keep selecting 'Next' until you get to the 'Select Generic RTF report ID' and select 'RTF Report'. Select 'Next' and then select the 'Display Script'. This is the Display script for the dashboard screen (from step 1).

Display script

  1. Select 'Next' and save the menu. You will return to the 'Select Menu' screen -- select the new 'Quick View' Menu you created in steps c and d. Continue to select 'Next' and 'Finished' until you are back at the display script edit screen. 
  2. Add a new command after your new print command. Select 'General', then 'Space'. Add 5 spaces, then select 'OK'. Select 'Finished' to return to the dashboard display.
  3. Refresh the dashboard (you may have to close it and re-open it). Select the link and preview the report.


To add a Print button to the bottom of a treatment dashboard:

  1. Select to Configuration from the Office View. The Configuration List appears.
  2. Select Treatment Type. The Select Treatment Type screen appears.
  3. Select the treatment type that corresponds to the treatment dashboard that you want to add the button to, for example, Referral. The Treatment Type Definition screen appears.
  4. Select the Services tab, then select Add Service.
  5. Select EPRO Report from the list of services. The new service is added to the bottom of the list of services for this treatment type.
  6. Select the EPRO Report from the list of services then select Edit Service
  7. Change the Visible Before Workplan Completes flag to 'Yes'.
  8. Change the Button Title to reflect that this report will go to the printer. For example, Referral-Print.
  9. Change the Button bitmap to button_printer.bmp.
  10. Select OK.
  11. Select the EPRO Report service again and select Configure Service.
  12. Select Report Only, then Next.
  13. Enter a destination, if desired: SCREEN or PRINTER. Select Next.
  14. Enter a printer id or leave the Printer parameter blank to use the default printer. Select Next.
  15. Select the report that you want to print, then select Next.
  16. Select Next three times until the Print/Send Now parameter appears. If you want the document to automatically print to the selected or default printer, select Yes. If you want to be able to choose the printer, leave the parameter as No.
  17. Select Next until all remaining parameters have been bypassed. Select Finished.
  18. If you want to reorder the buttons on the dashboard, select the EPRO Report service and select Move. Move the new button to the desired location and select OK.
  19. Select OK to save the changes to the treatment type.
  20. Select Finished to return to the Configuration list.

To configure a workplan step to send a report to the printer:

  1. Select to Configuration from the Office View. The Configuration List appears.
  2. Select Workplan, the Workplan Configuration screen appears.
  3. Select the workplan you want to configure and select Edit from the menu. The Workplan Definition screen appears.
  4. Select the step of the workplan that creates the report and select Edit Step from the menu.
  5. Select the EPRO report service step, then select Configure Service from the menu.
  6. Select Report Only, then Next.
  7. Enter a destination, if desired: SCREEN or PRINTER. Select Next.
  8. Enter a printer id or leave the Printer parameter blank to use the default printer. Select Next.
  9. Select the report, then select Next.
  10. Select Next three times until the Print/Send Now parameter appears. If you want the document to automatically print to the selected or default printer, select Yes. If you want to be able to choose the printer, leave the parameter as No.
  11. Select Next until all remaining parameters have been bypassed. The Workplan Step Definition screen reappears.
  12. Select OK to save the changes. The Workplan Definition screen reappears.
  13. Select OK twice to return to the Configuration List.
