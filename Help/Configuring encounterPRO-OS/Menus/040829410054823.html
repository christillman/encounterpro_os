How to: Configure for Object-level Confidentiality

Purpose

In order to use the confidentiality function in EncounterPRO, buttons that flag the objects as confidential or public must be configured for each object type. Flagging objects as confidential limits who can view the items in the patient's chart and prevents them from appearing on reports. Objects that can be flagged as confidential include:

·         A specific assessment, such as Pregnancy

·         A specific treatment, such as a prescription or lab test

·         An encounter

Note:  Flagging an assessment does NOT automatically flag any associated treatments as confidential. For reports that display Assessments with Treatments (ATAT), if the assessment is confidential the assessment will not appear and the associated treatments will not appear either. On reports that list treatments separate from their associated assessments (AATT), the treatments will appear unless they are also individually flagged as confidential.

Before You Begin

Treatments can be flagged as confidential or public. In order for the buttons Make Public/Make Private to be available, they must be configured for each treatment type that your practice might want to make confidential. For example, the buttons must be configured separately for Prescription Drugs, Labs, Referrals, etc.

Details:

Steps
Follow the steps below. This configuration is best done using a 'Test' patient.
To Add the Confidentiality Buttons for Assessments:

  1. Select the assessment description on the Patient Chart to display an assessment dashboard.
  2. Press F3 to display the Workplan Status screen. Turn on Configuration Mode and select OK. The menu name should appear in small type above the menu on the dashboard.
  3. Select the menu name. The Menu Properties screen appears.
  4. If there is no Local Copy, select Create Local Copy. Select Edit Menu. The Menu Properties screen appears.
  5. Select New to add a button to set an assessment as confidential (Make Private).
  6. Select Service, then select Assessment. A list of assessment-level services appears.
  7. Select New Note. Enter the parameters for this service as follows:
    • Context object = Assessment
    • Standard Note Type = leave blank
    • Special Note Type = Confidential
    • Preconfigured Treatment Titles or Keys = leave blank
    • Other Note Title or Key = Deny
    • Note Text = This assessment has been flagged as Confidential.
    • Select Finished.
  8. Change the Button Title to 'Make Private.'
  9. Select Finished. The Menu Properties screen reappears.
  10. Select New to add a button to remove the confidentiality flag for an assessment (Make Public).
  11. Select Service, then select Assessment. A list of assessment-level services appears.
  12. Select New Note. Enter the parameters for this service as follows:
    • Context object = Assessment
    • Standard Note Type = leave blank
    • Special Note Type = Confidential
    • Preconfigured Treatment Titles or Keys = leave blank
    • Other Note Title or Key = Allow
    • Note Text = This assessment has been flagged as Public.
    • Select Finished.
  13. Change the Button Title to 'Make Public.'
  14. Select Finished. The Menu Properties screen reappears.
  15. Select OK then Finished. The dashboard screen reappears.

To Add the Confidentiality Buttons for Encounters:

  1. Display the Patient Demographics screen (please use a 'Test' patient).
  2. Press F3 to display the Workplan Status screen. Turn on Configuration Mode and select OK. The menu name should appear in small type above the menu on the screen. (Note nothing happened when F3 was selected)
  3. Select the All Encounters tab. A list of the open encounters appears. If there are no open encounters, select All.
  4. Select the bar for an encounter to display the menu.
  5. Select the menu name in the upper left. The Menu Properties screen appears.
  6. If there is no Local Copy, select Create Local Copy. Select Edit Menu. The Menu Properties screen appears.
  7. Select New to add a button to set an encounter as confidential (Make Private).
  8. Select Service, then select Encounter. A list of encounter-level services appears.
  9. Select New Note. Enter the parameters for this service as follows:
    • Context object = Encounter
    • Standard Note Type = leave blank
    • Special Note Type = Confidential
    • Preconfigured Treatment Titles or Keys = leave blank
    • Other Note Title or Key = Deny
    • Note Text = This encounter has been flagged as Confidential.
    • Select Finished.
  10. Change the Button Title to 'Make Private.'
  11. Select Finished. The Menu Properties screen reappears.
  12. Select New to add a button to remove the confidentiality flag for an encounter (Make Public).
  13. Select Service, then select Encounter. A list of encounter-level services appears.
  14. Select New Note. Enter the parameters for this service as follows:
    • Context object = Encounter
    • Standard Note Type = leave blank
    • Special Note Type = Confidential
    • Preconfigured Treatment Titles or Keys = leave blank
    • Other Note Title or Key = Allow
    • Note Text = This encounter has been flagged as Public.
    • Select Finished.
  15. Change the Button Title to 'Make Public.'
  16. Select Finished. The Menu Properties screen reappears.
  17. Select OK then Finished. The Patient Demographics screen reappears.

To Add the Confidentiality Buttons for Treatments:

  1. Select Configure from the Office View screen. The Configuration list appears.
  2. Select Treatment Types. The Select Treatment Type screen appears.
  3. Select the treatment type for which you want to configure the confidentiality buttons. For example, Prescription Drugs. The Treatment Type Definition screen appears.
  4. Select the Services tab.
  5. Select Add Service. The Select Service screen appears.
  6. Select Treatment as the Context Object on the upper right of the screen.
  7. Select New Note. The New Note service appears on the list of Other Services on the Treatment Type Definition screen.
  8. Select the bar for New Note. A menu of options appears.
  9. Select Configure Service. Enter the parameters for the Make Private service as follows:
    • Context object = Treatment
    • Standard Note Type = leave blank
    • Special Note Type = Confidential
    • Preconfigured Treatment Titles or Keys = leave blank
    • Other Note Title or Key = Deny
    • Note Text = This encounter has been flagged as Confidential.
    • Select Finished.
  10. Select the bar for New Note. A menu of options appears.
  11. Select Edit Service.
  12. Change the Button Title to Make Private and Visible before Workplan Completes = Yes.
  13. Select OK. The Treatment Type Definition screen reappears.
  14. Select Add Service. The Select Service screen appears.
  15. Select Treatment as the Context Object on the upper right of the screen.
  16. Select New Note. The New Note service appears on the list of Other Services on the Treatment Type Definition screen.
  17. Select the bar for New Note. A menu of options appears.
  18. Select Configure Service. Enter the parameters for the Make Public service as follows:
    • Context object = Treatment
    • Standard Note Type = leave blank
    • Special Note Type = Confidential
    • Preconfigured Treatment Titles or Keys = leave blank
    • Other Note Title or Key = Deny
    • Note Text = This encounter has been flagged as Public.
    • Select Finished.
  19. Select the bar for New Note. A menu of options appears.
  20. Select Edit Service.
  21. Change the Button Title to Make Public and Visible before Workplan Completes = Yes.
  22. Select OK. The Treatment Type Definition screen reappears.
  23. Select OK. The Select Treatment Type screen reappears.
  24. Repeat steps 3 through 23 to configure the Make Private/Make Public buttons for another treatment type.
