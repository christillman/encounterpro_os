How to: Display the Filename of a Menu Button Icon

Solution:
Details:

If you need to know the exact system filename of a particular button icon, follow these steps:

  1. Display a menu that includes the icon you want to use.
  2. Turn On Config Mode.
  3. Right-click the button with the icon you desire. The button filename is displayed.
