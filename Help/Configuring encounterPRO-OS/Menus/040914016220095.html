How to: Add a Solution Center Button to the Utilities Menu

Solution:

In version 4, there was a button on the Utilities menu that accessed the PDF version of the EncounterPRO User's Guide. The User's Guide has been retired, but the Solution Center is available. If you want to add a button to access the Solution Center to the Utilities menu, follow the steps below.

NOTE: Remember that you can also access solutions from most ePRO screens by selecting F1. 

Details:

There are two stages to enabling the button:

  • Changing the URL for the User Manual in Preferences
  • Adding the button

Follow these steps to change the preference:

  1. Select the Configuration tab. The Configuration list appears.
  2. Select Preferences. The Preferences screen appears.
  3. Select Computer Preferences as the Preference Type on the left.
  4. Make sure that the Global level is selected at the bottom of the screen.
  5. Scroll down to locate the User Manual Location. Select the bar for that preference.
  6. Clear the existing link.
  7. Paste in the following link: http://solutioncenter.encounterpro.net:8080/selfservice/login_ranone.jsp?lang=en
  8. Select Finished.
  9. Select OK. The Configuration list reappears.

Follow these steps to add the button:

  1. Turn on Configuration Mode.
  2. Select the Utilities tab. The Utilities list appears.
  3. Select the name of the menu in small type at the bottom of the screen. The Menu Properties screen for the Utilities list appears.
  4. If needed, select Make Local Copy.
  5. Select Edit menu. The Menu Properties screen appears.
  6. Select New. The Menu Items Properties screen appears.
  7. Select Service, then General. A list of General Services appears.
  8. Select User Manual, then Finish.
  9. Change the button title to: Solution Center.
  10. Change the button bitmap to: button_solution_center.bmp
  11. Select Finished, then OK.
  12. Select Finished. The Utilities list reappears.
  13. Select your new button to make sure it launches the Solution Center.
