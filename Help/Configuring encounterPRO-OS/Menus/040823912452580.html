Details: Description
Use this screen to modify menus by adding or removing buttons or rearranging the order of the buttons. You must create a Local Version in order to modify a menu.

How to Access This Screen
Access this screen by selecting the menu name in small print on most menus. Configuration Mode must be turned on. See How to: Turn on Configuration Mode.
