How to: Configure a Menu Button to Print Instead of Using Document Manager

Purpose

Whether or not a report is routed to the printer or sent to Document Manager is determined by how the report is ordered.  When Document Manager is 'turned on'' the default for all EPRO Report services is to send documents to Document Manager rather than directly to a printer. If the practice would like to set up certain reports to use the simple printer option, that can be acomplished as well.

Reports can be ordered using several methods and the configuration of the specific method used determines if the report will be output to the printer or use Document Manager. The most common methods include:

  • a button on a menu
  • a button on the bottom of a dashboard
  • a hotspot on the top of a dashboard
  • a workplan item

Before You Begin

You should decide which menu(s) you want to add buttons to which will generate and print reports. One possible option would be to replace the View Reports menu with a Print Reports menu. The button to access that menu would be useful on the Patient Chart menu and from the Treatment Room menu.

It is important when you have buttons that will both print reports and order documents that you clearly label what the buttons will do.

Details:

Steps
Follow these steps:
To change a button on a menu to display the print menu or send directly to the printer:

  1. Turn on Config mode. See How to: Turn on Configuration Mode.
  2. Navigate to the screen displaying the menu you want to change. Notice the small text next to the menu buttons, this is the system name of the menu.
  3. Select the small text with the name of the menu to access the menu configuration screen. All menus have a JMJ Version that you cannot edit. If you want to change a menu, you must first create a local copy.
  4. Select the Create Local Copy button.

    NOTE: If you later decide that you want to revert to the JMJ Version, select the Local Copy and select Inactivate, then select the JMJ Version and select Activate.

  5. If you already have a Local Copy, be sure that bar is highlighted (darkened), then select the Edit Menu button. The Menu Properties screen appears.
  6. Select New. A selection box appears for you to choose the type of button function.
  7. EncounterPRO reports are a service, select Service. A list of service types appears.
  8. Select the context for the report:
    • General
    • Patient
    • Encounter
    • Treatment

    A list of services for the level chosen appears.
  9. Select the EPRO Report service.
  10. If there are further parameters to define for this service a screen appears to allow you to enter the desired parameters. Select Finish. The Menu Item Properties screen reappears.
  11. Select Report Only, then Next.
  12. Enter a destination, if desired: SCREEN or PRINTER. Select Next.
  13. Enter a printer id or leave the Printer parameter blank to use the default printer. Select Next.
  14. Select the report, then select Next.
  15. Select Next three times until the Print/Send Now parameter appears. If you want the document to automatically print to the selected or default printer, select Yes. If you want to be able to choose the printer, leave the parameter as No.
  16. Select Next until all remaining parameters have been bypassed. Select Finished. The Menu Item Properties screen reappears.
  17. At this point, you may want to change the Button Title (this appears on the menu) to indicate the report name and that it will go to the printer. For example, Take Home-Prt. When done making changes, select Finished. Your new button appears as the last button on the menu.
  18. To reorder the buttons on the menu, select the button, then select the Move button. Use the arrow buttons to move the button into the desired position. When finished moving buttons, select OK, then Finished. The screen reappears with the new button.

Changing the Button Icons

With the Menu Properties displayed, select the button for which you want to change the icon (picture).

  1. Select the Edit button. The Menu Item Properties screen appears.
  2. Type the filename for the new button icon in the Button field.
  3. Select Finished. The Menu Properties screen reappears. A small thumbnail version of the new button icon appears.
  4. Select OK, then Finished to return to the original screen. The new button icon is displayed.


TIP: You need to know the exact system filename of the button icon. If there is another menu that uses the icon you want to use, display the Menu Properties of that menu, then select the bar for the button with the icon you desire and select Edit. The button filename is displayed in the Button field.
