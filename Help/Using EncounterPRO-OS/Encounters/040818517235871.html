How to: Review Past Encounters

Purpose
Past encounters can be reviewed from several places in EncounterPRO. The most common being from the Patient Chart and the Patient Demographics screen.

Details:

Steps
Follow the steps below:
To review past encounters from the Patient Chart:

  1. From the Patient Chart, a list of past encounters appears on the left side of the main text area.
  2. Review the list of previous encounters. The color of the bar indicates the encounter owner and the encounter description.
  3. If you want to view a summary report of the encounter, select the (Sigma) button to the right of the colored bar. The encounter report appears. You can Print, Edit or View this report. When done reviewing the report, select OK to return to the Patient Chart.
  4. If you want to view the Patient Chart for a past encounter, select the colored bar. The main text area will display the information from that encounter. Note that the blue text indicates information recorded during that past encounter. Select the top bar on the Encounters list to return to the current encounter.

To review past encounters from the Patient Demographics screen:

  1. From the Chart tab, search for the desired patient.
  2. Select the patient name bar from the list of matching patients. The Patient Demographics screen appears.
  3. Select the Open Encounters tab. If there are any open encounters for the patient, a bar appears for each open encounter. The color of the bar indicates the encounter owner for that encounter.
  4. If the encounter you want to review is not listed with the Open encounters, select the All button to display all encounters for this patient. (This list does not include canceled encounters.)
  5. If you want to review a summary report of the encounter, select the  (Sigma) button to the right of the colored bar. The summary report appears. You can Print, Edit or View this report. When done reviewing the report, select OK to return to the encounter list.
  6. If you want to view the Patient Chart for a previous encounter, select the colored bar for that encounter. A menu of options appears. Select View Chart. The Patient Chart for that encounter appears. Note that the blue text indicates information recorded during that past encounter. Select the Exit tab to close the chart.
