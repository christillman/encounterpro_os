How to: Document a Telephone Encounter

Purpose
The Telephone tab is used to document all telephone calls received or initiated regarding any patient. Telephone encounters are accomplished by passing Tasks back and forth to various personnel in the practice.

Details:

Steps
Follow the steps below:
To create a telephone encounter:

  1. From the Office view, select the Telephone tab.
  2. If there are multiple telephone rooms, select a room. (Multiple patients can be "seen" in the same telephone room).
  3. Select the Check In button.
  4. Search for the patient using one of the available search options. For example, you can enter a partial last name using the keyboard or select the ABC button to use the touchscreen capability, then select Enter. A list of matching patients appears.
  5. Select the name bar of the desired patient. The Compose EPRO Task screen appears.
  6. Either accept the default recipient (Primary Provider) or select the colored bar and select the desired user or role.
  7. Enter a more descriptive Subject.
    TIP: Select the three dot (...) button next to Subject and create a pick list of commonly used subjects such as, Refill Request, Urgent, Needs to Speak to Nurse, etc.
  8. A template of information for the message appears in the Message text area. Enter the information for this call.
  9. Select Send. A patient-related Task appears in the Telephone room.

Completing a Telephone Task
When you are assigned a Task from a telephone call, you can complete the item or reassign it to another user.

To complete a Task for a telephone call:

  1. Access the telephone Task by selecting the Telephone tab from the Office View then select the service bar for the Task. The Telephone Task screen appears.
  2. The original message and all subsequent replies and comments appear in the Message gray area.
  3. If you want to review the patient's chart, select the View Chart button to go directly to the patient's chart. The Patient Chart screen appears with the telephone encounter displayed. You can perform any of the tasks needed (refill a prescription, order a new medication, enter a progress note, etc.). Select Exit to return to the Telephone Task screen.
  4. If you do not need to review the patient's chart, select the Open Item button to go directly to the telephone call dashboard. Select Finished to return to the Telephone Task screen.
  5. If you want to review the patient's demographic information, select View Patient. The Patient Demographics screen appears. Select Finished to return to the Telephone Task screen.
  6. Update the Comment/Reply text area. At this point, you can do one of the following:
    • If there is no task remaining for this call, enter a Comment then select Close with Comment. The system will prompt you to enter a comment about the disposition of this task if you leave the Comment box blank.
    • If you will complete the task later, enter a Comment and select I'll Be Back.
    • If you want another user to complete the task, enter a Comment describing what you want the new user to do, then select Send to.
    • If you want the previous person to complete the task, enter a Comment describing what you want the previous user to do and then select Reply to

    TIP: Get in the habit of always entering your progress in the Comment/Reply box.
  7. If the Task was closed, the Task service bar is closed and the Telephone Encounter is closed and sent to the Checkout Room.
    If you chose I'll Be Back, Reply to, or Send to, the Task service bar remains and the encounter remains in the Telephone Room.
