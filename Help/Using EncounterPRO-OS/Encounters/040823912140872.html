Description

This screen displays the Encounter Summary report. It is especially useful when reviewing past encounters since all the information collected for the specific encounter is displayed on the screen. This report can be printed, if desired.

How to Access This Screen
Access this screen by selecting the Sigma (Ó) button next to the encounter bar on the Patient Chart or on the encounter list on the Patient Demographics screen.
