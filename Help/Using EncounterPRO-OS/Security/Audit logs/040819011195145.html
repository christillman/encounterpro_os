Description
One of the new features in EncounterPRO version 5 is the ability to view the audit log for individual users. This information is visible to those users who have access to the User Configuration screens.

The User Audit tab on the User Configuration screen displays all actions taken by the specific user on the Audit Date selected. The log displays the following information:
  • Time and date of event
  • Who performed the event
  • What database was used Type of event (Security, Progress, Service, etc.)
  • Description of the event
  • Patient event was performed on, if applicable
Details:

Other Audit Options
The User Audit screen will allow you to track the actions of a specific user on a specific date. There are other audit displays in version 5 that will allow you to track actions for a specific:

  • Treatment
  • Assessment
  • Observation
  • Document (using Document Manager)

Access the dashboard screen for the treatment, assessment, or observation for which you want to review the activity log. Select the Properties tab. The Audit log for that item is displayed.
