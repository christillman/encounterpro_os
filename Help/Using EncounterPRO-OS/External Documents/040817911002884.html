How to: Post Documents to a Treatment

Purpose
Lab reports and other documents that pertain to a specific treatment need to be posted (attached) to that treatment. You can do this by posting directly into the patient's chart or by using the outstanding treatment lists on the Utilities list. If you have multiple treatment reports for multiple patients, many practices prefer to post using the outstanding treatments list. See How to: Scan and Post Directly to a Patient's Chart if you want to post multiple items for a single patient.

Before You Begin
You must set up your workstation with the scanner as an External Source to be able to scan and post. See How to: Configure a Workstation to Scan for instructions.

Details:

Steps
Follow the steps below:
To Post a Previously Scanned Attachment to a Treatment:

  1. From the Utilities list, select the appropriate Outstanding Labs/Tests button. A list of outstanding labs, tests, or other treatments appears.
  2. Select the item to which you will post the attachment. You may have to filter or search by patient name. The Treatment dashboard screen appears.
  3. Select the Attachments tab.
  4. Select the Star (New) button. A list of External Sources appears.
  5. Select EncounterPRO Attachment as the External Source. The Attachments screen appears.
  6. Select the To Be Posted tab. A list of all previously scanned, but unposted attachments appears.
  7. Select the item you want to preview from the list on the left. A thumbnail version (preview) appears in the white box on the right. Confirm that this is the item you want to attach to the current treatment.
  8. Select the Select button. The system will ask if you want to remove the item from the to-be-posted list. In most cases, the answer will be Yes. If this is an item you want to post to more than one treatment or more than one patient, select No to leave it in the To Be Posted list.
  9. You may be prompted to select a folder for this attachment. Select a folder from the list on the right, then OK.
  10. Depending on the folder selected, an Attachment Workplan screen may appear. (Folders can have workplans attached that are automatically launched when an item is saved to that folder.) If you want to launch the workplan (very important if this is a lab integration practice), select Finished. If you do not want to launch the workplan, select No Workplan This Time. A bar appears with the name of the attachment (defaulted from the treatment) and the date attached.
  11. Select Finished save the attachment and return to the Outstanding Treatments list..

To Scan and Post an Attachment to a Treatment:

  1. From the Utilities list, select the appropriate Outstanding Treatment button. A list of outstanding labs, test, or other treatments appears.
  2. Select the item to which you will post the attachment from the list. You may have to filter or search by patient name. The Treatment dashboard screen appears.
  3. Select the Attachments tab.
  4. Select the Star (New) button. A list of External Sources appears.
  5. Select TWAIN Compatible Scanner as the External Source. The scanning dialog appears. This box will vary based on the scanner installed.
  6. Set the scanning settings for the document you are scanning. Generally, you will be scanning Black and White documents that are Letter Sized. Select the 150 dpi setting (scanning at a higher dpi setting will create attachments that require much more storage space). Select the Scan button to begin scanning.
  7. Depending on the folder selected, an Attachment Workplan screen may appear. (Folders can have workplans attached that are automatically launched when an item is saved to that folder.) If you want to launch the workplan (very important if this is a lab integration practice), select Finished. If you do not want to launch the workplan, select No Workplan This Time. A bar appears with the name of the attachment (defaulted from the treatment) and the date attached.
  8. The Treatment dashboard Attachments screen reappears with a gray bar for the new attachment.
  9. Repeat steps 2 through 7 to scan each item. When finished scanning, select Finished. The Office View reappears.
