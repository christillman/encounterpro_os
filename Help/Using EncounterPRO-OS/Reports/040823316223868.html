Definition
Document Management is a new feature in enounterPRO version 5. There are two major benefits of Document Management:

  • Enables multiple routing options for EncounterPRO reports, including electronic prescriptions, secure email, faxing, etc. Document Management is essential to the implementation of e-prescribing.
  • Saves each report generated in the patient's chart as an attachment. Currently reports that are generated and printed are not saved in the patient's chart.
Steps in the Process
There are three main steps in the Document Management process:
  1. Order the document (either automatically or manually). Documents are ordered the same way as reports are ordered.
  2. Create the document. The document is automatically created when a user either wants to View or Edit or when the document is Sent.
  3. Send the document. Documents are sent to Recipient via Routes. Depending on the type of document, there will be one or more Recipients to choose from and each Recipient can have one or more Routes. For example, a Prescription can be sent to the recipient's Patient or Pharmacy (with e-prescribing). If the recipient 'Patient' is chosen, the routes available might be Printer. If the Pharmacy is chosen, the routes available might be Fax or E-Prescribing.
