Standard Reports in Peds Standard

Solution Title: Standard Reports in Peds Standard
Overview:




























































































































































































































































































































































































Report type Name Description
General Activity for Select Date by Patient Lists all user activity for a given patient and encounter date
General Activity For Select Date by User Lists all activity by a given user for a given encounter date
General All Patient Tasks A list of all Patient-contexted (not specific to an encounter or treatment) Tasks created within a date range
General Assessments by Age Lists the count of how many times each assessment was ordered within a given date range for patients born within a given date range
General Assessments Most Used by Doctor Lists how many times each assessment was ordered by a given doctor within a given date range
General Assessments Most Used by Practice Lists how many times each assessment was ordered by the practice within a given date range
General Average Service Times Lists the average time that each user takes to perform a given service (from begin time to end time) during a given date range, followed by a list of how long it took each user to perform each service for each patient
General Billing For Select Date by Doctor Lists all ICD9s and CPTs billed by a user with date and patient for a given encounter date
General Billing over date range Lists all ICD9s and CPTs billed with date and patient over a given date range
General Deleted Folder Audit Report Lists users who have attachments in folders which have been deleted
General Drug Usage Report For a given drug ordered in a given date range, lists patient, date of birth, date ordered and status
General Dx/Meds Used By Doctor For a given user and date range, lists how many times each drug was ordered for which diagnosis
General Encounters for Date by Office Lists encounters done at a given office on a given date with provider, patient, and encounter type
General Encounters for Date by Provider Lists encounters done by a given user on a given date
General Encounters by Role Lists all encounters in a given date range owned by users with a given role
General General Audit Report Lists all services done by a given user in a given date range
General General VFC Usage Lists all vaccines with 'VFC' in their description administered at a given office for a given date range
General Location/Doc Enc. For Select Date Lists  Encounters with a given discharge date, ordered by location, with whether the encounter was billable and posted
General Meds/Dx Used By Doctor For a given doctor and date range, lists how many times each drug was ordered for each assessment
General Observation EM Coding For a given observation, lists the E&M credits given for each part
General Observation Treatments By Doctor For a given user, date range, and observation, lists each patient and date the observation was ordered
General Office Meds Most Used by Doctor Lists how many times each office drug was ordered by a given user
General Office Meds Most Used by Practice Lists how many times each office drug was ordered
General Open Lab/Test By Doctor Lists all open labs/tests, the order date and patient billing_id, along with associated diagnosis, ordered by a given user within a given date range
General OrderedBy Open Labs Report Lists all open labs ordered by a given user
General Outstanding Patient TODO Items Lists all outstanding patient-contexted tasks with sender and reciever, along with how many hours the task has been open
General Patient Assessment Report Lists all patients diagnosed with a given diagnosis within a given date range
General Patient Bills A given patient's billing history
General Patient Drug Report A list of patients who are currently on a given drug
General Patient Encounters For Select Date Lists patient encounters with a given discharge date, with descroption, bill flag, start and end date and who owned by
General Patient Record Integrity Check Lists possible duplicate patients by name, birthday, SSN, billing id, bad names and birthdates and more
General Patient Referrals by Office For a particular office and date range, lists all of the referrals made
General Patient Referrals by Specialty Lists all patients referred to a particular specialty in a given date range
General Patient Visits by Insurance Lists all patient encounters in a given date range, sorted by patient insurance
General Patient WaitingRoom Report Lists information about how long patients are in the waiting room in a given date range, with averages by provider
General Patients by DOB Lists all patients whose date of birth is in a given date range
General Patients by Visit Type lists all encounters in a given date range, with visit code, sorted by visit type
General Patients Past Due Well Visit By Age Gives a list of patients who are in a given age range who have not had a well visit in a given number of months
General Patients With No Primary Provider List of patients with no primary provider
General Patients with Select Insurance Lists name, dob, and billing id for patients with a given billing authority
General Patients_with_Dx_rtf Lists patients diagnosed with a given assessment in a given date range
General RX Meds Most Used By Doctor For a given user and date range, lists how many times each drug was ordered
General RX Meds Most Used by Practice Lists how many times each drug was ordered in the database's history
General Script Audit Lists all prescriptions ordered in a given date range and status
General Select Date Bills Lists billing id, name, icd-9, cpt, and date and time for encounters done on a given date
General Selected Type Open Treatment Report Displays all open treatments of a given treatment type
General Send Out Labs Lists labs ordered on a given date and the treatment's status
General Today's Bills Lists billing id, name, icd-9, cpt, and date and time for encounters done today
General Treatment Count for Period For a given date range and observation, counts how many times it has been ordered per user
General Usage by Vaccine For a given vaccine and date range, lists patients who have been given the vaccine
General User Privilege Report Lists all service privileges for users
General Vaccine by Provider Lists patient and shot details for vaccines ordered by a given user
General Vaccine Usage Lists how many of which vaccine and lot number have been used in a given date range
Patient All Encounters (With Parameters) Prints a (specified in configuration) specified encounter report for each of a patient's encounters that meets parameters
Patient All Encounters (Without Parameters) Prints a (specified in configuration) specified encounter report for each of a patient's encounters
Patient Assessment History Lists the patient's open and closed diagnoses and dates
Patient Billing Sheet Patient's bill for the most recent encounter
Patient Canceled Treatments Shows any of the patient's treatments cancelled other than on the date of creation or cancelled by other user than the creator
Patient Cholesterol Flowdata A flowsheet of the patient's cholesterol lab results
Patient Height/Weight Flowsheet A flowsheet of the patient's height and weight
Patient Immunization Maker Lists the patient's immunizations with details
Patient Immunization Report Lists the patient's immunizations with parent signatures
Patient Lipids/Glucose Flow Report A flowsheet of the patient's lipid/glucose lab results
Patient Outstanding Treatment Report Lists open referrals, counseling, follow-ups, ongoing filenotes and instructions and special services
Patient Patient All Medications Lists a patient's medication history
Patient Patient Audit Report Audits activity in a chart in a given date range
Patient Patient Breath Flow Flowsheet for a patient's 
Patient Patient Labs/Tests All Lists the patient's lab/test history
Patient Patient Labs/Tests Select Lists the patien's history for a particular observation
Patient Patient Open Labs Report Lists the patient's outstanding labs
Patient Patient Procedure History Lists open labs, procedures, cardio tests, CAT scans, MRIs, nuclear tests, tests, ultrasounds, and xrays
Patient PMH Report The patient's PFSH
Encounter Asthma Action Report Lists the Asthma Action plan for the patient, with red, yellow, and green meds
Encounter Camp Physical Lists hgt, wgt, bp, ROS, current sick dxs, current meds, remarks, and a clearance statement
Encounter Encounter SOAP Note A/P Gives specifics of what happened in an encounter- treatments listed after associated diagnosis
Encounter Encounter SOAP Note Long Gives detailed information about what happened in an encounter
Encounter Encounter SOAP Note Short Gives brief information about what happened in an encounter
Encounter General Preview Report Gives provider a brief patient history and information as to what staff has done for the patient so far in the encounter
Encounter Lab Referral Prescription A lab referral slip in 1/4 sheet format
Encounter Lab Results Letter to Patient, Detailed Long format letter to patient with lab results
Encounter Lab Results Letter to Patient, Short Short format letter to patient with lab results
Encounter Lab/Test Report Lists open assessments and labs and tests for that encounter
Encounter Open Letter to Patient Formatted letter for the user to edit and type content into
Encounter Patient Current Medications Lists patient's current medications
Encounter Patient Take Home Report A/P A patient-friendly report of what happened at the encounter
Encounter School Physical Age, weight, height, address, immunization status, PMH, ROS, Physical, Remarks
Encounter SendOut Tests Lists labs and tests and associated diagnoses ordered in that encounter that are send-out
Encounter Small Return Form A school/work return slip
Treatment Drug Prescription Prescription
Treatment General Treatment Report Lists the treatment's information
Treatment Referral Treatment Prescription Letter w/ SOAP A letter to refer the treatment to a specialist in letter format
Treatment Referral Treatment Prescription No SOAP A letter to refer the treatment to a specialist in short format with no SOAP note
Treatment Referral Treatment Prescription w/ SOAP A letter to refer the treatment to a specialist in short format with a SOAP note










Assessment Assessment Report Gives information on that patient's assessment and all related treatments and notes
