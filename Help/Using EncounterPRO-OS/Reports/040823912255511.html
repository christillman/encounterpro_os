Description
Use this screen to select a report to print or create a document (if you are using Document Manager). If you have the permission to configure reports, you can select the icon to the left of the report name and configure the report.

When searching for a report, be sure to select the Context Object at the top of the screen. The available context objects will depend upon how you accessed this screen. If you access this screen from a Patient Chart, you can select from General, Patient, and Encounter. If you access this screen from a treatment dashboard, you can select from General, Patient, Encounter, and Treatment.

There is a different Common short list for each report context.

How to Access This Screen
Access this screen by selecting the Pick Report button from one of the EncounterPRO menus.
