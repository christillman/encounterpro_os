Description
This screen displays medications prescribed during previous visits and allows the clinical staff to document medications prescribed by providers outside the practice and OTC medications the patient is currently taking. Medications on this screen appear in the Meds box on the Vitals screen and Patient Chart screen.
Providers and clinical staff should ensure that the list of medications on this screen are maintained. If a medication is discontinued, it needs to be closed so that it no longer appears on this screen.

How to Access This Screen
Access this screen by:

  • Selecting the Meds box at the top center of the Patient Chart
  • Selecting the Meds box on the lower left of the Vitals screen
  • As part of the clinical staff workplan, this screen can automatically display (depends on specialty)
