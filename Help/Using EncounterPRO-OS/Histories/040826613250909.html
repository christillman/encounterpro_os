How to: Edit a Pick List.

Purpose
Pick list screens are easily configured 'on-the-fly,' that is, from within an encounter. You can continue to build on this list over time.

There is a limit to the number of entries you can add to the common pick list. Pick lists appear on several screens, such as Chief Complaint, Notation of Review, Patient Notes, Encounter Notes, etc. Below is an example of a Pick List screen.



Before You Begin
There are four lists available on a Pick List screen:

  • Common Generic
  • Common Specific
  • Personal Generic
  • Personal Specific

If you are making changes to the Common lists shared by all users, you should confirm that the changes are agreeable to those users. If you are making changes only to your Personal lists, those changes are only visible to you and will not effect other users.

TIP: If there is an entry on the Specific list, that will display before the Generic list, if there is an entry on the Personal list, that will display before the Common List. So, if you have a Personal-Specific list that will always display first.

Details:

Steps
Follow the steps below:

  1. Display the Pick List screen. The pick list appears on the left side of the screen.
  2. Decide which list you want to update:
    • Common Generic: This is a shared list. Be sure that Common List is displayed at the bottom of the screen and Generic is highlighted at the top of the screen.
    • Common Specific: This is a shared list. Be sure that Common List is displayed at the bottom of the screen and Specific is highlighted at the top of the screen.
    • Personal Generic: This is not a shared list. Select the Common List button at the bottom of the screen to toggle to Personal List and highlight Generic at the top of the screen. (If there is anything in this list, it is the default that is first displayed when the user accesses the Chief Complaint screen.)
    • Personal Specific: This is not a shared list. Select the Common List button at the bottom of the screen to toggle to Personal List and highlight Specific at the top of the screen.
  3. Type the new entry in the text box.
  4. Select the Add to List button (This will add all of the text in the text box as a single entry). The new item is added to the bottom of the existing list.
  5. If you want to reorder the list, select the notepad icon to the far left of the item you want to move. A menu appears.
  6. Select the Move button. You can move the item up or down one position at a time or to the very top or very bottom of the list using the buttons that appear.
  7. If you want to remove an item from the pick list, select the notepad icon to the far left of the item you want to delete. Select the Remove Top 20 button (trashcan).
  8. When all entries have been completed, select I'm Finished.
