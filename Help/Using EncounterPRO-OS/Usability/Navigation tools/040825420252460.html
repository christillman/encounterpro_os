How to: Upgrade EncounterPRO on a Fat Client

Purpose

When a new version of the EncounterPRO client is installed on the server, all the thin client workstations are automatically upgraded. The 'fat client' workstations will need to be upgraded manually by the EncounterPRO Administrator. This task must be performed by a user who is on site.

Details:

Steps
Follow these steps:

  1. When the EncounterPRO client has been upgraded on the server, the next time a user attempts to log in to EncounterPRO on a fat client a message appears letting the user know that the client needs to be upgraded.
  2. Select Install Upgrade. The install process begins.
  3. Accept the defaults.
  4. When the upgrade is complete a sucess message appears.
