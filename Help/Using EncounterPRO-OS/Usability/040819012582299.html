Concept: Common vs. Personal Lists

Description
Common lists are used throughout the practice and are accessible by all EncounterPRO users. Each list that has a Common list also gives the user the option of creating a personal list. Personal lists are only used by and accessible by  the user who created them.

There are several areas in EncounterPRO that allow users to create their own personal version of lists:

  • Short lists
  • Treatment lists (for an assessment)
  • Standard Exams
  • Assessmrnt lists
Details:

Uses
Personal lists allow a user to have their own version of a list, accessible only to themselves. This can be very useful when:

  • A user has a different specialty than a majority of the providers in the practice
  • A user has specific treatments that they prefer to use that are not popular enough to be on the Common list
  • A user wishes to create their own standard exam templates to allow one-touch documentation of their physical examinations
  • A user wants to arrange his/her short list by most used items while the Common list is alphabetical or vice versa

Default to Personal
If there is anything entered on the Personal  list, that list will display when the screen is initially presented. The user will need to select the Common button or tab to display the Common list.

TIP: When you initially select the Personal list option for a short list or treatment list, the system will ask if you want to start with a copy of the Common List. Generally, most users find that they prefer to decline this option and build their personal list from scratch, since they can always access the Common list with one-click.
