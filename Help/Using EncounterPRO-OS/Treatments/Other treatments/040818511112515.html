How to: Order Patient Materials.

Purpose
Patient materials can be any document, handout, diagram which has been imported or scanned into EncounterPRO. Providers typically "order" patient materials as part of the treatment plan for a particular assessment, such as during a well child visit. If desired, patient materials can be ordered without being associated with a specific assessment.

Before You Begin
Before the provider can order patient materials as part of a treatment plan, the document must be imported or scanned into EncounterPRO. See How to: Add Patient Education Materials for detailed instructions.

Details:

Steps
Follow the steps below:
To order patient materials associated with a specific assessment:

  1. From the Patient Chart, access the treatment list for the desired assessment by doing one of the following:
    • Select the New Sick or New Other DX button on the right of the chart and select the desired assessment
    • Select the small icon next to the description of an existing assessment (open assessment) on the Patient Chart
    • Select the description of the existing assessment (open assessment) then select the blue description from the dashboard screen

    The treatment list for the selected assessment appears.
  2. If the patient material you want to order appears on the treatment list, select it. You can select as many treatments from the list as needed.
  3. If the patient material you want to order does not appear on the treatment list, select the New Treatment button on the right of the list screen. The Select Patient Education Materials screen appears.
  4. Select the desired patient material from the short list or use the search options to locate the desired items.
  5. As you select items from the list, they appear in the Selected Items box. When you have selected all items for this patient, select OK. The treatment list reappears with the new materials selected (highlighted)
  6. If you want to save a patient material to the treatment list for future use, select the small icon to the left of the description.
  7. Select the Yes button below Update List, then OK. The treatment list may automatically sort the list to place the new treatment at the top of the list.
  8. Finish selecting or ordering treatments for this assessment. When the treatment plan is complete, select Finished. The Patient Chart reappears.
    Most practices are configured to have a service bar appear for the clinical staff to print the patient materials.

To order patient materials not associated with a specific assessment:

  1. From the Patient Chart, select the Order Treatments button from the right side of the chart. (You may have to select the Other Items button first to display the entire menu.)
  2. If you want the instruction to appear only for the current encounter, select Encounter Instruction. If you want the instruction to continue to appear until the assessment is closed, select Ongoing Instruction. An observation comment screen (with pick list) appears.
  3. Select the instruction from the pick list, or type the instruction in the white test box.
  4. Select OK. The Patient Chart reappears.
    Most practices are configured to have a service bar appear for the clinical staff to print the patient materials.
