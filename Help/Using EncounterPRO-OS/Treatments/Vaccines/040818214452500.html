Purpose
Ordering a vaccine will create a service bar to notify the clinical staff to prepare and/or perform the treatment.

Before You Begin
Along with each vaccine, a vaccination administration code needs to be ordered. See  Screen: How to: Order Vaccines Administration Codes. for details on selecting the appropriate administration codes.

Details:

Steps
Follow the steps below:
To Order the Vaccine from the Well Assessment:

  1. From the treatment list for the well assessment, select the desired vaccine(s) from the list.
  2. If the vaccine you want to order does not appear on the treatment list, select New Treatment. A menu of treatment types appears.
  3. Select the Vaccine button. The Select Vaccines Procedures screen appears. Select the desired vaccine from the short list. If the vaccine you want to order does not appear on the list, use the search options to locate the desired vaccine.
  4. Select the appropriate vaccine administration procedures from the treatment list. (See  Screen: How to: Order Vaccine Administration Codes.) for details.
  5. When finished selecting vaccines and administration codes, select OK. Service bars will be created for the administration of each vaccine.

To Order the Vaccine from the Immunization Status tab:
If the provider or clinical staff orders the vaccines directly from the ProVacs, someone will need ot order the appropriate vaccine administration codes separately.

  1. From the Immunization Status screen, select Order Vaccine. The Select Vaccine Procedures screen appears.
  2. Select the vaccines that you want to administer today. Each vaccine appears in the Selected Items list.
  3. Select OK. The Immunization status screen reappears. Service bars will be created for the administration of each vaccine.
  4. Be sure that the appropriate vaccine administration codes are ordered. See  Screen: How to: Order Vaccine Administration Codes. for details.
