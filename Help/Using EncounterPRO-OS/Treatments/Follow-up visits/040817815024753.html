How to: Schedule Follow-ups and Referrals

Purpose
After the clinical staff and provider have completed all in-office services for a patient, the system will usually send the patient to Checkout. If, however, the provider ordered a follow-up appointment or a referral for the patient, the system can create a service bar for the front desk staff to perform those tasks. The patient will not move to the Checkout room until these services are finished.

Details:

Steps
Follow the steps below:

  1. Check the Office View to see if any patients have a service bar for a follow-up or referral. The service bar will usually appear in the color assigned to the front desk personnel.
  2. Enter the room in which the patient name bar appears. Select the follow-up or referral service bar. The treatment dashboard screen for the follow-up or referral appears.
  3. Perform the actual scheduling of the follow-up visit in your scheduling system. Perform the actions required at your office for a referral.
  4. Select the Scheduling Note link on the treatment dashboard. A screen appears allowing you to type in the scheduling information. If a pick list has been created, you can select the appropriate item from the left side of the screen. Select OK when finished. The treatment dashboard reappears.
  5. Select Finished. The dashboard screen closes. If this was the last service for the patient, the patient will move to Checkout.
