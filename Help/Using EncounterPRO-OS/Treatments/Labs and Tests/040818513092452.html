How to: Track Send-out Labs

Purpose
Labs and tests that are processed off site must be tracked so that the results can be reviewed, relayed to the patient, and providers can be alerted when abnormal results are returned.
Your practice needs to have a process in place that determines the following:

  • Who is responsible for tracking labs and tests that are sent out?
  • How will providers be notified of lab results?
  • How will patients be notified of lab results?
  • Who is responsible for closing the treatment?

The answers to these questions will vary from practice to practice and sometimes provider to provider. It is important that all the personnel understand which tasks they are expected to perform so that nothing "falls through the cracks."

Before You Begin
If your practice has lab integration, see  Screen: How to: Track and Manage Lab Results - Lab Integration.  This solution discusses tracking off site lab results in a non-integrated environment.

Details:

Steps
Follow the steps below:

  1. From the Office View, select the Utilities tab. The Utilities list appears.
  2. Select the Outstanding Labs/Tests button. The Outstanding Labs/Tests screen reappears. The list is sorted by date ordered with the newest at the top.
  3. To filter the list, you can do any of the following:
    • To filter by ordering provider, select the bar that reads to the right of Ordering Provider. The Select User screen appears. Select the provider you want to filter the list for.
    • To filter by patient name, select the bar below By Name. A keyboard popup appears. Type the patient's last name (or partial last name) and select OK.
    • To filter by office location, select the bar below by Office. A list of office locations appears. Select the office you want to filter the list for.
  4. To change the filter, select the Clear filter button on the right of the screen. Repeat step 3 to filter the list again.
  5. Locate the lab/test that you want to review. Select the bar for the test description. The treatment dashboard for that treatment appears.
  6. If you want to send a Task to the ordering provider to alert him/her that the results are available and need to be reviewed, follow these steps:
    1.  From the treatment dashboard, select Inbox Task.
    2. The recipient defaults to the encounter owner (for the encounter in which the lab was ordered). If you need to change the recipient, select the bar for Encounter Owner and select the desired provider from the list.
    3. Optional. Add any message needed.
    4. Select Send.

    A Task appears in the recipient's Task List for this treatment.
  7. If you want to add a review note, select one of the links at the top of the screen. Enter the text of the note and select OK.
  8. If you want to close this treatment, select at the top of the screen or the Close Treatment button at the bottom of the screen.
  9. To return to the Outstanding Labs/Test list, select Finished.
