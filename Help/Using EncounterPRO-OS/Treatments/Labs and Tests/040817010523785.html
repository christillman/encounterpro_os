Details: Description
This screen is used to review and manage outstanding (open) treatment items such as labs and tests, referrals, x-rays, etc. If a treatment item remains open at the end of an encounter, it will appear on one of the Outstanding Items screens.
Each of these screens can be filtered and sorted by a number of criteria:
  • Ordering provider
  • Patient name
  • Date ordered
  • Name of treatment (drug, lab, etc.)


NOTE: If your practice is integrated with a Lab, the outstanding labs will still appear on the Outstanding Labs and Tests screen in addition to appearing on the Daily Lab Report screen.

How to Access this Screen
Access this screen by selecting one of the following from the Utilities List:

  • Outstanding Labs and Tests
  • Outstanding Referrals
  • Outstanding RX Drugs
  • Outstanding Other Treatments
