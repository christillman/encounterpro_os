How to: Copy a Treatment

Purpose
Copying a treatment list saves time when the original treatment list is close to or the same as the list you want for another assessment. There are several reasons you might want to copy a treatment list:

  • Copy the Common list for an assessment to start your Personal list
  • Copy the Common list for an assessment to start your Personal or Common list for a different assessment
  • Copy your Personal list for an assessment to start your Personal or Common list for a different assessment

Before You Begin
Decide which treatment list you will be copying and what the destination list will be.

NOTE: You cannot copy another user's Personal List, you can only copy your own Personal List from one assessment treatment list to another.

Details:

Steps
Follow the steps below:

  1. From the Patient Chart, do one of the following:
    • Select the New Sick DX or New Other DX button and select the assessment which you want to copy the treatment list from. 
    • Select an existing assessment description on the chart and select Order Treatment. The treatment list for the assessment appears. This is the assessment you will be copying the treatment list to.
  2. Select the destination of the treatment list you want to copy, either the Personal or Common tab.
  3. Select the Copy from button in the lower right of the screen. A list of options appears.
  4. You can copy from:
    • The Common/Personal list of the current assessment. (if you are viewing the Common list, you can copy from the Personal list and vice versa.)
    • The Default list for the practice.
    • Another Assessment Common list
    • Another Assessment Personal list
  5. If you select one of the options to copy the list from another assessment, the Select Assessment screen appears. Select the assessment you want to copy the treatment list from.
  6. If there are any treatments already listed on the current screen, the system prompts you to choose to:
    • Remove the exiting items before copying the list
    • Append the items from the copied list to the current list

    The treatments from the selected list are copied (either replacing or appended to) the currently displayed list.
  7. At this point, you can add other treatments to the list or remove any copied treatments that you do not want to keep. See How to: Update a Treatment List for details.
  8. Select Finished to return to the Patient Chart.
