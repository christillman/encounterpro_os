How to: Rearrange Assessments on the Patient Chart.

Purpose

EncounterPRO will automatically order the assessments on the Patient Chart and on the Billing either ascending (first chosen in today's encounter is at the top) or descending (last chosen in today's encounter is at the top). This is determined by a General Preference, New Assessment Sort. The default is to sort assessment ascending. See Screen: How to: Set the New Assessment Sort Order.

If an assessment was ordered in the 'wrong' order and does not appear on the Patient Chart or Billing as desired, the assessments can be rearranged as needed.

The encounter owner or billing specialist can rearrange the assessments on the Patient Chart and on the Billing Review screen. This ensures that the primary diagnosis for the visit is at the top of the list when there is more than one billable assessment for an encounter.

NOTE: The Assessment Move button must be added to the Billing Review screen. See How to: Add the Assessment Move button.


Details:

Steps
Follow these steps:

  1. From the Patient Chart, select Exit to access the Billing tab.
  2. Select the gray area with the billing summary. The Billing Review screen appears. The billable assessments are listed below the Primary Visit Charge.
  3. Select the Assessment Move button (you may need to select Other Options first). The Assessment Sort Order screen appears.
  4. Select the assessment that you want to re-order in the list.
  5. Use the arrow keys to move the selected assessment up or down the list of assessments.
  6. Repeat steps 4 and 5 to move additional assessments on the list.
  7. When the assessments are in the desired order select Finished. The Billing Review screen reappears. The new order of the assessments may not be displayed immediately.
  8. Select Finished to return to the Patient Chart/Billing tab. The new order of assessments should be reflected in the gray text box.
