Description
Chart alerts can be used for information that you want to be sure the office personnel will notice. Alerts are commonly created for items such as:
  • Latex allergy
  • Similar or same named patients (for example, multiple patients named John Smith)
  • The patient or patient's parent has a disability, such as hearing loss
  • Reminders that a patient needs an immunization form or return to work form before they depart the office today
IMPORTANT: Remember that patient confidentiality ultimately dictates what should and should not be included within the alert. The alert screen will display whenever any user accesses the patient's chart.
Details:

Using Chart Alerts
When a chart alert appears, you have two options:

  • If you want to retain the alert so that it continues to appear whenever any user accesses the chart, click OK
  • If you want to remove an alert, select the gray alert text bar. Confirm that you want to delete the alert. The alert will no longer display automatically when the chart is accessed. (Removing an alert is restricted to providers and clinical staff.)
  • If you want to remove a reminder, select the Dismiss button to the right of the reminder text. Dismissing reminders is available to all users with access to clinical data (patient charts).
