Details: Description
This screen is used to export all or part of a patient's chart to a location on the workstation or network. This screen is used to export the records of a single patient. If there are multiple patient charts that need to be transferred or exported, use the Multi-Chart Export function.
You can choose to export any or all of the following:
  • Assessment history
  • PFSH
  • All medications report
  • All encounter reports
  • A list of attachments
  • Actual attachments

How to Access This Screen
Access this screen by selecting the Chart Export button at the bottom of the Patient Demographics screen.
