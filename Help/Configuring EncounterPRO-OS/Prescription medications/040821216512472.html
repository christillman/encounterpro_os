How to: Configure Prescriptions to Display Generic Drug Names

Purpose
A Prescription Preference can be set that will include the generic drug name on all prescriptions (the system will pull this information from the Generic Name field on the Prescription Medication Configuration screen).


Before You Begin
You must set this preference globally (if you have sufficient permissions) which will change the setting for all users in the practice. Be sure to consult with the other providers in the practice before making global preference changes.

Details:  Steps
Follow the steps below:
  1. From the Configuration tab, select the Preferences tab. The Preferences screen appears.
  2. Select Prescription Preferences on the left.
  3. You can only set this default for the entire office. Make sure that Global is selected  at the bottom of the screen.
  4. Select Display Drug Generic Names.
  5. To always print the generic drug name on prescriptions, select Yes.
  6. Select Finished. The Preferences list appears.
  7. Select OK to return to the Configuration list.

NOTES:

1) This preference only takes affect when EncounterPRO is restarted.
2) It only affects new or modified drug treatments.  Reprinting the prescription or refilling without modifying the prescription will re-use the same text that appears on the Patient Chart screen.
3) The EncounterPRO screens (Order Meds) will not look any differently, but the printed script will display the preference changes.
