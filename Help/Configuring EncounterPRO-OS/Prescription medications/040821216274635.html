How to: Configure the Signature Prompt for Prescriptions

Purpose
When the practice uses signature capture devices to authorize prescriptions, a prompt appears for the ordering provider to sign for each prescription. A Prescription Preference allows the configuration user to determine the text of that prompt.

If the practice always uses the signature stored on file for prescriptions, this preference does not need to be set.


Before You Begin
You must set this preference globally (if you have sufficient permissions) which will change the setting for all users in the practice. Be sure to consult with the other providers in the practice before making global preference changes.

Details:

Steps
Follow the steps below:

  1. From the Configuration tab, select the Preferences tab. The Preferences screen appears.
  2. Select Presecription Preferences on the left.
  3. You can only set this default for the entire office. Make sure that Global is at the bottom of the screen.
  4. Select Rx Signature Prompt.
  5. Enter the text of the signature prompt, then select Finished. The Preferences list appears.
  6. Select OK to return to the Configuration list.
