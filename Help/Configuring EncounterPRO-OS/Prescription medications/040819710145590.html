How to: Configure a Prescription Medication with Irregular Dosing

Purpose
There are certain medications that have an irregular dosing schedule that need to be configured differently than those with a steady dosing schedule. One such medication is Azithromycin 5 Day Dose Pack, where the patient takes 2 pills the first day then 1 pill for the next 4 days. Since EncounterPRO administration rules do not allow for a variable dosing schedule, the drug package and administration rules are, in effect, disabled. The variable dosing schedule is handled by creating a default Special Instruction for the medication/package combination.

Before You Begin
Review the instructions for Adding a Prescription Medication Package before beginning.
You will need to perform several steps:

  1. Create the Take as Directed Package
  2. Create the Administration Rule for the new package
  3. Create the default Patient Instructions for the drug/package
Details:

Steps
Because you will be creating default Special Instructions, it is easiest to perform this procedure from a patient context, rather than from the Configuration list. Create an encounter for a Test patient, then follow the steps below:
To create the Take as Directed Package:

  1. From the Patient Chart screen, select the Any Treatment button.
  2. Select Prescription Drug from the menu. The Select Prescription Drug screen appears.
  3. Locate the medication you need to configure from the short list or use the Search options to locate it.
  4. Select the small icon to the left of the medication name. A menu of options appears.
  5. Select Edit Config. The Single Drug Definition screen appears.
  6. Create a 'Take as Directed' Package.

    a.  Select Packages. 
    b. Select Add Package. The Select Dosage Form list appears.
    c. Select a dosage form of 'Titrate/Special Dose.' Remember that the dosage form appears on the Assessment Treatment screen but does not display on the Patient Chart or print out on the prescription. Choose the Titrate/Special Dose dosage form is so that users will be able to distinguish it at order time from traditional drug packages.
    d. Select the New Package button.
    e. Select an Administer Method.
    f. Enter a dosage amount using the Each __ Contain __ of the drug.  remember this unit, since you will need to create an Administration Rule with the same unit. A typical configuration is Each 1 dose Contains 1 dose of drug.
    g. Edit the Package Description to reflect the dosage/package you are creating. This Package Description is what prints out on the prescription.
    h. Select OK, a second screen will appear with the title 'Package for (Drug name)'
    i. Select the 'Take As Directed' option near the bottom of the screen. Click on the button and change 'No' to 'Yes.' This will suppress all dosing information when the end user prescribes the drug package.
    j. Enter a Default Dispense Amount and select the Unit.
    k. Select the Dispense Choices box and enter at least one dispense choice.
    l.Select OK.

    The Single Drug Definition screen reappears with the new package listed.

To Create the Administration Rule:
You will need to create an Administration Rule that is compatible with the drug package you created above. Use the same unit as you selected in step 6 above.

  1. Select the Administration tab. There must be at least one Administration Rule for any given drug package.
  2. Select the appropriate rule (ASDIR, etc.). The Administration screen for the new drug appears.
  3. Enter the remaining information needed for the administration rule:

    a. Enter the Amount. In this situation you could use 1 Dose Equals 1 Dose or something similar.
    b. Select the Unit. (The administration unit for the Administration Rule and Take As Directed package created above must be the same to be compatible.)
    c. Accept the generated description or edit the description to meet your needs

  4. Select OK. The Select Prescription Drug screen reappears.

To Create the Special Instructions:
Default instructions may be created only on the Prescription Drug Ordering screen.

  1. Select the prescription medication for which you just created the new Take As Directed package.
  2. Select the new Administration Rule and be sure to select the Titrate/Special Dose as the package.
  3. Select OK to order the medication. The Drug Ordering screen appears.With the Take As Directed package, there is no dosing information. This will be handled by creating default instructions for this drug.
  4. Select the Special Instructions gray box then select Patient Instructions. The Patient Instructions screen appears.
  5. Enter the dosing instructions in the white text area then select OK. The text appears in the Special Instructions box.
  6. Select the Special Instructions box again and select Set as Default.
  7. Select the Set Default Patient Instruction option.
  8. Select the Set Default for Drug/Package option.
  9. Select Finished to order the medication.
