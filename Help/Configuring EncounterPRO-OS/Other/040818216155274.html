How to: Add a Health Maintenance Rule

Purpose
EncounterPRO is delivered with many common Health Maintenance Rules (HMR) already configured.
A procedure must be attached to each rule to activate the schedule. If you want this health maintenance rule to apply only to those patients with specific assessments, you need to attach the assessments to the rule.

Before You Begin
Decide which procedures will 'satisfy' the rule and which assessments or criteria will flag a patient as eligible for the rule.

Details:

Steps

Follow these steps:

  1. Select Configuration from the Office View. The Configuration List appears.
  2. Select Health Maintenance Rules. The Health Maintenance Rules screen appears.
  3. Select New Maintenance Rule. Enter a description for the new rule. This description appears on the Patient Chart/Maint tab.
  4. Select OK. The Health Maintenance Rule criteria screen appears.
  5. Enter the HMR criteria:
    • Sex
    • Race
    • Age Range
    • Interval
    • Warning Days
    • Attach Assessments
  6. Select OK. The new rule appears at the bottom of the Health Maintenance Rules list.
  7. Select the bar for the new rule. A menu appears.
  8. If the new rule applies to patients with a specific diagnosis or set of diagnoses,
  9. Select Diagnoses. The Health Maintenance Assessments screen appears. 
    1. Select Add Assessment. The Select Assessments screen appears.
    2. Select all assessments that this new rule applies to. Each will appear in the Selected Items list.
    3. Select OK. The Health Maintenance Assessments screen reappears with a list of the assessments for this rule.
    4. Select OK. The Health Maintenance Rules screen appears.
  10. Select the bar for the new rule. A menu appears.
  11. Select Procedures. The Health Maintenance Procedures screen appears.
    a. Select Add Proc. The Select Procedures screen appears.
    b. Select the primary procedure that satisfies the new rule. The Health Maintenance Procedures screen reappears.
    c. If there is more than one procedure that will satisfy the rule, select Add Proc and repeat step b until all procedures have been selected and appear on the list.

    NOTE: When one of the procedures in the list has been performed, that will satisfy the list and re-set the interval. If more than one procedure must be performed, you will need to create a new HMR for each procedure.
    d. Select OK. The Health Maintenance Rules screen appears.
  12. Select OK. The Configuration List reappears.
