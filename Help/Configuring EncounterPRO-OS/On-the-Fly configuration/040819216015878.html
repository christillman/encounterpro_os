How to: Modify an Existing Drilldown Observation

Purpose
Drilldown screens are used throughout EncounterPRO. They are all configured similarly regardless of the screen type - PFSH, ROS, Physical Examination, or other observation. You can alter existing observations easily 'on the fly.' In fact, modifying observations is best done in this manner, rather than through the configuration tab, since you can quickly check your work to make sure you have made the modifications as desired.

Before You Begin
You should already be familiar with drilldown screens. See Screen: Drilldown screen for details.

Most drill down observations consist of one or more composite observations which in turn are composed of several simple observations. When using the drill down screen, you will drill down the levels of the composite observations to the simple observations, at which point you select or enter results.

Details:

Steps
Follow the steps below:

  1. From the drill down screen, drill down to the level you want to change.
  2. Select the Other Options button. A menu of options appears. Select the Configure button. If the observation you want to modify is a composite observation (multi-level) the Composite Observation Definition screen appears.
  3. If you want to change one of the constituent observations, select the bar with the observation name. A menu of options appears.
    NOTE: If it is a composite observation (an * appears to the right of the observation name) select Cancel to return to the observation data entry screen and drilldown further to reach the level you want to modify. Repeat step 2.

  4. At this point, you can do any of the following:
    • Modify the observation name or update the available results. Select the Edit Observation button, go to Changing the Available Results below.
    • Change the appearance/format or behavior.  Select the Attributes button. The Edit Service bar determines the format of the observation. Go to Changing Observation Attributes below.
    • Delete the observation from the composite.  Select the Remove Observation button
    • Change the order of the child observations. Select the Move button and move the selected observation up or down in the list
  5. Select OK to save your changes and return to the original observation data entry screen.

Changing the Available Results
From the Edit Observation screen you can change the description, select collect or perform procedures, or alter the available results.

  1. From the Observation Definition screen select the gray Results box. The Results Definition screen appears.
  2. If you want to remove a result, select the applicable result on the list at the left and select the Delete button.
  3. If you want to change the order in which the results display, select the result you want to reorder, select the Move button, then move the result up or down as needed.
  4. If you wish to use a pre-set result set you must first delete all existing results (see step 2). Then select the Add Result Set button and choose the desired result set from the list.
  5. If you want to add a new result, select the New Result button.

    a.  Type the result text in the Description field.
    b.  Select the Result Type: Normal or Abnormal (noteworthy)
    c.  Select whether or not you want the Result Title to appear on charts and reports. Select Yes or No.
    d.  If the result will be a selection only and does not require the user to enter a value (text, date, or number) select No for Result Has Value.
    e.  If the result requires the user to enter a value, select Yes for Result Has Value. Select the type of value that will be entered:

    • Text
    • Date
    • Numeric. If numeric choose the Unit.

    f.  If the Result Type is Abnormal (step B), then select the Severity level:

    • N/A (result is notable, but not necessarily abnormal)
    • Normal
    • Borderline
    • Abnormal
    • Severe

      g.  Enter the text description of the normal range for this result in the Normal Range field. This is text only and will not be used to calculate any abnormal or severity flags. It is very useful for lab results and appears on the results table in the Normal Range column.
      NOTE: If you want to set ranges for this results that will be used to trigger flags and severity levels, see Screen: Setting Observation Result Ranges.

    • Repeat steps 2 - 5 as needed to make the result set for this observation complete and as desired.
    • If you want to save this set of results for future use in another observation, select the Set Result Set button.

      a.  Choose Create a new result set.
      b.  Enter the name of this new set and select OK. The name of the new set is added to the bottom of the Result Set list.

      • Select OK to save the results. Select OK again to return to the list of observations. Select OK again to return to the original drilldown data entry screen. (Depending on how many levels you drilled down, you may need to select OK additional times.)
      • Check your modifications by selecting the observation you just changed. Your changes should appear. Note: Other users will not see any change to the observation until they have closed and re-started EncounterPRO.

      Changing the Observation Attributes
      From the Edit Observation Attributes screen you can change the way the observation displays and behaves.

      1. From the Observation Attributes screen 
      2. To control what happens when the user selects a result for the observation, select one of the following options for On Result:
        • Move to the Next (simple observation in the composite)
        • Move Up one level
        • Do Nothing. If the user can select more than one results for the observation, be sure to choose Do Nothing so that the screen does not advance or go up one level before the user completes their data entry.
      3. To determine the display format of the observation, select one of the options for Edit Service. Commonly used displays are:
        • Edit ROS Yes/No
        • Edit ROS Drilldown
        • Edit Physical Yes/No
        • Edit Physical Drilldown
        • Edit Vital Signs
        • Edit PFSH Drilldown
        • Edit PFSH Yes/No
        Generally you will choose a service that begins with the word, 'Edit' and names the type of observation you are modifying (physical, ROS, Vitals) and then the display format (Drilldown or Yes/No).
        NOTE: If you want the observations to display in a Yes/No format, the entire Composite Observation must be Yes/No.
      4. If you want to set a default unit preference for this observation, select Unit Preference and select one of the options.
      5. When finished, select OK to save your changes and return to the Composite Observation Definition screen. Select OK to return to the original drilldown data entry screen.
