How to: Maintain User Roles

Purpose

User roles can be maintained by Super Users at a practice. Roles can be:

  • Added
  • Disabled
  • Enabled
  • Given a new description (the Role Name cannot be changed, only the description)
  • Assigned a new color

Before You Begin

It is important to be very careful when disabling any of the user roles delivered with EncounterPRO as there may be tasks in a workplan that are assigned to that role. If a user role is disabled and the system has a task assigned to that role, a task is created for the Error Handler role to determine what should be done with that task.

Details:

Steps
Follow these steps:
To add a new user role:

  1. From the Office View select the Configuration tab. The Configuration list appears.
  2. Select Users. The Select User list appears.
  3. Select Roles at the top left of the screen as the Actor Type.
  4. Select New Role.
  5. Enter the name for the new role and select OK. The Role Definition screen appears.
  6. Enter a description for the new role in the Description box.
  7. Select the white bar below Color and select the color you want to assign to this role.
  8. Select OK. The Select User screen reappears.

To modify an existing user role:

  1. From the Office View select the Configuration tab. The Configuration list appears.
  2. Select Users. The Select User list appears.
  3. Select Roles at the top left of the screen as the Actor Type.
  4. Select the role you want to modify from the list. Use the Search Options if needed to locate the desired user role.
  5. Select the role from the list. The Role Definition screen appears.
  6. You can change the following:
    • Color
    • Description
    • Status
  7. When finished, select OK. The Select User screen reappears.


To disable or enable a user role:

  1. From the Office View select the Configuration tab. The Configuration list appears.
  2. Select Users. The Select User list appears.
  3. Select Roles at the top left of the screen as the Actor Type.
  4. If you want to enable a previously disabled user role, select the Active button for Status to change to Inactive, otherwise leave it as Active. (Disabled user roles will only appear on the Inactive list.)
  5. Select the role you want to enable or disable from the list. Use the Search Options if needed to locate the desired user role.
  6. Select the role from the list. The Role Definition screen appears.
  7. Select the Status button to toggle from Disabled to Enabled.
  8. When finished, select OK. The Select User screen reappears.
