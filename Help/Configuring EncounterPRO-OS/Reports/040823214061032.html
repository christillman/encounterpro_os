How to: Import a New Report from a File

Purpose

EncounterPRO version 5 offers a second method of adding reports to your system. In addition to the Reports Library, reports can be imported from a special file created when a user exports an EncounterPRO report to a file. The file type is .jmjcfg.

Shareware Reports

EncounterPRO users who have developed their own reports can choose to share those reports with the EncounterPRO community by posting their exported report files in an accessible location. It is preferred that the reports are shared via the Reports Library. See How to: Import a Report from the Reports Library.

CAUTION: Shareware reports are not warranted or maintained by EHR Product Support. If there is a defect in a shareware report, there will be a fee charged if you wish EHR Product Support to troubleshoot or repair the report.

Details:

Steps
Follow these steps:
To import reports from a file:

  1. From the Office View, select the Configuration tab. The Configuration list appears.
  2. Select Reports. The Select Report screen appears.
  3. Select the Import from File button. A Select File screen appears.
  4. Navigate to the location of the report file (.jmjcfg) and select it.
  5. Select Open.
    NOTE: If the report has been created by an EncounterPRO user rather than EHR, Inc. the report will not be warranted or maintained by EHR Product Support.
  6. If you want to add the new report to the Common short list or your Personal short list you will need to search for the report using the Description or Category Search Options, then select the small icon to the left of the report name and select either Common or Personal Short List button.
  7. Repeat steps 3 through 6 to import additional reports.
  8. When finished, select Cancel to return to the Configuration list.
