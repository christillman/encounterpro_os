Triage Protocol: Prescriptions Printing With Wrong Format/Information

Triage Protocol: Triage Protocol: Prescriptions Printing With Wrong Format/Information
Overview:

Prescriptions are printing with incorrect format and/or information. Examples include text running off the page, signature missing or too small, wrong header information, etc.

Possible Causes

The one possible cause of printing problems is that something changed in the printer setup - a new printer was added, printers were moved, etc.  This type of issue will usually need to be sent to Level 2 to resolve.

Symptoms:
Resolution:

Get Additional Information

q  What printer are you trying to print to?

q  Ask caller to fax you 1-2 examples of the RX with the wrong formatting/information. Fax number is 770-919-9112

Send issue to Level 2
