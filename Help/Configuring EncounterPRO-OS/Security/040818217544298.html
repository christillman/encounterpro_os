How to: Inactivate an EncounterPRO User

Purpose
Rather than deleting users, you can change their status to Inactive. This applies to users no longer employed by the practice or those working on a temporary basis. You can re-activate the users after a waiting period, if needed. Users who are inactive cannot access the system.

Before You Begin
If you inadvertently inactivate a user, you will need to contact EHR Product Support to reactivate them before the waiting period expires.

Details:

Steps
Follow the steps below:

  1. From the Configuration tab, select Users. The Select User to Edit screen appears. The display defaults to Active users.
  2. Select the name bar of the user you wish to make inactive. You may need to use one of the Search options to locate the desired user. The User Definition screen appears.
  3. Select the Active Status button in the upper right of the screen to change it to Inactive. A message appears prompting you to confirm that you want to inactivate this user and notifying you that the user cannot be reactivated for a certain period of time, generally 7 days.
  4. Select Yes to confirm that you want to inactive this user. The User Definition screen reappears.
  5. Select OK to return to the Select User to Edit screen.
  6. Select OK to return to the Configuration list.
