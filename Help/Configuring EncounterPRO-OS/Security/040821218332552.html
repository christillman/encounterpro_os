How to: Restrict Access to Patient Records by Office Location

Purpose
If you have multiple office locations you can restrict specific users from being able to access patient records for specific office locations. When a user is restricted from viewing patient records at a specific office location, patients who have that office location as their primary office will not appear in searches or on screens such as Outstanding Labs and Tests.

Before You Begin
You must have the appropriate authority and permissions to configure users in order to perform this function.

Details:

Steps

  1. Select Configuration from the Office View. The Configuration List appears.
  2. Select Users. The Select User to Edit screen appears.
  3. Select the user you want to restrict access for from the list or use the Search options to locate the desired user. The User Definition screen appears.
  4. Select the Permissions tab.
  5. Select the office location for which you want to restrict this user from viewing patients.
  6. Scroll down the list of Office Privileges to the bottom of the list.
  7. Locate the privilege for User May View Patients in This Office. Select the Yes to change it to No.
  8. Repeat steps 5 through 7 if you want to restrict this user from viewing patients at additional office locations.
  9. When finished, select OK.
