How to: Configure a Preferred Room for a Specific Workstation

Purpose

In practices which have a permanent workstation in a treatment room it is useful to set the preferred room Computer Preference. This means that when a user logs in to that workstation, the preferred room is immediately displayed - saving 'clicks' since the user does not have to navigate from the Office View.

Before You Begin

Only users with specific permissions are authorized to update Preferences.

Details:

Steps

To configure a preferred room, follow these steps:

  1. Select Configuration from the Office View. The Configuration List appears.
  2. Select Preferences. The Preferences screen appears.
  3. Select Computer Configuration as the Preference Type on the left.
  4. Select Computer as the Preference Level on the bottom. The workstation you are currently logged into appears in the Preference Key.
  5. If you want to set the preferred room for a different computer, select the Preference Key and select the desired computer from the list.
  6. Select Preferred Room. A parameters screen appears.
  7. Select the Office location from the list on the left.
  8. Select the desired room from the Select Room list on the right.
  9. Select OK.
