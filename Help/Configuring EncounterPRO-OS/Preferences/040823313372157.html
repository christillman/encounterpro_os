Definition

Preferences allow the EncounterPRO Administrator or Super User to effect certain changes to the system's behaviors without resorting to programming. In many ways, the preferences will allow the practice to tailor the EncounterPRO system to their preferred method of documenting encounters. Certain preferences can be set at the user level, allowing each user in the practice to use their favorite screens and display options.

NOTE: Only the EncounterPRO Administrator or Super User, under the direction of EHR Product Support should make any alterations or changes to these settings.  The performance of the system can be dramatically changed by alteration of the preferences.

Preference Types
There are five Preference Types:

  • Attachment Preferences
  • Billing Preferences
  • Color Preferences
  • Computer Configuration
  • EpIE Preferences
  • For Future Use
  • General Preferences
  • Maintenance Preferences
  • Msg/Task Preferences
  • Prescription Preferences
  • Reports/Documents Preferences
  • Security Preferences
  • Server Configuration
  • User Favorite Preferences
  • Version 2,4 Only Preferences
  • Workflow Configuration

Preference Levels
Each of the preference types can be applied to up to five different levels. The level controls the scope of the configuration change being made. The levels are:

  • Global -- the preference applies to the entire network and will be the same on all office workstations at all locations.
  • Office -- the preference applies to the entire office location and will be the same at all workstations at that location. The bar below the Level bar displays the office location for the current workstation. A different office location can be selected, if needed.
  • Computer -- the preference applies to the current workstation (the computer the user is logged into) and will apply to all users logging on to that workstation. The bar below the Level bar changes to Computer and identifies the name of the computer at this workstation. A different workstation can be selected, if needed.
  • Specialty -- the preference applies to users with the same specialty only. The bar below the Level bar will display the specialty of the current user. A different specialty can be selected, if needed.
  • User -- preference applies to this user only. The bar below the Level bar will be labeled User and will default to the name of the user currently logged on. Another user can be selected, if the preference needs to be changed for someone other than the user logged in.

Preference Hierarchy

The lower levels will override the preferences set a the higher levels. That is, the Global preference can be set one way and if the Computer preference is different, the system will use the Computer preference on that workstation. The hierarchy is represented on the Preferences screen as the levels go from Global on the left (highest) to User on the right (lowest).

NOTE: Not every preference can be set at every level. Italic writing means that preference is not available at the current level.

Examples of Commonly Used Preferences
The follow lists are some of the more commonly changed preferences in the EncounterPRO system. There are many more preferences available which do not appear in the lists below. Download this Spreadsheet of Preferences to get a list of all the preferences by Preference Type with levels available and description.

Attachment Preferences

  • Allow Folder Delete - allow users to delete attachment folders
  • Attachment Default Folder (context) - the default folder in which to put new documents and attachments when the folder is not otherwise specified

Billing Preferences

  • Default Encounter Type - set the default encounter type for encounters created via the PMS integration

Color Preferences

  • These preferences should not be changed by EncounterPRO users

Computer Preferences

  • Audio Priority Sound - enable the audible alert for high and urgent priority messages and tasks
  • Auto Logoff When Idle - determine the length of time a workstation can be idle before the system automatically logs the user out
  • Preferred Room - set a preferred treatment room in EncounterPRO as the preferred room for a specific workstation
  • Window Resizable/Start Maximized - enable the window resizing options

EpIE Preferences

  • These preferences should not be changed by EncounterPRO users

For Future Use

  • These preferences should not be changed by EncounterPRO users

General Preferences

  • Configuration Mode - turn on/off Config Mode
  • Default Search Mode  - set the search mode to Begins with or Contains
  • Logon Screen Browser URL - set a website as the default for the EncounterPRO logon screen display
  • Enable Scribing - turn on scribing at the practice

Maintenance Preferences

  • Perform Daily Sync - perform the data sync operation during the daily maintenance routine
  • Purge Recent Picks - purge the freeform recent pick lists during the nightly maintenance routine
  • Purge Recent Picks Keep # - the number of recent picks to keep when the lists are purged.  Default is 30.

Msg/Task Preferences

  • Audio Priority Sound - enable/disable audio alert when office view contains high-priority service
  • Default Filter for My ToDo List - the default service to filter for on the My Todo Items screen

Prescription Preferences

  • RX Frequency Display - show only the code (e.g. BID), or description, or both code and description
  • RX Signature Prompt - Text of the prompt for authorization and verification of prescription
  • Display RX Dispense Amount Textually - display dispense amounts both textually and numerically (e.g. fifty three (53))  or display dispense amounts only numerically.

Security Preferences

  • Allow Multiple Logons - allow/restrict users to log on to more than one workstation simultaneously

Subscription Preferences

  • Preferences used for any external service the preference is subscribed to

User Favorite Preferences

  • Favorite Chart Display Mode - determine how the default Patient Chart screen displays treatments and assessments
  • Favorite Observation (CC, HPI, ROS, Physical, etc.) - select the exact observation (screen) that will be used for that type of observation
  • Favorite Display Mode for Observation - determine the format of the favorite observations, drilldown or Yes/No

Workflow Preferences

  • Favorite Role for Scheduling Referrals - role of the staff personnel who will be responsible for scheduling referrals
  • Supervisor Review - determine if supervisor review is mandatory

NOTE: Some preferences will be apparent to the user making the change immediately, others require you to restart the EncounterPRO EHR. Other users in the practice will generally need to restart the EncounterPRO EHR.
