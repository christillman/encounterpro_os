How to: Default the Task List to Show All Items.

Purpose
The kinds of Tasks that are displayed on the Task List are controlled by Msg/Task Preferences. The default display is to show EncounterPRO Tasks, not All.

TIP: A common situation with the default showing only EncounterPRO Tasks is that users are confused and unable to locate their tasks: Where are my Task Items? The tab says I have 63 items, but nothing is displayed!

Before You Begin
You can set this Preference globally (if you have sufficient permissions) which will change the setting for all users in the practice. Be sure to consult with the other providers in the practice before making global preference changes.

Details:

Steps
Follow the steps below:

  1. From the Configuration tab, select the Preferences tab. The Preferences screen appears.
  2. Select Msg/Task Preferences on the left.
  3. You can set this default for the entire office, the computer, or the individual user. Select the level you want to change the preference for at the bottom of the screen:
    • Global - changes the preference for all users
    • Computer - changes the preference for all users who log in to the current workstation
    • User - changes the preference for the current user only, is effective on all workstations
  4. Select Default Filter for My ToDo List.
  5. If you want the default to show All tasks, select the Clear button. If you want the default to show a specific type of task, select the gray bar and then select the type of task.
  6. Select Finished. The Preferences screen reappears.
  7. Select OK to return to the Configuration list.
