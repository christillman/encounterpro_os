How to: Set the New Assessment Sort Preference

Purpose

EncounterPRO will automatically order the assessments on the Patient Chart and on the Billing either ascending (first chosen in today's encounter is at the top) or descending (last chosen in today's encounter is at the top). This is determined by a General Preference, New Assessment Sort. The default is to sort assessment ascending.

Before You Begin

Since this preference is only available at a Global level, the preference chosen will affect all EncounterPRO users at all locations for your practice. Be sure that you have discussed this change with the providers and staff before changing the default sort.

Details: Steps
Follow these steps:
  1. From the Office View, select the Configuration tab. The Configuration List appears.
  2. Select Preferences. The Preferences screen appears.
  3. Select the General Preference on the left and a list of available preferences is displayed.
  4. This preference is only available at the Global level.
  5. Select the New Assessment Sort preference from the available list. A preference parameters screen appears.
  6. Select the sort order that you want to use:
    • Ascending - new assessments appear in the same order in which they are ordered during the encounter
    • Descending - new assessment appear in reverse order (last ordered appears first)
  7. Select Finished. The Preferences screen reappears.

NOTE: Some preferences will be apparent to the user making the change immediately, others require you to restart the EncounterPRO EHR. Other users in the practice will generally need to restart the EncounterPRO EHR.
