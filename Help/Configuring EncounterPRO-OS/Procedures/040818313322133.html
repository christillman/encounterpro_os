How to: Add Patient Send Out Options to Collect Procedures

Purpose
Collect Procedures delivered in the database have the "specimen sent to:" and the "patient referred to" option at the top of the collect specimen procedure.

After you receive your database, if you create any new labs, test, x-ray, etc templates in your database, the following steps needs to be performed for each collect procedure that you create.

Before You Begin
You can configure these options from the actual treatment observation screen using a test patient or go through the Configuration screens. It is often easier to modify observations in a patient context as you can immediately see the results of your changes.

Details:

Steps
Follow the steps below:
To Modify the Observation from a Patient Context:

  1. Check a test patient into the office. See How to Check in a Patient for detailed instructions.
  2. Go to the Patient Chart and select Other Items. A menu of additional options appears.
  3. Select Any Treatment. A menu of treatment types appears.
  4. Select the treatment type for this item. For example, Lab, Test, or X-ray. The selection screen for the treatment type you chose appears.
  5. Select the lab or treatment from the shortlist, if the treatment is not listed there use your search options to locate the treatment to be modified. You can select multiple treatment of this type if needed. Each item selected appears in the Selected Items list.
  6. Select OK. A service bar for the collection appears on the Office View.
  7. Select Exit from the Patient Chart, then I'll Be Back. The Office View appears.
  8. Select the service bar for the collection. It will be labeled "Collect or Refer Out Treatment." The collect observation appears.
  9. If this observation is missing either the Patient Referred to or Specimen Sent to options, select the Other Options button at the bottom of the screen.
  10. Select Configure. The Observation Definition or screen appears.
  11. From the Observation Definition screen, select the Perform button. A list of options appears.
  12. Select Property. The Results and Locations boxes are replaced with a list of properties for this observation.
  13. Select in the gray box for Properties. The Properties screen appears.
  14. Select New Results.
  15. Select the EncounterPRO Property button and choose User Defined Patient Property. A list of patient properties appears.
  16. Select Patient Referred to: then select OK.
  17. Repeat steps 14 and 15 and select Specimen Sent to, then OK.
  18. Select OK. The results definition screen reappears.
  19. Select OK. The Observation Definition screen reappears.
  20. Select OK. The collection observation reappears with the new options at the top of the screen.

    NOTE:  Since you are configuring the observation you will see the results of your changes immediately. Other users will need to log out and log back into EncounterPRO to see the changes.

To Modify the Observation from the Configuration Screens:

  1. From the office view screen select the Configuration tab. The Configuration list appears.
  2. Select Observations from the list. A list of treatment types appears.
  3. Select the treatment type for this item. For example, Lab, Test, or X-ray. The Treatment Type you select appears as the Treatment Type for the search options.
  4. Select the lab or treatment from the shortlist, if the treatment is not listed there use your search options to locate the treatment to be modified. A menu of option appears.
  5. Select Edit Observation. The Observation Definition or Composite Observation Definition screen appears, depending on the treatment selected.
  6. From the Observation Definition screen, select the Perform button. A list of options appears.
  7. Select Property. The Results and Locations boxes are replaced with a list of properties for this observation.
  8. Select in the gray box for Properties. The Properties screen appears.
  9. Select New Results.
  10. Select the EncounterPRO Property button and choose User Defined Patient Property. A list of patient properties appears.
  11. Select Patient Referred to: then select OK.
  12. Repeat steps 9 and 10 and select Specimen Sent to, then OK.
  13. Select OK. The results definition screen reappears.
  14. Select OK. The Observation Definition screen reappears.
  15. Select OK. The Select Observation screen reappears.

    NOTE: Since you are configuring the observation you will see the results of your changes immediately. Other users will need to log out and log back into EncounterPRO to see the changes.
