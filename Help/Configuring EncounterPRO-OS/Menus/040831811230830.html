How to: Add No RX and Meds Revd Buttons (Progress Notes)

Purpose

It can be useful to have 1-touch progress notes on the Allergies and Current Medications screens so that the clinical staff can document that they have reviewed the information with the patient. The 1-Touch buttons that are useful on the Current Medications screen are:

  • 1-Touch No RX - the patient reports no medications
  • 1-Touch Meds Rev'd - the current medications list has been reviewed and updated

Before You Begin

Adding the button for the progress note is only half of the procedure. Editing the display script for the report or screen that you want to show the progress note is required, otherwise you will not see the progress notes.

IMPORTANT: If you are not familiar with editing display scripts, do not attempt to perform this procedure. Contact EHR Product Support for assistance. 

 

Details:

Steps
Follow the steps below:

To add the menu button (this procedure is best done using a Test or Practice patient):

  1. Turn Config Mode On.
  2. Display the Current Medications screen.
  3. Select the menu display name. The Menu Properties screen appears.
  4. If a Local Copy does note already exist, select Create Local Copy.
  5. Select Edit Menu.
  6. Select New. The Menu Item Properties screen appears.
  7. Select Service, then Patient. A list of Patient-level services appears.
  8. Select New Note. The parameters screen appears. Enter the following:
    • Context object = Encounter
    • Standard Note Type = Blank
    • Special Note Type = Medication Check
    • Preconfigured Treatment Titles or Keys = Blank
    • Other Note Title or Key = No RX (This is the title of the note.)
    • Note Text = The patient/parent states that he/she is taking no medications. (This is the text of the note.)
  9. Edit the name of the button to reflect its function. Suggested name: 1-Touch No RX.
  10. Select Finished. The Menu Properties screen reappears.
  11. Repeat steps 6 thoguth 8 using the following parameters:
    • Context object = Encounter
    • Standard Note Type = Blank
    • Special Note Type = Medication Check
    • Preconfigured Treatment Titles or Keys = Blank
    • Other Note Title or Key = RX Reviewed (This is the title of the note.)
    • Note Text = Current medications list reviewed and updated. (This is the text of the note.)
  12. Edit the name of the button to reflect its function. Suggested name: 1-Touch Meds Rev'd.
  13. Select the new button bar on the left, then select the Move button.
  14. Use the arrow keys to re-order the menu buttons as desired.
  15. Select OK. The Menu Properties screen reappears.
  16. Select Finished. The Current Medications screen reappears. You may need to refresh the screen to see the new button.

NOTE: Adding the button will NOT automatically make the progress note visible. The display scripts for the desired reports and screens must be edited in order for the progress note to display. If you are not familiar with editing display scripts, contact EHR Product Support for assistance.

To edit the display script (this procedure is best done using a Test or Practice patient):

  1. Turn Config Mode On.
  2. Display the screen or preview the report on which you want the new progress note to appear. Suggestions: The Quick View Summary, the SOAP Note report, etc.
  3. Scroll down on the display to locate the Allergies section of the display.
  4. Right click in the Current Medications sub-title. The Display Script screen appears with the name of the display script as the title.
  5. If a local copy does not already exist an Edit button appears in the upper right of the screen. Select Edit to create an editable local copy of the display script.
    NOTE: Certain reports and display are no editable from a patient context. If that is the case, the Edit button will not appear on the screen and you will not be able to edit the current display script. Note the exact name of the display script and leave the encounter. Go to the Configuration tab and select Display Scripts. Search for the exact display script name you recorded, then select the bar for that display script and the Edit button. Make the changes as described below, then return to the patient encounter to check your work.
  6. Scroll down to find the highlighted (darkened) command for the Current Medications sub-title.
  7. Add the following commands to the display script:
    • Format = fontsize=1-,xbold,margin=0/0/0
    • Encounter Progress
      • Note Type=Medication Check
      • Check Current Encounter Only=Y
      • Display Stype=Standard
  8. Your script should look much like this:

  9. Select Finished. The viewable screen or report appears.
  10. Go to the Current Medications screen and select the 1-Touch No New button you added previously.
  11. Return to the edited display script. The progress notes should appear in the Current Medications section of the report/screen.
