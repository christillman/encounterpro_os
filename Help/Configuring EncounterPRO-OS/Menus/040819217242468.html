Purpose
A practice may decide that they want to modify the exiting EncounterPRO menu on a specific screen. Adding buttons to or changing the existing buttons is fairly straightforward. Remember that when you change the menu buttons, that change will affect all the users in the practice.

NOTE: The changes made to a menu will not appear for the other EncounterPRO users until they close and re-start EncounterPRO.

Before You Begin
The easiest method of customizing menus is to start from the screen containing that menu. Turn on Configuration Mode.
If you plan on adding a button you will need to know a little information about the function the button will activate.
Is the button you want to add a:
Service
Services are General or Patient-level. Services generally either display a new screen or perform a function. See Services
Treatment Type
Add a treatment type button if you want to be able to display a screen that allows the user to select a particular treatment type. See Treatment Types
Treatment
Add a Treatment button if you want to have a button that orders a particular treatment. You will select the treatment type, then the specific treatment. For example, you can create a button that orders a Hematocrit lab or one that orders a Sports Physical.
Treatment List
Add a button to access a Treatment List if you want a menu of treatments or treatment selection list to appear.
Menu
Add a button for a menu to display a secondary menu. For example, you can add a Print Reports button on a screen that will display the Reports menu.

Some menu names are a little tricky to understand from the names in the list. You may need to locate that menu and find the menu name or select the space to the left of the menu bar and select Edit to display the Menu Properties to see which buttons are contained in that menu.

Details:

Steps
Follow the steps below:

  1. Navigate to the screen displaying the menu you want to change. Notice the small text next to the menu buttons, this is the system name of the menu.
  2. Select the small text with the name of the menu to access the menu configuration screen.
  3. All menus have a JMJ Version that you cannot edit. If you want to change a menu, you must first create a local copy. Select the Create Local Copy button.
    NOTE:  If you later decide that you want to revert to the JMJ Version, select the Local Copy and select Inactivate, then select the JMJ Version and select Activate.
  4. If you already have a Local Copy, be sure that bar is highlighted (darkened), then select the Edit Menu button. The Menu Properties screen appears. From this screen you can add new buttons, remove unwanted buttons, and change the order of the buttons.

Removing a Button from a Menu

  1. With the Menu Properties displayed, select the button you want to remove.
  2. Select Delete. The system will prompt you to confirm that you want to delete the selected item. Select Yes.

Adding New Button

  1. Select New. A selection box appears for you to choose the type of button function. Each button function falls into one of five categories:
    • Service
    • Treatment Type
    • Treatment
    • Treatment List 
    • Menu
  2. If you want to create a button for a service select Service, then:
    1. A list of service types appears. Select General or Patient. A list of services for the level chosen appears.
    2. Select the desired service (example: Chart Export). If there are further parameters to define for this service a screen appears to allow you to enter the desired parameters. Select Finish. The Menu Item Properties screen reappears.
    3. At this point, you may want to change the Button Title (this appears on the menu). When done making changes, select Finished. Your new button appears as the last button on the menu.
  3. If you want to create a button that will allow the users to select a treatment from a list (such as a list of labs), select Treatment_Type. A list of treatment types appears.
    1. Select the treatment type.
    2. At this point, you may want to change the Button Title (this appears on the menu). When done making changes, select Finished. Your new button appears as the last button on the menu.
  4. If you want to create a button that will allow order a specific a treatment (such as a Urinalysis). Select Treatment. A list of treatment types appears.
    1. Select the treatment type for the desired treatment
    2. Select the desired treatment from the list.
    3. The system will prompt you to decide if this treatment will be a past treatment or if you want it to order a new treatment. Select No if you want the button to order a New treatment.
    4. At this point, you may want to change the Button Title (this appears on the menu). When done making changes, select Finished. Your new button appears as the last button on the menu.
  5. If you want to create a button that will display a menu of treatment types for the user to select (much like the Any Treatment button displays), select Treatment_List. A list of treatment lists appears.
    1. Select the treatment list.
    2. At this point, you may want to change the Button Title (this appears on the menu).
    3. When done making changes, select Finished. Your new button appears as the last button on the menu.
  6. If you want to create a button that will display another menu for the user to select (much like the Pick Reports or Add History buttons on the Patient Chart displays), select Menu. The Select Menu screen appears.
    1. Select the menu from the short list or use the Search Options to locate it.
    2. At this point, you may want to change the Button Title (this appears on the menu).
    3. When done making changes, select Finished. Your new button appears as the last button on the menu.
    TIP: To determine the exact menu name, display a screen that contains the desired menu and, while in Config Mode, display the menu and make a note of the system menu name.
  7. To reorder the buttons on the menu, select the button, then select the Move button. Use the arrow buttons to move the button into the desired position. When finished moving buttons, select OK, then Finished. The screen reappears with the new button.

Changing the Button Icons

  1. With the Menu Properties displayed, select the button for which you want to change the icon (picture).
  2. Select the Edit button. The Menu Item Properties screen appears.
  3. Type the filename for the new button icon in the Button field.
  4. Select Finished. The Menu Properties screen reappears. A small thumbnail version of the new button icon appears.
  5. Select OK, then Finished to return to the original screen. The new button icon is displayed.

    TIP: You need to know the exact system filename of the button icon. If there is another menu that uses the icon you want to use, display that menu, turn On Config Mode, then right-click the button with the icon you desire. The button filename is displayed.
