Description
Use this screen to configure a new assessment or modify an existing assessment. The screen has several tabs at the bottom to divide up the information required.

  • Definition - the basic information about this assessment including the ICD-9 code, Category (used for searches), and Location Domain
  • Long Description - the longer description of the assessment which appears on the properties dialog and can appear in certain reports
  • Acuteness/Auto Close - the designation of the assessment as acute or chronic and the auto close rules, if any are desired
  • Health Maintenance - reserved for future use
Details:

How to Access This Screen
Access this screen by selecting an Assessment on the Select Assessment to Edit screen, then selecting Edit Assessment from the menu that appears.
