How to: Set Treatment Mode to Create High Priority Task for Abnormal Lab Results

Solution:

This feature automates a visual cue that alerts the EncounterPRO users that there is a high priority Task. Makes it easy to find high priority Tasks on the Task list.

The Tasks tab displays visual cues when the user has pending tasks that are high priority. The treatment workplan definition is able to either:

  • include Abnormal Results on the Edit Criteria screen, OR
  • allow the setting of different priority for normal/abnormal on the Edit Flags screen

 

Details:

Moved 'Abnormal Flag' and 'Severity' from the Edit Flags screen to the Edit Criteria screen. Setting 'Abnormal Results Present' to 'Yes' will cause the workplan item to be skipped unless abnormal results are present for that treatment. Setting 'Abnormal Results Present' to 'No' will cause the workplan item to be skipped even if abnormal results are present.

1) Go to the Treatment Type Configuration screen.

2) Locate the treatment type, for example Lab.

3) Select the Treatment Modes tab. The list of valid treatment modes for this treatment type appears.

4) Select the treatment mode you want to update, select Edit from the menu. The workplan for the selected treatment mode appears.

5) If needed, select Create Local Copy then select Edit Workplan.

6) If there is already a step in the workplan that creates a review Task, select that Step and Edit Step. Otherwise, create a new step for the Review.

7) Edit the Criteria for the Review Service. Select Yes. This will cause this step to only be launched if abnormal results are present for that treatment (abnormal as set in the observation result definition - see How to: Configure Result Ranges).

8) Select the Minimum Result Severity from the drop-down list of severities. This will cause this step to only be launched if the severity of the abnormal result for that treatment is equal to or greater than the severity level chosen.

9) Select OK. Select OK again to return to the Workplan Definition screen.

10) Select OK to save your changes.
