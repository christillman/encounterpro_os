Purpose

Audit logs allow the Practice Manager, Super Users, and EHR personnel to track activity on the encounterPRO system. Permission to view the audit logs is restricted.

Audit Logs Available

There are several types of audit logs available:

  • By User - review all activities on the system by a specific user for a specific period of time
  • By Encounter - review all activities performed during a specific encounter
  • By Practice - review all activities performed on the system for a specific period of time
  • By Treatment - review all activities associated with a specific treatment for a specific patient

 

Details:

Steps
Follow these steps:
To view the audit log for a specific user:

  1. From the Office View, select the Configuration tab. The Configuration List appears.
  2. Select Users. The Select User screen appears.
  3. Select the user for whom you want to review the audit log. The User Definition screen appears.
  4. Select the Audit tab. All activities on the system for today are shown.
  5. If you want to review activities for a different date, use the arrow keys to go back or forward one day at a time. You can also select the date to display a calendar that allows you to select a specific date.
  6. When finished reviewing the information, select OK to return to the Select User screen.
  7. Repeat steps 3 through 6 to review activities for additional users.

To view the audit log for the entire practice:

  1. From the Office View, select the Utilities tab. The Utilities List appears.
  2. Select Audit Records. The Audit Records screen appears. All activities on the system for today are shown.
  3. If you want to review activities for a date range, select the Audit From and Audit To Dates. You can use the arrow keys to go back or forward one day at a time. You can also select the date to display a calendar that allows you to select a specific date.
  4. When finished reviewing the information, select OK to return to the Select User screen.

To view the audit log for a specific encounter:

  1. Display the Patient Demographics screen for the desired patient.
  2. Select the Open Encounters tab. A list of open encounter (if any) appears.
  3. Select All Encounters. A list of all encounters for this patient appears.
  4. Locate the encounter for which you want to view the audit log in the list of previous encounters on the left of the Patient Chart.
  5. Select the colored bar for that encounter. A menu of options appears.
  6. Select Encounter Audit. The encounter audit log appears. All activities on the system for the selected encounter are shown.
  7. When you are finished reviewing the information, select OK. The Patient Demographics screen reappears.
  8. If you want to review activities for a different encounter, repeat steps 4 through 6.

To view the audit log for a specific treatment:

  1. From the treatment dashboard for the treatment in question, select the Properties tab. The Audit list appears with the latest acitivites performed on this treatment at the top.
  2. When finished reviewing the information, select Finished.
