Description
While EncounterPRO has always documented the audit trails to preserve the integrity of the database, most audit information was only available through EHR Product Support. This screen displays the audit information for the entire practice by date. This log shows the following information for each event:

  • Date and time
  • User who performed the action
  • Office location
  • Type of event
  • Description of event
  • Patient the event was related to, if any

The date or a range of dates can be chosen using the Audit From and Audit To dates at the top of the screen.

How to Access this Screen

Access this screen from the Utilities list by selecting the Audit Records button.
