How to: Reactivate an encounterPRO User

Purpose
If you need to re-activate a user that was previously inactivated, you can do so after the waiting period has expired.

Details:

Steps
Follow the steps below:

  1. From the Configuration tab, select Users. The Select User to Edit screen appears. The display defaults to Active users.
  2. Select the Inactive Status button to change the display to show inactive users. The list now shows only those users who have been made inactive.
  3. You can use the search options to locate a specific user or select By Name and leave the search criteria blank to display all inactive users.
  4. Select the name bar of the user you want to re-activate. The User Definition screen appears.
  5. The Inactive Status button to change it to Active.
  6. Select OK. The Select User to Edit screen reappears.
  7. Select OK. The Configuration list reappears.
