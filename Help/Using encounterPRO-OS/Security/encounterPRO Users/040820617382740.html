Steps
To change your supervisor, follow these steps:

  1. Select Utilities from the Office View.
  2. Select Edit Supervisor. The Change Supervisor screen appears. The current supervisor for the currently logged in user appears.
  3. If you want to select a new supervisor, select the gray bar for Change Supervisor To. The Select User screen appears.
  4. Select a user to serve as your supervisor. The Change Supervisor screen reappears. (Note: You cannot select yourself as your supervisor or any user that you are currently supervising.)
  5. Select OK. The Utilities list reappears. This user will remain your supervisor until you select a new supervisor from the Utilities list or a different user is assigned as your supervisor on the User Definition screen.

NOTE: If you want to remove the supervisor entirely for a user, that must be performed on the User Configuration screen by an authorized user.
