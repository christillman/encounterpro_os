Description
This screen is used to change your EncounterPRO passcode. You will be prompted to enter the new passcode twice to ensure that you typed it correctly.
To change another user's passcode, authorized users use the User Configuration screen.

How to Access This Screen
Access this screen by selecting Change Password on the Utilities List.
