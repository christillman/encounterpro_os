Description
Use this screen to add new users and maintain information for existing users. Additionally, this screen allows authorized users to review the audit log for other users.
The user information is grouped onto five tabs:

  • Profile - basic information identifying the user to the encounterPRO system such as the username, password, specialty, etc.
  • Person - name and degree information
  • Contact - address, telephone, and email address information
  • Roles - the set of roles assigned to an individual user. Each role assigned to the user will also appear as a tab on their Task list.
  • Permissions - the special permissions (if any) assigned to an individual user
  • Routes - used with Document Management to determine which types of document routes are allowed for this user
  • ID Numbers - license numbers and other ID numbers
  • Audit - the complete record of all tasks performed by the user

How to Access This Screen
Access this screen by selecting Users from the Configuration List.
