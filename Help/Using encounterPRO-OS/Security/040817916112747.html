How to: Log into EncounterPRO

Purpose
Before opening EncounterPRO, you need to log in to your local computer first. Your local computer's operating system will either be Windows 95, Windows 98, Windows 2000, Windows XP or Windows NT.

Before You Begin
EncounterPRO controls access to the patient records through a personal identification number called a passcode. You need to know your passcode before logging on. See your system administrator if you do not know your passcode.

Details:

Steps
Follow the steps below:

  1. Double-click the EncounterPRO icon on the desktop.
    NOTE: This is the only time you will need to double-click while using EncounterPRO.

    Alternatively, you can single click the icon, then press Enter on the keyboard. EncounterPRO opens to a Secured/Non-Secured Log on screen.
  2. Select the Log on icon at the top right of the Login screen.
  3. Enter your passcode using the number pad that appears or the number pad on the keyboard. The Office View appears.

    NOTE: As an added security measure, EncounterPRO can be configured to log off automatically after you perform each function. If this happens, follow the above procedures to log back on.

Multiple Log-ins
There is a Security Preference that can be set that will prevent a user from having more than one active EncounterPRO session. If that preference is activated (Allow Multiple Logons = No), then the following steps can be followed:

If the user is currently logged in to EncounterPRO, then the system will prompt the user to either:

a. Cancel the login request
b. Override the lock-out by closing the existing session and starting a new session on the current workstation.
