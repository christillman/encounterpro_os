Course Description: encounterPRO Basics

Solution Title: Course Description: encounterPRO Basics
Overview:

This course is designed to introduce the functions in encounterPRO version 5 that are generally performed by all users in the practice, regardless of their role. This course is required for all encounterPRO version 5 users.

Audience: All users
Prerequisites: 
-- Introduction to encounterPRO-OS

Agenda:

  • Looking up a patient chart
  • Using Chart Alerts
  • Using encounterPRO Messages
  • Using Tasks
  • Changing your encounterPRO password
  • Documenting telephone encounters
  • Using the Holding Room (Chart room)
  • Using Document Manager
