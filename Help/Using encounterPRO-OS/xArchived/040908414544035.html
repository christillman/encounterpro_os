How to Find Release Notes

Solution Title: How to Find Release Notes
Overview:
The latest Release Notes are always in the same network directory:
\\dataserv\ehrifiles\documentation\ePRO-OS documentationelease_notes v5 documentationelease_notes
Sort the directory by Modified Date to find the most recently updated files. In general, when there are several releases, there will be a file for each. For example, there is a separate file for notes from 5.1 and another for 5.2 marked Release Notes 5.1 BETA and Release Notes 5.2 BETA.
Once there is a Production release, the files will be combined into a single file and the name of the file will include FINAL.
Where Else Can You Get the Release Notes?
The Release Notes will also be attached to the solution, "Release Notes for Version X.X" in the Solution Center. Search for "Release Notes" and you will find it. It is also in the FAQ list for the Solution Center and the Support Analyst Portal.
