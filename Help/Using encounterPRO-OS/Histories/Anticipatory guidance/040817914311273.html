Purpose
Most child development histories are configured into the clinical staff's workflow, therefore the screen will display automatically at some point during a well visit. Alternatively, the provider can access it from the Patient Chart by selecting the Dev tab.

Before You Begin
If any development stage information has been documented previsouly, and the child was behind in meeting milestones that information can be brought up-to-date.

Details:

Steps
Follow the steps below:

  1. From the Patient Chart, select the Dev/Growth Chart tab. The Growth Chart appears.
  2. Select the Development button. A new patient will have no information to display. If the patient has been seen previously, the existing developmental information is brought forward and displays in the gray area.
  3. Select the appropriate age from the list on the left, then select anywhere in the gray box on the right to display a menu of options. Select an option from the menu to add new results or review/change existing information.
  4. The development histories have a series of Yes/No questions as shown on the following screen. To set all answers to Yes (the normal value), click Set All. You can then select No for any answers as needed. Select Finished when done to return to the Development screen.
