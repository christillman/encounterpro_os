How to: Change from English to Metric Readings

Purpose
encounterPRO allows you to enter and display Vitals in either English or Metric units. If you are using English measurements, the button to the right of the Height field will read Eng; if you are using Metric measurements, the button will read Met.

NOTE: The preferred method can be set as the default for your practice.

Before You Begin
Which system is the preferred method for documenting Vitals at your practice, English or Metric?

Details:

Steps
Select the Met/Eng button to the right of the Height field to toggle between English and Metric measurements.
You can enter the measurements in English and then select the Eng button to change the display to Metric and vice versa.
