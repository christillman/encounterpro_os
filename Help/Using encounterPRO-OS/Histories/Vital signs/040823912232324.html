Description
The Growth Chart screens display the CDC Growth Chart for infants and children. The chart can display the graph points for several measurements:

  • Weight vs. Age
  • Length or Stature vs. Age
  • Length or Stature vs. Weight
  • Head circumference vs. Age (infants up to 24 months old)
  • BMI vs. Age (children older than 2 years old)

You can view the data collected at all encounters or only those collected during well encounters by selecting the Visit Type button. (A well encounter is defined by ordering a well assessment, not by the encounter type.)

Note: The CDC growth curve screens will only show a single data point in any two-week timeframe. The system will suppress multiple data points that occur in the same two-week time period because the CDC growth data is provided in half-month increments. The growth-percentiles screen available from the Data button does not suffer from this problem because it does not need to show CDC data. If you need to see the percentile increase/decrease for clinical reasons, use the percentiles screen.

How to Access This Screen
Access this screen by selecting the Dev tab on the Patient Chart, then selecting CDC Growth Charts.
