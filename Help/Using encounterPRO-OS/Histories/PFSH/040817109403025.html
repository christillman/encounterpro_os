Details: Description
This screen is used to display the patient's past medical, family, and social history (PFSH). This is a persistent history, meaning that the information entered builds and continues to display from encounter to encounter. This is in contrast to histories such as the Review of Systems which close at the end of an encounter and appear blank at the start of the next encounter.

Review the previously entered information in the gray text area. The bars on the left will indicate if information is entered for that category by the text; bold indicates information has been entered for the category, red text indicates abnormal results were entered. You have several options at this point:
  • If you want to note that you have reviewed the information, select the gray area then select the Reviewed with No Changes button.
  • If you want to add to or make changes to the information already entered, select the gray area then select the Reviewed with Changes button. The drilldown screen appears to allow you to add or alter information.
  • If there is no information entered, select the gray area then select New Results. The drilldown screen appears to allow you to enter new information.
How to Access This Screen
Access this screen by:
  • Selecting PFSH from the Add History menu on the Patient Chart
  • Selecting the small icon to the left of the PFSH description on the Patient Chart
  • Selecting the description (text) for PFSH on the Patient Chart, then selecting the description again on the dashboard screen
  • It can be automatically displayed as a step in the encounter workplan
