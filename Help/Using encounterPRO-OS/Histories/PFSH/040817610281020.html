How to: Document the PFSH.

Purpose
Documenting the PFSH (Past Medical, Family, Social History) is often part of the clinical staff workflow. This history is different from the HPI (History of Present Illness) in that the entries are cumulative and persistent. When information is entered in the PFSH, it will remain available for all subsequent encounters so that the clinical personnel can review this information easily.

Before You Begin
If you have a new patient intake form, you can use that to quickly and easily enter the information for the patient. Your practice will have guidelines about how much of the PFSH needs to be filled in.

If this is a subsequent visit, you can:

  • Review the information making no changes
  • Review the information and make changes
  • Add a comment
Details:

Steps
Follow the steps below:

  1. Display the PFSH screen. The screen defaults to show only the abnormal entries across all the history categories. A new patient will have no information to display.

  2. Select anywhere in the large gray box on the right. A menu appears with five options:
    • Review No Changes - Notes that you reviewed the history and did not make any changes. You will receive credit in the E & M coding for reviewing the history.
    • Review with Changes - Notes that you reviewed the history and opens the history observations so that you can make changes (corrections or additions) to existing information. You will receive credit in the E & M coding for reviewing the history.
    • New Results - Allows you to document new information within the histories for a patient. Use this option the first time you are entering PFSH information for the patient. You will receive credit in the E & M coding for documenting the history.
    • Enter Comments - Allows you to enter free text comments about the history. It can be used to enter additional information.
    • Cancel - Closes the pop up menu

  3. To enter information select the New Results or Reviewed with Changes button. The History drill down screen appears to allow you to fill in the appropriate information.

    NOTE: Some history screens will have one or more Standard Exam buttons (located on the top right corner). This button allows you to quickly populate all the normal values for the history in question. The values set will depend on how that standard exam button was configured. Select the Standard Exam button, if one exists, to populate the normal values for the patient. You can then use the drill down selections to change any items as needed for the individual patient.

  4. Select the drill down bar of the specific history to modify some of the answers. After you drill down and select your answer, use the four navigation buttons at the bottom of the screen to navigate through the remaining histories.
    • Top - Takes you to the top level of the history. If Top and Finished appear on the same screen, always select Top so that you can review your work before closing the history.
    • Up - Takes you up one level in the history
    • Next - Takes you to the next simple observation
    • Prev - Takes you to the previous observation

  5. When you are finished making your selections in the history screens, select Top. Review the information entered and repeat step 4 for any category you missed. When done entering information, select Finished. The PFSH screen reappears with the history information you selected displayed on the right side of the screen.

  6. Select I'm Finished to complete this service. The screen for the next service appears, or, if there are no more services in the workplan for your role, the Office view reappears.
