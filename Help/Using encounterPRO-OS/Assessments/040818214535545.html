How to: Close Assessments and Treatments

Purpose
After a patient has been diagnosed and treatments have seen ordered for that assessments, most of these items will remain open until they are manually closed by a provider or a staff member.

For example, a patient was diagnosed with sinusitis during last week's visit and a prescription was written for Claritin with multiple refills. When the same patient returns for a follow-up or for another office visit in few weeks, the sinusitis assessment and the prescription are still displayed on the Patient Chart because they are still open.

Treatments attached to an assessment will close when the associated assessment is closed. You can also close treatments while leaving the associated assessment open. For example, if you have a patient diagnosed with asthma, you can close certain treatments as you refine the treatment plan while leaving the assessment open.

Auto Closing Assessments
Well and Sick assessments can be set to automatically close after the encounter or after a specified number of days, months, or years. This is based on the preferences of the practice and is done as part of configuration. See How to Modify an Assessment for instructions.

NOTE: You can close multiple treatments simultaneously from the Patient Chart screen by holding down the CTRL key then clicking on the description of each treatment. A Close button appears at the bottom of the Chart screen. Select Close and all the selected treatments are closed.

Before You Begin
Your practice needs to determine which role has the responsibility for closing assessments: providers or clinical staff.

NOTE: If you inadvertently close a treatment that should remain open, you can reopen that treatment. See How to: Reopen a Closed Treatment. This is available only for closed treatments, not assessments. 

Details:

Steps
Follow the steps below:
To Close an Open Assessment:

  1. Select the description for the open assessment on the Patient Chart (can be either blue or black text) or on the Quick View screen. An assessment dashboard screen appears. The treatments associated with the selected diagnosis appear on the dashboard screen.
  2. To close the assessment, select the Close or Resolved/Close link at the top of the dashboard screen. The Order Workplan or Cancel icons appear, select the Order Workplan icon. The Close Assessment screen appears. This screen allows you to enter information about the assessment and treatment effectiveness.
  3. Select one of the options for rating the outcome of the assessment:
    • Very Satisfactory
    • Satisfactory
    • Unsatisfactory
    • Very Unsatisfactory
  4. If you do not want to include this assessment in the billing for the current encounter, change the Bill With This Encounter selection from Yes to No.
  5. If you want to make a notation about the effectiveness of the treatments associated with this assessment, select the N/A button for each treatment then select the effectiveness level for that treatment from the list. If you want to leave a specific treatment open after the assessment is closed, select the N/A button for that treatment then select Leave Open.
  6. Select Close Assessment. The dashboard reappears and the assessment is noted as (Closed).
  7. Select Finished. The Patient Chart reappears. This assessment and its associated treatments will no longer appear on the Patient Chart display for subsequent encounters.
    Note: Closing an assessment will automatically close all its attached treatments, unless you specify that a treatment should be left open.

To Close an Open Treatment (while leaving the assessment open):

  1. Select the description of the open treatment on the Patient Chart (can be either blue or black text) or on the Quick View screen. A treatment dashboard screen appears.
  2. To close the treatment, select or the Close Treatment button.
  3. Select Finished. The treatment is noted on the Patient Chart as (closed). This treatment will no longer appear on the Patient Chart for subsequent visits.

To Close Multiple Open Treatments (while leaving the assessment open):

  1. From the Patient Chart screen, hold down the CTRL key and select the description of each open treatment (can be either blue or black text). A Close button appears at the bottom of the Chart screen.
  2. To close the selected treatments, select the Close button.
  3. The treatment is noted on the Patient Chart as (closed). These treatments will no longer appear on the Patient Chart for subsequent visits.
