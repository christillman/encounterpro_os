How to: Document a Chronic Problem.

Purpose
When a patient has a chronic problem, such as asthma or hypertension, you will want to note the progress and status of those assessments at each encounter. EncounterPRO offers one-touch progress notes to easily track the status of a chronic problem. When you enter a progress note for an existing assessment, that assessment will appear on the billing screen. You have the option of billing for it or not billing for it.

Before You Begin
When you update the treatment plan or close an existing assessment, that assessment is automatically included in the billing for the encounter. Determine if you want to bill for that assessment during the current encounter or not.
Details:

Steps
Follow the steps below:
To Update a Chronic Assessment:

  1. From the Patient Chart, select the description of the open assessment. The assessment dashboard screen appears.
  2. To add a progress note for this diagnosis, select the Assessment Note button or link at the top of the screen and a menu of progress notes appears.
  3. Select the appropriate one-touch button (Resolved, Resolving, Improved, or Stable, etc.) or select Progress Note to access a screen that will allow you to type in your comments. Select OK when finished. The dashboard screen reappears with the progress note.
  4. When you are done, select Finished. A notation that a progress note was attached to the assessment appears on the Patient Chart.

To Remove an Assessment from the Current Encounter Billing:
Anytime you 'touch' an assessment, that assessment will appear on the billing for the current encounter. Adding a progress note as above will cause that assessment to bill. If you do not want to include that assessment on today's bill, follow these steps:

  1. From the Patient Chart, select Exit. The Billing Review screen appears. The assessment for which you entered a progress note will appear on the billing screen.
  2. If you do not want to bill for this assessment on the current encounter, select the gray area with the billing charges. A second billing screen appears.
  3. Select the gray bar for the encounter visit code. The Encounter Charge Edit screen appears.
  4. De-select the assessment from the Associated Assessments list near the bottom of the screen. Select Finished. The Billing Review screen reappears.
  5. Select I'm Finished to approve the billing and send the patient to Checkout.
