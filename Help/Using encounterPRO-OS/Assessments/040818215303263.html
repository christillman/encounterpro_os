How to: Inactivate an Assessment from the Database

Purpose
If there are assessments in the database that your practice does not use, or you find that duplicate assessments have been inadvertently added, you can inactivate them.

Before You Begin
If you inactivate an assessment from the database, you will be able to re-activate it later if needed. Any patients who have been diagnosed with the assessment will not be affected,  the assessment will still appear on the chart.

Details:

Steps
Follow the steps below:

  1. From the Configuration tab, select the Diagnosis button. The Select Assessment to Configure screen appears.
  2. Select the assessment you want to modify. You may have to use the search options to locate the desired assessment. A menu of options appears.
  3. Select Edit Assessment from the menu. The Select Assessment to Configure screen appears.
  4. Select the Active status to change it to Inactive.
  5. Select OK. The assessment is inactivated and cannot be ordered going forward.
