How to: Search for an Assessment

Purpose
Providers will typically select an assessment from the Assessment short list. Occasionally, a patient will present with a problem that is not used often enough to rate a place on the short list. In those cases, the user will need to search for the desired assessment.

Details:

Steps
Follow the steps below:

  1. From the Patient Chart, select either the Sick DX or Other DX button. If you select the Other DX button, select the type of assessment from the menu. The Select Assessment screen appears with the short list displayed.
  2. If the assessment you want to order is not on the short list, you can use one of the search options:
    • Description
    • ICD Code
    • Category.
  3. Select the search option button that corresponds to the type of search you want to perform.
  4. If you select Description, a typing entry pad appears. Enter a partial description of the assessment. You can search for matching items in the database using either the Begins With or Contains options. Select OK. A list of matching assessments appears.
  5. If you selected ICD Code as the search option, a dialog box appears. Enter the ICD-9 code (at least the first 3 digits) for the desired assessment, then select OK. A list of matching assessments appears.
  6. If you selected Category, a list of assessment categories appears. Select the category that you think best fits the desired assessment. A list of matching assessments appears.
  7. The list of matching assessments that is displayed contains those matches that are also assigned to your specialty. If you do not see the assessment you are searching for in the list of matches, select the Specialty button above the Category button. A list of all matches in the entire database appears.
  8. If the desired assessment is still not in the list of matching assessments, repeat steps 4-7 using a different search option or a shorter partial name.
