How to: View/Print Growth Charts

Purpose
Growth charts are plotted for children based on the CDC Growth Chart data. The charts can be printed or viewed from the Patient Chart. The growth charts available are:

  • Weight vs. Age
  • Length/Stature (Height) vs. Age
  • Length/Stature (Height) vs. Weight
  • BMI vs. Age
  • Head circumference vs. Age (for infants)

Note: The CDC growth curve screens will only show a single data point in any two-week timeframe. The system will suppress multiple data points that occur in the same two-week time period because the CDC growth data is provided in half-month increments. The growth-percentiles screen available from the Data button does not suffer from this problem because it does not need to show CDC data. If you need to see the percentile increase/decrease for clinical reasons, use the percentiles screen.

Before You Begin
If the child was a premature infant, be sure to enter the Weeks Premature on the Patient Demographics screen. This enables encounterPRO to calculate the Gestational Adjusted Age (GAA). The GAA will be used on the growth charts until the GAA reaches 2 years, at which point the actual age of the patient is used. See How to: Document Adjusted Age for Premature Infants for details.

Details:

Steps
Follow the steps below:

  1. From the Patient Chart, select the Dev tab.
  2. Select the Growth Charts (CDC) button. The CDC Growth Chart appears. The default display is the Weight vs. Age chart. Points on the chart include today's data and those measurements taken at Well visits (visits with a Well Assessment). If you want to view all measurements for this patient, select the Well button on the upper right to toggle between Well only and All visits.
  3. If you want to view one of the other graphs, select the button at the bottom of the screen:
    • Stature/Length vs. Age
    • Stature/Length vs. Weight
    • BMI vs. Age (children over age 2 years)
    • Head Circumference vs. Age (infants only)
  4. If you want to print a graph, display the graph, then select the Print button. If more than one printer is attached to your workstation a selection list appears. Select the printer and OK.
  5. When finished viewing the growth chart, select OK. The Growth chart reappears.
