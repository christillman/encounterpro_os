How to: Log Out of encounterPRO.

Purpose
For security reasons you should always log out of encounterPRO when leaving your station. This will prevent unauthorized access to medical records and will ensure that each subsequent user logs in at the station so that they see their own preferences, messages, and services.

Details:

Steps
To log out, select the Logoff tab at the bottom right of the Office view or any Exam Room screen. The log on screen reappears.

NOTE: A Security Preference can be set to automatically log off users if they are inactive for a specified period of time. This is done during configuration.
