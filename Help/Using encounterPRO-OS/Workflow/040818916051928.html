Description
Services often either display a new screen or perform a function. Services can be General (across the practice) or Patient-level. Services are one of the main building blocks in workplans allowing users or the system to perform certain operations. Service bars are often created when a service is ordered as part of a workplan.
Services can be used to:

  • Add buttons to menus
  • Secure specific functions (Authorize Services)
