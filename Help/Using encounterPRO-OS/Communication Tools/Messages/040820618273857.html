How to: Un-delete a Message.

Purpose

If an encounterPRO message as been 'deleted' inadvertently, it can be easily retrieved. Messages are not actually deleted, they are moved to a Deleted Messages folder. Un-deleting a message means that the message is moved from the Deleted Messages folder to one of the other available folders.

 

x

Details:

Steps

To un-delete a message, follow these steps:

  1. Select Messages from the Office View.
  2. Select the bar for the folder that is currently displayed (generally Inbox). A list of folders appears.
  3. Select Deleted Messages from the list of available folders.
  4. Select the gray bar for the message you want to un-delete. The message is displayed.
  5. Select the Move to Folder button at the bottom of the screen. A list of available folders appears.
  6. Select the desired folder. The message will be moved from the Deleted Messages folder to the selected folder.
