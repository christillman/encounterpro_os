How to: Create a Message

Purpose
The steps to create a message will depend on whether you want this message to be linked to a specific patient's chart or not.

You can create a non-patient-related message to send information to one or more users in the office. For example, if you want to remind all the nurses of a staff meeting or you want to send all the providers a tip on how to use an encounterPRO feature.

You can create a patient-related message when you want to notify someone in the office of information concerning a patient. While patient-related messages and Tasks both appear in the patient's chart, you should use a message only when there is no task associated with the information. If there is a task you want a person in the office to perform for this patient, such as calling in a prescription or getting a copy of an immunization form, create a Task instead.

Before You Begin
Remember, Messages are informational only. If you want someone in the office to perform a task, create a Task.

Details:

Steps
Follow the steps below:
To create a non-patient related message:

  1. Select the Messages tab from the Office view.
  2. Select the New Msg button near the bottom of the screen.
  3. Select the gray bar for To and Select the user(s) and/or role(s) to receive the message. You can send messages to multiple users and multiple roles. (Tasks can only be sent to a single user or single role).
  4. Enter a description of the message in the Subject line (required).
  5. Enter additional text in the Message area, if needed.
  6. Select Send to dispatch the item. The item will appear in the Messages list for the recipients selected.

To create a patient-related message:

  1. If you want to send an intra-office message related to a specific patient, do one of the following:
    • From any treatment room screen, select the patient name bar, then select the Inbox Message button
    • From the Patient Chart, select Other Items, then select the Inbox Message button
    • From the Charts tab, locate the patient's chart, select the name bar, then select the Inbox Message button from the Patient Demographics screen
    • From the Patient Chart, select the Messages/Task tab. Select Other Items, then select either Inbox Message or In Office Message

    NOTE: Use the Inbox Message button if the patient is not "in the office."
  2. Select the user(s) and/or role(s) to whom you want to send this message.
  3. Enter a description of the message in the Subject line (required). Since this information is displayed on the Inbox, try to make the subject as descriptive as possible.
  4. Enter additional text in the Message area, if needed.
  5. Select Send to dispatch the message.
