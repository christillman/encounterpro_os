How to: Create a High Priority Message.

Purpose
The steps to create a high priority message will depend on whether you want this message to be linked to a specific patient's chart or not.

You can create a non-patient-related message to send information to one or more users in the office. For example, if you want to remind all the nurses of an important deadline.

You can create a patient-related high priority message when you want to notify someone in the office of urgent information concerning a patient. While patient-related messages and Tasks both appear in the patient's chart, you should use a high priority message only when there is no task associated with the information. If there is an urgent task you want a person in the office to perform for this patient, such as calling the emergency room, create a high priority Task instead. See How to: Create a High Priority Task.

Before You Begin
Remember, Messages are informational only. If you want someone in the office to perform a task, create a Task.

NOTE: There is a Preferences that controls the audible cues that notify users when a high or urgent priority message or task is waiting. See How to: Set the Audible Priority Alert for details.

Details:

Steps
Follow the steps below:
To create a non-patient related message:

  1. Select the Messages tab from the Office view.
  2. Select the New Msg button near the bottom of the screen.
  3. Select the gray bar for To. Select the user(s) and/or role(s) you want to send this message to. You can send messages to multiple users and multiple roles. (Tasks can only be sent to a single user or single role.)
  4. Select the N/A bar for Priority. A list of priority levels appears.
  5. Select the Priority for this message:
    • N/A
    • Low
    • Normal
    • High - message bar is marked with one red exclamation points (!)
    • Urgent - message bar is marked with two red exclamation points (!!)
  6. Enter a description of the message in the Subject line (required).
  7. Enter additional text in the Message area, if needed.
  8. Select Send to dispatch the item. The item will appear in the Messages list for the recipients selected.

To create a patient-related message:

  1. If you want to send an intra-office message related to a specific patient, do one of the following:
    • From any treatment room screen, select the patient name bar, then select the Inbox Message button
    • From the Patient Chart, select Other Items, then select the Inbox Message button
    • From the Charts tab, locate the patient's chart select the name bar, then select the Inbox Message button from the Patient Demographics screen
    • From the Patient Chart, select the Messages/Task tab. Select Other Items, then select either Inbox Message or Inoffice Message

    NOTE: Use the Inbox Message button if the patient is not 'in the office.'
  2. Select the user(s) and/or role(s) to whom you want to send this message.
  3. Select the N/A bar for Priority. A list of priority levels appears.
  4. Select the Priority for this message:
    • N/A
    • Low
    • Normal
    • High - message bar is marked with one red exclamation points (!)
    • Urgent - message bar is marked with two red exclamation points (!!)
  5. Enter a description of the message in the Subject line (required). Since this information is displayed on the Inbox, try to make the subject as descriptive as possible.
  6. Enter additional text in the Message area, if needed.
  7. Select Send to dispatch the message.
