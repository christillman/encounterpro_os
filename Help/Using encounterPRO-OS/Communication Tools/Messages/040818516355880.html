How to: Reply to Messages

Purpose
Messages in encounterPRO are used as a sort of internal email system. Messages should be informational only; if someone in the office needs to perform a task, then a Task should be used.
Details:

Steps
Follow the steps below:

  1. From the Office View, select the Messages tab. Your Inbox list of unread messages appears.
  2. Select the message you want to reply to from the list. The Message screen appears.
  3. Select Reply. The Compose Message screen appears. The recipient will be filled in from the sender of the previous message.
  4. Enter any reply needed in the Message text area.
  5. Select Send. The original message reappears.
  6. Do any combination of the following:
    • Select Print to print a copy of the original message
    • Select Move to Folder to place a copy of this message in another message folder
    • Select Forward to send the original message to another user
    • Select I'll be Back to leave the original message in the Inbox
    • Select I'm Finished to move the original message to the Read Messages folder.
