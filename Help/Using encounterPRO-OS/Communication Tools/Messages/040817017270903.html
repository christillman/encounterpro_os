Description
This screen is used to manage your EncounterPRO Messages. You can read, reply, create, and organize your Messages from this screen.

Each message has a gray bar that lists the date of the message, the sender, the subject, and the first few lines of the message text. Select the gray bar to view the entire message.

How to Access this Screen
Access this screen by selecting Messages from the Office View.
