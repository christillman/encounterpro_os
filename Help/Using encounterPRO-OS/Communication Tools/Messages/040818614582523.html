How to: Create a Messages Folder.

Purpose
The encounterPRO Messaging feature is delivered with three folders to organize your messages:
  • Inbox (unread)
  • Read
  • Sent

Many users find it useful to further organize their messages in a series of personally created folders.

Before You Begin
Moving or saving a message into a user-created folder actually creates a copy of that message in the new folder. The original message will still appear in the Read or Sent messages folder.

Details:


Steps
Follow the steps below:

  1. From the Office view, select the Messages tab.
  2. Select the Edit Folder List button at the bottom of the screen. The Edit Message Folders screen appears.
  3. Select New Folder. A dialog box appears to allow you to enter the new folder name.
  4. Enter the folder name and select OK. The new folder appears on the list.
  5. Repeat steps 3 and 4 to create additional folders.
  6. Use the Move and Sort buttons to arrange the list of folders.
    • Select the folder and select Move, then use the up and down arrows to position the folder in the list
    • Select Sort to alphabetize the list of folders.
  7. When you are satisfied with the folder list, select OK. The Message list reappears.
