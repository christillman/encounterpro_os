Messages and Tasks
EncounterPRO's internal messaging system operates much like email. The messaging system has two types of items: Messages and Tasks. You can send and receive messages and assign Tasks among the users in the practice. There are several important differences between the two types of items:

Messages

Tasks

 
Should be used for Information Only.


Should be used when you have a task that needs to be performed.


Can be sent to multiple individuals and/or roles.


Can be assigned to a single individual or single role only.


When you send a message to one or more users encounterPRO creates a copy of the message in each of those users' inboxes; each user must individually open the message and read it to clear it from their inbox. Likewise, if you send the message to one or more roles, encounterPRO will place a copy of the message in each user's inbox who is assigned to the selected roles.


When you assign a Task item to a role, such as Any Provider, the item appears on the Task list for that role; when one user takes ownership of that item it is removed from the Task list for that role and moved to the Task list for that individual user.

When to Use...
Use messages to...

  • Communicate from user to user
  • Communicate generic information about a patient

Use Tasks to...

  • Assign a task to yourself or another user
  • Assign a task related to a patient
  • Place the Task in the patient's medical record
  • Allow the recipient to access the patient's chart with a single click
  • Allow the recipient to access the appropriate screen to perform a service or task with a single click (for example, review lab results)
  • Assign a task to a role, such as phone nurse
  • Return to a task that you did not finish (select I'll Be Back)
  • Enter results for a lab or test

NOTE: Both messages and Tasks are considered legal documents and are never deleted from the database. Patient-related messages and Tasks are permanently saved as part of the patient's medical record. Also, keep in mind that messages and Tasks are not private. Certain users in your practice and all EHR support personnel have access to all messages and Tasks.
