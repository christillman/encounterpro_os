How to: Forward Tasks

Purpose
Tasks (called ToDo Items in versions 2 and 4) are used to assign tasks to either a role or a specific user. There may be times when you cannot work on a task that was assigned to you or you need to send it to another role or user to complete.

Before You Begin
If the task was an in-office task with a service bar showing on the Office View, then it will remain an in-office task. If the task was an inbox task, it will remain an inbox task. Only inbox tasks will appear in the Task list accessed from the Task tab of the Office View.

Details:

Steps
Follow the steps below:
To Forward a Task from the Task list:

  1. From the Office View, select Tasks. The Task list appears.
  2. Select the Task that you want to forward from the My Tasks list. The Task opens.
  3. Enter the reason you are forwarding this task in the Comment/Reply area.
    TIP: Never close or forward a Task without noting something in the Comment or Disposition field.
  4. Select the Send to button at the bottom of the Task
  5. Select the user or the role you want to forward this Task to. You can only assign Tasks to a single user or single role. The Task window disappears.

To Forward a Task from the Office View:

  1. From the Office View, select the service bar for the Task. For example, the service bars in the Telephone room are almost all Tasks.
  2. Enter the reason you are forwarding this task in the Comment/Reply area.
    TIP: Never close or forward a Task without noting something in the Comment or Disposition field.
  3. Select the Send to button at the bottom of the Task
  4. Select the user or the role you want to forward this Task to. You can only assign Tasks to a single user or single role. The Task window disappears.
