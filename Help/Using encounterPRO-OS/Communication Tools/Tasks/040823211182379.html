Description

The internal commnuication system in encounterPRO allows a user to designate the priority of the message or Task they are sending or forwarding. The priority levels available are:

  • N/A (the default)
  • Low
  • Normal
  • High - the message or task has one red ! and the tab on the Office View where that message or Task is viewable will also have one red ! on the tab name.
  • Urgent - the message or task has two red !! and the tab on the Office View where that message or Task is viewable will also have two red !! on the tab name.

NOTE: In addition to the graphic indicators (!!) on a high or urgent priority message or Task, a Msg/Task Preference can be set for each workstation that will activate an audible alert when a high or urgent priority message is waiting for attention. See How to: Set the Audible Priority Alert.

Examples

Some uses for high and urgent priority include:

  • To notify a provider when abnormal lab results have been entered
  • To notify the primary provider when a patient has been admitted to the hospital
  • To notify the office staff of a security issue
