Purpose
EncounterPRO Tasks are used to assign specific tasks to a role or specific user. There are two types of Tasks in encounterPRO:

  • In office - In office tasks create service bars visible from the Office View
  • Inbox - Inbox tasks appear on the Task list accessible from the Tasks tab on the Office View
Details:

Steps
Follow the steps below:
To manage in office tasks:

  1. From the Office View, locate a task that is assigned to you specifically or to one of your roles.
  2. Select the room that the patient is located in.
  3. Select the service bar for the task you want to work on. The Task screen appears.
  4. At this point, there are several options:
    • Select View Patient to view the Patient Demographics screen
    • Select View Chart to view the Patient Chart
    • Select Open Item to display the dashboard for the item attached to the Task
    • Type a comment in the Comment/Reply text box.
  5. When you are ready to dispose of this Task, do one of the following:
    • Select Send to.. to forward this Task to a specific user or role
    • Select I'll Be Back to leave the Task open
    • Select Close with Comment (you must enter a comment first)

To manage inbox tasks:

  1. From the Office View, select the Tasks tab. The My Tasks list displays. A tab appears for each role that the user has been assigned.
  2. If the task you want to work on does not appear on the My Tasks list, select the tab for the role to which the task was assigned.
  3. Select the bar for the task you want to work on. The Task screen appears.
  4. At this point, there are several options:
    • Select View Patient to view the Patient Demographics screen (if this Task is patient-related)
    • Select View Chart to view the Patient Chart (if this Task is patient-related)
    • Select Open Item to display the dashboard for the item attached to the Task
    • Type a comment in the Comment/Reply text box.
  5. When you are ready to dispose of this Task, do one of the following:
    • Select Send to.. to forward this Task to a specific user or role
    • Select I'll Be Back to leave the Task open
    • Select Close with Comment (you must enter a comment first)
