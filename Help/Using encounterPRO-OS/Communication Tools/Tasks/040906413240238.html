How to: View Another Users Task List

Purpose
Managing the efficiency of your office will sometimes require the practice manager or supervisory users to monitor another user's task list. The will also be occassions when a user is not in the office because of vacation or illness when the practice manager will want to reassign that user's high and urgent priority tasks to another user. If you can view another user's task list, you can forward tasks to another user or role as needed. See How to: Forward Tasks for instructions.

Supervisors will have a tab on their Task List for each user they supervise.

Before You Begin

Your user profile must be set to allow you the privilege of viewing other users' task list. All Super Users automatically have this privilege.  To set this privilege for other users, go to the User Definition screen for that user and selectthe Permissions tab. Select Yes for the 'User may see Task List of any user' privilege. The user will need to log out and log back in to EncounterPRO to see the change in permissions.


Details:

Steps
Follow the steps below:

  1. From the Office view, select the Tasks tab.
  2. A tab labeled 'Other' appears along with the tabs for your roles and users you supervise.
  3. If the user you want to view the Task List for is one on the users you supervise, select the tab with that user's name.
    If the user you want to view the Task List for is not one of the users you supervise, select the Other tab. 
    The Task List appears.
  4. Select a task from the list. At this point, you can do one of the following:
    • Perform the task and complete it
    • Forward the task to another user
    • Leave the task for the original user
