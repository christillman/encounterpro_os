How to: Create a High Priority Task.

Purpose
EncounterPRO Tasks (formerly called ToDo Items) are used to assign a task to another user or role. The system also generates Tasks as needed for specific workplans, such as reviewing lab results. Additionally, individual users can create Tasks for each other. Tasks can have a priority assigned to them:

  • N/A (default)
  • Low
  • Normal
  • High - the task bar will have one red ! to indicate high priority
  • Urgent - the task bar will have two red !! to indicate urgent priority

NOTE: Use a Task if there is an action required by the recipient. Use a Message if you are communicating information only, with no action required. See Messages vs. Tasks for more information.

Before You Begin
The steps to create a Task depend on whether you want to link the Task to a specific patient's chart. You can create a patient-related Task if there is a task that needs to be performed for a specific patient. For example, call in a refill, review the lab results, etc.

You can create a non-patient-related Task  to assign general office tasks to a user or role. For example, you could create a Task for reordering exam room supplies and assign that to the administrative role.

Details:

Steps
Follow the steps below:
To create a non-patient-related Task:

  1. Select the Tasks tab from the Office view.
  2. Select the New Item button near the bottom of the screen.
  3. Select the gray bar next to To: and select the user or role you want to assign this Task to. Tasks can be assigned to a single role or a single individual.
  4. Select the priority level for the Task in the upper right of the screen:
    • NA
    • Low
    • Normal
    • High
    • Urgent
  5. Enter a description of the item in the Subject line (required).
  6. Enter additional text in the Message area, if needed.
  7. Select Send to save and send the Task.

To create a patient-related Tasks:

  1. If you want to create a Task related to a specific patient, do one of the following:
    • From any room screen, select the patient name bar (left side,) then select the Task button.
    • From the Patient Chart, select Other Items, then select the Task button (In-Office is used when the patient is physically in the office, In-Box is used if the patient is not in the office.)
    • From Charts tab, locate the correct patient's chart, select the Other Options button, then select the Task button.
  2. The recipient may default to the Encounter Owner or the Primary Provider. If you want to assign this task to a different user or role select the colored bar, then select the user or role you want to assign this item to.
  3. Select the priority level for the Task in the upper right of the screen:
    • NA
    • Low
    • Normal
    • High
    • Urgent
  4. Enter a description of the item in the Subject line (required).
  5. Enter additional text in the Message area, if needed.
  6. Select Send to send the item.
