How to: Remove Confidentiality From an Object

Purpose

Objects that can be flagged as confidential include:

·         A specific assessment, such as Pregnancy

·         A specific treatment, such as a prescription or lab test

·         An encounter

NOTE: Flagging an assessment as Public does NOT automatically change the Confidential flag for any associated treatments. On reports that list treatments separate from their associated assessments (AATT), if the associated treatments have been previously flagged as Confidential, they will not appear unless the Confidential flag is individually removed for each treatment.

If the user wants to make an item confidential, select the Make Private button to add a progress note that Denies access to that item. To remove the confidentiality flag from an object, a second progress note is added that Allows access to the item. (These buttons must be configured by EHR Product Support or a Super User at the practice.)



Details: Steps
Follow these steps:

To make a confidential assessment Public:

  1. From the Patient's Chart, select the description of the assessment to display the assessment dashboard.
  2. Select the button for Make Public. A progress note is added that the item has been flagged as Public.

To make a confidential encounter Public:

  1. From the Patient Demographics screen, select the All Encounters tab. A list of open encounters appears.
  2. If the encounter that you want to make private does not appear on this list, select All Encounters. A list of all encounters for the current patient appears.
  3. Select the bar for the desired encounter. A menu of options appears.
  4. Select the button for Make Public. A progress note is added that the item has been flagged as Public.

To make a confidential treatment Public:

  1. From the Patient's Chart, select the description of the treatment to display the treatment dashboard.
  2. Select the button for Make Public. A progress note is added that the item has been flagged as Public.
