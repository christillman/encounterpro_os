How to: Document the Physical Examination.

Purpose
The Physical Examination screen varies from specialty to specialty and practice to practice. Most physical exams are configured as drilldown screens. Many practices create Standard Exams to make documenting common results faster.

Before You Begin

Be familiar with the various Standard Exams available to you on the physical exam screens at your practice.

Details:

Steps
Follow the steps below:

  1. From the Patient Chart, do one of the following:
    • Select the Physical button on the right of the chart screen
    • Select the small icon to the left of the blue description of the Physical Exam
    • Select the blue description of the Physical Exam in the Subjective and Objective portion of the chart, the dashboard appears. Select the blue description for the Physical Exam.

      The Physical Exam screen appears.
  2. The Physical Exam has body systems listed on the drilldown bars on the left. Document the examination findings by doing any combination of the following:
    • Select the right side of the bar labeled "Drill Down." The next level of observations appears. Either select the drill down again, select a result, or enter a result.
      Use the navigation buttons on the bottom right to go Up one level, to the Next observation or to the Top level of the physical exam.
    • Select the left side of the bar where the body system description is located. The defaults set for that system are populated.
    • Select one of the Standard Exams on the right side of the screen. The results for that standard exam are populated.
    • Select the Previous Exam button at the top right of the screen. A list of previous examinations appears.
      Select the desired examination. The results from that instance of the exam are populated. (You may want to select Review Physicals at the bottom of the screen to review the physical examination from previous encounters to be sure you are selecting the correct one.)
  3. Be sure to select Top to go to the top level of the physical exam and review the findings.
  4. When all the results for the exam are entered, select Finished. The Patient Chart reappears.
