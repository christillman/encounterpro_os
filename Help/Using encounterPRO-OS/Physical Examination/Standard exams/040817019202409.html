Description
Use this screen to view the list of current standard exams or set up a new one. This screen lists currently available standard exams, both common (can be used by the entire practice) and personal (can only be used by the current user). This screen is also where you can establish criteria for a particular standard exam, such as age range, gender or race.
The standard exams that are created are particular to the drilldown observation screen for which they are created.

How to Access This Screen
Access this screen by selecting the Std Results button on any drilldown screen.
