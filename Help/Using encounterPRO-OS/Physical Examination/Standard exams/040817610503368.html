How to: Create a Standard Exam.

Purpose
Standard exams can be a great time saver when entering physicals, ROS, or lab results. They allow one-touch population of results on a drill-down screen. For example, you can set up a standard exam button for All Normal results for a physical to quickly record all normals for the patient. Additionally, you can set up the expected results for an Otitis Media physical or a Newborn physical. After selecting the standard exam button to populate the results, the provider or nurse can change any of those results which do not apply to the current patient as needed.

Before You Begin
You must have the specific permissions to be able to create common standard exams which can be used by the entire office. All clinical users have the ability to create their own personal standard exams that are visible only to them.


TIP: Changing the Common Standard Exams should only be done with the agreement of all users who will be using that exam.

If you decide to create your own Personal Standard Exams, choose a name that is NOT the same as the Common Standard Exam to reduce confusion. Many users prefix the exam name with their initials to clearly identify Personal Standard Exams.

Details:

Steps
It is best to perform this procedure on a Test patient, rather than a live patient. Follow the steps below:

  1. Check in a Test Patient and navigate to the Patient Chart screen.
  2. Go to the drill down observation screen for which you want to create the Standard Exam. For example, the physical exam or ROS.
  3. Select the Std Results button. The Standard Exams screen appears listing all exams saved for this observation.
  4. If you want this standard exam to be available to all users, select Common. (You must have the access rights to create common exams.) If you want this standard exam to be available only to yourself, select Personal.
  5. Select the Add Standard Exam button.
  6. Select Create New Exam then enter a name for your exam.
    TIP:  Always use a unique name for your standard exams. If you are creating a personal exam, append your initials to the name of the exam.
  7. Select OK. The observation screen reappears. The new standard exam appears in the list on the right of the screen.
  8. Select the bar for your new exam on the right of the screen. The system asks whether you want to apply the results from the standard exam. Since the new exam is empty, select No. The exam bar remains highlighted (darkened).
  9. Select and enter the results you want to save as this standard exam.
  10. Select the Set button then select Set Personal or Set Common (depending on your choice in step 4). The system asks whether you want to save the current results to the selected exam.
  11. Select Yes.
  12. To check that the results were saved, select the Clear All button to remove all results.
  13. Select the bar for the new exam on the right of the screen (to turn off highlight), then re-select it. The saved results are applied.
  14. If the results are not the way you want them, repeat steps 9 -13.
