Purpose
If you have documents, pictures, and other items in a Windows-compatible file format, those items can be imported into encounterPRO and posted to a patient's chart or, to a specific treatment much like scanned items.

Before You Begin
When using the File Import function you will need to know the location on your workstation or network of the files you want to import.
Your workstation must be configured for File Import as an External Source. See Setting up External Sources for instructions.

Details:

Steps
Follow the steps below:
To Import to the Import and Post Attachments screen:

  1. From the Office view, select the Utilities tab. The Utilities list appears. The options available depend on your security access rights.
  2. Select Scan/Post Images. The Import and Post Attachments screen appears.
  3. Select the Import button. The Import Attachments screen appears.
  4. Select File Import.
  5. Enter a Comment Title for the item you want to import or select one from the list. This title will appear on the bar on the Import and Post Attachments list. Select OK. The Select Attachment File screen appears.
  6. Locate the file you want to import using the Windows File Open dialog box. Select the file then select OK.
  7. The Images to Post screen reappears with the new item added to the top of the list.
  8. Repeat steps 3 - 7 to import additional items. When finished importing, select OK.
  9. The imported file can be posted to a patient's chart just as any scanned other document.

To Import and Post to a Specific Assessment or Treatment:

  1. From the Patient Chart, select the item (assessment or treatment) to which you want to attach the file. The dashboard screen appears.
  2. Select the Attachments tab. The list of items attached to this assessment or treatment appears.
  3. Select the New (Star) button on the right. A list of external sources appears.
  4. Select File Import.
  5. Enter a Title for the item you want to import or select one from the list. This title will appear on the bar on the attachments list. Select OK. The Select Attachment File screen appears.
  6. Locate the file you want to import using the Windows File Open dialog box. Select the file then select OK.
  7. The attachments list reappears with the new item added to the top of the list.
  8. Repeat steps 3 - 7 to import additional items. When finished importing, select OK.
