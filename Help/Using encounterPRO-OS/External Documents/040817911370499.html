How to: Scan and Post Directly to a Patient's Chart

Purpose
Another approach to scanning is to scan a document directly into a specific patient's chart. This differs from scanning in bulk as you are usually working on one patient's documentation at a time. You could use this approach when you have several documents to scan for a single patient.

Before You Begin
Each scanner is slightly different. You can select the settings for each document that you scan. In order to maximize scanning speed and minimize the data storage space required, the recommended settings are:

  • 150 dpi
  • Black and white (unless color is specifically called for)

See Screen: Concept: Selecting Scanner Settings.

NOTE: You can scan only from a station that has a scanner physically connected to it. The scanner must be configured for this station under the External Sources screen. See Screen: How to Configure a Workstation to Scan Screen: for instructions.

Details:

Steps
Follow the steps below:

  1. Select the Charts tab from the Office view. Locate the desired patient using the search option.
  2. Select the patient name bar. The Patient Demographics screen appears.
  3. Confirm that this is the correct patient.
  4. Select the View Chart button. The Patient Chart appears.
  5. If you want to post the attachment to a specific treatment:  
    1. Locate the treatment to which you need to attach the document
    2. Select the description of the item. A dashboard screen appears.
    3. If a Scan button does not appear on the dashboard screen, select the Other Options button, then select the Scan button.

    The scanning dialog appears. Go to Step 8.
  6. If you want to post the attachment to a specific assessment:
    1. Locate the assessment to which you need to attach the document
    2. Select the description of the assessment. A dashboard screen appears.
    3. If a Scan button does not appear on the dashboard screen, select the Other Options button, then select the Scan button.

    The scanning dialog appears. Go to Step 8.
  7. If you want to post the attachment to a folder in the patient's chart:
    1. Select the Scan tab.
    2. Select the Star (New) 'button. The scanning dialog appears. Go to Step 8.
  8. Set the scanning settings for the document you are scanning.
    Generally, you will be scanning Black and White documents that are Letter Sized. Select the 150 dpi setting (scanning at a higher dpi setting will create attachments that require much more storage space).
  9. Select the Scan button to begin scanning.
  10. Depending on the item/folder selected, an Attachment Workplan screen may appear (folders can have workplans attached that are automatically launched when an item is saved to that folder).

    If you want to launch the workplan (very important if this is a lab integration practice), select Finished.

    If you do not want to launch the workplan, select No Workplan This Time. A bar appears with the name of the attachment (defaulted from the treatment) and the date attached.
  11. Repeat steps 5 through 10 to scan and post each item. When finished scanning, select Finished. The Patient Chart reappears.
  12. Select the Exit tab to close the chart.
