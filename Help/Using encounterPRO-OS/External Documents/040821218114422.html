How to: Add a To Be Posted List.

Purpose
Scanned items appear on the To Be Posted list ready to be posted to a specific patient and/or treatment. Some practices have more than one location with scanners and it can be convenient and more efficient to have a To Be Posted list for each location.

When multiple To Be Posted lists have been created, be sure when scanning that you have the desired list selected.

Details:

Steps
To add a new To Be Posted list, follow these steps:

  1. Turn on Config Mode.
  2. Select Utilities from Office View. The Utilities list appears.
  3. Select Scan/Post Images. The To Be Posted screen appears.
  4. Select the Configure button at the top right of the screen.
  5. Select the New Folder button near the bottom right of the screen.
  6. Enter the name of the new folder and select OK. A new bar appears for that folder in the Posting List.
  7. If you want to re-order the folders, select the bar for the folder you want to move and select the Move button. Use the arrow keys to move the selected folder up and down on the list. The top folder will be the top list (default display) on the To Be Posted screen.
  8. Select Finished. The To Be Posted screen reappears with a tab for the new folder.
