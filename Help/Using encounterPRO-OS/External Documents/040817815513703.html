Purpose
Before using the scanner, you must configure your workstation to indicate to encounterPRO that a scanner is attached. You can configure the workstation directly from the specific scanning station or from any other encounterPRO workstation.

Before You Begin
You may not have security rights to configure your station. If this is the case, a user with local administrator rights on the workstation must perform the following task.

Scanner Settings
Each scanner is slightly different. You can select the settings for each document that you scan. In order to maximize scanning speed and minimize the data storage space required, the recommended  scanner settings are:

  • 150 dpi
  • Black and white (unless color is specifically called for)

See Screen: Concept: Selecting Scanner Settings.

NOTE:  You can scan only from a station that has a scanner physically connected to it. The scanner must be configured for this station under the External Sources screen. See Screen: How to: Configure a Workstation to Scan.

Details:

Steps
Follow the steps below:

  1. Select the Configuration tab from the Office View. The Configuration list appears.
  2. Select External Sources. The External Sources Available by Computer screen appears.
  3. The computer you are currently using is highlighted on the list on the left. The External Sources configured for this computer are highlighted on the right. If you are using the scanning workstation, go to step 4. If you are using a different workstation, select the scanning workstation from the list of computers on the left. The list shows each unique computer name/log in combination in the office.
  4. Locate the row for Scanner/TWAIN Compatible Scanner on the right. It may be on page 2. Be sure that it is highlighted (darkened).
  5. If multiple users will be logging into the scanning workstation, you need to repeat steps 3 and 4 for each combination of computer name/log in name.
  6. Select Finished. The Configuration list reappears.
