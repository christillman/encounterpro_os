How to: Post a File Directly to an Assessment

Purpose
There may be situations where a document, photo, or other file pertains to a specific assessment and the user wants to attach that file directly to the assessment. For example, if the patient has a laceration or other injury and you take a digital photo of the injury you can attach that photo directly to the assessment rather than just putting it in a folder in the patient's chart.

Details:

Steps
Follow the steps below:

  1. From the Patient Chart locate the assessment you want to attach the file to.
  2. Select the text description of the assessment. If this is a new assessment for today's encounter, the text will appear blue. If it is an existing assessment, the text will appear black. A menu of options appears.
  3. Select Dashboard. The assessment dashboard appears.
  4. Select the Attachments tab.
  5. Select the New (Star) button on the right. A list of external sources appears.
  6. Select the external source of the item:
    • File import : if the file existing on the network or workstation
    • Scanner : if you are working from the scanner workstation
    • Camera : if you are working from the workstation that has the camera attached
  7. Select the item you want to attach to the current assessment.
  8. Select OK or Finished (depending on your selection in step 6). A bar appears on the attachments tab for the item.
  9. If you have additional items to attach to this assessment repeat steps 5 - 8. When all items have been attached, select Finished to return to the Patient Chart.
