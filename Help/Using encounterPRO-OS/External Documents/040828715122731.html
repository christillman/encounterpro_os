Description
Scanning and then posting documents in encounterPRO allows any paper document to be attached
to a patient's electronic health record and viewable from any encounterPRO session.

Settings on The Scanner

  • The scanner needs to be attached to a fat-client workstation running encounterPRO locally on the workstation's hard drive (some practices have attached the scanner to the network directly - this is a fine option that can avoid the fat-client workstation if the local system administrator can manage the scanner's connectivity)
  • The scanner needs to be TWAIN or WIA compliant (TWAIN is recommended especially for multiple page scanning)
  • We recommend the scanner's default settings be set to 150 dots per inch (dpi) - providing plenty of detail for viewing on the screen and printing the documents
  • We recommend the scanner's default settings be set to black and white (note that grayscale is not the same thing and will produce larger images taking more space in the database).

In choosing the scanner, particular attention should be paid to whether or not feeding multiple
documents into the scanner at one time is desired. Also consideration should be given to the size of
the scanner and the space needed to open and close the top of the scanner (if a flatbed scanner is to
be used). Consult with EHR Product Support for a list of recommended scanners.

Assuming the above four requirements are met, once the scanner allows a document to be
successfully scanned and viewed outside of encounterPRO, encounterPRO is ready to be
configured to work with that scanner.  See How to: Configure a Workstation to Scan.
