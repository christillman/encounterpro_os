How to: Use Document Manager.

Purpose
Document manager allows the practice to better control the creation and dispersal of patient-related documents. Documents differ from traditional encounterPRO reports in the following ways:

  • Reports are temporary and once printed, must be re-generated to view later
  • Document are persistent and are automatically saved to folders in the Patient Chart

Documents can be ordered for:

  • A specific treatment (prescription or referral letter)
  • A specific encounter (Patient Take Home, Encounter SOAP Note), or
  • A specific patient (Immunization Report, Current Medications)
When a document is ordered for a treatment or encounter, icons appear on the Office View and Patient Chart that indicate the status of those documents:
  •   Documents ordered and ready to send
  •   Documents sent or completed

Saving Documents to the Patient's Chart
When a document is sent or printed using Document Manager, a copy of that document is saved as an attachment in a folder in the Patient Chart. The folder is selected based on the Attachment Preferences set for attachment folders or the folder specified on the report button used to order the document. See How to: Set the Attachment Folder Preference for details.

Details:

Steps
When documents are ordered:

  1. Select the Documents ordered icon on the Patient Chart or Office View or select the Document Management button on the menu. The Document Management screen appears.
  2. If the document status is Ordered, you have several options:
    1. Select View to preview the document
    2. Select Other, then Edit to access the dashboard for the document
    Note: Changes can be made to prescriptions even after they appear on  the Document Manager screen. However, the user making the changes must be RX-certified as that user will be considered the ordering provider for the purposes of the signature on the prescription and eligibility for sending via SureScripts-RxHub.

  3. When you are satisfied that the document is ready to send/print confirm the recipient. If the recipient that defaulted is not the method you want to use for this document, select the recipient bar. A list of available recipients for the document appears.
  4. Select the desired recipient.
  5. If more than one route is available for the recipient selected in step 4, select the Via Route bar. A list of available routes appears.
  6. Select the desired route for this document.
  7. If more than one address is available for the route you selected in step 6, select the Address bar.
  8. Select the Send/Print button (the name of the button is determined by the recipient selected in step 4).
  9. Repeat steps 2 through 8 for all documents on the Document Management screen that have not yet been sent.
  10. When all documents have been sent or completed, select OK.
