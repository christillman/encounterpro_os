Purpose
There are many reports that you can print for a patient. Reports are divided into four areas:

  • General - gathers and analyzes data across the entire practice. See Printing General Reports for instructions
  • Patient - gathers and analyzes data for a specific patient across multiple encounters
  • Encounter - provides data for a specific encounter
  • Treatment - provides data for a specific treatment. Treatment reports are often set to automatically print, such as Prescriptions and Lab Referrals. See Printing Prescriptions for instructions

This solution will describe the procedure to print Patient and Encounter-level reports.

Printing Reports for a Patient Not in the Office
If the patient is not in the office, you can still print reports at any time. From time to time, a patient will need a copy of his or her immunization records, current medications, etc.

Steps
Follow the steps below:
To Print an Encounter or Patient Report for a Patient in the Office:

  1. From the Office view, select the patient name bar (left hand side) representing the patient for whom you want to print an encounter report.
  2. Select Print Report. The Reports menu appears. This menu lists your most commonly used reports.
  3. If the report you need has a button, select it. Otherwise, select Pick Report. The Select Report screen appears.
  4. Select the type of report you want to print: Patient or Encounter. The short list for that report type appears.
  5. To select the report you want to print, do one of the following:
    • Select the button assigned to the desired report
    • Search for the desired report, then select it
  6. Select OK. The Print menu appears. You can preview or print the report by selecting the appropriate icon.

To Print a Patient-level Report for a Patient Not in the Office:

  1. From the Office view, select the Charts tab.
  2. Search for the patient for whom you want to print the report(s).
  3. Select the patient bar representing the patient for whom you want to print a report. The Patient Demographics screen appears.
  4. Select Pick Report, then select Patient. (To print an Encounter-level report for a patient not in the office, go to the next procedure.)
  5. Select the report you want to print, and then select Finish. (If you want to print a complete copy of the patient's records, select All Encounters Without Parameters).
  6. Select OK. The Print menu appears. You can preview or print the reports by selecting the appropriate icon.

To Print an Encounter-level Report for a Patient Not in the Office:

  1. From the Office view, select the Charts tab.
  2. Search for the patient for whom you want to print the report(s).
  3. Select the patient bar representing the patient for whom you want to print a report. The Patient Demographics screen appears.
  4. Select the Open Encounters tab. All currently open encounters are listed.
  5. Select All to view a list of all encounters for this patient.
  6. Select the encounter for which you want to print a report. A menu of options appears.
  7. Select Print Report. The Reports menu appears. This menu lists your most commonly used reports.
  8. If the report you need has a button, select it. Otherwise, select Pick Report. The Select Report screen appears.
  9. Select the type of report you want to print: Patient or Encounter. The short list for that report type appears.
  10. To select the report you want to print, do one of the following: Select the button assigned to the desired report.
  11. Search for the desired report, then select it.
  12. Select OK. The Print menu appears. You can preview or print the report by selecting the appropriate icon.
