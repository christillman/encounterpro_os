Purpose
There are many reports that you can print for a patient. Reports are divided into four areas:

  • General - gathers and analyzes data across the entire practice.
  • Patient - gathers and analyzes data for a specific patient across multiple encounters
  • Encounter - provides data for a specific encounter
  • Treatment - provides data for a specific treatment. Treatment reports are often set to automatically print, such as Prescriptions and Lab Referrals. See Printing Prescriptions for instructions

Printing General Reports from the Outbox

You can generate and print General Reports from the Utilities list using the steps below, or you can generate and dispatch General Reports from the Outbox tab. You must configure the Order Document menu on the Outbox with the General Reports you want to be able to produce on the Outbox tab. See How to: Create the Order Documents Menu for the Outbox for information on configuring that menu.

Details:

Steps
To print a General Report from the Utilities list, follow the steps below:

  1. From the Office view, select the Utilities tab.
  2. Select General Reports. The Select Report screen appears.
  3. Select a report from the short list or use one of the search options to locate the desired report.
  4. If the report requires certain parameters, a screen will appear. For example, many reports require a Begin and End date. Proceed through the parameters by selecting Next.
  5. When you have completed the required parameters, select Finish. 
  6. The Print menu appears. You can preview or print the reports by selecting the appropriate icon.

To print a General Report from the Outbox, follow the steps below:

  1. From the Office view, select the Outbox tab.
  2. Select Order Document. The menu of General Reports appears.
  3. Select a report from the menu.
  4. If the report requires certain parameters, a screen will appear. For example, many reports require a Begin and End date. Proceed through the parameters by selecting Next.
  5. When you have completed the required parameters, select Finish. 
  6. The document appears in the Outbox list. You can select a recipient (generally an encounterPRO user) and a route (generally a printer) or accept the defaults.
  7. Select Send/Print for that document or select the Send All/Print All button at the bottom of the screen to dispatch all pending documents.
