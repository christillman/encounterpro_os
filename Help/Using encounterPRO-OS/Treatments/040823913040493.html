How to: Cancel Treatments

Purpose
Treatments can be canceled if they are ordered in error. The user name and date/time that the treatment was canceled is retained and displays in the audit logs. If a treatment has been canceled inadvertently, it can be uncanceled. See How to: Uncancel Treatments and Assessments.

Details:

Steps
Follow these steps:

  1. Display the Patient Chart for the patient for which you want to cancel a treatment.
  2. Select the description of the treatment you want to cancel. The dashboard screen appears.
  3. Select the Cancel Treatment button (trashcan at the bottom of the dashboard screen). The system prompts for you to enter a reason for canceling the treatment.
  4. Enter the reason for canceling and select OK.
  5. The system prompts you to be sure that you want to cancel the treatment. Select Yes. The status is changed to Canceled.
  6. Select Finished to return to the Patient Chart screen. The treatment may still display, but with (Canceled) in the description.
