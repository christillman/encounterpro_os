How to: Create a Default Treatment List

Purpose
The Default Treatment List is available as one of the tabs on every treatment list for every assessment at the practice. The Default Treatment List allows the practice to create a list of commonly used treatments that can be accessed from every assessment treatment list.

Details:

Steps
The default treatment list is available for ALL assessments. This procedure should be performed from a test or example patient.

  1. Display the Patient Chart screen.
  2. Select the new Sick DX button.
  3. Select any assessment from the list.
  4. Select OK. The Treatment list screen appears. Note that there are three tabs on the treatment list: Personal, Common, and Default.
  5. Select the Default List tab.
  6. If you want to add a single treatment to the list, select New Treatment, then search for the desired treatment. The Default Treatment list reappears.
  7. Select the icon to the left of the newly added treatment and then select Yes to update the list. Selecting Yes will permanently add the treatment to the Default Treatment List.
  8. If you want to add a treatment group, such as a group for Follow-up Visits, select New Group.
    1. Enter the name of the treatment group, then select OK. A bar for the treatment group appears.
    2. Select the small icon to the left of the treatment group name. A menu of options appears.
    3. Select New Treatment. A menu of treatment types appears.
    4. Select the desired treatment type, for example, Follow-up.
    5. Complete the treatment ordering screen, if needed, then select Finished. (Certain treatments, such as Follow-ups, Office Medications, Prescription Medications, etc. will require additional information on an ordering screen before they can be added to the treatment group.) The treatment appears below the treatment group name.
    6. Repeat b through f to add additional treatments to the group.
  9. If you want to add an individual treatment to the Default treatment list, select the New Treatment button on the right. (The system may prompt you to determine if this treatment is part of an existing treatment group. If the treatment you are adding is NOT part of an existing treatment list, select No.) A menu of treatment types appears.
    1. Select the desired treatment type. A Select Treatment screen appears.
    2. Select the desired treatment from the screen.
    3. If you want to select additional treatments of this same treatment type, you can select them from the list. Each treatment selected appears in the Selected Items list.
    4. Select OK. If the treatments selected require additional information, a treatment ordering screen will appear. Complete the ordering screen for each treatment selected, then select Finished. The treatments appear on the Default treatment list.
  10. Repeat steps 6 and 7 until your Default treatment list is complete.
  11. If you do not want to actually order all of these new treatments, you can de-select (remove the dark highlighting) each treatment or select the 'clear all' button on the right side of the treatment screen.
  12. Select Finished. The treatment list changes are saved.
