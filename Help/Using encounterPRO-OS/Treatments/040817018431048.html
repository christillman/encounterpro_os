Details: Description
When you select the type of treatment you want to order, the Select Treatment screen appears. The screen displays the short list for the treatment type you selected.

EncounterPRO has many types of treatments you can order. Some of the most commonly ordered treatment types are:
  • In-Office drugs
  • Prescription drugs
  • Vaccinations
  • Labs and Tests (both send-out and in-house)
  • Follow-ups
  • Referrals
How to Access This Screen
Access this screen by:
  • Selecting the Order Treatment (or Any Treatment) button, then any button on the resulting menu
  • Selecting New Treatment on the Treatment List screen for any assessment, then selecting any button on the resulting menu
