How to: Document Vaccine Administration.

Purpose
When the provider or clinical staff orders immunizations for a patient, several service bars are created. These service bars allow you to enter the relevant information about the vaccine and access the Vaccine Consent screen (when configured to accept signatures). You can view and select these service bars from the OfficeView.

Before You Begin

You should have the following information available:

  • Vaccine maker
  • Lot number
  • Expiration Date
  • Literature Date
  • Reference NDC code *

The NDC code can be configured for the vaccine on the Medication Definition screen, but if it does not default, it can be entered when each dose is administered.

Details:

Steps
Follow the steps below:

  1. From the Office View, a service bar appears to notify the staff that a vaccine has been ordered. Select the service bar for the first vaccine. The Vaccine screen appears.
  2. Select the Vaccine Maker bar and select the appropriate maker from the list.
  3. Type the Lot Number in the data box (A number may already be there from a previous immunization. Make sure the vial number is correct and change if necessary.)
    Note: Lot numbers default from the previous number entered for that specific vaccine at that specific office location.
  4. If literature information has been given, select the Gave Literature button to fill in a previously entered date of publication. Verify that this is correct and if not, enter the correct date in the text field.
  5. Select the Location bar and select the area where the vaccine will be administered.
  6. If the NDC number does not default, select the three dot button (...) and enter the Reference NDC code for that vaccine.
  7. Select Immunization Given.
  8. Repeat steps 1 through 7 for each vaccine ordered.
  9. Select the Vaccine Consent service bar to allow the patient or parent to read the consent information or have it read to them.
  10. If no signature device is installed on that workstation, have the patient or parent select Accept without Signature. If a signature device is installed, select Sign to display the signature area. Have the patient or parent use the mouse or touchpen to sign in the area provided.
  11. Select OK when finished signing. The Office View reappears.
