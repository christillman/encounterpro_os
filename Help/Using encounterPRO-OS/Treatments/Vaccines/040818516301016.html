Purpose
The ProVacs tab on the Patient Chart will indicate which vaccines the patient is currently due to receive, which ones have been completed, and which ones he/she is ineligible for. The information on the ProVacs tab is based on the CDC Immunization Guidelines for 2008, the patient's age, and the patient's vaccine history.

NOTE:  The information presented on the ProVacs is meant to assist the provider and clinical staff in evaluating the patient's immunization status. Only medical personnel can make the final decision as to whether a patient should receive a particular immunization at any encounter.

Before You Begin
If the patient's past vaccinations have not been manually entered into the encounterPRO system, the ProVacs cannot accurately calculate the patient's immunization status.

Details:

Steps
Follow the steps below:

  1. From the Patient Chart, select the Vax tab. The Immunization Status screen appears.
  2. Review the patient's status for each of the vaccines listed.
  3. If you want to review the rules used to calculate the suggested Due Date, select the bar in the Due Date column corresponding to the specific vaccine. The schedule for that specific vaccine is listed.

    If you want to review the CDC rules for this vaccine schedule, select the Show Rules button. Select OK to return to the Immunization Status screen.
  4. If you want to review all the vaccinations administered to this patient, select the View Detail button. Select Finished to return to the Immunization Status screen.
  5. At this point, you can do one of the following:
    • Select the View Chart tab to return to the chart view
    • Select the Order Vaccine button to order the vaccines needed for today's encounter
    • Select View Detail, then New to manually enter past vaccines. See How to: Document Past Immunizations for details.
