Purpose
When a patient is immunized, the practice can bill for both the vaccine and the administration of the vaccine. (Patients who receive VFC vaccines generally are billed only for the administration of the vaccine.) Vaccine Administration Codes are a special type of Procedure in EncounterPRO.

If you want to charge vaccine administration codes for a particular encounter, follow the guidelines below:


Before You Begin
Vaccine Administration Codes are billable treatments. Be sure to either order the admin codes from a treatment list so that they are automatically associated with an assessment or manually associate the admin codes with the appropriate assessment, otherwise it may not bill correctly. See How to: Associate Treatments With a Specific Assessment for details.

Details:

Steps
Follow the steps below:

  1. From the Patient's Chart do one of the following:
    • Select existing vaccine or well assessment to display the assessment dashboard. Select Any Treatment, a menu of treatment types appears.
    • Order a new Vaccine assessment and select New Treatment, a menu of treatment types appears.
  2. Select Procedures. The Select Procedure screen appears.
  3. Select Vaccine Admin Codes as the Procedure Type.
  4. Select the desired admin code from the short list or use the search options to locate it. You can select multiple codes if needed.
  5. Select OK. Depending on your configuration service bars for each admin code ordered may appear for the clinical staff in the Office View.
