Description
This screen displays the vaccines that have been ordered during the current encounter and gives the parent or guardian the opportunity to review the consent statement and indicate approval for the immunizations. If a signature device is attached or the workstation is able to capture signatures, the parent or guardian can sign for approval and that signature is captured as an attachment in the patient's chart.

How to Access This Screen
Access this screen by selecting the Vaccine Signature service bar.
