Description
Prescription drugs are the most commonly ordered treatment type in EncounterPRO. The drug database delivered with the system is extensive. However, you may find it necessary to add drug packages or administration rules as new packages are introduced by the manufacturer. Before you add new drugs, be sure you have thoroughly searched the database, in case the drug you need is in the database under a different specialty.

The solutions in this Solution Center concern configuring simple drugs only. If you need to configure a compound drug or drug cocktail, contact Product Support for assistance.
