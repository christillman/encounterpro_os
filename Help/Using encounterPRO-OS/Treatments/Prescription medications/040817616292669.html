How to: Order a Prescription Medication

Purpose
Prescription drugs are one of the most commonly ordered treatments in EncounterPRO.

Before You Begin
You might want to check the patient's allergies to make sure that there are no drug allergies.

Details:

Steps
Follow the steps below:

  1. From the Select Treatment screen for any assessment: If the desired drug appears on the treatment list, select the drug from the treatment list. Go to step 6.
  2. If the drug you want to order does not appear on the treatment list, select New Treatment. A menu appears.
  3. Select Prescription Drugs. The Select Prescription Drug screen appears.
  4. Select the drug(s) you want to prescribe from the list. If the drug you want to prescribe is not on the short list, use the Search Options to locate the desired drug. For each drug selected, you may be prompted to select the appropriate package. When finished selecting drugs for the specified assessment, select OK.
  5. Select any remaining treatments for this assessment, then select Finished.
  6. A dosage screen appears for each drug selected with the default dosage information.
  7. On this screen confirm the following (to change any information, select the gray bar and choose from the list of options or enter an amount):
    • Package (You must select a package first)
    • Dose
    • Frequency
    • Duration
    • Total dispense
    • Number of refills allowed
    • Special instructions, if any. You can enter instructions for the patient and pharmacist, if needed
  8. When finished, select the Finished button. The dosage screen for the next drug appears.

Depending on your configuration options, the prescription will print automatically when ordered or at the end of the encounter.
