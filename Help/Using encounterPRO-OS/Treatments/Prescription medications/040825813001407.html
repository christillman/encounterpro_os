How to: Search for a Pharmacy.

Purpose
You can use any one of several different search options to find a pharmacy that is not on the patient's Care Team or your short list:
  • Name
  • Zip Code
  • Care Team

Pharmacies that have been selected for this patient in the past will automatically appear on the Care Team list. If there are any pharmacies on the Care Team, that list is initially displayed. If there are no pharmacies on the Care Team list, the Personal or Common short list is initially displayed.

Before You Begin
There is a General Preference called 'Actor Filter - Use Zip Code Filter' that allows the user to enter a list of applicable Zip codes at search time. This will restrict the Name search to those pharmacies who match the name entered as well as the Zip code. If the desired consultant is not located within one of the Zip codes in the filter, you can temporarily disable it for the current search.

If you are using e-Prescribing with SureScripts,  the EPiE engine downloads the pharmacies to your EncounterPRO database during the nightly maintenance job. Only pharmacies that are registered with SureScripts will be downloaded.

Details:

Steps
Follow the steps below:

  1. On the Document Manager screen, select Pharmacy as the recipient for a NewRX or e-Refill Request.
  2. The Select Pharmacy screen appears. Either the Care Team list or the Short List is initially displayed.  
  3. If the desired pharmacy does not appear on either the Care Team or Short List, select one of the search options:
    • By Name
    • By ZIP Code
  4. If you selected By Name, select the Begins With or Contains option for this search.
  5. Use the keyboard (either on the touchscreen or the regular keyboard) to enter a partial name or the ZIP code. The system uses an exact matching algorithm, so a partial name is going to match more items than a longer name.
  6. Select OK. A list of matching items appears.  If the Use Zip Code Filter preference is active, the last ZIP codes used on the Search Pharmacy screen are automatically displayed and used to filter the list of matching pharmacies.
  7. If the desired pharmacy is not found on the list of matches, you can do either of the following:

    • Select the bar for Zip to disable the Zip code filter for this search
    • Select the small Edit button below the Zip bar to enter a new list of ZIP codes for the search. (Enter multiple ZIP codes separated by a comma.)

  8. If the item is still not found in the list of matches, repeat the search using either a shorter partial name or select a different search option.

NOTE: If you have performed a thorough search and the desired pharmacy is not in the EncounterPRO database, search for the pharmacy on the SureScripts web site (www.surescripts.com).   If the pharmacy does not exist in the SureScripts pharmacy directory then the pharmacy will not be in our directory and is not eligible for e-prescribing. If the pharmacy does exist in SureScripts, then the customer should get the complete details (full name, address, phone number) and contact EncounterPRO Product Support with the details of the missing pharmacy.
