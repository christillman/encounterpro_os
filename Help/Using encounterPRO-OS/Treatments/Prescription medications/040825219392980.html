Description
This screen appears when a refill request has been received electronically via SureScripts-RxHub. When the request is successfully matched with an individual patient and a specific prescription treatment, encounterPRO sends a Refill Request Task to the ordering provider's Task List. The provider uses this screen to communicate his/her response to the refill request.

Details:

How to Access This Screen
Access this screen by selecting the Refill Request Task on the Task list.
