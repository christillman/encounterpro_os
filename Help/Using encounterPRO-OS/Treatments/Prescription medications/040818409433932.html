How to: Change Dosage from CC to Teaspoons.

Purpose
Some providers prefer to write prescriptions with the dosage in CC/MLs and some prefer to use teaspoons. If the drug package is set up with both units of measure, you can select the preferred dosage when ordering the prescription.

Before You Begin
If you cannot change the unit of measure on the drug prescription screen, you will need to update the drug package configuration. See How to: Add a New Prescription Medication Package for detailed information.

Details:

Steps
Follow the steps below:

  1. Select the prescription drug from the treatment list or from the Select prescription drug screen.
  2. Select the small pill icon to the left
  3. Select the Edit Config icon
  4. The Single Drug Definition screen appears (the drug name will be listed in the Trade and Generic Name boxes.)  select the package for this patient from the list of available packages.
  5. Select the Dose bar. A numeric entry pad appears.
  6. Select the unit of the dose. A list of available units appears.
  7. Select the preferred dosage unit.
  8. Enter the number of units per dose, then select OK.
  9. Complete the prescription by selecting the Frequency, Duration, Total Dispense Amount and number of refills.
  10. Select Finished.
