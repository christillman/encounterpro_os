How to: Search for a Prescription Medication.

Purpose
Before you add a new drug to the database, you should perform a thorough search of the database. Search the prescription drugs for your specialty as well as All specialties to be sure the drug is not already in the database.

Details:

Steps
Follow the steps below:

  1. From the Office screen, select the Configuration tab. The encounterPRO Configuration list appears.
  2. Select Drugs. The Drug Configuration screen appears.
  3. Select one of the search option buttons.
    • If you know the Trade Name of the drug, select that button, enter the trade name or a partial name and select OK. Note the option to search by Begins With or Contains.
    • If you know the Generic Name, select that button, enter the generic name or a partial name and select OK. Note the option to search by Begins With or Contains.
    • If you want to search by type of drug, select the Category button then select a category from the list.
  4. The matching drugs within your specialty are shown. If no matches were found under your specialty, select the Specialty button above the Category button to toggle it to All to search the entire database.
  5. If the drug is not in the database, proceed to the instructions on adding a new simple drug.
