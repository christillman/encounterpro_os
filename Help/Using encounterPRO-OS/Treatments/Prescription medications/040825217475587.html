How to: Order a Prescription Medication Using e-Prescribing

Purpose
Prescription drugs are one of the most commonly ordered treatments in encounterPRO. If your practice is using e-Prescribing, registered providers can send new prescriptions and refills electronically. See Concept: e-Prescribing for details.

Before You Begin
The ordering provider must register with SureScripts-RxHub before he/she can send prescriptions electronically. This is accomplished on the User Definition screen in encounterPRO. See How to: Register for SureScripts-RxHub e-Prescribing for details.

IMPORTANT: When you first begin using e-Prescribing there will be some prescriptions that fail to reach the pharmacy. Sometimes this is due to incomplete information, sometimes it is due to a problem with the SureScripts registration. There is a solution, Troublesheeting: e-Prescribing with SureScripts , that each SureScripts user should read and be familiar with in order to make the transition smoother. 

Details:

Steps
Follow the steps below:

  1. From the Select Treatment screen for any assessment: If the desired drug appears on the treatment list, select the drug from the treatment list. Go to step 6.
  2. If the drug you want to order does not appear on the treatment list, select New Treatment. A menu appears.
  3. Select Prescription Drugs. The Select Prescription Drug screen appears.
  4. Select the drug(s) you want to prescribe from the list. If the drug you want to prescribe is not on the short list, use the Search Options to locate the desired drug. For each drug selected, you may be prompted to select the appropriate package. When finished selecting drugs for the specified assessment, select OK.
  5. Select any remaining treatments for this assessment, then select Finished.
  6. A dosage screen appears for each drug selected with the default dosage information.
  7. On this screen confirm the following (to change any information, select the gray bar and choose from the list of options or enter an amount):
    • Package (You must select a package first)
    • Dose
    • Frequency
    • Duration
    • Total dispense Number of refills allowed
    • Special instructions, if any. You can enter instructions for the patient and pharmacist, if needed
  8. When finished, select the Finished button. The dosage screen for the next drug appears.
  9. When all the prescriptions are complete, a NewRX bar appears for each prescription on the Document Manager screen. Any user with access to Document Manager can select the patient's preferred pharmacy as the recipient and select SureScripts as the route.

NOTE: Both the provider ordering the prescription and the pharrmacy must be registered with SureScripts. If SureScripts is not available as a route, encounterPRO will display the reason why it is not available.
