How to: Edit a Prescription

Purpose
It may be necessary at some point to alter a prescription after ordering. You can change the dosage, package, frequency, add special instructions, etc.

Before You Begin
You cannot change the drug ordered. If you selected the wrong drug, you will need to cancel the original prescription and order a new treatment. See How to: Cancel a Prescription Medication for instructions.

Details:

Steps
Follow the steps below:

  1. You can access the treatment dashboard for the prescription by:
    • Selecting the Meds box at the top of the Patient Chart then selecting the description of the drug.
    • Locating the prescription treatment on the Patient Chart and selecting the description text.

    The prescription treatment dashboard appears.

  2. Select at the top of the dashboard or the Edit button from the menu at the bottom of the screen. The prescription screen appears with the name of the drug at the top of the screen.
  3. If you need to change the package, select the desired Package. This will automatically change the Dose and Total Dispense amounts.
  4. If needed, select Dose Based On and change as necessary. This will also change the Dose and Dispense amounts.
  5. If needed, select Dose and change the dosage as necessary. This will change the Dispense amount and remove Dose Based On, as it is no longer applicable.
  6. If needed, select Frequency and select the desired frequency from the list.
  7. If needed, select Duration and select the number and unit of time for the duration.
  8. If needed, override Total Dispense amount by selecting the bar and either selecting another dispense choice from the list or selecting and entering the new amount. Indicate the amount of samples given by selecting the bar for In Office and either selecting another dispense choice from the list or selecting and entering the new amount.
  9. If needed, enter Special Instructions for this prescription:
    1. Click in gray box marked Special Instructions and select Patient or Pharmacist.
    2. Choose the instructions from Pick-List or type in the white text box.
    3. Select OK. The prescription screen reappears.
    4. To save instructions for future use, select the Special Instructions box again.
    5. Select Set as Default.
    6. Select which instruction you want this to be the default for:
      • Set Default Patient Instruction
      • Set Default Pharmacist Instruction
    7. Select which combination you want this to be the default for:
      • Drug
      • Drug + package
      • Drug + Administration rule
  10. To indicate refills allowed, select No Refills and enter appropriate amount or to indicate PRN simply select PRN to highlight it.
  11. To indicate that the brand name is required select No to change Brand Name Required to Yes.
  12. Select Finished. The treatment dashboard reappears. Depending on your workflow, the prescription may automatically reprint. If not, you can select or Epro Report to manually print the prescription.
  13. Select Finished to return to the Patient Chart or Current Medications screen.
