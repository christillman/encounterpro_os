Description
Patient Materials can be any document or website that you want to be able to order as part of a treatment plan.

The items will need to be imported, scanned, or have a URL address. Once loaded into the EncounterPRO system, Patient Materials can be added to treatment plans for easy ordering, or ordered individually as needed.
If you are using the Document Management feature, you may have the option of emailing or faxing these items as well as printing. See Concept: Document Management for details.

Details:

Uses
Things like the Bright Futures handouts (www.brightfutures.aap.org) and aftercare instructions are ideal candidates for patient materials. It can be helpful to add patient materials to a treatment list for a specific assessment so that the materials associated with a well visit age range or treatment are at the provider's fingertips.
