Purpose
Procedures are a type of treatment, often concerning repairing injuries. Common procedures in encounterPRO are applying casts or splint, destruction of warts or skin lesions, ear piercing, cauteriziation, etc.

Before You Begin
Procedures are almost always billable items. Be sure to either order the procedure from a treatment list so that it is automatically associated with an assessment or manually associate the procedure with the appropriate assessment, otherwise it may not bill correctly. See How to: Associate Treatments With a Specific Assessment for details.

Details:

Steps
Follow the steps below:

  1. From the Patient's Chart do one of the following:
    • Select the Other Items button on the right, then select Procedures.
    • Order a new Sick Assessment and select New Treatment, then select Procedures. The Select Procedure screen appears.
  2. Select the desired procedure from the short list or use the search options to locate it. You can select multiple procedures if needed.
  3. Select OK. Service bars for each procedure ordered will appear for the clinical staff in the Office View.
