Description
This screen is used to allow clinical staff to manually enter lab and test results. The actual observations displayed will depend on which lab or test was ordered.

How to Access This Screen
Access this screen by:

  • Selecting the Enter Results service bar for the lab or test from the Treatment Room screen
  • Selecting the small icon to the left of the lab or test description on the Patient Chart
  • Selecting the description of the lab or test on the Patient Chart, then selecting the description again from the dashboard
  • Selecting the lab or test from the Outstanding Labs and Test list (from the Utilities List), then selecting the description from the dashboard
