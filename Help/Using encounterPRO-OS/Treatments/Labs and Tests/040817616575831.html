How to: Order a Lab or Test

Purpose
Your practice will configure its labs and tests to indicate which are send out and which are done in house. In house labs and tests will create collection and entry of results service bars for the appropriate staff member when ordered. The procedure for ordering labs, tests, x-rays and other tests is similar.

Before You Begin
Will this lab or test be processed in the office or sent out?
You can order labs and other treatments without an assessment, however, you will then need to manually associate the treatment to the correct assessment later in order for the billing to be accurate.

Details:

Steps

To Order the Lab or Test:

  1. From the Treatment List screen for any assessment, if the desired lab or test appears on the treatment list, select it. Go to Step 6.
  2. If the lab or test you want to order does not appear on the treatment list, select New Treatment. A menu of treatment types appears.
  3. Select the Lab, Test, Biopsy, X-ray or other test button as appropriate. The Select screen for the treatment type chosen appears.
  4. Select the items you want to order from the list. If the item you want to order does not appear on the list, use the Search Options to locate it.
  5. Select as many items as you need. Each will appear in the Selected Items box. When finished, select OK. The Treatment List screen reappears.
  6. Select any other treatments needed for this assessment, then select Finished. Service bars appear on the Office View for each treatment ordered.

To Collect or Refer Out the Specimen:

  1. If the lab or test ordered has a treatment mode that requires a collect service, a service bar appears in the Office View.
  2. Select the Collect Specimen service bar. The collect screen appears for the treatment. Collect screens will differ from treatment to treatment.
  3. If the specimen was collected in the office, select Collected. If the specimen will be sent to an outside laboratory for processing, select Specimen Sent To then select the laboratory that will process the specimen. If your practice is integrated with a lab, such as Quest or LabCorp, be sure to select that provider. If the patient was sent to an outside provider, select Patient Sent To, then select the laboratory that the patient will go to for the specimen collection.
  4. Select Finished. If this lab is being processed in the office, an Enter Results service bar appears in the Office View.

To Enter the Results:

  1. If the lab or test ordered is processed in the office, an Enter Results service bar appears in the Office View (after the collect service bar is completed).
  2. Select the Enter Results service bar. The results screen appears for the treatment. Results screens will differ from treatment to treatment giving you the appropriate selections for each lab..
  3. Enter the results for the lab or test.
  4. Select Finished. A Review Labs service bar can be configured to notify the provider that results are ready to be reviewed.
