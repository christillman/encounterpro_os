How to: Search for a Consultant.

Purpose
You can use any one of several different search options to find a consultant that is not on the short list:
  • Name 
  • Zip Code 
  • Care Team
  • Specialty/All

Before You Begin
You can set a preference to change the default search from Begins With to Contains. It is often easier to search using the Contains option.

NOTE:  There is a General preference called 'Actor Filter - Use Zip Code Filter' that allows the practice to enter a list of applicable Zip codes. This will restrict the Name search to those consultants who match the name entered as well as the Zip code in the preference. The applicable Zip codes will appear on the Search screen. If the desired consultant is not located within one of the Zip codes in the filter, you will need to temporarily disable that preference.

Details:

Steps
Follow the steps below. This procedure begins on the Select Consultants screen. You can get to this screen from several different methods such as the Configuration list or selecting a consultant for a referral.

  1. From the Select Consultants screen select the Specialty bar and select the specialty for the desired consultant.
  2. Select one of the search options:
    • Name
    • Zip Code

    An entry keypad appears.
  3. Select the Begins With or Contains option for this search.
  4. Use the keyboard (either on the touchscreen or the regular keyboard) to enter a partial name. The system uses an exact matching algorithm, so a partial name is going to match more items than a longer name.
  5. Select OK. A list of matching consultants appears. 
  6. If the consultant you need does not appear in the list of matching items, you can disable the Zip code filter by selecting the gray bar listing the Zip codes. A list of all consultants matching the search criteria appears.
  7. If the consultant you need still does not appear in the list of matching items, select the Specialty button to toggle to All. This will show all the matching consultants in the entire database.
  8. If the item is still not found in the list of matches, repeat the search using either a shorter partial name or select a different search option.
