How to: Order a Referral.

Purpose
Referrals can be ordered as part of the treatment plan for a specific assessment, as part of a general preventative visit schedule, etc. After the provider orders the referral and specifies the timeframe and purpose, the treatment workplan displays a service bar for the front desk staff or referral specialist. If no service bar is desired, that is a configuration option.

Before You Begin
Referrals may be handled differently based on your specialty or your practice's preferences. One way to order referrals is described below.

It is helpful, but not required, that you know which provider you want to refer the patient to. You can also select a specialty with no consultant and let the patient or the referral specialist choose the specific provider.

Details:

Steps
Follow the steps below:

  1. If the referral you want to order appears on the Treatment List screen for the appropriate assessment, select it from the treatment list. Go to Step 6.
  2. If the referral you want to order does not appear on the treatment list, select New Treatment. A menu of treatment types appears.
  3. Select the Referral button. A list of Specialties appears.
  4. Select the specialty for this referral. A list of consultants in this specialty appears.
  5. Select a specific provider from the list or select the option. Office Staff: Please Enter Consultant. The Referral screen appears.
  6. Enter the following information for this referral:
    • Directive for the referral:
      • Please Evaluate
      • Please Evaluate For - select the gray bar and select the evaluation criteria
      • Please Rule Out - select the gray bar and select the evaluation criteria
    • Timeframe: When you want the referral appointment scheduled
    • Description: Accept the automatically generated description or enter a new description
  7. When finished, select OK. The Referral Confirmation screen appears.
  8. You can change any of the referral information, if needed. If you want to select a specific treatment for the referral visit, select the Add Treatment button and select the desired treatment.
  9. When satisfied with the referral instructions, select the Finished button. A service bar appears to schedule the referral.
