How to: Set the Default Treatment Mode for a Specific Treatment

Purpose
A treatment mode is basically a treatment workplan. The treatment mode assigned to a specific treatment will control which service bars appear and for which roles. For example, the treatment mode selected as the default for a Rapid Strep screening will usually trigger service bars to collect and perform that test in the office.

Treatment modes are set for all treatments in a treatment type through configuration. Individual treatments can have a default treatment mode different from the default for the entire treatment type, if needed. You set the default for an individual treatment "on-the-fly" from the treatment selection screen. This solution covers setting the default treatment mode for an individual treatment "on-the-fly." See How to: Configure Treatment Types to determine the available treatment modes and set the default for the entire treatment type.

Before You Begin
The default treatment mode for a specific treatment is office location specific. That is, if you have multiple office locations, the default for individual treatments can be different if that is needed.

You may want to use a Test patient to set the default treatment modes for specific treatments in your office.

Details:

Steps
Follow the steps below:

  1. Display the select treatment screen by doing one of the following:
    • From the Patient Chart select the Any Treatment button
    • From an assessment's treatment list screen select the New Treatment button
  2. Select the treatment type for the treatment you want to set the default treatment mode for. The Select Treatment screen appears. For example, if you select Labs, the Select Lab Observation screen appears.
  3. Select the treatments for which you want to set a default treatment mode. Each treatment will appear in the Selected Items box.
  4. Select a treatment in the Selected Items box. A list of available treatment modes for this treatment type appears.
  5. Select the desired treatment mode for this specific treatment. The system prompts you to determine if you want to set this as the default for this treatment or set this as the treatment mode for this specific encounter.

    NOTE: You must have the required access privileges to set default treatment modes.

  6. Select Set as Default. The new treatment mode appears in small type on the bar in the Selected Items list.
  7. Repeat steps 4 - 6 to set the default for additional treatments of the same treatment type.
  8. Select OK when finished. This orders the new treatment with the new default treatment mode. Depending on the treatment mode, service bars may appear in the Office View.
