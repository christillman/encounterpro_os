Enhancements in Version 5

Solution:

Advantages of encounterPRO EHR Version 5 Compared to encounterPRO EHR Version 2

Select a link to view a solution containing detailed information about that function in version 5.

  1. All the following functionality, plus still just as fast to chart a patient (see video of demo on www.encounterpro.com )

  2. Immunizations (CDC catch up logic, faster entry of multiple past immunizations)
    See How to: Evaluate a Patient's Immunization Status and How to: Document Past Immunizations

  3. Build-A-Note (see exactly what your encounter note looks like, before the end of the encounter!)
    See Screen: Patient Chart

  4. Much more customizable workflow (user-editable workplans)

  5. Self-Service support (Solution Center and EPRO University)
    See How to: Use the Solution Center

  6. PDF Form Generator (populate those complicated forms directly from encounterPRO)

  7. Sharable Report Library (download reports other people created, contribute your own)
    See Concept: Reports Library and How to: Import a Report from the Reports Library

  8. E&M Calculator (what item in chart counts for what)
    See Concept: How encounterPRO Calculates E and M Coding and Screen: E and M Coding Rules

  9. E-Prescribing (send prescription to pharmacy, respond to request to renew/refill)
    See Concept: e-Prescribing and How to: Order a Prescription Medication Using e-Prescribing


  10. Lab Results (automatically, directly, electronically downloaded from several Clinical lab companies)
    See Concept: Lab Integration and How to: Track and Manage Lab Results - Lab Integration

  11. State Vaccine Registry upload (automatic, behind the scenes, driven by workflow engine)

  12. Multi-Office view (Office View and Tasks list show Tasks from other office locations)
    See Concept: Multiple Office Locations and Screen: Office View

  13. Expanded PMS options (over 30 partners, including a Web-based ASP option)

  14. Enhanced security (who can do, or view, what permissions, plus audit trail)
    See Concept: encounterPRO Security and How to: View Audit Logs

  15. CCHIT Certification

  16. Enhanced Growth Charts (English/Metric, printing, adjustments for premature infants)
    See How to: View/Print Growth Charts and How to: Document Adjusted Age for Premature Infants

  17. Chart Export (convenient electronic Adobe PDF format, put on thumb drive/CD, printable)
    See How to: Export a Patient Chart and How to: Export Multiple Patient Charts

  18. CARE (Clinical Active Reporting Environment: hit Pay-for-Performance targets)

  19. High and urgent priority tasks (optional audible alert)
    See Concept: High Priority Messages and Tasks
