How to: Exit an encounterPRO Screen

Purpose
EncounterPRO is designed so that the program moves you automatically to a new screen when you execute a choice on the current screen. The following instructions apply when you have finished with the screen you are working on and the program does not move automatically to a new screen.

Before You Begin
If there is a menu or selection list displayed, you must close it first before you can take any other action on the screen. Selecting any icon on the menu or selecting Close or Cancel will close the menu or list.

Details:

Steps
There are several ways to exit an encounterPRO screen:

  • OK -- this will save your changes and close the current screen
  • Cancel -- this will exit without saving your changes
  • Done -- this will save your changes and close the current screen
  • Exit -- this will save your changes and close the current screen
  • I'll Be Back -- this will save your changes and allow you to return to this screen later to add more information or make additional changes
  • Finished, or I'm Finished -- this will save your changes and close the current screen. You may not be able to return to this screen to make additional changes
