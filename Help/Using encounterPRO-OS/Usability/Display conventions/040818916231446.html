Description
Dashboard screens are used to display a great deal of information and allow specific actions for the following:
  • Assessments
  • Treatments
  • Histories
  • Observations
Details:

Examples
The easiest way to access a dashboard screen is to select the description for the desired item on the Patient Chart screen (as opposed to selecting the icon for the item which displays the data entry screen). If you are using the Quick View screens, select the date next to the description of the item to access the dashboard; the description on the Quick View screen displays the data entry screen.



What's on the Dashboard?
The options on the dashboard depend on the type of item you are viewing the dashboard for.

Assessments

  • Order a new treatment
  • Rediagnose (select a new assessment)
  • View the assessment history
  • Add a DX note
  • Close the assessment
  • Create a Task for a staff person that is attached to this assessment
  • Cancel the assessment
  • Add attachments
  • Review the audit log for this assessment
  • Review the status of this assessment
  • Change the assessment from Acute to Chronic or vice versa
  • Change the Begin date for this assessment
  • Access the dashboard for any of the treatments associated with this assessment

Treatments

  • Access the results entry screen (for labs, tests, etc.)
  • Order a refill (for prescriptions)
  • Cancel the treatment
  • Close the treatment
  • Mark the treatment as reviewed (add a progress note)
  • Create a Task for the staff that is attached to this treatment
  • Create a Task for the billing specialist that is attached to this treatment
  • Print a report for this treatment
  • Add attachments
  • Review the workplan for this treatment and see which steps remain and which have been completed
  • Review the audit log for this treatment
  • Change the Begin date for this treatment
  • Associate this treatment with one or more open assessments
  • Determine if this treatment is billable during the current encounter

Histories

  • Add a treatment or follow-up note
  • Cancel the history
  • Create a Task for the staff that is attached to this history
  • Review the workplan for this history and see which steps remain and which have been completed
  • Add attachments
  • Associate this history with one or more open assessments
  • Determine if this history is billable during the current encounter
  • Review the audit log for this history
  • Change the Begin date for the history

Observations (such as the physical examination)

  • Add a treatment or follow-up note
  • Cancel the observation
  • Review past observations of this type
  • Create a Task for the staff that is attached to this observation
  • Review the workplan for this observation and see which steps remain and which have been completed
  • Add attachments
  • Associate this observation with one or more open assessments
  • Determine if this observation is billable during the current encounter
  • Review the audit log for this observation
  • Change the Begin date for the observation
