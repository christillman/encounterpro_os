Description
Use this screen to determine why an encounter did not send the billing information to the integrated billing system and to re-send the billing information.

The screen displays the posting history with the date and time dispatched and the status. If the billing was not accepted by the integrated Practice Management System, the reasons for failure are displayed.

How to Access This Screen
Access this screen by selecting Retry Posting from the menu that appears when you select a patient name bar from the Checkout room.
