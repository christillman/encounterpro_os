How to: Change the Encounter Description

Purpose
The encounter description appears in several places throughout encounterPRO. Typically, the encounter description matches the encounter type. However, some practices find that changing the encounter description can be a useful way of communicating between the front office and the clinical staff. See Using the Encounter Description for details.

Before You Begin
The encounter description will appear on:

  • All encounter reports
  • The name bar on the Waiting Room
  • The encounter list
  • All encounter history screens and reports
Details:

Steps
Follow the steps below:

  1. The encounter description can be changed at virtually any point in the encounter without interfering with the workflow. You can access the Edit Encounter screen from the menu displayed when you select the patient name bar from one of these locations:
    • The Waiting Room
    • Any treatment room
  2. Select the Edit Encounter button. The Edit Encounter Information screen appears.
  3. Select the gray bar for the Description. The Encounter Description screen appears.
  4. Type the new description or select from the pick list.
  5. When you are finished changing the description, select OK. The Edit Encounter screen reappears.
  6. Select Finished to save your changes.
