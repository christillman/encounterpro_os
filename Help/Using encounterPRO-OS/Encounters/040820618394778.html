How to: Uncancel an Encounter

Purpose
Encounters that have been previously canceled can be retrieved, by users with the proper privledges.
 

Details:

Steps

To retrieve a previously canceled encounter, follow these steps:

  1. On the Charts tab, search for the patient with the encounter you want to un-cancel.
  2. Select the patient name bar. The Patient Demographics screen appears.
  3. Select the Open Encounters tab. The list of currently open encounters appears.
  4. Select the Canceled button on the lower portion of the tab. A list of all canceled encounters for this patient appears.
  5. Select the colored bar for the previously canceled encounter that you want to un-cancel.
  6. The system prompts you to be sure that you want to uncancel the selected encounter. Select Yes.
  7. The encounter status is changed from canceled to Closed and it will appear on the All encounters list.
