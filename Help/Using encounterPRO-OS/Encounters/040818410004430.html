How to: Review Closed Assessments, Treatments, and Histories.

Purpose
Even after an assessment or treatment is closed, you can still review that information as needed.

Before You Begin
Your procedure will depend on the type of historical information you want to review. If you want to review:

  • Closed assessments: start on the Summary tab
  • Closed treatments associated with an assessment: start on the Summary tab
  • Closed histories (vitals, ROS, physicals, etc.): start on the Review menu
  • Closed treatments by type: start on the Review menu
Details:

Steps
Follow the steps below:
To Review Closed Assessments and Treatments from the Summary Tab:

  1. From the Patient Chart, select the  Summary/Rev tab. The Summary display appears.
  2. Select the box for Problems. The assessment history screen displays. The default displays both Open and Closed assessments.
  3. To view only closed assessments, select the Closed button below Assessments. The display shows closed assessments and their associated treatments. To view only the assessments, select the Display Mode third from left with the thick lines. 
    NOTE: The Assessment Type column to the right allows the user to filter out unwanted assessment types.
  4. To view only closed treatments, select the Closed button below Treatments. The display shows closed treatments with the assessments they were associated with. To view only the treatments, select the Display Mode to the far right with the thin lines.
  5. Select Finished to return to the Summary tab.

To Review Closed Histories and Treatments from the Review Menu:

  1. From the Patient Chart, select the Other Items button. A menu of options appears.
  2. Select Review Menu. A menu of the various histories, assessments, and treatments that are available for review appears.
  3. Select one of the items from the menu to review that type of observation or treatment. The History Review screen appears.
  4. Select Finished to return to the Patient Chart.
