Description
Generally, when you check in or 'arrive' a patient in the billing and scheduling system this will automatically check the patient into encounterPRO. If you are using encounterPRO stand alone, or wish to work on a Test patient during configuration, follow these steps to create a new encounter.

How to Access This Screen
Access this screen by selecting New Encounter from the Patient Demographics screen.
