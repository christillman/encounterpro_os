Description
This screen displays current and previous patient encounters. Use the buttons at the bottom of the screen to change the list of encounters from Open to All or Canceled.

An encounter bar displays for each encounter. The encounter bar displays the encounter date, visit type, and the provider's name color-coded to the provider who 'owns' this visit.

Select the encounter bar to display a menu of options for that encounter. Select the sigma (Σ) button to view a summary report of the encounter.

How to Access This Screen
Access this screen by selecting the Encounters tab on the Patient Demographics screen.
