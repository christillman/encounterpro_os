How to: Document an Entry of Records Encounter.

Purpose
Entry of Records encounters are used to scan documents into a patient's chart or manually enter data into the chart when the patient is not physically in the office.

Before You Begin
Some practices will have an Entry of Records room (usually available from the Chart Room or Holding Room). When that room is available, the patient can be checked into that room directly and the encounter type will default to Entry of Records, saving the user several steps.

Details:

Steps
Follow the steps below:

  1. From the Office view, select the Chart Room or Holding Room tab.
    NOTE:  Entry of Records encounters are used when the patient is not physically in the office, so we do not want to put the patient into a treatment room.
  2. Select a room, if there is a specific Entry of Records room, select that. The Room View appears.
  3. Select the Check-in button at the bottom of the screen. The Patient Search screen appears.
  4. Search for the patient using one of the available search options. For example, you can enter a partial last name using the keyboard or select the ABC button to use the touchscreen capability, then press Enter. A list of matching patients appears.
  5. Select the patient name bar. The Create New Encounter screen appears.
  6. Confirm that the Encounter Type is Entry of Records. If not,
    a. Select the bar for Encounter Type. The Select Encounter Type screen appears.
    b. Select Indirect as the Encounter Mode in the upper right of the screen.
    c. Select Entry of Records from the list.
    d. Select OK.

  7. Select Finished. The Patient Chart appears. At this point, you can scan documents directly into the chart from the Scanned Items tab, or use the other functions to document information for this patient.
  8. When the information has been added, select Exit Chart. The Review Billing screen appears. There should be no billable items for this encounter.
  9. Select Finished.
