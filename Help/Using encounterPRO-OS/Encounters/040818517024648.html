How to: Change the Encounter Owner

Purpose
The encounter owner determines the provider of record for this encounter. The encounter owner's ID will be used in the billing system for all encounters he/she 'owns.'

Certain Tasks may be pre-ordered in the workplan for the encounter owner and some treatment workplans (such as with lab integration) will assign tasks to the owner of the encounter in which the treatment was ordered.

Changing the encounter owner using the following procedure can also be accomplished by having the new provider Take Over the encounter. See How to Take Over an Encounter for details.

Before You Begin
If there are any preferences set at the provider level, such as favorite physical exam or favorite Quick View, those items are determined when the encounter is started (with the Get Patient service). If the encounter owner is changed after the encounter workplan has already started, then the system will user the preferences for the original encounter owner  the preferences will not be changed to the current owner.

Details:

Steps
Follow the steps below:

  1. The encounter owner can be changed at virtually any point in the encounter without interfering with the workflow. You can access the Edit Encounter screen from the menu displayed when you select the patient name bar from one of these locations: 
    • The Waiting Room
    • Any treatment room
  2. Select the Edit Encounter button. The Edit Encounter Information screen appears.
  3. Select the colored bar for the Encounter Owner. The Select User screen appears.
  4. Select the desired user from the short list or use the search options.
  5. When you are finished changing the Encounter Owner, select OK. The Edit Encounter screen reappears.
  6. Select Finished to save your changes.
