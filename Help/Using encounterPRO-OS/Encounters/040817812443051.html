How to: Manually Check in a Patient

Purpose
Checking in a patient depends on whether or not you are using an integrated billing and scheduling system. If your practice has integrated EncounterPRO with a PMS, the exact method used will further depend on the specific billing and scheduling system. Generally, when you check in or "arrive" a patient in the billing and scheduling system this will automatically check the patient into EncounterPRO.

There will be situations when the patient will need to be manually checked in to EncounterPRO, such as:

  • EncounterPRO is not integrated with a scheduling system
  • The integration with the scheduling system is not working for some reason
  • When you want to practice using EncounterPRO functions

Before You Begin
If you are using EncounterPRO stand alone, or wish to work on a Test patient during configuration, follow these steps.
You will need to know:

  • Which provider the patient is here to see
  • What type of visit this is (sick, well, etc.)
Details:

Steps
Follow the steps below:

  1. From the Charts screen, search for the patient.
  2. Select the patient name bar to display the Patient Demographics screen. Confirm that this is the correct patient.
  3. Select the New Encounter button at the bottom of the screen. If the patient has any outstanding follow-up visits or appointments pending, the system will prompt you to indicate if today's visit is related to that follow-up.
    • If today's encounter is not related, select the button for "This encounter is not related to a followup order"
    • If today's encounter is related to the follow-up, select OK.

    The Create New Encounter screen appears.
  4. Be sure that the Encounter Type is correct for this particular visit. If you need to change it, select the gray bar, then select from the available encounter types.

    NOTE: It is very important to select the correct Encounter Type. The Encounter Type controls the screens and tasks displayed to the clinical staff and providers. If the wrong Encounter Type is chosen, it will be more difficult to document the visit.
  5. Select the appropriate Encounter Owner for this visit. If today's visit is with a different provider than the one defaulted (the patient's primary provider), select the colored name bar, then select the appropriate provider from the list.
  6. If the patient is new to your practice, select the New/Established button so that it reads "New."
  7. Select Finished. If your office has a sick and well waiting room, a menu appears for you to select the appropriate waiting room for this patient.

    NOTE: You must select either Sick or Well. If you select Cancel, the patient will go to the Nowhere room.
  8. The patient appears in the Waiting Room. The color of the patient name bar will be the color of the provider chosen as the encounter owner.
