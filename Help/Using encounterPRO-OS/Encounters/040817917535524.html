How to: Document an Encounter from the Patient Chart

Purpose
There are four parts to documenting an encounter using the Patient Chart:
  1. Reviewing or documenting the histories (HPI, PFSH, ROS).
  2. Entering the physical examination findings.
  3. Update existing assessments and treatments.
  4. Selecting new assessments and treatments.
Before You Begin
You can also use the Quick View screens to document an encounter. See How to Document Using the Quick View Summary.
Details:

Steps
Follow the steps below:
To Review or Document Histories:

  1. From the Office View, select the Examination service bar. The Patient Chart appears.
  2. Scroll through the information on the Patient Chart, then select the blue title for any history for which you want to review or enter results.
  3. If you selected the PFSH, a history review screen appears.
    • To make changes to the information, select the large gray text area, then select either Reviewed with Changes or New Results to display an entry screen.
    • To review without making changes, select the large gray text area then select Reviewed No Changes. When finished reviewing the history, select the I'm Finished button to return to the PFSH History Display screen.
    • Select Finished to return to the Patient Chart.
  4. If you selected the HPI (History of Present Illness), a HPI screen appears. The HPI can be either a drill down or Yes/No format.
    • If this is a drill down screen., select the bar on the right side to display another level of observations or selections.
    • If this is a Yes/No format screen the HPI select the Yes or No button for each selection. Or use the Select All button under the column to set all selections to one or the other option. When you have completed the HPI, select Finished. The Patient Chart reappears.

    Note: In order to get full E & M credit for the history documented and reviewed in an encounter, you MUST document something in the HPI and Chief Complaint screens.
  5. If you selected the ROS (Review of Systems), a ROS screen appears. The ROS can be either a drill down or Yes/No format.
    • If this is a drill down screen., select the bar on the right side to display another level of observations or selections.
    • If this is a Yes/No format screen the ROS select the Yes or No button for each selection. Or use the Select All button under the column to set all selections to one or the other option.
    When you have completed the ROS, select Finished. The Patient Chart reappears.

To Document the Physical Examination:

  1. Select the blue title for the Physical Examination, the Exam screen appears. The physical exam screens are also available in both the Drill-down and Yes/No formats. Most often, the drill down format is used. The Physical Exam that appears is based on the workplan and the user's preferences.
  2. If this is a drill down screen document the findings for the examination using any combination of the following:
    • Select the bar on the right side to display another level of observations or selections.
    • Use the Standard Exam buttons on the right side of the screen. If the findings for today's encounter are similar to one of the pre-set standard exams, select that button to populate the screen. Make any changes needed.
    • Use the Previous Exam button in the upper right of the screen. Select Prev Exam and a list of previous examinations appears. Select the encounter where the physical examination findings are similar to today's. The screen is populated with the findings from that examination. Make any changes needed.
  3. When finished entering information for the physical, select the Top button (when it appears), review the information entered, then select the Finished button. The Patient Chart reappears.

To Order Assessments and Treatments:

  1. If you want to update an existing assessment, select the blue date of the assessment to access the dashboard screen. From this screen you can add/change the treatment plan, add a progress note, or close the assessment.
    • Select to add a progress note to the assessment for today's visit
    • Select to close the assessment. You can choose to bill or not bill the assessment as part of today's visit.
  2. If you want to select a new sick assessment, select the New Sick DX button. The Select Sick Assessments screen appears.
  3. Select the assessment(s) from the list or use the search options to locate the desired assessment. You can select multiple assessments, as needed. When finished, select OK.
  4. For each assessment selected, a treatment list appears to allow you to order treatments for that diagnosis. The treatment list is pre-populated with commonly used treatments for the assessment listed at the top of the screen. To order treatments for this patient, select as many items from the list as desired.
  5. If a treatment you want to order does not appear, select the New Treatment button. A menu of treatment types appears. Select the button for the type of treatment you want to order. The appropriate treatment selection screen appears.

    NOTE:  Follow-ups and referrals are considered treatments and should be ordered as part of the treatment plan at this point.
  6. Select the treatments you want to order from the list or use the search options to locate the desired treatments. When finished selecting items, select OK to return to the treatment list.
  7. When finished selecting treatments for this assessment, select the Finished button. Certain treatments such as prescription drugs, follow-ups, and referrals require additional information. If you chose a treatment that requires additional information, those screens will be presented to you automatically.
  8. If more than one assessment was chosen, the treatment list for the next assessment appears. Repeat steps 4 - 7 to order treatments for each assessment. When all the treatment screens have been completed, the SOAP screen reappears. If no other assessments need to be documented or treatments updated, you can complete the encounter.
