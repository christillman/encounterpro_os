How to: Document an Encounter Using the Quick View Summary

Purpose
There are three parts to documenting an encounter that you can perform using the Quick View Summary:
  1. Reviewing or modifying the history (HPI, PFSH, ROS)
  2. Entering the physical examination findings
  3. Updating existing assessments and treatments
Before You Begin
Your practice may not have the Quick View screens enabled. This is a configuration setting. If you want to use the Quick View screens, contact EHR Product Support and schedule a consultation.

To select new assessments, you will need to select Finished and complete the encounter from the Patient Chart or the Quick View DX/Trt screen.
Details:

Steps
Follow the steps below:
To Review or Modify the Histories:

  1. From the Quick View Summary screen, select the blue title for any history for which you want to enter results or modify the information entered by the clinical staff. The appropriate history entry screen appears.
  2. If you selected the PFSH, the PFSH History Display screen appears.
    • To make changes to the information, select the large gray text area, then select either Reviewed with Changes, or New Results to display an entry screen.
    • To review without making changes, select the large gray text area then select Reviewed No Changes.

    When finished reviewing the history, select the I'm Finished button to return to the Quick View Summary.
  3. If you selected the ROS (Review of Systems), the Review of Systems screen appears. The ROS screen can be either a drilldown or Yes/No format, depending on your practice's preferences.
  4. When you have completed the ROS, select Finished. The Quick View Summary reappears.

To Document the Physical Examination:

  1. From the Quick View Summary, select the blue title for the Physical Examination, a screen similar to the following appears: The physical exam screens are also available in both the drill-down and Yes/No formats. The Physical Exam that appears is based on the workplan and the user's preferences.
  2. When finished entering information for the physical, select the Top button (when it appears), review the information entered, then select the Finished button. The Quick View Summary screen reappears

To Update an Existing Assessment:

  1. From the Quick View Summary, select the blue date of the assessment to access the dashboard screen. From this screen you can add/change the treatment plan, add a progress note, or close the assessment.
    • Select to add a progress note to the assessment for today's visit
    • Select to close the assessment. You can choose to bill or not bill the assessment as part of today's visit.
  2. To select new assessments, you will need to select Finished and complete the encounter from the Patient Chart or the Quick View DX/Trt screen.
