Description
When creating a new encounter manually or changing an encounter type (from the Waiting Room), this screen allows you to select the encounter type for the visit. If the encounter type you want does not appear on the short list, use the search options to find the desired type.

How to Access This Screen
Access this screen by selecting the Encounter Type bar on the Create New Encounter or Edit Encounter screens.
