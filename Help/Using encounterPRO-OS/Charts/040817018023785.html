Description
When you want to add or select a portrait use this screen. This screen displays a gray bar for each existing patient portrait file for the current patient. Select the button with a star to add a new portrait.

How to Access This Screen
Access this screen by:

  • Selecting the portrait box on the Patient Chart screen
  • Selecting the portrait box on the Patient Demographics screen
  • Selecting the Portrait button from any menu on which it appears
