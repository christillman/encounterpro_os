How to: Document a Patient Name Change or Alias.

Purpose
There are several reasons you might need to set up an alias for a patient:

  • Patient is adopted and has records under both birth name and adopted name
  • Patient is married and has a name change
  • Patient has a legal name change

Setting up an alias allows you to search for either name in the database.

When you search for a patient who has an alias in the system, you can search using either name and find the patient. The patient name bar will list the alias type next to the name you searched for plus display the other aliases for that patient.

Before You Begin
You will give each name an Alias Type. You should decide what terms you want to use at your practice. Commonly used Alias Types include: Birth Name, Married Name, Adopted Name, Legal Name.

Details:

Steps
Follow the steps below:

  1. Access the Patient Demographics screen by:
    • Select Chart from the Office View, search for the desired patient. Select the patient name bar.
      OR
    • From the Patient Chart, select Other Items, then select Edit Patient Data.

    The Patient Demographics screen appears.
  2. Select the Alias tab. The Alias screen appears.
  3. The name entered on the General Info tab automatically appears in the alias list. To add a new alias, select the Add New Alias button. A dialog box appears.
  4. Enter the Alias Type. For example, Married Name, Birth Name, etc.
  5. Select OK. The Patient Alias screen appears.
  6. Enter the information for this name, then select Finished. The new name appears in the Patient Alias list.
  7. Select Finished to return to the screen you started on in step 1.
