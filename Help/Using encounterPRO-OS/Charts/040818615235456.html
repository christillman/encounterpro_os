How to: Export a Patient Chart.

Purpose
An individual patient's chart can be exported to a folder on the workstation or network. From there, the files can be saved to a portable medium, such as a CD ROM or flash drive, or printed. You can export the entire chart, including all attachments, or select from a group of items in the chart to produce a subset of the documentation.

If you need to export the files for a group of patients, you can use the multiple chart export function. See How to: Export Multiple Patient Charts for details.

Before You Begin
Determine the destination on the workstation or network for the exported files.

Details:

Steps
Follow the steps below:

  1. From the Office view, select the Charts tab.
  2. Search for the patient using one of the available search options. For example, you can enter a partial last name using the keyboard or select the ABC button to use the touchscreen capability, then press Enter. A list of matching patients appears.
  3. Select the patient name bar. The Patient Demographics screen appears.
  4. Select the Chart Export button. The Export Patient Chart screen appears.
  5. Determine where you want the files to be saved. The default is c:\temp\PatientCharts where a folder for the patient will be created. If you want to save the files in a different location, select the ellipsis button (...) and locate the desired location for the exported files.
  6. Select which parts of the chart you want to include:
    • Patient Assessment History
    • Patient PFSH Report
    • Patient All Medications
    • Encounter Reports
    • Attachments list
    • Attachments (each attachment will be saved into a sub folder)
  7. Select Export. A progress bar appears as the files are being exported.
  8. When the files have all been saved, a message appears. Select OK. The Patient Demographics screen reappears.
  9. Select Finished to return to the Patient Search screen.
