How to: View a Patient's Chart

Purpose
Many times you will need to review a patient's chart. Most tasks in EncounterPRO can be accessed via the patient's chart.

Before You Begin
Is the patient in the office? The steps to access the Patient Chart  screen will depend on whether the patient is in the office or not. Remember, a patient is considered 'in the office' if they are in any room of the Office view: Exam rooms, Telephone room, Chart Room/Holding, Checkout, etc.

Details:

Steps
Follow the steps below:
To View a Patient's Chart for a patient in the Office:

  1. Locate the patient in the Office view.
  2. Select the room where the patient is currently located.
  3. Select the patient name bar (left hand side), then select the View Chart button from the menu.

ToView a Patient's Chart for a patient not in the Office:
NOTE: The following steps can also be used while the patient is in the office.

  1. From the Office view, select the Charts tab.
  2. Search for the patient using one of the available search options. For example, you can enter a partial last name using the keyboard or select the ABC button to use the touchscreen capability, then press Enter (on the computer keyboard). A list of matching patients appears.
  3. Select the patient name bar. The Patient Demographics screen appears.
  4. Select the View Chart button at the bottom of the screen.

Other Ways to Access the Patient Chart:
You can view a patient's chart from many areas in EncounterPRO, including:

  • From the Vitals screen, select View Chart
  • From a Treatment Room menu, select View Chart
