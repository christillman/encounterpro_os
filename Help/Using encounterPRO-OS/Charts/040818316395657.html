How to: Export Multiple Patient Charts

Details:

Purpose
In version 4, EncounterPRO introduced the ability to export a single chart to a folder on the network or workstation hard drive. In version 5, you have the added ability to export a set of charts at one time. You select a set of charts by selecting patients from a list or by primary provider.

This function allows you to select multiple patient charts to export either to a set of files saved either on the network or workstation hard drive or a CD-ROM, or send it directly to the printer.

Also, if you want to print the charts en masse, then you will probably find it useful to get a relatively affordable printing manager. The software that EHR tested with this batch printing function can be found at http://www.traction-software.co.uk/batchprint/index.html.

Steps
Follow the steps below:

  1. From the Office View, select Utilities. The Utilities List appears.
  2. Select the Export Charts button. The Multi-Patient Chart Export screen appears.
  3. Select the destination directory for the patient reports and attachments. The default will create a sub-folder for each patient in the c:\temp\Patient Charts\ directory.

    If you want to select a different directory, select the three-dot button to display the Browse for Folder dialog appears. Locate the desired directory, then select OK.
  4. Select the patients you want to export. You can select patients one of two ways:
    • By patient name
    • By Primary Provider
  5. If you want to select patients by name, select By Patient. The Patient Search screen appears.
    1. a. Enter the search criteria for the first patient.
    2. b. Select the patient name bar. The patient name appears in the Select Patients list on the right.
    3. c. Repeat steps a and b until you have selected all patients.
    4. d. Select OK.

      5. The Multi-Patient Chart Export screen reappears.
  6. If you want to select patients by the Primary Provider, select Pick Provider, the Select User screen appears. Select one provider from the list. Select OK. The Multi-Patient Chart Export screen reappears.
  7. Select the time frame to exclude recently exported charts. The default is 7 days. If a patient in the list of selected patients has been exported more recently than the time frame selected, the system will ask you to confirm that you want to include that chart in this export.

    You can choose No to remove that patient from the list, Yes to include the specific patient named in the message, or Yes to All to include this patient and any others that were recently exported in the list.
  8. Choose what information and reports you want to include in the patient chart export by select Yes or No in the Include box. The default is Yes.
  9. Select the export options:
    • Export to files: confirm that the http://www.knowledgepak.com/kpaksonline/kpol.asp?ViewLink=true&k2dockey= in the upper portion of the screen is the one you want to use for this export function. If you want to change the path, select the three dot button to the right.
    • Send to Printer: the default is No. Select the No button to change it to Yes. Fields appear for you to select the destination printer and whether you want to print a separator page between patient files.
    • Both export to files and print: You can select both options for a single export function.
  10. Select Export Now. A Progress box will appear. When the export is complete a message will display showing how many charts were exported.
  11. Select OK to return to the Utilities list.
