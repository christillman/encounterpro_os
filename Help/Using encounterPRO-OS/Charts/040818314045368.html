How to: Document Adjusted Age for Premature Infants.

Purpose
With premature infants it is important to track their adjusted age for things like growth charts until age 24 months. With encounterPRO you can track both the actual age and Gestational Adjusted Age (GAA).

Note: The CDC growth curve screens will only show a single data point in any two-week timeframe. The system will suppress multiple data points that occur in the same two-week time period because the CDC growth data is provided in half-month increments. The growth-percentiles screen available from the Data button does not suffer from this problem because it does not need to show CDC data. If you need to see the percentile increase/decrease for clinical reasons, use the percentiles screen.

Before You Begin
You need to know how many weeks premature that the child was born.

Details:

Steps
Follow the steps below:

  1. Access the Patient Demographics screen by:
    • Select Charts from the Office View, search for the desired patient. Select the patient name bar. OR
    • From the Patient Chart, select Other Items, then select Edit Patient Data.

    The Patient Demographics screen appears.
  2. Select the Age bar. A dialog box appears. The patient's date of birth and actual age are displayed.
  3. Enter the number of weeks premature that the infant was born. The system calculates the Gestational Adjusted Age (GAA) by subtracting the weeks premature from 40 weeks.
  4. Select OK. The calculated age and GAA appear on the Patient Demographics screen. The GAA will be used to calculate the age for the growth percentiles for the patient until the GAA reaches 24 months.
  5. Select Finished to return to the screen you started on in step 1.
