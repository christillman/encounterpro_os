Purpose
When encounterPRO is not integrated with a billing and scheduling system, you will need to manually add each patient to the database.  If your practice is integrated, all new patients will be added automatically from the data entered in the PMS system.
Details:

Steps
Follow the steps below:
To Create a New Patient Record:

  1. Select the Charts tab from the Office view. The search screen appears.
  2. Select the New Patient button. The system prompts you to confirm that you want to create a new patient record.
  3. Select Yes. The New Patient screen appears.
  4. Enter the basic information for this patient:
    • Name
    • Address
    • Patient Status (Select Active)
    • Sex
    • Birth date
    • Race
    • SSN
    • Email
    • Phone

    The Patient Demographics screen appears.
  5. encounterPRO can assign a billing ID, or you can manually enter one. Select the three dot (...) button to manually enter the patient Billing ID.
  6. There is more information that can be entered on the various tabs of the Patient Demographics screen. See Screen: How to: Update Patient Demographic Information for complete details..
  7. When finished entering information for this patient, select Finished. The Patient Search screen reappears.
