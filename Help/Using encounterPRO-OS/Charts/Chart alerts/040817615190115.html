How to: Delete a Chart Alert

Purpose
Chart alerts (both alerts and reminders) are part of the legal medical record and are never deleted entirely from the database. However, once the alert is no longer needed, it can be removed from view. The time, date, and user who removed an alert is also saved for auditing purposes.

Before You Begin
There are two types of chart alerts, Alerts and Reminders. Only users with proper access rights (providers and clinical staff) can remove alerts. Any user can remove a reminder.

Details:

Steps
To remove a chart alert:
Follow the steps below:

  1. Access the Chart Alert screen by doing one of the following:
    • From the Patient Demographics screen, select the Alert button located at the bottom of the screen.
    • From within the Patient Chart, select the Other Items button, then select the Alert button.
    • From the Waiting Room, select the patient name bar, then select the Alert button.
    The Chart Alerts screen appears.
  2. Select the gray alert text bar. Confirm that you want to delete the alert. The alert will remain in the patient's file, but will no longer display automatically when the chart is accessed.

To remove a reminder:
Follow the steps below:

  1. Access the Chart Alert screen by doing one of the following:
    • From the Patient Demographics screen, select the Alert button located at the bottom of the screen.
    • From within the Patient Chart, select the Other Items button, then select the Alert button. From the Waiting Room, select the patient name bar, then select the Alert button.
    The Chart Alerts screen appears.
  2. Select the Dismiss button to the right of the reminder. Confirm that you want to delete the reminder. The reminder will remain in the patient's file, but will no longer display automatically when the chart is accessed.
