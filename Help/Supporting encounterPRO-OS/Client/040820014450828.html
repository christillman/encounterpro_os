How to: Link Tables in Access 2007

Solution: Tables have not been linked.
Details:
  1. Open Microsoft Access.
  2. Click on the 'new blank database' icon.
  3. Name the database CPR and click 'create'.
  4. When that opens, click the 'External data' tab at the top.
  5. Click and drop down on the 'More' menu item select ODBC database.
  6. Select the 'Link to the data source by...' radio button then click ok.
  7. Click 'new', beside DSN name.
  8. Select the lowest option 'SQL Server'.
  9. Name it CPR, then click next then click finish.
  10. In this window, click the drop down the 'Server' menu and select the name of the SQL server then click next.
  11. Click next.
  12. On this window, select the 'change the default database to' radio button.
  13. Drop down and select the 'CPR' database.
  14. Click next then click finish.
  15. Click 'test data source' if successful, click ok.
  16. Click ok on the Data source screen if 'CPR' is available and highlighted.
  17. Click 'select all' then click ok.
  18. At this point you may click 'ok' on the first few popups, but i usually just hold the enter key and let them all fly by. Otherwise you may be clicking for quite some time.
  19. When it finishes, the last table that should have been opened by default when the linking finished should be x_translation _set or something like it. You can start working by selecting the tables you need to work on to your left in the 'all tables' section.

NOTE: For further reference, you may refer to the knowledge base article for linking tables in Access 03.
