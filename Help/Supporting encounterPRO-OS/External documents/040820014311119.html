Error: Unable to Verify Attachment Location

Error Message: Unable to verify attachment location. You may not be able to use attachment functionality.
Cause:
Solution: Setting preference within Encounterpro
  1. Select Configuration
  2. Select Preferences
  3. Select Attachment Preferences
  4. Attachment Path = Folder Share Name (Usually CPRFILES)
  5. Attachments Server = Computer Name (Usually Application Server)


If setting the preference in the interface does not work try:
Open cpr database
Go to b_Attachment_functionality
Make sure server name is correct
Also verify CPRFILES is shared folder
