How to: Fix Chart Export Error

Solution: The display script attached to the medication report was invalid and that is what caused the text control error.
Details:

If you go to Authorize services and set the Patient level, you'll find Chart Export.  Once you've selected chart export there is a "configure service" button at the bottom of the page, select this button and you can set up the parameters for the chart export.

1.       Replace display script 645 with a valid display script (found in c_display_script) for patient medication report

2.       Added the display script for the vaccine history (discovered that there was no vaccine history in the export L)

3.       Set the default encounter report which is the last param in the config service wizard.  For some reason it was blank in their db but not in the testing.  I'm not sure if the "script" we run against the database has anything to do with these issues or not.

Symptoms: Text control error when trying to export a patient chart
