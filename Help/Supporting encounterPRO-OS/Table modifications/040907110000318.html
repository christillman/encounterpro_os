How to: Change the Owner/Office ID of a Practice

Solution Title: How to: Change the Owner/Office ID of a Practice
Overview:

There is a script called Update owner_id.sql that can be run when you need to change the owner/office id of a practice.

Information About the Script

This script should be run after hours because it will perform a table scan of the affected tables.

  • Replace the xxx here with the owner_id we're changing FROM (i.e. the WRONG customer_id) SET @ll_from_customer_id = xxx
  • Replace the yyy here with the owner_id we're changing TO (i.e. the CORRECT customer_id) SET @ll_to_customer_id = yyy
