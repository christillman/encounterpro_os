Error: Upgrading Mod Level

Error Message: Cannot insert the value NULL into column ...
Cause:
Solution:

Anytime you get a 'Cannot insert the value NULL' error or a 'Invalid column name' error then follow these steps:

 

1)      Click 'Check Now' again

2)      Scan the Hotfixes tab for any hotfixes that have an error status, execute them

3)      Retry the upgrade

 

If there is a later production release of Epro Server, consider upgrading.
