Billing not working with LCR interface

Cause:
Solution:

This article explains what to do if a practice with an LCR billing interface is not working. Initialize the default value of 'N' for batch_mode for 'o_message_log' table.

Details:

This article explains what to do if a practice with an LCR billing interface (particularly Shands) is not working.

Resolution
Initialize the default value of 'N' for batch_mode for 'o_message_log' table.

See also Support issue 20673
