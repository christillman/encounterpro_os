How to: Add a Secondary Visit Charge to the Pick List

Solution: A secondary visit code is missing from the pick list on the billing screen.
Details:

To add a secondary visit charge to the list, add it in c_procedure as a secondary type, and the service as secondary, and add it to the c_domain table with the first column that fits *sec*
