How to: Get a copy of the billing file that was exported to the PM System

Solution:

EncounterPRO will never send a partial bill. It sends complete billable items listed in the EncounterPRO billing screen. Find a path where EncounterPRO saved a copy of all the bills. If you know the patient# and discharge date then look up the path and find a message written for that date time or do a search by patient#. For HL7 systems, look in windows registry to find a path where it's saving copy all the bills. For non-HL7 systems, find a path in c_component_attribute table by looking at the attribute value for 'hold_bills_directory'.

Details:

EncounterPRO will never send a partial bill. It sends complete billable items listed in the EncounterPRO billing screen. Find a path where EncounterPRO saved a copy of all the bills. If you know the patient# and discharge date then look up the path and find a message written for that date time or do a search by patient#.

HL7 systems
Look in windows registry (In 'Hkey_Local_Machine', select 'Software', click 'AHC', select 'Transport' and click 'EPROHOLDMESSAGES') to find a path where it's saving copy all the bills.

To save a copy of bills , set the value for 'hold_outgoing' attribute in c_component_attribute table to 'TRUE' to turn it ON or 'FALSE' to turn it OFF. If the row does not exist in c_component_attribute, add it.

dbo_c_Component_Attribute

 component_id attribute_sequence  attribute  value 
 TRN_OUT_FILECOPY_HL7 1000086  hold_outgoing  TRUE 

Non-HL7 systems
Find a path in c_component_attribute table by looking at the attribute value for 'hold_bills_directory'.

To save a copy of bills , set the value for 'hold_outgoing' attribute in c_component_attribute table to 'Y' to turn it ON or 'N' to turn it OFF. If the rows do not exist in c_component_attribute, add them.

c_Component_Attribute

 component_id attribute_sequence  attribute  value 
TRN_OUT_FILECOPY 1000047  hold_bills  Y 
TRN_OUT_FILECOPY  1000048  hold_bills_directory  c:\jmj\pmshare\holdbills
