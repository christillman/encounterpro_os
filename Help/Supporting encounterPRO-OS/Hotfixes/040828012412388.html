Error: Logging incoming message

Error Message: Error logging incoming message.
Cause:
Solution: Check to make sure the directory has FULL permissions on the folder where it is reading / writing the messages. Check in the o_message_subscription table to make sure the 'address' field holds a valid path to import and export messages.
Details: This article explains why the transport in throws an error 'error logging incoming message'.

1) Check to make sure the directory has FULL (read & write) permissions on the folder where it is reading / writing the messages.
2) Check in the o_message_subscription table to make sure the 'address' field holds a valid path to import and export messages. It is important to check whether the file delimiters are entered properly. There are cases where it ignores the extension (say *.txt or *.* in case all files needs to be read) and in such case it is possible to get the 'error logging incoming' message.
