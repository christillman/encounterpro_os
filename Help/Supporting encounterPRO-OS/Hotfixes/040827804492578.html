Missing folder on the post to screen

Cause: This is a known bug in our vs4 client.
Solution:

When you add a folder in 'Folders' under 'Configuration', it only creates the folder in the patient's chart. The folder is not accessible from the scan/post folder list. Verify folder is defined in 'dbo_c_folder'. Add folder to 'dbo_c_folder_selection'. Create a new record in 'c_folder_selection'. Set object context to 'patient'. Set Auto Select Flag to 'N'.

Details:

When you add a folder in 'Folders' under 'Configuration', it only creates the folder in the patient's chart. The folder is not accessible from the scan/post folder list. To resolve the problem:

1) Verify the folder is already defined in 'dbo_c_folder'.
2) The folder(s) will need to be added to 'dbo_c_folder_selection'.
3) Using the folder ID create a new record in 'c_folder_selection'.
4) Set object context to 'patient'.
5) Set Auto Select Flag to 'N'.
6) The Sort Sequence controls the placement of the folders within the list.
EXAMPLE: Adding a number one integer higher than the highest patient context sort sequence number will put the folder at the bottom of the list of folders.
