Error: Cannot Insert the Value Null (Insert Fails)

Error Message: Cannot insert the value NULL or Invalid column name
Cause:
Solution:

Follow these steps:

  1. Click "Check Now".
  2. Scan the Hotfixes tab for any hotfixes that have an error status, execute them.
  3. Retry Installing the Report.
  4. If the error still occurs then: If their Mod Level is < 127 or they are running an old Epro Server (< 6.1.10) then consider upgrading. If they are already a recent ML/Epro Server and Check Now doesn't fix it, then try running sp_rebuild_constraints on the table in question (c_Report_Definition this time). If that doesn't fix it, then try to reproduce it here in a testing database.
