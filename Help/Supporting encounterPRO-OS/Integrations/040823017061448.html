Troubleshooting Lab Integration: Import Service is Timing Out.

Cause: This problem is a consequence of not having the 'JMJ Lab Results' external source properly identified on the client's database server.
Solution: Open EncounterPRO. Go to 'Configuration' tab and select 'External Sources'. Find 'JMJ Lab Results' external source and make sure that it is highlighted.
Details:

To properly configure the external sources of the server:
1) Log onto the server and open EncounterPRO.
2) Go to the 'Configuration' tab and select 'External Sources'.
3) Find the 'JMJ Lab Results' external source and make sure that it is highlighted.
