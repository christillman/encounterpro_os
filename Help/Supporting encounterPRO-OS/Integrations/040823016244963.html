Troubleshooting Lab Integration: Problems with XML files.

Cause: Both problems are related to a missing table entry that directs .xml files to a specific folder.
Solution: The following entry needs to be added to the table c_Folder_Selection.
Details:

The following entry needs to be added to the table:

c_Folder_Selection

folder_selection_id context_object context_object_type attachment_type extension folder auto_select_flag sort_sequence
1000711 Patient     XML Lab Reports Y  

Article created on 12/22/2004 by Kris.Griffith

Article last updated on 5/29/2007 by adam.zangara

Symptoms: .XML files being posted in patient chart. Manual post of .XML lab results requests a folder selection.
