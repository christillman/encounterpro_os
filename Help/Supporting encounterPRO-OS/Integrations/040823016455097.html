Troubleshooting Lab Integration: No Daily Referral Report.

Cause: A button for the Daily Referral report needs to be added to the Utilities list for each lab integration enabled at the practice.
Solution: A button needs to be added to the utilities screen. The new button is a general service called 'referrals by day'. The following entry can be added to the tables to eliminate the selection of labs.
Details: A button needs to be added to the utilities screen. The new button is a general service called 'referrals by day'. The following entry can be added to the tables to eliminate the selection of labs (for practices that only send specimens to one lab).
NOTE: Substitute the consultant ID to appropriate lab if different from LabCorp.

dbo_o_Service_Attribute
service attribute_sequence user_id attribute value
Referrals by Day 1000644   specialty_id $LAB
Referrals by Day 1000645   consultant_id LAB_CORP

Article created on 12/22/2004 by Kris.Griffith

Article last updated on 5/29/2007 by adam.zangara

Symptoms: The practice does not have 'Daily' Referral report for each of it's lab integrations.
