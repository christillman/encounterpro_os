Problem: Users at Practice Have Different SOAP Screen Views

Cause: One or more users have a different Specialty on the User Definition screen and are seeing a different SOAP screen/Patient chart.
Solution: Make sure that all users who have the same specialty have been assigned to the same Specialty in the User Definition screen.
Details:

At Gwinnett Pediatrics & Adolescent Medicine they had this problem:

User T.M. had a different view on SOAP screen than other users.  Took screen shots for different users viewing the same patient, Test-Robby. Compared the two and discovered "Standard Chart Menu SOAP Screen Default Adult" displayed in the bottom right hand corner for user T.M. and "Standard Chart Menu SOAP Screen Peds" displayed for user B.  From Config/Users, compared user profiles.  T.M.'s profile for "Specialty Tab" located in the right hand corner was set to "Pediatricians" and all other users were set to "Pediatrics".  Changed T.M.'s user profile to reflect "Pediatrics" and resolved issue.
