How to: Take Out Top Tab - Fixing Flow Tab Display

Solution: Taking out a top tab (example: H&P on the SOAP screen)
  1. Go to c_chart_section_page.
  2. Filter for the desired tab in the description field .
  3. For all of the rows returned, add 4 zeroes to the end of the chart_id (so the record won't be read).
Fixing the Flow tab:
  1. Go to c_chart_section_page_attribute.
  2. In Value, filter for *general*general* .
  3. Will return rows with the value %General %General Favorite Quick View, PT Flow% .
  4. Delete the first %General so that it just has %General Favorite Quick View, PT Flow% .
  5. The value will now look for the preferred PT Flow script.
