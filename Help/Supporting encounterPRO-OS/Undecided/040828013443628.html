How to: Set up a New Office

Solution:

To add a new office to 2.0, create new record in c_office. For 2.0 add following records to o_report_attribute if they don't already exist. Copy all records in o_workplan_start and change the office_id. Run below query against database UPDATE c_Office SET server = NULL.

Details:

Below are the steps for adding a new office to 2.0:

1) Create new record in c_office. The office_id field is what is linked in the other tables. The default is 0001 so if this is the second location it will be 0002.
2) For 2.0 add following records to o_report_attribute if they don't already exist. (If the practice name is the same for both location you can skip the text.heading attribute. If the address is the same you can skip the text.pratice_name heading)

5) Follow instructions in KB Article, Adding Rooms and Group Tabs to EncounterPRO.
