Error: A Recent Back-up Is Missing or Cannot Be Found

Error Message: A recent back-up is missing or cannot be found
Cause:

Each time users log in they receive the message ?A recent back-up is missing or cannot be found?
Users get this message normally when EncounterPro does not detect a recent DB back-up.

The job ?Epro Back-up? failed to run/complete successfully. There are several things that can cause this job to fail. The most common causes are not enough space for SQL to store the back up log on the target drive or the back up job was interrupted.

Solution: Ensure that EPRO DB Back-up Job is properly scheduled, and that the target drive has sufficient space to allow for log creation and storage.
Details:
  1. The back up job can be restarted by opening Enterprise Manager (SQL), expand the following:
    • Microsoft SQL Servers
    • SQL Server Group
    • (Local) (Windows NT)
      • Management
      • SQL Server Agent (Note: ensure that Server is connected 'Green Arrow')
      • Click 'Jobs'
      • Jobs will be displayed to the right
  2. Right click on the 'EPRO DB Backup' job and select 'View Job History'. This will display a history of the jobs activity i.e., when it last ran and the status (Succeeded/Failed).
  3. At this point, if the job indicates that it failed to complete on its most recent attempt: Restart the job by right clicking on it again and selecting 'Restart Job'.
  4. Refresh the job and the 'last run' column should change to something like 'running or in-progress'.
  5. Depending on the size of the DB it may take up to an hour for the EncountrerPro to stop displaying the missing Back-up message (until the back-up job completes).
  6. Exit Enterprise Manager.


NOTE:  If the EPRO DB Back-up indicates that it successfully completed as scheduled but the missing Back-up message is still displayed refer to KB Article #10215
If the EPRO DB Backup Fails to successfully complete check the Event Viewer for errors and take appropriate steps to correct the conditions causing the errors, then repeat steps listed above.
