How to: Check scanner outside encounterPRO

Solution: If the scanner is not working within EncounterPRO, you need to find out if it will work 'outside' EncounterPRO. You can do this via MS Word. This is the first thing that Support will need to know to troubleshoot the scanning problem.
Details:

1) Open 'MS Word'.
2) Select the 'Insert' menu.
3) Select 'Insert Picture' and click 'Scanner or Camera'.
4) A dialog box appears. The scanner should be in the drop down list. If it's not shown, scroll in the list to find the problem device.
5) Put a piece of paper in the scanner and select 'Insert'.
