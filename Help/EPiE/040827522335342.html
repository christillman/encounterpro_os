How to: Setup a Customer Subscription to an Interface on CMA

Solution:
Details:

HOW TO SUBSCRIBE AN INTERFACE

Step1: Go to the EpIE Customer Management webpage and then go the customer you like to setup the interface.

 

Then click new subscription and fill up all the required entries and complete the subscription.

HOW TO SETUP EPIE ACCOUNT FOR A CUSTOMER

 

Click on create new user account and assign EPIE user and password. These credentials are used to authenticate the message transfer thru EPIE from Customers. So these credentials need to be mapped in EncounterPRO at the respective customer sites.

Enter the EPIE user name [usually like 'EPIE'. (example: EPIE9991)] , Password (usually $!Epro&&*) and email (usually the 'EPIE'@encounterpro.com ).
