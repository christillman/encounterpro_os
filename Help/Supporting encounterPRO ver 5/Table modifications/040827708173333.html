How to: Remove all patient data from an EncounterPRO database

Solution:

You would like to get a copy of a customer's database and create a new demo database or install it as the starting point for a different customer.

1) Get a copy of the customer's database.  Never perform this process on a running production database.

2) Restore the copy on a server where you can safely modify it.

2) Download the following script and execute it in the new (copied) database:

Remove Patient Data.sql
