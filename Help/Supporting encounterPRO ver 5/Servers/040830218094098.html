How to: Install Support for Datawindow PDF Output

Solution:

When the Document Manager is used to send or print reports, EncounterPRO needs to create a file that will get sent or printed.  This file is saved as a permanent record of what was sent.

Most report types, such as RTF or PDF, automatically have a file format that is appropriate for saving a permanent copy of the report.  The Datawindow report type, which is used for the traditional EncounterPRO Prescription report, does not have a native file format.  The solution for Datawindow reports is to install a tool called a PDF Distiller which EncounterPRO will use to convert the Datawindow report output into a PDF document for permanent storage.

In addition to the PDF Distiller, EncounterPRO also needs a PostScript printer driver to create documents from datawindow reports.

Details:

1.    Download the latest production version of GhostScript:

 

The installation programs can be found on the FTP Site here:

EP_Upgrades/Production/Components/GhostScript

 

For 32-bit systems, use a version labeled '-32bit'.  For 64 bit systems, use a version labeled '-64bit'.

 

2.    Run the GhostScript installation program from a privileged windows account (e.g. a local administrator)

a.    The installation directory can be redirected as desired

b.    Shortcuts and Asian TrueType fonts are not needed

 

3.    Create a PostScript printer profile

a.    Go to 'Printers and Faxes' and click the 'Add a printer' button on the left side.  This will bring up the Add Printer Wizard.

b.    On the 'Local or Network Printer' page, select 'Local printer attached to this computer'.  UNCHECK 'Automatically detect and install my Plug and Play printer'

c.    On the 'Select a Printer Port' page, select the radio button 'Use the following port' and in the drop-down list select 'File: (Print to File)'

d.    On the 'Install Printer Software' page, click 'Have Disk…'

e.   On the 'Install From Disk' screen, enter the following (or browse to where the Sybase Datawindow PS driver is):

            C:\Program Files\Common Files\JMJTech Common\Sybase Datawindow PS

f.    Back on the 'Install Printer Software' page, select 'Acrobat Distiller' and click 'Next'

e.    On the 'Use Existing Driver' page (if it shows up), select 'Keep existing driver'.

f.     On the 'Name Your Printer' page, replace whatever is in the Printer name box with EXACTLY THIS NAME: Sybase Datawindow PS.  Select 'No' under 'Do you want to use this printer as the default printer?'.

g.    On the 'Printer Sharing' page, select 'Do not share this printer'

h.    On the 'Print Test Page' page, select 'No'.

i.      On the Completing the Add Printer Wizard page, click 'Finished'.

j.    If a warning appears about the Printers software having not passed Windows Logo testing, click 'Continue Anyway'
