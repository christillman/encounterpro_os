Error: Unable to Start Transport Out Service - Type Mismatch Accessing External Object Property

Error Message: Error in timer ding function (Type mismatch accessing external object property value at line 291 in function chrg of object u component outgoing filecopy hl7.)
Cause:

Mostly this error shows up if the following attribute is left NULL.

Solution: Fill in a valid facility id value to fix this error and restart the transport out.
Details:

The above mentioned issue is no longer valid from Install Bundle 5.1.5 because it does not care about 'FacilityID' attribute for 'TRN_OUT_FILECOPY_HL7'.
