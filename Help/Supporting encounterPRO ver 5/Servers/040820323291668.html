Solution Path: Supporting encounterPRO ver 5 > Servers

Error: Check Now Fails and Check Remote Server Fails

Error Message: SQL Server does not exist or access is denied.
Cause: This error is caused by the EncounterPRO database server being unable to access the EncounterPRO sync server.
Solution: Check the availability of the EncounterPRO Sync Server by typing the following at a command line prompt: ping eprosync.jmjtech.com.
Details:

Check the availability of the EncounterPRO Sync Server by typing the following at a command line prompt:
ping eprosync.jmjtech.com
If an error message is issued then contact your network administrator or EncounterPRO technical services for assistance.
NOTE: Should check now fail, check to see if you can access ftp.jmjtech.com from the clients site. IF YOU CANNOT, then the problem is with us and you should notify our Technical Services dept.

Symptoms: The 'Check Now' button fails and the Check Remote Server fails with an error message like 'SQL Server does not exist or access is denied'.
