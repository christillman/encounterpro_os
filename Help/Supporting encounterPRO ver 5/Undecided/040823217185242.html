How to: Suppress the Create Encounter Screen for a Specific Treatment Room Check-in

Purpose

For some rooms, the practice wants to be able to skip the Create New Encounter screen when the user selects 'Check-In'. The new encounter should be created with the default values, as if the user had simply clicked 'Finished'.

For example, it would be a 'click saver' to allow the providers and office personnel to be able to select Check-in in the Provider-Initiated Phone room and skip the Create New Encounter screen. This technique will be most useful for specific treatment rooms, such as the Provider-initiated Phone room, the Entry of Records room, etc.

Details:

Steps
Follow these steps:

  1. From the Configuration tab
  2. SelectPreferences & turn on Config Mode.
  3. Go to the Telephone or Treatment Room you want to suppress the Create New Encounter screen from.
  4. Select the Checkin button. A 2-button menu appears. (If this menu does not appear, you will need to add it in o_menu_selection.)
  5. Edit the existing button to order a Service. Select General.
  6. Select Check-in Patient.
  7. Enter the parameters for the button as follows:
    • encounter type (PHONE for example)
    • default role (!PHONE for example)
    • Show Screen = No
  8. Save the menu.

NOTE: If the menu does not appear in step 3, go to the o_menu_selection table and search for a menu_context of Room Check-in. If it does not exist, add that row with menu_context = Room Checkin (2 words with a space), menu_key= such as $Telephone, and menu_id of menu you want to use.
