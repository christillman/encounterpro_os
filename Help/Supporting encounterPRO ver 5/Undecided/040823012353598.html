How to: Create a Reviewed button for Lab Dashboard

Solution: It is convenient to be able to document a progress note on the lab result treatment dashboard. With a Reviewed button, the reviewer can add a review note with one touch.
Details:

To create a 'Reviewed' button on the lab dashboard, do the following:

1) With 'Config Mode' turned on, right-click anywhere inside a lab dashboard.
2) Create a local copy if necessary then click 'Edit Display Script'.
3) Locate the 'Print' command for the 'Reviewed' link you want to enable. There are several available:
  a) 'Reviewed; WNL, Send to Staff' - adds a progress note that the lab results have been reviewed, the results are within normal limits, and sends a Task to the Staff.
  b) 'Reviewed; ABN, Send to Staff' - adds a progress note that the lab results have been reviewed, the results were abnormal, and sends a Task to the Staff.
  c) 'Reviewed' - adds a progress note that the lab results have been reviewed. No Task is generated.
4) Enable the 'Print' command for the 'Reviewed' button that you want to add to the dashboard.
5) Enable both 'Space' commands located above and below the 'Print' command you just enabled.
6) Click 'OK'.
7) Click 'Finish'.
NOTE: See also kb article #10607 ('Add a Print Button to Treatment Dashboard Screens').

TIP: All steps mentioned in Lab Dashboard Issues section work for any dashboards but you must perform steps for each of the dashboards. ie X-Ray dashboard, lab dashboard etc.
