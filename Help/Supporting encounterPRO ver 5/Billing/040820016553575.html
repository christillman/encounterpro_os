How to: Cancel All Billing Requests for a Date Range

Solution: Billing interface has never been turned on or had problems and stopped working. For any of these cases, always CANCEL ALL THE PREVIOUS BILLS BEFORE STARTING THE BILLING INTERFACE
Details:

IMPORTANT:  RUN THE QUERY DURING THE PRACTICE'S OFF-HOURS

VERSION 2
How to cancel queued billing requests for a given date range:

  1. Run the below query
    Update p_Patient_Encounter set billing_posted='Y' where encounter_date < [enter the date in single quotes] and bill_flag = 'Y' and billing_posted = 'N'
    Example: to cancel all the bills prior to '6/30/2005' then it'll be something like where encounter_date < '6/30/2005'
  2. Set EncounterPROBilling and EncounterPROTransportOut Status to OK in o_server_component
  3. Restart EncounterPROBilling and EncounterPRO TransportOut Services


VERSION 4 and VERSION 5
How to cancel queued billing requests for a given date range:

  1. Run the below query
    Update p_Patient_WP_Item set status = 'CANCELLED' WHERE dispatch_date < [enter the date in single quotes] AND ordered_service in ('SENDBILLING','RETRYPOSTING') AND status in ('DISPATCHED','STARTED')
    Example: to cancel all the bills prior to '6/30/2005' then it'll be something like where dispatch_date < '6/30/2005'
  2. Set EncounterPROBilling and EncounterPROTransportOut Status to OK in o_server_component
  3. Restart EncounterPROService Services (or EncounterPROBilling and EncounterPRO TransportOut)
