How to: Edit the Billing Courtesy Options on the Billing Edit Screen

Solution:

The options on the Billing Courtesy popup list are editable. Filter the c_Domain table on domain_id = 'COURTESY_CODE' and add/remove/edit records as needed.

Details:

The options on the Billing Courtesy popup list are editable.

Workaround:
To edit the options on the Billing Courtesy popup list:

1) Filter the c_Domain table on domain_id = 'COURTESY_CODE' and add/remove/edit records as needed.
2) The [domain_item] column holds the code that will actually be saved in the p_Patient_Encounter tables and sent to billing systems.
3) The [domain_item_description] column holds the value that is actually displayed to the user.
