How to: Set up Spirometer integration

Solution: This article details how to set up spirometer integration. See also How to: Configure a Spirometer interface.
Details:

1) Make sure that any existing Midmark or IQMark software is removed from the computer, including but not limited to any Midmark dependencies installed with pre-4.05.34 versions of EncounterPRO.
2) Install the Midmark 6.3 SDK.(\\Wwwserver\ftproot\EP_Upgrades\Production\Encounter_Production_CD\JMJ_Support\Programs\IQMarkSDK633)
3) You will find six (6) .RAR files. Extract any one of these and run the setup.exe file.
4) Install the latest EproLib4 and EproLibNET libraries.
5) Ensure the following exists in c_Component_Registry:

component_id component_type component description component_class id
ATCH_COM ATTACHMENT GENERIC COM Attachment Handler eprolib4.attachment {63F88597-27C1-4E59-BBB2-7B2C12487EE5}

6) Ensure the following exists in c_Attachment_Extension (fields not listed are unnecessary):

extension description default_attachment_type default_storage_flag component_id display_control
car Brentwood Spirometer Report FILE D ATCH_COM ActiveX

7) In EncounterPRO on computers that will be connected to spirometry devices, make sure that Spirometer is set up as a valid external source.
NOTE: See also support issues 13754, 17270, 17279 and 15432.
